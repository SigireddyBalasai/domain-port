# Hello, World!

This is my first blog post. It's written in **MDX** and processed by **Velite**.

```ts
function greet(name: string) {
  return `Hello, ${name}!`
}

console.log(greet("World"))
```

> Velite turns your MDX files into a type-safe data layer.
