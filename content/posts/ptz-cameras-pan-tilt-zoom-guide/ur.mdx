**PTZ (پین-ٹیلٹ-زوم)** کیمرے حرکت کرتے ہیں — وہ گھومتے، جھکتے اور زوم کرتے ہیں تاکہ کسی بھی زاویے کو کور کریں۔ ایک PTZ پورے ہال، بڑے کمرے، یا گیراج کے یارڈ کو ایک سے زیادہ فکسڈ کیمروں کے برابر کور کرتا ہے۔

یہ گائیڈ PTZ کیمروں، ان کی صلاحیتوں اور اپنی ضروریات کے لیے صحیح انتخاب کے بارے میں وہ سب کچھ بتاتا ہے جو آپ کو 2026 میں جاننے کی ضرورت ہے۔

تناظر دیکھتا ہے جبکہ PTZ تفصیل کے لیے زوم ان کرتا ہے۔

## PTZ کیمرے کیسے کام کرتے ہیں؟

PTZ دراصل دو چیزوں کا امتزاج ہے: ایک فکسڈ پوزیشن والا سیکیورٹی کیمرہ جس میں میکانکی طور پر حرکت کرنے والا ہیڈ ہوتا ہے، اور سافٹ ویئر جو حرکت کو کنٹرول کرتا ہے۔

| فیچر        | تفصیل                                                   |
| ----------- | ------------------------------------------------------- |
| **پین**     | 0–360° افقی گردش                                        |
| **ٹیلٹ**    | 0–90° (یا 180°) عمودی جھکاؤ                             |
| **زوم**     | آپٹیکل (لینس) + ڈیجیٹل (سافٹ ویئر) — تفصیل کے لیے کلیدی |
| **IR رینج** | رات کو کتنی دور دیکھتا ہے (100–500+ فٹ)                 |
| **رفتار**   | پری سیٹس: 100–400°/s؛ آپریٹر: 1–100°/s                  |
| **پری سیٹ** | پروگرام قابل پوزیشنز — مخصوص مقامات پر فوری سوئنگ       |
| **پیٹرن**   | خودکار گشت کا راستہ                                     |

## اہم PTZ خصوصیات

### آپٹیکل زوم

آپٹیکل زوم PTZ کا سب سے اہم پیرامیٹر ہے:

- **10x زوم** — چھوٹے گیراج، ذاتی استعمال
- **25x زوم** — تجارتی، اسکول، بڑے صحن
- **30x–36x زوم** — گودام، کھیت، کھلا علاقہ
- **45x+ زوم** — **محفوظ علاقے**: پارکنگ لاٹ، باڑ کی لکیر، نامزدگی

<Callout type="tip">

**30x زوم کا حساب:** 30x آپٹیکل @ 5mm فلیٹ زوم = 150mm → 100 فٹ پر چہرہ پہچاننے کے قابل۔ 45x @ 200mm → 200+ فٹ۔

</Callout>

### آٹو ٹریکنگ

پیشہ ورانہ PTZs خود بخود **AI تجزیہ** کے ساتھ حرکت کرنے والی اشیاء کو ٹریک کر سکتے ہیں:

| درجہ           | ٹریکنگ کا معیار                                       | قیمت کی حد    |
| -------------- | ----------------------------------------------------- | ------------- |
| **بنیادی**     | حرکت کا پتہ چلتا ہے، کسی بھی چیز کو فالو کرتا ہے      | $200–500      |
| **اعلیٰ**      | شخص/گاڑی کے فلٹر، ہموار فالو، زوم آٹو                 | $500–1,500    |
| **پیشہ ورانہ** | AI رفتار کی پیش گوئی، ایک سے زیادہ ٹریکس، زوم آپٹمائز | $1,500–5,000+ |

### پری سیٹ اور پیٹرن

- **پری سیٹ:** GPS وے پوائنٹ کی طرح — کیمرے کو منتخب مقام پر زاویہ اور زوم یاد رہتا ہے۔ مثال: "مین گیٹ"، "پارکنگ لاٹ A"، "لابی"
- **پیٹرن:** پری سیٹس کی ترتیب جسے PTZ خود بخود فالو کرتا ہے
- **گشت:** پیٹرن کے ذریعے مسلسل لوپ

## کنیکٹیویٹی اور پروٹوکول

| پروٹوکول                | کیبل     | زیادہ سے زیادہ رینج | رفتار     | استعمال                        |
| ----------------------- | -------- | ------------------- | --------- | ------------------------------ |
| **ONVIF Profile Q/S/T** | Ethernet | 330 فٹ              | 1 Gbps    | **سب سے عام** — نیا PTZ خریدیں |
| **Pelco-D/P**           | RS-485   | 4,000 فٹ            | 38.4 Kbps | پرانا، لیکن صنعت میں اب بھی    |
| **RS-422**              | RS-422   | 4,000 فٹ            | 10 Mbps   | صنعتی، زیادہ فاصلے             |
| **DHCP/اسٹیٹک**         | Ethernet | 330 فٹ              | 1 Gbps    | NVR/سافٹ ویئر کنٹرول           |

### ڈیزی چین (RS-485)

RS-485 پر ایک سے زیادہ PTZs کو ڈیزی چین کیا جا سکتا ہے:

```
کنٹرولر → [PTZ #1] → [PTZ #2] → [PTZ #3] (ہر ایک کو یونیک آئی ڈی)
```

**حد:** 32 ڈیوائسز زیادہ سے زیادہ، ہر ایک کا اپنا Pelco ID۔ **ONVIF** اتنا آسان نہیں — ہر کیمرے کو ایتھرنیٹ کی ضرورت ہے (جیسے سوئچ پورٹس)۔

## وائرنگ اور پاور

| پاور ٹائپ           | کیبل            | زیادہ سے زیادہ واٹج | بہترین کے لیے                  |
| ------------------- | --------------- | ------------------- | ------------------------------ |
| **PoE+ (802.3at)**  | Cat6 (≤330 فٹ)  | 30W                 | انڈور / چھوٹے PTZs (~$200–500) |
| **PoE++ (802.3bt)** | Cat6a (≤330 فٹ) | 60–90W              | ہیوی ڈیوٹی PTZ + ہیٹر + IR     |
| **24V AC**          | 18/2 تار        | 50–100W             | صنعتی PTZ روایتی (ہیوی ڈیوٹی)  |
| **12V DC**          | DC بیرل         | 12–24W              | ڈوم PTZs (چھوٹے)               |

<Callout type="warning">

**PoE PTZ پاور:** PTZ کے بڑے موٹرز سب سے زیادہ طاقت اس وقت لیتی ہیں جب وہ حرکت کر رہے ہوتے ہیں۔ اس بات کو یقینی بنائیں کہ آپ کا PoE سوئچ **ہر پورٹ کی پاور فراہم کر سکتا ہے**، نہ کہ صرف کل بجٹ۔ اگر پاور انڈر ہو تو PTZ حرکت نہیں کرے گا۔

</Callout>

## PTZ کیمرہ کی تنصیب

| ماؤنٹ        | بہترین کے لیے                     | وزن برداشت |
| ------------ | --------------------------------- | ---------- |
| سیفٹی ماؤنٹ  | پیشہ ورانہ / صنعتی — طاقت، بریکٹ  | 5–15 lbs   |
| وال ماؤنٹ    | انڈور / باہر — دیوار پر براہ راست | 2–8 lbs    |
| پول ماؤنٹ    | گیراج، کھیت — کھمبے سے لگائیں     | 5–20 lbs   |
| پینڈنٹ ماؤنٹ | چھت — ریستوران، مال، گودام        | 5–15 lbs   |

**تنصیب کے اصول:**

1. **پری سیٹس** — اینکر پوائنٹس تبدیل کرنے سے پہلے تمام پری سیٹس کو پروگرام کریں
2. **اوپن اینڈ ڈیٹیکشن** — پہلی اور آخری پری سیٹ میں اوورلیپ ہونا چاہیے
3. **زوم ٹیسٹ** — آخری زوم پر امیج معیار کی تصدیق کریں
4. **ڈیزی چین میں اضافہ** — RS-485 پری سیٹ IDs کا اپ اسٹریم اور ڈاؤن اسٹریم سے کوئی تنازع نہیں ہونا چاہیے
