Las cámaras PTZ (Pan-Tilt-Zoom) son los caballos de batalla de la seguridad profesional. A diferencia de las cámaras fijas que solo ven en una dirección, las PTZ pueden girar 360°, inclinarse hacia arriba/abajo y acercarse a los detalles. Son ideales para propiedades grandes, usos comerciales y vigilancia activa.

<Badge variant="outline">PTZ vs Fija</Badge> Una cámara PTZ puede reemplazar 2-4
cámaras fijas para cobertura, PERO solo puede apuntar en una dirección a la vez.
Para cobertura completa, combine PTZ + cámaras fijas.

---

## Cómo Funciona una PTZ

### Mecánica

| Componente               | Función                       | Especificaciones Típicas      |
| ------------------------ | ----------------------------- | ----------------------------- |
| **Motor Pan**            | Gira horizontalmente          | 0-360° (continuo o con tope)  |
| **Motor Tilt**           | Inclina verticalmente         | 0-90° (hacia arriba/abajo)    |
| **Lente de zoom**        | Amplía/ reduce el FOV         | 3x-30x óptico (+ digital)     |
| **Motor de zoom**        | Mueve los elementos del lente | Rápido (&lt;1s) o lento (>3s) |
| **Codificadores**        | Posición de seguimiento       | Precisión de ±0.1°            |
| **Junta tórica / junta** | Sello contra la intemperie    | IP66-67 (típico)              |

### Velocidad de Giro/Inclinación

| Velocidad  | Grados/segundo | Uso                                                                |
| ---------- | -------------- | ------------------------------------------------------------------ |
| **Lenta**  | 1-10°/s        | Patrullaje suave, seguimiento preciso                              |
| **Media**  | 10-50°/s       | Seguimiento de peatones, transiciones entre puntos preestablecidos |
| **Rápida** | 50-200°/s      | Seguimiento de vehículos, respuesta a alarmas                      |
| **Turbo**  | 200-400°/s     | Cámaras PTZ de velocidad (Dahua, Hikvision profesionales)          |

## Tipos de Zoom

### Zoom Óptico vs Digital

| Tipo                          | Cómo Funciona                                                        | Calidad                         | Alcance Típico                             |
| ----------------------------- | -------------------------------------------------------------------- | ------------------------------- | ------------------------------------------ |
| **Zoom óptico**               | Los elementos del lente se mueven físicamente para ampliar la imagen | Sin pérdida de calidad          | 3x-30x (estándar); hasta 50x (profesional) |
| **Zoom digital**              | La cámara recorta y amplía el centro del sensor                      | Pérdida de calidad (pixelación) | 4x-16x (sobre el zoom óptico)              |
| **Zoom inteligente (mezcla)** | Óptico hasta el máximo, luego digital suavemente                     | Buena (transición suave)        | Óptico + 4-8x digital                      |

### Alcance del Zoom (Identificación)

| Aumento | Identificación Facial | Identificación de Placa | Campo de Visión |
| ------- | --------------------- | ----------------------- | --------------- |
| **3x**  | 50 pies               | 30 pies                 | ~35°            |
| **5x**  | 100 pies              | 60 pies                 | ~20°            |
| **10x** | 200 pies              | 150 pies                | ~10°            |
| **20x** | 400 pies              | 300 pies                | ~5°             |
| **30x** | 600 pies              | 500 pies                | ~3°             |

## Funciones de las PTZ

### Puntos Preestablecidos (Presets)

Guarde posiciones específicas (pan, tilt, zoom) como puntos preestablecidos. La cámara puede saltar a cualquier preset al instante.

| Configuración Típica | Ubicación        | Propósito                 |
| -------------------- | ---------------- | ------------------------- |
| Preset 1             | Puerta principal | Identificar visitantes    |
| Preset 2             | Entrada          | Capturar matrículas       |
| Preset 3             | Puerta trasera   | Cubrir entrada secundaria |
| Preset 4             | Buzón            | Monitorear entregas       |

### Patrullaje (Crucero/Guardia)

La cámara recorre automáticamente una secuencia de puntos preestablecidos.

**Configuración:**

1. Establezca 4-8 puntos preestablecidos que cubran las áreas clave
2. Configure la velocidad de patrullaje (lenta = 30s/punto, rápida = 5s/punto)
3. Establezca el tiempo de pausa en cada punto (3-10s)
4. Active el patrullaje durante las horas de inactividad (noche, fines de semana)

### Seguimiento Automático

La cámara detecta movimiento (persona/vehículo) y lo sigue automáticamente.

| Tipo de Seguimiento     | Cómo Funciona                               | Funciona Mejor Para              |
| ----------------------- | ------------------------------------------- | -------------------------------- |
| **Pan Track**           | Sigue horizontalmente                       | Personas caminando               |
| **Pan+Tilt Track**      | Sigue en 2D                                 | Personas en patios, vehículos    |
| **Pan+Tilt+Zoom Track** | Sigue + zoom al objetivo                    | Identificar intrusos a distancia |
| **Auto-Zoom**           | Ajusta el zoom según el tamaño del objetivo | Mantener el objetivo enmarcado   |

<Callout type="warning">

**Limitación del seguimiento automático:** Solo puede seguir a un objetivo a
la vez. Si ocurren dos eventos, seguirá al primero o se detendrá confundido.
Combine PTZ con cámaras fijas para cobertura de respaldo.

</Callout>

### Máscaras de Privacidad

Las PTZ pueden tener zonas de privacidad que se oscurecen automáticamente para evitar que la cámara grabe ventanas de vecinos, áreas sensibles, etc.

## Instalación de PTZ

### Consideraciones de Montaje

| Aspecto          | Recomendación                                                                         |
| ---------------- | ------------------------------------------------------------------------------------- |
| **Altura**       | 10-15 pies (más alto que las cámaras fijas para evitar vandalismo)                    |
| **Superficie**   | Pared sólida o poste—la PTZ pesa más y el viento la afecta                            |
| **Alimentación** | PoE+ como mínimo (PoE estándar puede no ser suficiente para calefacción + motores)    |
| **Cable**        | Cat6 para exteriores (PoE+ necesita buena conductividad)                              |
| **Visibilidad**  | Asegúrese de que la cámara tenga un FOV claro en todas las direcciones—sin obstáculos |

### Cableado PoE+ para PTZ

Las PTZ consumen más energía que las cámaras fijas—especialmente con calefacción/ventilador y motores activos.

| Consumo Típico             | Sin Calefacción | Con Calefacción |
| -------------------------- | --------------- | --------------- |
| En espera (sin movimiento) | 8-12W           | 15-25W          |
| Movimiento activo          | 15-25W          | 25-40W          |
| Seguimiento + zoom         | 20-30W          | 30-50W          |

**Recomendación de PoE:** Use PoE+ (802.3at, 30W) como mínimo. Para PTZ con calefacción en climas fríos, use PoE++ (802.3bt, 60W).

## Automatización de PTZ

### Integración con NVR

| Función                      | Descripción                                                                             |
| ---------------------------- | --------------------------------------------------------------------------------------- |
| **Vinculación de alarmas**   | Cuando una cámara fija detecta movimiento, ordena a la PTZ ir al preset correspondiente |
| **Grabación de seguimiento** | El NVR graba desde la PTZ mientras sigue un objetivo                                    |
| **Pantalla dividida**        | Vea la vista general (cámara fija) + primer plano (PTZ) lado a lado                     |
| **Vinculación de audio**     | La PTZ emite un tono de advertencia cuando detecta un intruso                           |

### Integración con Home Assistant

```yaml
# Ejemplo: Cuando una cámara fija detecta una persona, PTZ va al preset de la puerta principal
automation:
  - alias: "PTZ Go to Front Door on Person Detection"
    trigger:
      - platform: mqtt
        topic: "frigate/front_door_fixed/person"
        payload: "ON"
    action:
      - service: ptz.goto_preset
        target:
          entity_id: camera.front_door_ptz
        data:
          preset: 1
      - service: camera.record
        target:
          entity_id: camera.front_door_ptz
        data:
          duration: 30
```

## Mantenimiento de PTZ

| Tarea                                 | Frecuencia             | Por Qué                                                      |
| ------------------------------------- | ---------------------- | ------------------------------------------------------------ |
| **Lubricar los engranajes del motor** | Cada 6-12 meses        | El polvo y la suciedad desgastan los engranajes              |
| **Limpiar el domo/lente**             | Mensualmente           | La suciedad degrada la calidad de imagen                     |
| **Verificar la junta tórica**         | Anualmente             | Las juntas de goma se degradan con los rayos UV              |
| **Prueba de giro completo**           | Mensualmente           | Verifique que la PTZ pueda girar 360° sin obstrucciones      |
| **Actualización de firmware**         | Cuando esté disponible | Corrige errores de seguimiento y problemas de conectividad   |
| **Reiniciar**                         | Cada 3-6 meses         | Las PTZ pueden desarrollar errores de software con el tiempo |
