## Глубокое погружение в платформы

### Amazon Alexa

**Нативная поддержка:** Ring, Blink, Arlo, Echo Show (как дисплей)  
**Skill требуется:** Reolink, Eufy, UniFi, Wyze, Tapo, Amcrest, TP-Link Kasa

### Apple HomeKit (HomeKit Secure Video)

**Нативная поддержка:** Eufy (HomeBase 3), Arlo (Pro 3/4/5S, Ultra), Logitech Circle View, Netatmo, Eve, Aqara, TP-Link Tapo (Matter), UniFi Protect (3.0+), Onvis, Wemo  
**Bridge Required:** Reolink (via Homebridge/Scrypted), Ring (via Homebridge), Wyze (via Homebridge), Dahua/Hikvision (via Homebridge/Scrypted)

<Callout type="info">

**HomeKit Secure Video (HSV) = Лучшая приватность.** Видео анализируется **на
устройстве** (Home Hub: Apple TV 4K, HomePod, iPad). **E2E шифрование** —
Apple не может посмотреть. 10-дневное скользящее хранение **включено в
iCloud+** (50GB = 1 камера, 200GB = 5 камер, 2TB = безлимит). Никакой
отдельной подписки.

</Callout>

<Accordion type="single" collapsible>
  <AccordionItem value="hsv-features">
    <AccordionTrigger>Функции и требования HSV</AccordionTrigger>
    <AccordionContent>

**Поддерживаемые разрешения:** 1080p, 2K, 4K (анализ даунскейлится до 1080p, хранятся в исходном)
**Activity Zones:** На камеру, настраиваются в Home app
**Face Recognition:** "Person detected" → "John detected" (если в People album)
**Опции записи:** Off / Detect Activity / Stream & Record
**Хранение:** iCloud+ (50GB/$0.99, 200GB/$2.99, 2TB/$9.99) — **не за камеру**
**Home Hub Required:** Apple TV 4K (2021+), HomePod (mini/2nd gen), iPad (always on)
**Локальный стриминг:** iPhone/iPad в той же Wi-Fi → прямая (нет интернета). Remote → через iCloud (зашифрован).
**Лимиты:** Макс 5 камер на 200GB плане (если не 2TB); нет 24/7 continuous (event-only); нет web view (только iOS/macOS); нет нативного Android.

    </AccordionContent>

  </AccordionItem>
  <AccordionItem value="homebridge">
    <AccordionTrigger>Мост для не-HomeKit камер (Homebridge / Scrypted / Hoobs)</AccordionTrigger>
    <AccordionContent>

**Homebridge** (Node.js, 2000+ плагинов): Бегает на Pi, Docker, NAS, mini PC

- `homebridge-camera-ffmpeg` → RTSP → HSV (транскодирует, добавляет motion)
- `homebridge-reolink` → Reolink нативный API (motion, PTZ, снапшоты)
- `homebridge-dahua` / `homebridge-hikvision` → ONVIF/ISAPI
- `homebridge-ring` → Ring камеры/звонки (polling, ~30s latency)

**Scrypted** (TypeScript, новее, быстрее): Нативные HomeKit, Google, Alexa плагины

- Лучше RTSP обработка, меньше латентность, плагин маркетплейс
- Бегает на том же железе что Homebridge

**Hardware для Bridge:** Raspberry Pi 4 (4GB+) / 5, Intel N100/N305 mini PC ($120), старый ноутбук, Synology/QNAP Docker.
**Latency:** Homebridge FFmpeg ~2–5s; Scrypted ~1–3s; Нативный HSV ~0.5–1s.
**HSV через Bridge:** Камера должна поддерживать RTSP. Bridge transcodes → HSV → iCloud. **Считается в лимит HSV камер.**

</AccordionContent>

  </AccordionItem>
</Accordion>

### Matter (Новый стандарт)

**Matter 1.3 (2024)** добавляет **Camera** device type. **Matter 1.4 (2025)** добавляет **Doorbell** type.

<Accordion type="single" collapsible>
  <AccordionItem value="matter-status">
    <AccordionTrigger>Текущая поддержка Matter камер (2026)</AccordionTrigger>
    <AccordionContent>

**Сертифицированные Matter камеры:**

- **TP-Link Tapo C120 / C220 / C320WS** (Matter over Wi-Fi)
- **Eve **Nanoleaf Smart Cam\*\* (Matter)
- **Eve Cam** (Thread + Matter)
- **Google Nest Cam (wired, 2024)** / **Nest Doorbell (wired, 2024)** — Matter bridge через Google Home
- **EufyCam 3 / SoloCam S340** — Matter firmware update (2024)

**Что дает Matter:**

- **Multi-admin:** Добавьте в Apple Home, Google Home, Alexa, SmartThings одновременно
- **Local Control:** Никакого облачного релея для базовых on/off/stream (Thread/Wi-Fi direct)
- **Стандартизированные команды:** Live stream, motion events, privacy mode

**Что Matter НЕ делает пока:**

- 4K стриминг (макс 1080p в спеке)
- PTZ контроль (нет в 1.3/1.4)
- Face recognition / package detection (только vendor extensions)
- Continuous recording (event clips only)
- NVR интеграция

**Вердикт (2026):** Matter камеры работают для **базового multi-platform live view + motion alerts**. Для записи, AI, PTZ, 4K → все еще нужно native integration или bridge.

## Local Control / No-Cloud: Продвинутый путь

<Callout type="tip">

**Local = Быстро, Приватно, Работает Офлайн.** RTSP/ONVIF/WebSocket →
Frigate/Scrypted/Blue Iris → Home Assistant → Dashboards/Автоматизации.
Никакой вендорский облак никогда не видит ваше видео.

</Callout>

```
    ▼

    │                    │
    ▼                    ▼
┌─────────────┐    ┌─────────────┐
│ Home Asst.  │    │  Scrypted   │
│ (Automation)│    │ (HomeKit/   │
│  - Lights   │    │  Google/    │
│  - Locks    │    │  Alexa)     │
│  - Alerts   │    │             │
└─────────────┘    └─────────────┘
    │                    │
    ▼                    ▼

                    Google Home /
                    Alexa App
```

### Железо для локального стека

| Компонент          | Бюджет          | Рекомендуемый                          | High-End                       |
| ------------------ | --------------- | -------------------------------------- | ------------------------------ |
| **Server**         | RPi 4 4GB ($80) | Intel N100/N305 Mini PC ($130)         | i5-13400 / Ryzen 5700G ($400)  |
| **AI Accelerator** | CPU only (slow) | **Google Coral USB ($60)** / M.2 ($40) | Coral PCIe x4 / Hailo-8 ($150) |
| **Storage**        | 1× 4TB HDD      | 2× 8TB RAID 1                          | 4× 14TB RAID 6 / ZFS           |
| **Network**        | 1 Gbps          | 2.5 Gbps switch + NIC                  | 10 Gbps backbone               |

### Требования камер для локального стека

- **RTSP Support** (стандарт: `rtsp://user:pass@ip:554/stream1`)
- **ONVIF Profile S/T/G** (discovery, PTZ, events)
- **Sub-streams** (D1/720p для AI, 4K для записи)
- **No cloud dependency** для стрима (некоторые камеры требуют cloud auth token — избегайте)

**Лучшие ONVIF/RTSP камеры (2026):**

- Reolink (all PoE, most Wi-Fi): `rtsp://ip:554/h264Preview_01_main`
- Dahua / Hikvision / Hanwha / Axis / Uniview / Vivotek / Milesight
- UniFi G4/G5 (RTSP via Protect settings)
- Amcrest / Lorex / Annke / Montavue (OEM Dahua/Hik)
- **Avoid:** Ring, Blink, Nest, Arlo (pre-2024), Wyze (limited), Eufy (no RTSP on battery cams)

## Рецепты автоматизаций (Copy-Paste Ready)

### Home Assistant (YAML)

```yaml
# Porch Light on Person (after sunset)
automation:
  - alias: "Porch Light - Person at Front Door"
    trigger:
      - platform: mqtt
        topic: frigate/front_door/person
        payload: "on"
    condition:
      - condition: sun
        after: sunset
        after_offset: "-00:30:00"
    action:
      - service: light.turn_on
        target:
          entity_id: light.porch
        data:
          brightness_pct: 80
      - delay: "00:05:00"
      - service: light.turn_off
        target:
          entity_id: light.porch

  # Deterrence: Siren + Lights on Person at Driveway (night)
  - alias: "Deterrence - Driveway Person Night"
    trigger:
      - platform: mqtt
        topic: frigate/driveway/person
        payload: "on"
    condition:
      - condition: time
        after: "22:00:00"
        before: "06:00:00"
    action:
      - service: siren.turn_on
        target:
          entity_id: siren.outdoor
      - service: light.turn_on
        target:
          entity_id: light.floodlights
        data:
          brightness_pct: 100
      - service: notify.mobile_app
        data:
          title: "🚨 Driveway Intrusion"
          message: "Person detected at driveway at {{ now().strftime('%H:%M') }}"
```

### Alexa Routine (App → Routines)

```
WHEN: Smart Home Device → [Front Door Camera] → Motion Detected → Person
IF: Time → Between Sunset and Sunrise
THEN:
    1. Smart Home → [Porch Light] → Turn On → 80% → 5 minutes
    2. Alexa Says → "Person at front door" → Device: [Kitchen Echo]
    3. Smart Home → [Floodlight] → Turn On → 100% → 3 minutes
```

### Google Home Automation (Home App → Automations)

```
STARTER: Device → [Driveway Camera] → Event → Person detected
CONDITION: Time → Between 10:00 PM and 6:00 AM
ACTIONS:
    1. Device → [Floodlights] → Turn On → Brightness 100%
    2. Device → [All Speakers] → Broadcast → "Person detected at driveway"
    3. Script → Log to Google Sheets (via Make/Zapier/webhook)
```

### Apple HomeKit (Home App → Automation)

```
WHEN: A Sensor Detects Something → [Front Door Camera] → People Detected
CONDITION: Time → After Sunset
DO:
    1. Scenes → [Porch Lights On] (pre-created scene: porch 80%, entry 50%)
    2. Wait → 5 minutes
    3. Scenes → [Porch Lights Off]
```

## Platform Comparison: Which Should You Choose?

| Priority                   | Best Platform                           | Why                                                  |
| -------------------------- | --------------------------------------- | ---------------------------------------------------- |
| **Privacy / No Cloud**     | **HomeKit (HSV) + Local Bridge**        | E2E encryption, iCloud+ included, on-device AI       |
| **Voice + Displays**       | **Alexa** (Echo Show ecosystem)         | Most skills, best routine UI, cheap displays         |
| **Automation Power**       | **Google Home** (script editor)         | Best starter types, complex logic, Nest Hub grid     |
| **Local + Multi-Platform** | **Home Assistant + Frigate + Scrypted** | Total control, Matter bridge, no vendor lock-in      |
| **Easiest Setup**          | **Ring / Blink / Nest** (single brand)  | Native, no bridge, but subscription + cloud          |
| **Business / Multi-Site**  | **UniFi Protect**                       | Centralized, no cloud fee, HSV + Alexa/Google skills |
| **Matter Future-Proof**    | **TP-Link Tapo / Eve / Nanoleaf**       | Thread + Matter, works everywhere                    |

## Migration Path: From Cloud to Local

1. **Keep cloud cams** for now (Ring doorbell, Blink outdoor)
2. **Add PoE cams** (Reolink, Dahua) → NVR (UniFi/Reolink) or Frigate server
3. **Bridge cloud cams** via Homebridge/Scrypted → HomeKit/Google/Alexa
4. **Move automations** to Home Assistant (or HA + Alexa/Google routines)
5. **Disable cloud recording** on bridged cams; use local NVR/Frigate
6. **Cancel subscriptions** when local recording verified
7. **Future cams:** Buy ONVIF/RTSP + Matter only
