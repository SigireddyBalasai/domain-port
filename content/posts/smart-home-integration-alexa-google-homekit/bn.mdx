**নেটিভ সাপোর্ট:** Ring, Blink, Arlo, Echo Show (ডিসপ্লে হিসেবে)  
**Skill প্রয়োজন:** Reolink, Eufy, UniFi, Wyze, Tapo, Amcrest, TP-Link Kasa

### Apple HomeKit (HomeKit Secure Video)

**নেটিভ সাপোর্ট:** Eufy (HomeBase 3), Arlo (Pro 3/4/5S, Ultra), Logitech Circle View, Netatmo, Eve, Aqara, TP-Link Tapo (Matter), UniFi Protect (3.0+), Onvis, Wemo  
**ব্রিজ প্রয়োজন:** Reolink (Homebridge/Scrypted-এর মাধ্যমে), Ring (Homebridge-এর মাধ্যমে), Wyze (Homebridge-এর মাধ্যমে), Dahua/Hikvision (Homebridge/Scrypted-এর মাধ্যমে)

<Callout type="info">

**HomeKit Secure Video (HSV) = সেরা গোপনীয়তা।** ভিডিও **অন-ডিভাইস** বিশ্লেষিত
(Home Hub: Apple TV 4K, HomePod, iPad)। **এন্ড-টু-এন্ড এনক্রিপ্টেড** — Apple
দেখতে পারে না। iCloud+-এর সাথে 10 দিন রোলিং স্টোরেজ **অন্তর্ভুক্ত** (50GB = 1
ক্যাম, 200GB = 5 ক্যাম, 2TB = সীমাহীন)। কোনো আলাদা সাবস্ক্রিপশন নেই।

</Callout>

**অ্যাক্টিভিটি জোন:** প্রতি-ক্যামেরা, Home অ্যাপে সেট করুন
**ফেস রিকগনিশন:** "ব্যক্তি শনাক্ত" → "জন শনাক্ত" (যদি People অ্যালবামে থাকে)
**রেকর্ডিং অপশন:** বন্ধ / কার্যকলাপ শনাক্ত / স্ট্রিম ও রেকর্ড
**স্টোরেজ:** iCloud+ (50GB/$0.99, 200GB/$2.99, 2TB/$9.99) — **প্রতি ক্যামেরায় নয়**
**Home Hub প্রয়োজন:** Apple TV 4K (2021+), HomePod (mini/2য় জেন), iPad (সবসময় চালু)
**লোকাল স্ট্রিমিং:** একই Wi-Fi-তে iPhone/iPad → সরাসরি (কোনো ইন্টারনেট নেই)। দূরবর্তী → iCloud-এর মাধ্যমে রিলে (এনক্রিপ্টেড)।
**সীমাবদ্ধতা:** 200GB প্ল্যানে সর্বোচ্চ 5 ক্যামেরা (2TB না হলে); কোনো 24/7 একটানা নয় (শুধু ইভেন্ট); কোনো ওয়েব ভিউ নয় (শুধু iOS/macOS); কোনো নেটিভ Android নয়।

**Homebridge** (Node.js, 2000+ প্লাগইন): Pi, Docker, NAS, মিনি পিসিতে চলে

- `homebridge-camera-ffmpeg` → RTSP → HSV (ট্রান্সকোড, মোশন যোগ করে)
- `homebridge-reolink` → Reolink নেটিভ API (মোশন, PTZ, স্ন্যাপশট)
- `homebridge-dahua` / `homebridge-hikvision` → ONVIF/ISAPI
- `homebridge-ring` → Ring ক্যামেরা/ডোরবেল (পোলিং, ~30 সেকেন্ড লেটেন্সি)

**Scrypted** (TypeScript, নতুন, দ্রুত): নেটিভ HomeKit, Google, Alexa প্লাগইন

- ভালো RTSP হ্যান্ডলিং, কম লেটেন্সি, প্লাগইন মার্কেটপ্লেস
- Homebridge-এর মতো একই হার্ডওয়্যারে চলে

**ব্রিজের জন্য হার্ডওয়্যার:** Raspberry Pi 4 (4GB+) / 5, Intel N100/N305 মিনি PC ($120), পুরনো ল্যাপটপ, Synology/QNAP Docker।
**লেটেন্সি:** Homebridge FFmpeg ~2-5 সেকেন্ড; Scrypted ~1-3 সেকেন্ড; নেটিভ HSV ~0.5-1 সেকেন্ড।
**ব্রিজের মাধ্যমে HSV:** ক্যামেরাকে RTSP সমর্থন করতে হবে। ব্রিজ ট্রান্সকোড → HSV → iCloud। **HSV ক্যাম সীমার দিকে গণনা করে।**

### Matter (নতুন স্ট্যান্ডার্ড)

**Matter 1.3 (2024)** **ক্যামেরা** ডিভাইস টাইপ যোগ করে। **Matter 1.4 (2025)** **ডোরবেল** টাইপ যোগ করে।

**সার্টিফাইড Matter ক্যামেরা:**

- **TP-Link Tapo C120 / C220 / C320WS** (Wi-Fi-র উপর Matter)
- **Eve Cam** (Thread + Matter)
- **Google Nest Cam (wired, 2024)** / **Nest Doorbell (wired, 2024)** — Google Home Matter ব্রিজের মাধ্যমে
- **EufyCam 3 / SoloCam S340** — Matter ফার্মওয়্যার আপডেট (2024)

**Matter আপনাকে কী দেয়:**

- **মাল্টি-অ্যাডমিন:** একসাথে Apple Home, Google Home, Alexa, SmartThings-এ যোগ দিন
- **লোকাল কন্ট্রোল:** বেসিক অন/অফ/স্ট্রিমের জন্য কোনো ক্লাউড রিলে নেই (Thread/Wi-Fi সরাসরি)
- **স্ট্যান্ডার্ডাইজড কমান্ড:** লাইভ স্ট্রিম, মোশন ইভেন্ট, প্রাইভেসি মোড

**Matter কী এখনও করে না:**

- 4K স্ট্রিমিং (স্পেকে সর্বোচ্চ 1080p)
- PTZ নিয়ন্ত্রণ (1.3/1.4-এ নেই)
- ফেস রিকগনিশন / প্যাকেজ ডিটেকশন (শুধু ভেন্ডর এক্সটেনশন)
- একটানা রেকর্ডিং (শুধু ইভেন্ট ক্লিপ)
- NVR ইন্টিগ্রেশন

**রায় (2026):** Matter ক্যামেরা **বেসিক মাল্টি-প্ল্যাটফর্ম লাইভ ভিউ + মোশন অ্যালার্টের** জন্য কাজ করে। রেকর্ডিং, AI, PTZ, 4K-এর জন্য → এখনও নেটিভ ইন্টিগ্রেশন বা ব্রিজ প্রয়োজন।

**লোকাল = দ্রুত, ব্যক্তিগত, অফলাইনে কাজ করে।** RTSP/ONVIF/WebSocket →
Frigate/Scrypted/Blue Iris → Home Assistant → ড্যাশবোর্ড/অটোমেশন। কোনো ভেন্ডর
ক্লাউড কখনো আপনার ভিডিও দেখে না।

</Callout>

### লোকাল স্ট্যাকের জন্য ক্যামেরার প্রয়োজনীয়তা

- **RTSP সাপোর্ট** (স্ট্যান্ডার্ড: `rtsp://user:pass@ip:554/stream1`)
- **ONVIF Profile S/T/G** (আবিষ্কার, PTZ, ইভেন্ট)
- **সাব-স্ট্রিম** (D1/720p AI-র জন্য, 4K রেকর্ডিংয়ের জন্য)
- **স্ট্রিমের জন্য কোনো ক্লাউড নির্ভরতা নেই** (কিছু ক্যামেরার ক্লাউড অথ টোকেন প্রয়োজন — এড়িয়ে চলুন)

**সেরা ONVIF/RTSP ক্যামেরা (2026):**

- Reolink (সব PoE, বেশিরভাগ Wi-Fi): `rtsp://ip:554/h264Preview_01_main`
- Dahua / Hikvision / Hanwha / Axis / Uniview / Vivotek / Milesight
- UniFi G4/G5 (Protect সেটিংসের মাধ্যমে RTSP)
- Amcrest / Lorex / Annke / Montavue (OEM Dahua/Hik)
- **এড়িয়ে চলুন:** Ring, Blink, Nest, Arlo (2024-পূর্ব), Wyze (সীমিত), Eufy (ব্যাটারি ক্যামে RTSP নেই)

## অটোমেশন রেসিপি (কপি-পেস্ট রেডি)

### Home Assistant (YAML)

```yaml
# সূর্যাস্তের পরে সামনের দরজায় ব্যক্তি → পোর্চ লাইট
automation:
  - alias: "Porch Light - Person at Front Door"
    trigger:
      - platform: mqtt
        topic: frigate/front_door/person
        payload: "on"
    condition:
      - condition: sun
        after: sunset
        after_offset: "-00:30:00"
    action:
      - service: light.turn_on
        target:
          entity_id: light.porch
        data:
          brightness_pct: 80
      - delay: "00:05:00"
      - service: light.turn_off
        target:
          entity_id: light.porch

  # রাতে ড্রাইভওয়েতে ব্যক্তি → সাইরেন + লাইট
  - alias: "Deterrence - Driveway Person Night"
    trigger:
      - platform: mqtt
        topic: frigate/driveway/person
        payload: "on"
    condition:
      - condition: time
        after: "22:00:00"
        before: "06:00:00"
    action:
      - service: siren.turn_on
        target:
          entity_id: siren.outdoor
      - service: light.turn_on
        target:
          entity_id: light.floodlights
        data:
          brightness_pct: 100
      - service: notify.mobile_app
        data:
          title: "Driveway Intrusion"
          message: "Person detected at driveway at {{ now().strftime('%H:%M') }}"
```

### Alexa রুটিন (অ্যাপ → Routines)

```
WHEN: Smart Home Device → [Front Door Camera] → Motion Detected → Person
IF: Time → Between Sunset and Sunrise
THEN:
    1. Smart Home → [Porch Light] → Turn On → 80% → 5 minutes
    2. Alexa Says → "Person at front door" → Device: [Kitchen Echo]
    3. Smart Home → [Floodlight] → Turn On → 100% → 3 minutes
```

### Google Home অটোমেশন (Home App → Automations)

```
STARTER: Device → [Driveway Camera] → Event → Person detected
CONDITION: Time → Between 10:00 PM and 6:00 AM
ACTIONS:
    1. Device → [Floodlights] → Turn On → Brightness 100%
    2. Device → [All Speakers] → Broadcast → "Person detected at driveway"
    3. Script → Log to Google Sheets (via Make/Zapier/webhook)
```

### Apple HomeKit (Home App → Automation)

```
WHEN: A Sensor Detects Something → [Front Door Camera] → People Detected
CONDITION: Time → After Sunset
DO:
    1. Scenes → [Porch Lights On] (পূর্ব-নির্মিত সিন: porch 80%, entry 50%)
    2. Wait → 5 minutes
    3. Scenes → [Porch Lights Off]
```

## মাইগ্রেশন পাথ: ক্লাউড থেকে লোকালে

1. **আপাতত ক্লাউড ক্যাম রাখুন** (Ring ডোরবেল, Blink আউটডোর)
2. **PoE ক্যাম যোগ করুন** (Reolink, Dahua) → NVR (UniFi/Reolink) বা Frigate সার্ভার
3. **ক্লাউড ক্যাম ব্রিজ করুন** Homebridge/Scrypted → HomeKit/Google/Alexa-তে
4. **অটোমেশন সরান** Home Assistant-এ (বা HA + Alexa/Google রুটিন)
5. **ব্রিজড ক্যামে ক্লাউড রেকর্ডিং নিষ্ক্রিয় করুন**; লোকাল NVR/Frigate ব্যবহার করুন
6. **সাবস্ক্রিপশন বাতিল করুন** যখন লোকাল রেকর্ডিং যাচাই করা হয়েছে
7. **ভবিষ্যতের ক্যাম:** শুধু ONVIF/RTSP + Matter কিনুন
