اسمارٹ ہوم انٹیگریشن سیکیورٹی کیمروں کو **صرف کیمرے** سے **پورے سمارٹ ہوم کا حصہ** بنا دیتا ہے — دروازہ کھلنے پر لائٹ آن کرنا، کھڑکی ٹوٹنے پر الارم بجانا، یا "Alexa، سامنے والا دروازہ دکھا" کہہ کر دیکھنا۔

یہ گائیڈ **Alexa، Google Home، اور Apple HomeKit** کے ساتھ اپنے کیمروں کو مربوط کرنے کا طریقہ بتاتی ہے۔

---

## کیوں انٹیگریٹ کریں؟

1. **وائس کنٹرول:** "Hey Google، گیراج دکھائیں"
2. **آٹومیشن:** موشن ڈیٹیکٹ → لائٹ آن
3. **نوٹیفکیشن:** کھڑکی کھلنے پر دروازہ لاک
4. **ایک انٹرفیس:** تمام کیمرے ایک ڈیش بورڈ میں
5. **آنے والے کو دیکھیں:** ڈور بیل → Fire TV

### Alexa کے ساتھ انٹیگریشن

**مراحل:**

1. کیمرہ ایپ → Amazon اکاؤنٹ سے لنک کریں
2. Alexa ایپ → Skills → کیمرہ سکل تلاش کریں → فعال کریں
3. ڈیوائس ڈسکوری → Alexa ڈیوائس ڈھونڈتی ہے
4. گروپس → کیمرے کو کمرے میں شامل کریں

**Alexa روٹینز:**

| محرک        | کارروائی  | مثال                                            |
| ----------- | --------- | ----------------------------------------------- |
| موشن ڈیٹیکٹ | لائٹ آن   | جب ڈرائیو وے کا کیمرہ حرکت دیکھے → پورچ لائٹ آن |
| ڈور بیل بجے | اعلان     | جب ڈور بیل بجے → "کوئی سامنے والے دروازے پر ہے" |
| وائس        | لائیو ویو | "Alexa، سامنے والا دروازہ دکھا"                 |
| سینسر       | کیمرہ آن  | جب فرنٹ ڈور سینسر کھلے → کیمرہ ریکارڈنگ شروع    |
| شیڈول       | کیمرہ آف  | رات 11 بجے → کیمرہ بند (پرائیویسی)              |

**بہترین روٹینز:**

- "میں جا رہا ہوں" → تمام کیمرے آن، لائٹس بند، دروازے لاک
- "میں واپس آ گیا" → تمام کیمرے بند، لائٹس آن، آرام
- "گڈ نائٹ" → بیڈروم کیمرے بند، بیرونی کیمرے آن

### Google Home کے ساتھ انٹیگریشن

**مراحل:**

1. کیمرہ ایپ → Google اکاؤنٹ سے لنک کریں
2. Google Home ایپ → ڈیوائس شامل کریں → ورکس وید گوگل
3. ڈیوائس ڈسکوری → گوگل ڈیوائس ڈھونڈتا ہے
4. گروپس → کمرے تفویض کریں

**Google روٹینز:**

| محرک        | کارروائی     | مثال                                        |
| ----------- | ------------ | ------------------------------------------- |
| موشن ڈیٹیکٹ | کیمرہ اسکرین | جب کیمرہ حرکت دیکھے → Nest Hub پر لائیو ویو |
| وائس        | کیمرہ آن     | "Hey Google، گیراج دکھائیں"                 |
| ڈور بیل     | نوٹیفکیشن    | جب ڈور بیل بجے → پش نوٹیفکیشن               |
| اسسٹنٹ      | روٹین        | "گڈ مارننگ" → کیمرے بند، شیڈز کھولیں        |

### Apple HomeKit کے ساتھ انٹیگریشن

**مراحل:**

1. کیمرہ ایپ → HomeKit کوڈ اسکین کریں
2. Home ایپ → کیمرہ ظاہر ہوتا ہے
3. زونز → سرگرمی کے زونز سیٹ کریں
4. آٹومیشن → منظر نامے بنائیں

**HomeKit خصوصیات:**

| فیچر                     | تفصیل                           |
| ------------------------ | ------------------------------- |
| **HomeKit Secure Video** | کلاؤڈ اسٹوریج (iCloud+ کے ساتھ) |
| **آٹومیشن**              | موشن ڈیٹیکٹ → لائٹ آن           |
| **منظر نامے**            | "گڈ نائٹ" → کیمرے آن            |
| **قابلیت**               | Siri، Home ایپ، Apple TV        |
| **پرائیویسی**            | اینڈ ٹو اینڈ انکرپٹڈ            |

| **AI**        | مقامی (Apple)        | کلاؤڈ   | کلاؤڈ   |

| **آٹومیشن کام نہیں کرتی**     | محرک اور کارروائی کی تصدیق کریں  |
| **نوٹیفکیشن نہیں آتے**        | ایپ کی اجازتیں چیک کریں          |
| **آڈیو کام نہیں کرتا**        | مائک / اسپیکر والیوم چیک کریں    |

## Matter: متحدہ مستقبل

Matter ایک نیا سمارٹ ہوم معیار ہے جو تمام پلیٹ فارمز پر ڈیوائسز کو مربوط کرتا ہے۔ **TP-Link Tapo C120** اور **Aqara Camera Hub G3** پہلے سے Matter کو سپورٹ کرتے ہیں۔

**Matter کیمروں کے فوائد:**

- **کراس پلیٹ فارم:** Alexa + Google + HomeKit ایک ساتھ
- **مقامی:** کلاؤڈ کی ضرورت نہیں، LAN پر چلتا ہے
- **محفوظ:** اینڈ ٹو اینڈ انکرپشن
- **آسان:** QR کوڈ اسکین کریں، خود بخود کنیکٹ

## بہترین طریقے

1. **ایک پلیٹ فارم چنیں** — تینوں کو ایک ساتھ استعمال نہ کریں
2. **وائی فائی کی منصوبہ بندی کریں** — تمام ڈیوائسز ایک نیٹ ورک پر ہوں
3. **نام رکھیں** — "سامنے والا دروازہ"، "گیراج" (واضح ہوں)
4. **پرائیویسی** — اندرونی کیمروں کو غیر فعال کرنے کا شیڈول بنائیں
5. **بیک اپ** — انٹرنیٹ کے بغیل لوکل رسائی کا منصوبہ رکھیں

## انٹیگریشن کے عمومی سوالات

**س: کیا ایک پلیٹ فارم پر ایک سے زیادہ کیمرے استعمال کر سکتے ہیں؟**
ج: جی ہاں — زیادہ تر پلیٹ فارم ایک سے زیادہ کیمروں کو سپورٹ کرتے ہیں۔

**س: کیا سمارٹ ہوم انٹیگریشن کے لیے انٹرنیٹ ضروری ہے؟**
ج: کلاؤڈ انٹیگریشن کے لیے جی ہاں۔ مقامی (HomeKit/LAN) کے لیے نہیں۔

**س: کیا Alexa اور Google Home دونوں ایک ساتھ استعمال کر سکتے ہیں؟**
ج: جی ہاں — اگر کیمرہ دونوں کو سپورٹ کرتا ہے۔ لیکن ایک پرائمری رکھیں۔

**س: کیا سمارٹ ہوم انٹیگریشن محفوظ ہے؟**
ج: جی ہاں — TLS اور اینڈ ٹو اینڈ انکرپشن کے ساتھ۔

**س: کیا پرائیویسی کے خدشات ہیں؟**
ج: جی ہاں — کیمرے کو غیر فعال کرنے والے روٹینز بنائیں، خاص طور پر اندرونی کیمروں کے لیے۔

**س: کیا Matter کیمروں کے ساتھ کام کرتا ہے؟**
ج: جی ہاں — TP-Link Tapo C120 اور Aqara Camera Hub G3 Matter کو سپورٹ کرتے ہیں۔

## ہر پلیٹ فارم کے لیے مرحلہ وار سیٹ اپ

### Alexa سیٹ اپ

1. Amazon Alexa ایپ → Skills → اپنے کیمرہ کا سکل شامل کریں
2. کیمرہ ایپ میں Amazon اکاؤنٹ سے لنک کریں
3. Alexa ایپ میں "Discover Devices" چلائیں
4. کیمرہ کو کمرے میں تفویض کریں
5. روٹین بنائیں: جب کیمرہ حرکت دیکھے → "لائٹ آن کریں"

### Google Home سیٹ اپ

1. Google Home ایپ → "Add" → "Works with Google"
2. اپنے کیمرہ کا برانڈ منتخب کریں
3. Google اکاؤنٹ سے لنک کریں
4. ڈیوائس ڈھونڈیں اور کمرے میں تفویض کریں
5. روٹین: "Hey Google، سامنے والا دروازہ دکھائیں"

### HomeKit سیٹ اپ

1. کیمرہ ایپ → HomeKit کوڈ اسکین کریں (کیمرہ پر یا باکس پر)
2. Apple Home ایپ → کیمرہ ظاہر ہوتا ہے
3. زونز اور آٹومیشن سیٹ کریں
4. iCloud+ میں HomeKit Secure Video فعال کریں
5. Siri سے کہیں: "Siri، گیراج دکھاؤ"

- **Google:** Google Nest Cam + Google Home
- **Apple:** EufyCam 3 + HomeKit Secure Video
- **مستقبل:** TP-Link Tapo C120 + Matter
