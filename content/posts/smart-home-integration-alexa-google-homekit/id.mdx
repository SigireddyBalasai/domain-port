### Apple HomeKit (HomeKit Secure Video)

**Dukungan Asli:** Eufy (HomeBase 3), Arlo (Pro 3/4/5S, Ultra), Logitech Circle View, Netatmo, Eve, Aqara, TP-Link Tapo (Matter), UniFi Protect (3.0+), Onvis, Wemo  
**Bridge Diperlukan:** Reolink (via Homebridge/Scrypted), Ring (via Homebridge), Wyze (via Homebridge), Dahua/Hikvision (via Homebridge/Scrypted)

<Callout type="info">

**HomeKit Secure Video (HSV) = Privasi Terbaik.** Video dianalisis **di perangkat**
(Home Hub: Apple TV 4K, HomePod, iPad). **Dienkripsi end-to-end** — Apple
tidak dapat melihat. Penyimpanan bergulir 10 hari **termasuk dengan iCloud+** (50GB = 1 kamera,
200GB = 5 kamera, 2TB = tak terbatas). Tidak ada langganan terpisah.

</Callout>

<Accordion type="single" collapsible>
  <AccordionItem value="hsv-features">
    <AccordionTrigger>Fitur & Persyaratan HSV</AccordionTrigger>
    <AccordionContent>

**Resolusi Didukung:** 1080p, 2K, 4K (diturunkan ke 1080p untuk analisis, disimpan pada resolusi asli)
**Zona Aktivitas:** Per kamera, atur di aplikasi Home
**Pengenalan Wajah:** "Orang terdeteksi" → "John terdeteksi" (jika di album Orang)
**Opsi Rekaman:** Mati / Deteksi Aktivitas / Streaming & Rekam
**Penyimpanan:** iCloud+ (50GB/$0,99, 200GB/$2,99, 2TB/$9,99) — **bukan per kamera**
**Home Hub Diperlukan:** Apple TV 4K (2021+), HomePod (mini/gen 2), iPad (selalu aktif)
**Streaming Lokal:** iPhone/iPad di Wi-Fi yang sama → langsung (tidak ada internet). Jarak jauh → relay melalui iCloud (dienkripsi).
**Keterbatasan:** Maks 5 kamera pada paket 200GB (kecuali 2TB); tidak ada 24/7 terus menerus (hanya peristiwa); tidak ada tampilan web (hanya iOS/macOS); tidak ada Android asli.

    </AccordionContent>

  </AccordionItem>
  <AccordionItem value="homebridge">
    <AccordionTrigger>Bridge Kamera Non-HomeKit (Homebridge / Scrypted / Hoobs)</AccordionTrigger>
    <AccordionContent>

**Homebridge** (Node.js, 2000+ plugin): Berjalan di Pi, Docker, NAS, mini PC

- `homebridge-camera-ffmpeg` → RTSP → HSV (mentranskode, menambahkan gerakan)
- `homebridge-reolink` → API asli Reolink (gerakan, PTZ, cuplikan)
- `homebridge-dahua` / `homebridge-hikvision` → ONVIF/ISAPI
- `homebridge-ring` → Kamera/bel pintu Ring (polling, ~30s latensi)

**Scrypted** (TypeScript, lebih baru, lebih cepat): Plugin HomeKit, Google, Alexa asli

- Penanganan RTSP lebih baik, latensi lebih rendah, pasar plugin
- Berjalan pada perangkat keras yang sama dengan Homebridge

**Perangkat Keras untuk Bridge:** Raspberry Pi 4 (4GB+) / 5, mini PC Intel N100/N305 ($120), laptop lama, Synology/QNAP Docker.
**Latensi:** Homebridge FFmpeg ~2–5s; Scrypted ~1–3s; HSV asli ~0,5–1s.
**HSV melalui Bridge:** Kamera harus mendukung RTSP. Bridge mentranskode → HSV → iCloud. **Dihitung ke batas kamera HSV.**

</AccordionContent>

  </AccordionItem>
</Accordion>

**Matter 1.3 (2024)** menambahkan tipe perangkat **Kamera**. **Matter 1.4 (2025)** menambahkan tipe **Bel Pintu**.

    <AccordionContent>

**Kamera Matter Bersertifikat:**

- **TP-Link Tapo C120 / C220 / C320WS** (Matter melalui Wi-Fi)
- **Eve Cam** (Thread + Matter)
- **Google Nest Cam (berkabel, 2024)** / **Nest Doorbell (berkabel, 2024)** — bridge Matter melalui Google Home
- **EufyCam 3 / SoloCam S340** — Pembaruan firmware Matter (2024)

**Apa yang Diberikan Matter kepada Anda:**

- **Multi-admin:** Tambahkan ke Apple Home, Google Home, Alexa, SmartThings secara bersamaan
- **Kontrol Lokal:** Tidak ada relay cloud untuk dasar on/off/stream (Thread/Wi-Fi langsung)
- **Perintah Standar:** Streaming langsung, peristiwa gerakan, mode privasi

**Apa yang BELUM Dilakukan Matter:**

- Streaming 4K (maks 1080p dalam spesifikasi)
- Kontrol PTZ (tidak dalam 1.3/1.4)
- Pengenalan wajah / deteksi paket (hanya ekstensi vendor)
- Rekaman terus menerus (hanya klip peristiwa)
- Integrasi NVR

**Verdik (2026):** Kamera Matter berfungsi untuk **tampilan langsung multi-platform dasar + peringatan gerakan**. Untuk rekaman, AI, PTZ, 4K → masih perlu integrasi asli atau bridge.

## Kontrol Lokal / Tanpa Cloud: Jalur Lanjutan

<Callout type="tip">

**Lokal = Cepat, Pribadi, Berfungsi Offline.** RTSP/ONVIF/WebSocket →
Frigate/Scrypted/Blue Iris → Home Assistant → Dasbor/Otomatisasi. Tidak ada
cloud vendor yang pernah melihat video Anda.

</Callout>

### Arsitektur: Tumpukan Kamera Sepenuhnya Lokal

```
Kamera (PoE/RTSP)
    │
    ▼
┌─────────────────────────────────────┐
│  Frigate (Docker)                   │
│  - Ingest RTSP                      │
│  - Deteksi gerakan (CPU)           │
│  - Deteksi objek AI (Coral TPU)  │
│  - Rekaman (MP4, tersegmentasi)       │
│  - Peristiwa MQTT (gerakan, orang, dll)│
│  - Tampilan langsung WebRTC (sub-detik)    │
└─────────────────────────────────────┘
    │                    │
    ▼                    ▼
┌─────────────┐    ┌─────────────┐
│ Home Asst.  │    │  Scrypted   │
│ (Otomatisasi)│    │ (HomeKit/   │
│  - Lampu   │    │  Google/    │
│  - Kunci    │    │  Alexa)     │
│  - Peringatan   │    │             │
└─────────────┘    └─────────────┘
    │                    │
    ▼                    ▼
Dasbor           Apple Home /
                    Google Home /
                    Aplikasi Alexa
```

### Perangkat Keras untuk Tumpukan Lokal

| Komponen           | Anggaran          | Direkomendasikan                       | High-End                       |
| ------------------ | ----------------- | -------------------------------------- | ------------------------------ |
| **Server**         | RPi 4 4GB ($80)   | Mini PC Intel N100/N305 ($130)         | i5-13400 / Ryzen 5700G ($400)  |
| **Akselerator AI** | CPU saja (lambat) | **Google Coral USB ($60)** / M.2 ($40) | Coral PCIe x4 / Hailo-8 ($150) |
| **Penyimpanan**    | 1× 4TB HDD        | 2× 8TB RAID 1                          | 4× 14TB RAID 6 / ZFS           |
| **Jaringan**       | 1 Gbps            | Switch 2.5 Gbps + NIC                  | Backbone 10 Gbps               |

### Persyaratan Kamera untuk Tumpukan Lokal

- **Dukungan RTSP** (standar: `rtsp://user:pass@ip:554/stream1`)
- **ONVIF Profile S/T/G** (penemuan, PTZ, peristiwa)
- **Sub-stream** (D1/720p untuk AI, 4K untuk rekaman)
- **Tidak ada ketergantungan cloud** untuk streaming (beberapa kamera memerlukan token auth cloud — hindari)

**Kamera ONVIF/RTSP Terbaik (2026):**

- Reolink (semua PoE, sebagian besar Wi-Fi): `rtsp://ip:554/h264Preview_01_main`
- Dahua / Hikvision / Hanwha / Axis / Uniview / Vivotek / Milesight
- UniFi G4/G5 (RTSP melalui pengaturan Protect)
- Amcrest / Lorex / Annke / Montavue (OEM Dahua/Hik)
- **Hindari:** Ring, Blink, Nest, Arlo (pra-2024), Wyze (terbatas), Eufy (tidak ada RTSP pada kamera baterai)

## Resep Otomatisasi (Siap Salin-Tempel)

### Home Assistant (YAML)

```yaml
# Lampu Teras saat Orang (setelah matahari terbenam)
otomatisasi:
  - alias: "Lampu Teras - Orang di Pintu Depan"
    trigger:
      - platform: mqtt
        topic: frigate/front_door/person
        payload: "on"
    condition:
      - condition: sun
        after: sunset
        after_offset: "-00:30:00"
    action:
      - service: light.turn_on
        target:
          entity_id: light.porch
        data:
          brightness_pct: 80
      - delay: "00:05:00"
      - service: light.turn_off
        target:
          entity_id: light.porch

  # Pencegahan: Sirine + Lampu saat Orang di Jalan Masuk (malam)
  - alias: "Pencegahan - Orang di Jalan Masuk Malam"
    trigger:
      - platform: mqtt
        topic: frigate/driveway/person
        payload: "on"
    condition:
      - condition: time
        after: "22:00:00"
        before: "06:00:00"
    action:
      - service: siren.turn_on
        target:
          entity_id: siren.outdoor
      - service: light.turn_on
        target:
          entity_id: light.floodlights
        data:
          brightness_pct: 100
      - service: notify.mobile_app
        data:
          title: "🚨 Intrusi Jalan Masuk"
          message: "Orang terdeteksi di jalan masuk pada {{ now().strftime('%H:%M') }}"
```

### Rutinitas Alexa (Aplikasi → Rutinitas)

```
KAPAN: Perangkat Rumah Pintar → [Kamera Pintu Depan] → Gerakan Terdeteksi → Orang
JIKA: Waktu → Antara Matahari Terbenam dan Matahari Terbit
MAKA:
    1. Rumah Pintar → [Lampu Teras] → Nyalakan → 80% → 5 menit
    2. Alexa Mengatakan → "Orang di pintu depan" → Perangkat: [Echo Dapur]
    3. Rumah Pintar → [Lampu Sorot] → Nyalakan → 100% → 3 menit
```

### Otomatisasi Google Home (Aplikasi Home → Otomatisasi)

```
PEMULA: Perangkat → [Kamera Jalan Masuk] → Peristiwa → Orang terdeteksi
KONDISI: Waktu → Antara 22:00 dan 6:00
TINDAKAN:
    1. Perangkat → [Lampu Sorot] → Nyalakan → Kecerahan 100%
    2. Perangkat → [Semua Speaker] → Siarkan → "Orang terdeteksi di jalan masuk"
    3. Skrip → Catat ke Google Sheets (via Make/Zapier/webhook)
```

### Apple HomeKit (Aplikasi Home → Otomatisasi)

```
KAPAN: Sensor Mendeteksi Sesuatu → [Kamera Pintu Depan] → Orang Terdeteksi
KONDISI: Waktu → Setelah Matahari Terbenam
LAKUKAN:
    1. Adegan → [Lampu Teras Nyala] (adegan yang sudah dibuat: teras 80%, pintu masuk 50%)
    2. Tunggu → 5 menit
    3. Adegan → [Lampu Teras Mati]
```

## Perbandingan Platform: Mana yang Harus Dipilih?

| Prioritas                  | Platform Terbaik                        | Mengapa                                               |
| -------------------------- | --------------------------------------- | ----------------------------------------------------- |
| **Privasi / Tanpa Cloud**  | **HomeKit (HSV) + Bridge Lokal**        | Enkripsi E2E, iCloud+ termasuk, AI di perangkat       |
| **Suara + Layar**          | **Alexa** (ekosistem Echo Show)         | Skill terbanyak, UI rutinitas terbaik, layar murah    |
| **Kekuatan Otomatisasi**   | **Google Home** (editor skrip)          | Jenis pemula terbaik, logika kompleks, grid Nest Hub  |
| **Lokal + Multi-Platform** | **Home Assistant + Frigate + Scrypted** | Kontrol total, bridge Matter, tanpa vendor lock-in    |
| **Pengaturan Termudah**    | **Ring / Blink / Nest** (merek tunggal) | Asli, tanpa bridge, tetapi berlangganan + cloud       |
| **Bisnis / Multi-Situs**   | **UniFi Protect**                       | Terpusat, tanpa biaya cloud, HSV + skill Alexa/Google |
| **Future-Proof Matter**    | **TP-Link Tapo / Eve / Nanoleaf**       | Thread + Matter, berfungsi di mana-mana               |

## Jalur Migrasi: Dari Cloud ke Lokal

1. **Pertahankan kamera cloud** untuk saat ini (bel pintu Ring, Blink luar ruangan)
2. **Tambahkan kamera PoE** (Reolink, Dahua) → NVR (UniFi/Reolink) atau server Frigate
3. **Bridge kamera cloud** melalui Homebridge/Scrypted → HomeKit/Google/Alexa
4. **Pindahkan otomatisasi** ke Home Assistant (atau HA + rutinitas Alexa/Google)
5. **Nonaktifkan rekaman cloud** pada kamera yang di-bridge; gunakan NVR/Frigate lokal
6. **Batalkan langganan** saat rekaman lokal terverifikasi
7. **Kamera masa depan:** Beli hanya ONVIF/RTSP + Matter
