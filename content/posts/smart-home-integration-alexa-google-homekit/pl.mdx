**Wymagany Skill:** Reolink, Eufy, UniFi, Wyze, Tapo, Amcrest, TP-Link Kasa

### Apple HomeKit (HomeKit Secure Video)

**Natywne Wsparcie:** Eufy (HomeBase 3), Arlo (Pro 3/4/5S, Ultra), Logitech Circle View, Netatmo, Eve, Aqara, TP-Link Tapo (Matter), UniFi Protect (3.0+), Onvis, Wemo
**Wymagany Bridge:** Reolink (przez Homebridge/Scrypted), Ring (przez Homebridge), Wyze (przez Homebridge), Dahua/Hikvision (przez Homebridge/Scrypted)

<Callout type="info">

**HomeKit Secure Video (HSV) = Najlepsza Prywatność.** Wideo analizowane **na
urządzeniu** (Home Hub: Apple TV 4K, HomePod, iPad). **Szyfrowane end-to-end**
— Apple nie może oglądać. 10-dniowe cykliczne przechowywanie **w zestawie z
iCloud+** (50GB = 1 kamera, 200GB = 5 kamer, 2TB = nieograniczone). Bez
osobnej subskrypcji.

</Callout>

<Accordion type="single" collapsible>
  <AccordionItem value="hsv-features">
    <AccordionTrigger>Funkcje i Wymagania HSV</AccordionTrigger>
    <AccordionContent>

**Obsługiwane Rozdzielczości:** 1080p, 2K, 4K (skalowane do 1080p do analizy, przechowywane w oryginalnej rozdz.)
**Strefy Aktywności:** Na kamerę, ustawiane w aplikacji Dom
**Rozpoznawanie Twarzy:** "Wykryto osobę" → "Wykryto Jana" (jeśli w albumie Osoby)
**Opcje Nagrywania:** Wył. / Wykrywaj Aktywność / Streamuj i Nagrywaj
**Pamięć:** iCloud+ (50GB/3,99 zł, 200GB/11,99 zł, 2TB/39,99 zł) — **nie na kamerę**
**Wymagany Home Hub:** Apple TV 4K (2021+), HomePod (mini/2. gen), iPad (zawsze włączony)
**Lokalne Streamowanie:** iPhone/iPad w tej samej sieci Wi-Fi → bezpośrednio (bez internetu). Zdalny → przez iCloud (szyfrowane).
**Ograniczenia:** Max 5 kamer na planie 200GB (chyba że 2TB); brak 24/7 ciągłego (tylko zdarzenia); brak widoku WWW (tylko iOS/macOS); brak natywnego Androida.

    </AccordionContent>

  </AccordionItem>
  <AccordionItem value="homebridge">
    <AccordionTrigger>Bridge dla Nie-HomeKit Kamer (Homebridge / Scrypted / Hoobs)</AccordionTrigger>
    <AccordionContent>

**Homebridge** (Node.js, 2000+ pluginów): Działa na Pi, Docker, NAS, mini PC

- `homebridge-camera-ffmpeg` → RTSP → HSV (transkoduje, dodaje ruch)
- `homebridge-reolink` → Reolink natywne API (ruch, PTZ, zrzuty)
- `homebridge-dahua` / `homebridge-hikvision` → ONVIF/ISAPI
- `homebridge-ring` → Ring kamery/domofony (polling, ~30s opóźnienie)

**Scrypted** (TypeScript, nowszy, szybszy): Natywne pluginy HomeKit, Google, Alexa

- Lepsze przetwarzanie RTSP, niższe opóźnienie, rynek pluginów
- Działa na tym samym sprzęcie co Homebridge

**Sprzęt dla Bridge:** Raspberry Pi 4 (4GB+) / 5, Intel N100/N305 Mini PC (480 zł), stary laptop, Synology/QNAP Docker.
**Opóźnienie:** Homebridge FFmpeg ~2–5s; Scrypted ~1–3s; Natywne HSV ~0.5–1s.
**HSV przez Bridge:** Kamera musi obsługiwać RTSP. Bridge transkoduje → HSV → iCloud. **Liczy się do limitu kamer HSV.**

</AccordionContent>

  </AccordionItem>
</Accordion>

### Matter (Nowy Standard)

**Matter 1.3 (2024)** dodaje typ urządzenia **Kamera**. **Matter 1.4 (2025)** dodaje typ **Domofon**.

<Accordion type="single" collapsible>
  <AccordionItem value="matter-status">
    <AccordionTrigger>Bieżące Wsparcie Matter dla Kamer (2026)</AccordionTrigger>
    <AccordionContent>

**Certyfikowane Kamery Matter:**

- **TP-Link Tapo C120 / C220 / C320WS** (Matter przez Wi-Fi)
- **Nanoleaf Smart Cam** (Matter)
- **Eve Cam** (Thread + Matter)
- **Google Nest Cam (przewodowa, 2024)** / **Nest Doorbell (przewodowa, 2024)** — bridge Matter przez Google Home
- **EufyCam 3 / SoloCam S340** — aktualizacja firmware Matter (2024)

**Co Daje Ci Matter:**

- **Multi-admin:** Dodaj do Apple Home, Google Home, Alexa, SmartThings jednocześnie
- **Lokalne Sterowanie:** Brak przekaźnika w chmurze dla podstawowego wł./wył./stream (Thread/Wi-Fi direct)
- **Standaryzowane Polecenia:** Transmisja na żywo, zdarzenia ruchu, tryb prywatności

**Czego Matter JESZCZE NIE ROBI:**

- 4K streaming (max 1080p w specyfikacji)
- Sterowanie PTZ (nie w 1.3/1.4)
- Rozpoznawanie twarzy / wykrywanie paczek (tylko rozszerzenia producenta)
- Ciągłe nagrywanie (tylko klipy zdarzeń)
- Integracja NVR

**Werdykt (2026):** Kamery Matter działają do **podstawowego multi-platform podglądu na żywo + alertów ruchu**. Do nagrywania, AI, PTZ, 4K → nadal potrzebna natywna integracja lub bridge.

## Lokalne Sterowanie / Brak Chmury: Zaawansowana Ścieżka

<Callout type="tip">

**Lokalne = Szybkie, Prywatne, Działa Offline.** RTSP/ONVIF/WebSocket →
Frigate/Scrypted/Blue Iris → Home Assistant → Dashboards/Automatyzacje. Żadna
chmura producenta nigdy nie widzi Twojego wideo.

</Callout>

### Architektura: W Pełni Lokalny Stos Kamer

```
Kamery (PoE/RTSP)
    │
    ▼
┌─────────────────────────────────────┐
│  Frigate (Docker)                   │
│  - RTSP ingest                      │
│  - Detekcja ruchu (CPU)             │
│  - Detekcja obiektów AI (Coral TPU) │
│  - Nagrywanie (MP4, segmentowane)   │
│  - Zdarzenia MQTT (ruch, itp)       │
│  - Podgląd na żywo WebRTC (sub-sek) │
└─────────────────────────────────────┘
    │                    │
    ▼                    ▼
┌─────────────┐    ┌─────────────┐
│ Home Asst.  │    │  Scrypted   │
│ (Autom.)    │    │ (HomeKit/   │
│  - Światła  │    │  Google/    │
│  - Zamki    │    │  Alexa)     │
│  - Alerty   │    │             │
└─────────────┘    └─────────────┘
    │                    │
    ▼                    ▼
Dashboard           Apple Home /
                    Google Home /
                    Aplikacja Alexa
```

### Sprzęt dla Lokalnego Stosu

| Komponent          | Budżetowy          | Zalecany                                     | High-End                          |
| ------------------ | ------------------ | -------------------------------------------- | --------------------------------- |
| **Serwer**         | RPi 4 4GB (320 zł) | Intel N100/N305 Mini PC (520 zł)             | i5-13400 / Ryzen 5700G (1.600 zł) |
| **Akcelerator AI** | Tylko CPU (wolne)  | **Google Coral USB (240 zł)** / M.2 (160 zł) | Coral PCIe x4 / Hailo-8 (600 zł)  |
| **Pamięć**         | 1× 4TB HDD         | 2× 8TB RAID 1                                | 4× 14TB RAID 6 / ZFS              |
| **Sieć**           | 1 Gbps             | 2.5 Gbps switch + NIC                        | 10 Gbps backbone                  |

### Wymagania Kamery dla Lokalnego Stosu

- **Obsługa RTSP** (standard: `rtsp://user:pass@ip:554/stream1`)
- **ONVIF Profile S/T/G** (odkrywanie, PTZ, zdarzenia)
- **Podstrumienie** (D1/720p do AI, 4K do nagrywania)
- **Brak zależności od chmury** dla streamu (niektóre kamery wymagają tokena uwierzytelniającego w chmurze — unikaj)

**Najlepsze Kamery ONVIF/RTSP (2026):**

- Reolink (wszystkie PoE, większość Wi-Fi): `rtsp://ip:554/h264Preview_01_main`
- Dahua / Hikvision / Hanwha / Axis / Uniview / Vivotek / Milesight
- UniFi G4/G5 (RTSP przez ustawienia Protect)
- Amcrest / Lorex / Annke / Montavue (OEM Dahua/Hik)
- **Unikaj:** Ring, Blink, Nest, Arlo (przed 2024), Wyze (ograniczone), Eufy (brak RTSP na kamerach bateryjnych)

## Przepisy Automatyzacji (Gotowe do Kopiowania)

### Home Assistant (YAML)

```yaml
# Światło Ganku przy Osobie (po zachodzie słońca)
automation:
  - alias: "Światło Ganku - Osoba przy Drzwiach Frontowych"
    trigger:
      - platform: mqtt
        topic: frigate/drzwi_frontowe/osoba
        payload: "on"
    condition:
      - condition: sun
        after: sunset
        after_offset: "-00:30:00"
    action:
      - service: light.turn_on
        target:
          entity_id: light.ganek
        data:
          brightness_pct: 80
      - delay: "00:05:00"
      - service: light.turn_off
        target:
          entity_id: light.ganek

  # Odstraszanie: Syrena + Światła przy Osobie na Podjeździe (noc)
  - alias: "Odstraszanie - Podjazd Osoba Noc"
    trigger:
      - platform: mqtt
        topic: frigate/podjazd/osoba
        payload: "on"
    condition:
      - condition: time
        after: "22:00:00"
        before: "06:00:00"
    action:
      - service: siren.turn_on
        target:
          entity_id: siren.na_zewnatrz
      - service: light.turn_on
        target:
          entity_id: light.reflektory
        data:
          brightness_pct: 100
      - service: notify.mobile_app
        data:
          title: "Naruszenie Podjazdu"
          message: "Wykryto osobę na podjeździe o {{ now().strftime('%H:%M') }}"
```

## Porównanie Platform: Którą Wybrać?

| Priorytet                      | Najlepsza Platforma                     | Dlaczego                                                         |
| ------------------------------ | --------------------------------------- | ---------------------------------------------------------------- |
| **Prywatność / Brak Chmury**   | **HomeKit (HSV) + Lokalny Bridge**      | Szyfrowanie E2E, iCloud+ w zestawie, AI na urządzeniu            |
| **Głos + Wyświetlacze**        | **Alexa** (ekosystem Echo Show)         | Najwięcej skilli, najlepsze UI rutyn, tanie wyświetlacze         |
| **Moc Automatyzacji**          | **Google Home** (edytor skryptów)       | Najlepsze typy starterów, złożona logika, siatka Nest Hub        |
| **Lokalne + Multi-Platform**   | **Home Assistant + Frigate + Scrypted** | Pełna kontrola, bridge Matter, brak blokady producenta           |
| **Najłatwiejsza Konfiguracja** | **Ring / Blink / Nest** (jedna marka)   | Natywnie, bez bridge, ale subskrypcja + chmura                   |
| **Biznes / Multi-Lokalizacja** | **UniFi Protect**                       | Scentralizowane, brak opłat za chmurę, HSV + Alexa/Google skills |
| **Przyszłościowy Matter**      | **TP-Link Tapo / Eve / Nanoleaf**       | Thread + Matter, działa wszędzie                                 |

## Ścieżka Migracji: Z Chmury do Lokalnego

1. **Zachowaj kamery chmurowe** na razie (Ring domofon, Blink zewnętrzna)
2. **Dodaj kamery PoE** (Reolink, Dahua) → NVR (UniFi/Reolink) lub serwer Frigate
3. **Podłącz kamery chmurowe** przez Homebridge/Scrypted → HomeKit/Google/Alexa
4. **Przenieś automatyzacje** do Home Assistant (lub HA + rutyny Alexa/Google)
5. **Wyłącz nagrywanie w chmurze** na podłączonych kamerach; użyj lokalnego NVR/Frigate
6. **Anuluj subskrypcje** gdy lokalne nagrywanie zostanie zweryfikowane
7. **Przyszłe kamery:** Kupuj tylko ONVIF/RTSP + Matter
