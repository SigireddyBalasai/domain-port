กล้องจะดีเท่าที่อุปกรณ์ยึด กล้อง $400 บนสกรู drywall สองตัวจะพัง คู่มือนี้ครอบคลุมทุกพื้นผิวภายนอกด้วยฮาร์ดแวร์ที่ถูกต้อง

<Badge variant="outline">หลักการสำคัญ</Badge> **ยึดเข้ากับโครงสร้าง
ไม่ใช่ผิวหน้า** กล้องภายนอกต้องยึดกับไม้โครง ผนังคอนกรีต หรือจันทัน
ไม่ใช่กับผนังไวนิลหรือไม้ฝาโดยตรง

---

## ข้อมูลอ้างอิงด่วน

| พื้นผิว          | สกรู / Anchor           | ดอกสว่าน     | น้ำหนักสูงสุด |
| ---------------- | ----------------------- | ------------ | ------------- |
| ไม้จริง / จันทัน | #8–#10 × 5–7.6 ซม.      | 3 มม. นำร่อง | 9+ กก.        |
| โครงโลหะ         | #8–#10 เกลียวในตัว      | 3 มม. นำร่อง | 6.8 กก.       |
| ไวนิล            | บล็อกยึด + สกรู         | 10 มม.       | 2.3 กก.       |
| ปูน/อิฐ/คอนกรีต  | Tapcon 6 มม.            | 5 มม. งานก่อ | 13+ กก.       |
| ชายคา            | Lag bolt 10 มม.×7.6 ซม. | 6 มม.        | 13+ กก.       |

## การติดตั้งเฉพาะพื้นผิว

### 1. ไม้ / กรอบ

1. หาตำแหน่งไม้โครง
2. ทำเครื่องหมายรู
3. เจาะนำร่อง 3 มม.
4. ติดตั้งขายึดด้วยสกรู #10 × 6 ซม.
5. อุดซิลิโคนรอบหัวสกรู
6. ติดตั้งกล้อง ตรวจสอบระดับ

### 2. ปูนฉาบ / อิฐ / คอนกรีต

ใช้สว่านกระแทก SDS+ และ Tapcon 6 มม.

1. ทำเครื่องหมายตำแหน่ง
2. เจาะด้วยดอกสว่านก่ออิฐ 5 มม.—ลึกกว่าสกรู 6 มม.
3. เป่าฝุ่นออก
4. ใส่ Tapcon—ขันด้วยสว่าน (อย่าใช้ไขควงมือ)
5. กันน้ำด้วยซิลิโคนรอบสกรู
6. ติดตั้งกล้อง

### 3. ชายคา

1. หาตำแหน่งจันทัน (ใช้เครื่องหาไม้หรือตอกดู)
2. ทำเครื่องหมาย—ให้ Lag bolt อยู่กึ่งกลางจันทัน
3. เจาะนำร่อง 6 มม. ผ่านชายคาเข้าจันทัน (ลึก 5 ซม.)
4. ขัน Lag bolt 10 มม. × 7.6 ซม. ด้วยประแจกระบอก
5. อุดซิลิโคนรอบหัวbolt

<Callout type="warning">

**ห้ามยึดกล้องเข้ากับแผ่นชายคาโดยตรง!** แผ่นชายคามีความหนาเพียง 6–12 มม. และไม่สามารถรับน้ำหนักของกล้องได้โดยไม่โก่งงอ ต้องยึด Lag bolt ให้ถึงจันทันเสมอ

</Callout>

### 4. ผนังไวนิล

1. ตัดไวนิลตรงตำแหน่งติดตั้ง (4×4 ซม.)
2. ติดตั้งบล็อกรองรับเข้ากับผนังไม้ด้านหลัง
3. ติดตั้งขายึดเข้ากับบล็อกรองรับ
4. ใส่แผ่นปิดทับไวนิล (ถ้ามี)
5. อุดซิลิโคนรอบขอบ

### 5. เสา (ไม้ / โลหะ)

1. ใช้ขายึดแบบเสาพร้อมแถบยางกันลื่น
2. พันแถบรอบเสา—จัดตำแหน่ง
3. ขันน็อตยึดให้แน่น (อย่าขันเกินไป—เสาโลหะอาจบุบ)
4. ต่อสาย—ใช้สายรัดกัน UV
5. ติดตั้งกล้อง

## เครื่องมือที่ต้องใช้

| พื้นผิว     | เครื่องมือที่จำเป็น                                       |
| ----------- | --------------------------------------------------------- |
| ไม้         | สว่านธรรมดา, ดอก 3 มม., ไขควง, ระดับ                      |
| คอนกรีต/อิฐ | สว่านกระแทก SDS+, ดอกก่ออิฐ 5 มม., ที่เป่าฝุ่น, ระดับ     |
| ชายคา       | สว่าน, ดอก 6 มม., เครื่องหาไม้, ประแจกระบอก, บันได, ระดับ |
| โลหะ        | สว่าน, ดอกไบคู่ 3 มม. (ขั้นบันได), น้ำมันตัด, ระดับ       |
| ไวนิล       | สว่าน, กรรไกรตัดไวนิล, ดอก 3 มม. + 10 มม., ระดับ          |
| เสา         | ไขควงกระบอก, ไขควง, ระดับ                                 |

## การจัดการสายเคเบิล

| วิธี           | เหมาะกับ            | ความยาก | ความสวยงาม     |
| -------------- | ------------------- | ------- | -------------- |
| ร้อยใต้ชายคา   | กล้องชายคา          | ยาก     | ซ่อนมองไม่เห็น |
| เดินในผนัง     | กล้องผนังภายนอก     | ยากมาก  | ดีที่สุด       |
| ท่อ PVC ภายนอก | อิฐ/คอนกรีต         | ปานกลาง | ดี             |
| รางร้อยสาย     | ภายในโรงรถ/ใต้ชายคา | ง่าย    | ปานกลาง        |
| สายลอย         | ชั่วคราว            | ง่ายมาก | แย่            |

## การป้องกันน้ำ

| จุด            | วิธีกันน้ำ                  |
| -------------- | --------------------------- |
| หัวสาย RJ45    | เทป Butyl ที่ขั้วต่อ        |
| รูเจาะ         | ซิลิโคนอุดรอบสกรูและสาย     |
| ขั้วต่อ DC     | เทปไฟฟ้าหุ้ม + เทป Butyl    |
| ขั้วต่อ PoE    | ขั้วต่อกันน้ำพร้อมโอริง     |
| จุดต่อใต้ชายคา | ไม่ต้อง—ชายคากันน้ำอยู่แล้ว |

<Callout type="note">

**เทป Butyl ก่อนขันสกรู:** การวางเทป Butyl ระหว่างขายึดกับผนังก่อนติดตั้งช่วยป้องกันน้ำซึมผ่านรูดอกสว่านได้ดีกว่าการอุดซิลิโคนอย่างเดียว

</Callout>
