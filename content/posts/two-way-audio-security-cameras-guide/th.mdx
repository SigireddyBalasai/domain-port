เสียงสองทางเปลี่ยนกล้องให้เป็นอินเตอร์คอม: พูดคุยกับพนักงานส่งของ ขัดขวางผู้บุกรุก ตรวจสอบเด็ก/สัตว์เลี้ยง หรือสื่อสารกับพนักงาน แต่ไม่ใช่ "เสียงสองทาง" ทุกระบบที่เท่าเทียมกัน — half-duplex (วอkeyทอล์กกี้) vs full-duplex (สายโทรศัพท์) คุณภาพการตัดเสียงรบกวน และข้อกำหนดการยินยอมตามกฎหมายแตกต่างกันอย่างมาก

<Badge variant="outline">ความแตกต่างหลัก</Badge> **Half-duplex** = push-to-talk
ทีละทิศทาง (กล้องราคาประหยัดส่วนใหญ่) **Full-duplex** = พูด/ฟังพร้อมกัน
เหมือนโทรศัพท์ (กล้องพรีเมียม, อินเตอร์คอมเฉพาะ)

**MEMS (Micro-Electro-Mechanical Systems)** — มาตรฐานในกล้องสมัยใหม่ เล็ก ทนทาน การตอบสนองความถี่เรียบ

<ul>
  <li>
    <strong>รอบทิศทาง</strong> — รับเสียง 360°; ดีสำหรับครอบคลุมห้อง,
    ไม่ดีสำหรับการโฟกัสเฉพาะ{" "}
  </li>
  <li>
    <strong>ทิศทาง (beamforming array)</strong> — 2–4 MEMS + DSP การบังคับ;
    โฟกัสที่เสียงพูด, ปฏิเสธเสียงพื้นหลัง
  </li>
</ul>

<p>
  <strong>การวางตำแหน่งสำคัญ:</strong>
</p>

<ul>
  <li>กริ่งประตู: ด้านล่าง, เอียงลง → จับภาพผู้มาเยือนที่ 0.9–1.5 ม.</li>
  <li>Turret/Dome: ใต้เลนส์หรือด้านข้าง → จับภาพรัศมี 3–6 ม.</li>
  <li>Bullet: ด้านหลัง/ช่องระบายอากาศข้าง → มักอู้อี้จากตัวกล้อง</li>
  <li>ในร่ม: ด้านหน้าบน → ดีที่สุดสำหรับการสนทนาในห้อง</li>
</ul>

## การตัดเสียงรบกวน: อะไรที่ใช้ได้จริง

<Callout type="note">

**สาย DSP ในกล้องดี:** Beamforming → การระงับสัญญาณรบกวนคงที่ (Wiener/Kalman)
→ การระงับไม่คงที่ (DNN) → AEC (full-duplex) → AGC (เกนอัตโนมัติ) → Limiter

</Callout>

| ประเภทสัญญาณรบกวน        | วิธีการระงับ                               | การลดทั่วไป |
| ------------------------ | ------------------------------------------ | ----------- |
| ลม                       | เมชกายภาพ + spectral gating                | 15–25 dB    |
| HVAC / พัดลม / การจราจร  | Spectral subtraction / Wiener              | 10–20 dB    |
| สุนัขเห่า / เด็กกรีดร้อง | DNN-based non-stationary (RNNoise, NSNet2) | 15–30 dB    |
| เสียงตัวเอง (ก้อง)       | Adaptive AEC (NLMS/Kalman)                 | 25–40 dB    |
| Feedback ลำโพง (หอน)     | Notch filtering + gain margin              | การป้องกัน  |

**การตลาด vs ความเป็นจริง:** "AI Noise Cancellation" = มักเป็นแค่ spectral gate **การระงับสัญญาณรบกวน DNN จริง** ต้องใช้ NPU (Ambarella CV25, Rockchip RV1126, Google TPU, Apple Neural Engine) ตรวจสอบสเปก: "RNNoise", "NSNet2", "DNN-based", "Deep learning noise reduction"

## กฎหมาย: การยินยอมฝ่ายเดียว vs สองฝ่าย

<Callout type="warning">

**กฎหมายบันทึกเสียงเข้มงวดกว่าวิดีโอ** วิดีโอ =
ความคาดหวังความเป็นส่วนตัวที่สมเหตุสมผล (แตกต่างกันไป) เสียง =
กฎหมายการดักฟังโทรศัพท์ในหลายรัฐ **ข้อกำหนดการยินยอมใช้กับ _การบันทึก_
ไม่ใช่การฟังสด**

</Callout>

## กล้องยอดนิยมตามกรณีการใช้งาน (2026)

### 1. กริ่งประตู Full-Duplex ที่ดีที่สุด: Google Nest Doorbell (แบบมีสาย, รุ่น 2)

**เสียง:** Full-duplex, AEC ยอดเยี่ยม, ลำโพง 2W, ไมค์ 3 ตัวพร้อม beamforming  
**ตัดเสียงรบกวน:** TPU-accelerated DNN (ซิลิคอน Google) — ดีที่สุดในการปฏิเสธลม/การจราจร  
**ฟีเจอร์การยินยอม:** "ประกาศผู้มาเยือน" (กริ่ง + "มีคนที่ประตู" ก่อนบันทึก); สลับการบันทึกเสียงในแอป Home  
**การรวมระบบ:** Google Home, Assistant, Nest Aware ($8/เดือน สำหรับวิดีโอ + เสียง 30 วัน)  
**ไฟ:** แบบมีสายเท่านั้น (16–24V AC) — ไม่มีแบตเตอรี่  
**ราคา:** ~$180  
**เหมาะที่สุด:** รัฐสองฝ่าย (CA/FL/IL), สนทนา full-duplex, บ้าน Google

### 2. กริ่งประตู Half-Duplex ที่ดีที่สุด: Ring Battery Doorbell Pro (2024) / Ring Stick Up Cam Pro

**เสียง:** Half-duplex PTT, ลำโพง 1.5W, beamforming ไมค์คู่  
**ตัดเสียงรบกวน:** DSP "Audio+" (spectral gate + DNN พื้นฐาน) — ปฏิเสธลมใช้ได้  
**ฟีเจอร์การยินยอม:** สลับ "บันทึกเสียง" ต่อกล้อง; "ยืนยันการเคลื่อนไหว" ลดตัวกระตุ้นเสียงปลอม  
**การรวมระบบ:** แอป Ring, Alexa ("Alexa, talk to front door"), Ring Protect ($4–20/เดือน)  
**ไฟ:** แบตเตอรี่ (Quick Release) + อุปกรณ์เสริมโซลาร์เซลล์ + ตัวเลือกเดินสาย  
**ราคา:** ~$180–230  
**เหมาะที่สุด:** ความยืดหยุ่นแบตเตอรี่, บ้าน Alexa, รัฐฝ่ายเดียว

### 3. ท้องถิ่น/ไม่ใช้คลาวด์ที่ดีที่สุด: Reolink Video Doorbell PoE / Reolink Argus 4 Pro

**เสียง:** Half-duplex, ลำโพง 1W, MEMS ไมค์เดียว (PoE) / ไมค์คู่ (Argus 4 Pro)  
**ตัดเสียงรบกวน:** Basic spectral subtraction — ใช้ได้ในชานเมืองเงียบ  
**ฟีเจอร์การยินยอม:** **ไม่มีคลาวด์** — เสียงบันทึกไปยัง NVR/microSD เท่านั้น; คุณควบคุมการเก็บรักษา  
**การรวมระบบ:** แอป Reolink, ONVIF, NVR, Home Assistant (ผ่าน RTSP/ONVIF), ไม่มี HomeKit  
**ไฟ:** PoE (กริ่งประตู) / แบตเตอรี่+โซลาร์ (Argus 4 Pro)  
**ราคา:** ~$130 (PoE) / ~$220 (Argus 4 Pro + โซลาร์)  
**เหมาะที่สุด:** ผู้ที่ให้ความสำคัญกับความเป็นส่วนตัวสูงสุด, เจ้าของ NVR, ไม่ต้องสมัครสมาชิก

### 4. Full-Duplex ในร่มที่ดีที่สุด: Google Nest Cam (แบบมีสาย, รุ่น 2)

**เสียง:** Full-duplex, ไมค์ 3 ตัว, ลำโพง 2W, Google TPU DNN noise suppression  
**กรณีการใช้งาน:** เบบี้มอนิเตอร์, ตรวจสอบผู้สูงอายุ, พูดกับสัตว์เลี้ยง — **คุณภาพสายโทรศัพท์**  
**ความเป็นส่วนตัว:** ไฟ LED ฮาร์ดแวร์เมื่อไมค์ทำงาน; ปิดการบันทึกเสียงเป็นค่าเริ่มต้น  
**ราคา:** ~$100  
**เหมาะที่สุด:** การสนทนาสองทางในร่ม, Google Home

### 5. อินเตอร์คอมธุรกิจที่ดีที่สุด: UniFi G4 Doorbell Pro / 2N IP Verso / Aiphone IX

**เสียง:** Full-duplex, ลำโพง 3W+, acoustic echo cancellation, การรวม SIP/VoIP  
**ฟีเจอร์:** รีเลย์ปลดล็อกประตู, คีย์แพด/RFID, ควบคุมลิฟต์, เสียงลิฟต์, หลายผู้เช่า  
**การปฏิบัติตาม:** 2N/Aiphone สร้างสำหรับรหัสพาณิชย์ (ADA, การรวมแผงอัคคีภัย)  
**ราคา:** G4 Doorbell Pro ~$300; 2N IP Verso ~$800+; Aiphone IX ~$1,200+  
**เหมาะที่สุด:** สำนักงาน, อาคารอพาร์ตเมนต์, ชุมชนที่มีรั้วรอบ, การรวมระบบควบคุมการเข้าถึง

### 6. Half-Duplex ราคาประหยัดที่ดีที่สุด: Wyze Cam v4 / Blink Outdoor 4 / TP-Link Tapo C120

**เสียง:** PTT, ลำโพง 0.5–1W, ไมค์เดียว, noise gate พื้นฐาน  
**คุณภาพ:** "ที่ดีที่สุดคือวอkeyทอล์กกี้" — ใช้ได้สำหรับ "พัสดุที่ประตู" ไม่ใช่สำหรับการสนทนา  
**คลาวด์:** จำเป็นสำหรับเสียง (Wyze/Blink); Tapo รองรับ microSD ท้องถิ่น  
**ราคา:** $30–60  
**เหมาะที่สุด:** กล้องรอง, รัฐฝ่ายเดียว, การสื่อสารความเสี่ยงต่ำ

## รายการตรวจสอบคุณภาพเสียง (ก่อนติดตั้ง)

1. **ดาวน์โหลดแอป, จับคู่กล้อง**
2. **ยืนห่าง 3 ม.** พูดปกติ → ให้คู่ฟังบนโทรศัพท์
3. **ยืนห่าง 0.9 ม.** พูด → ตรวจสอบ clipping/distortion
4. **เล่นเพลงใกล้กล้อง** (60 dB) → พูด → ตรวจสอบการระงับสัญญาณรบกวน
5. **จำลองลม** (พัดลมต่ำใกล้ไมค์) → พูด → ตรวจสอบการปฏิเสธลม
6. **ทดสอบ full-duplex:** ทั้งคู่พูดพร้อมกัน → ตรวจสอบเสียงก้อง/feedback
7. **ตรวจสอบความหน่วง:** นับ "1-2-3" → วัดหน่วงไปยังโทรศัพท์ (เป้าหมาย &lt;200 ms half, &lt;100 ms full)
8. **กล้องแบตเตอรี่:** พูด 60s → ตรวจสอบแบตเตอรี่ลด (ควร &lt;2% สำหรับ half-duplex)
