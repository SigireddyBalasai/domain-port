Двусторонний звук превращает камеру в интерком: общайтесь с курьерами, устрашайте нарушителей, проверяйте детей/питомцев или связываетесь с сотрудниками. Но не все "двусторонние звуки" одинаковы — полудуплекс (рация) vs фулл-дуплекс (телефон), качество шумоподавления и требования закона о согласии варьируются дико.

<Badge variant="outline">Ключевое различие</Badge> **Полудуплекс** = нажми-когда-говори, одно направление за раз (большинство бюджетных). **Фулл-дуплекс** = одновременный разговор/прослушивание как по телефону (премиум, выделенные интеркомы).

## Аудио архитектура: Что внутри

**MEMS (Micro-Electro-Mechanical Systems)** — Стандарт в современных камерах. Маленькие, прочные, плоский АЧХ.  
- **Омнидирекционные** — Ловят 360°; хорошо для покрытия комнаты, плохо для фокусного захвата  
- **Направленные (beamforming array)** — 2–4 MEMS + DSP наведение; фокусируются на голосе, отвергают фон  

**Размещение важно:**
- Дверной звонок: Нижняя грань, наклонена вниз → ловит посетителя на 3–5 футах
- Turret/Dome: Под объективом или сбоку → захватывает 10–20 футов радиуса
- Bullet: Задняя/боковая вентиляция → часто заглушена корпусом
- Внутренние: Верхняя передняя → лучшая для разговоров в комнате

**Ветра/погоды:** Ищите **акустическую сетку** (Gore-Tex / ePTFE) над микрофонным портом — блокирует воду, пропускает звук. Дешевые камеры пропускают это → дождь = статический шум.
    </AccordionContent>
  </AccordionItem>

**Типичные спеки:** 0.5–2W Class D amp + 28–40mm драйвер → 80–100 dB SPL @ 1 фут  
**Реальность:** 85 dB = комфортный разговор на 6 футах; 95 dB = слышно через 30 футов двора

**Источник питания имеет значение:**
- PoE (802.3af 15.4W) → достаточно для 2W аудио + камера
- 12V DC → нормально
- Батарейные камеры → **аудио убивает батарею** (amp тянет 200–500 mA). Ring/Eufy лимитят время разговора 30–60с на батарее.

**Искажения:** Дешевые Class D усилители клиппят на макс. громкости. Ищите **THD <1% @ 1W** или маркетинг "HD Audio" (обычно значит лучший amp + DSP).

  </AccordionItem>

| Режим | Как работает | Латентность | Лучше для | Камеры |
|------|--------------|---------|----------|---------|
| **Half-Duplex (PTT)** | Нажми кнопку → говори → отпусти → слушай | 200–500 ms | Дверной звонок, ворота, быстрое "оставь посылку" | Ring, Arlo, Blink, Wyze, Reolink (большинство), Eufy (большинство) |
| **Full-Duplex** | Одновременный TX/RX; эхо-отмена | 50–150 ms | Няня-монитор, уход за пожилыми, бизнес-интерком | Nest Doorbell (wired), Google Nest Cam, Arlo Ultra 2, UniFi G4 Doorbell Pro, Axis, 2N, Aiphone |

**Echo Cancellation (AEC) — дифференциатор:**  
- Полудуплексу не нужен AEC (мик выключен пока играет динамик)  
- Фулл-дуплекс **требует** адаптивный AEC (NLMS/RLS/Kalman) — без него слышишь свой голос с задержкой = непригодно

**Тест перед покупкой:** Позвони в камеру из приложения, говори пока оно играет. Если слышишь эхо/свист → AEC провалился.

## Шумоподавление: Что реально работает

<Callout type="note">
**DSP Цепочка в хороших камерах:** Beamforming → Stationary Noise Suppression (Wiener/Kalman) → Non-Stationary Suppression (DNN) → AEC (full-duplex) → AGC (auto gain) → Limiter
</Callout>

| Тип шума | Метод подавления | Типичное снижение |
|------------|-------------------|-------------------|
| Ветер | Физическая сетка + спектральный гейт | 15–25 dB |
| HVAC / Вентилятор / Трафик | Спектральное вычитание / Wiener | 10–20 dB |
| Лай собаки / Крик ребенка | DNN-based non-stationary (RNNoise, NSNet2) | 15–30 dB |
| Свой голос (эхо) | Адаптивный AEC (NLMS/Kalman) | 25–40 dB |
| Обратная связь динамика (свист) | Notch filtering + gain margin | Превенция |

**Маркетинг vs Реальность:** "AI Noise Cancellation" = обычно просто спектральный гейт. **Настоящее DNN шумоподавление** нужно NPU (Ambarella CV25, Rockchip RV1126, Google TPU, Apple Neural Engine). Ищите в спеках: "RNNoise", "NSNet2", "DNN-based", "Deep learning noise reduction."
    </AccordionContent>
  </AccordionItem>
</Accordion>

## Юридическое: Одностороннее vs Двухстороннее согласие

<Callout type="warning">
**Законы о аудиозаписи строже видео.** Видео = разумное ожидание приватности (вариабельно). Аудио = законы о прослушивании во многих штатах. **Требования согласия применяются к *записи*, а не к живому прослушиванию.**
</Callout>

### Законы США по согласии штатов (2026)

| Тип согласия | Штаты | Что означает |
|--------------|--------|---------------|
| **Одностороннее** (38 штатов + DC) | AL, AK, AZ, AR, CO, DE, DC, GA, HI, ID, IN, IA, KS, KY, LA, ME, MN, MS, MO, NE, NV, NJ, NM, NY, NC, ND, OH, OK, RI, SC, SD, TN, TX, UT, VT, VA, WV, WI, WY | Вы можете записывать, если *вы* участник разговора. Никакого уведомления не нужно. |
| **Двухстороннее / Все-участники** (12 штатов) | CA, CT, FL, IL, MD, MA, MI, MT, NH, OR, PA, WA | **Все участники должны согласиться.** Вывески + вербальное уведомление обязательны. |
| **Гибрид / Сложные** | VT (одностороннее но строгое), OR (все-участники для очного) | Проверьте местного юриста |

**Федеральный (Wiretap Act):** Одностороннее согласие базовая линия. Штаты могут быть строже.

- **Лучшая практика:** Полностью отключите аудиозапись в CA/FL/IL/MA/MD/PA/WA, если нет явного процесса согласия.

- **Добавление к договору:** "Арендатор может установить видео-дверной звонок; аудиозапись подлежит штатным законам о согласии; арендодатель не несет ответственности."

## Топ камер по сценариям использования (2026)

### 1. Лучший фулл-дуплекс дверной звонок: Google Nest Doorbell (Wired, 2nd Gen)
**Аудио:** Фулл-дуплекс, отличный AEC, 2W динамик, 3-микрофонный массив с beamforming  
**Шумоподавление:** TPU-ускоренное DNN (Google кремний) — лучший в классе подавление ветра/трафика  
**Функции согласия:** "Visitor Announcement" (зывень + "Кто-то у двери" перед записью); переключатель аудиозаписи в Home app  
**Интеграция:** Google Home, Assistant, Nest Aware ($8/мес за 30 дней видео + аудио)  
**Питание:** Только проводное (16–24V AC) — нет батареи  
**Цена:** ~$180  
**Лучше для:** Двухсторонних штатов (CA/FL/IL), фулл-дуплекс разговор, Google дома

### 2. Лучший полудуплекс дверной звонок: Ring Battery Doorbell Pro (2024) / Ring Stick Up Cam Pro
**Аудио:** Полудуплекс PTT, 1.5W динамик, двухмикрофонный beamforming  
**Шумоподавление:** "Audio+" DSP (спектральный гейт + базовый DNN) — неплохое подавление ветра  
**Функции согласия:** Переключатель "Audio Recording" на камере; "Motion Verification" снижает ложные аудио-триггеры  
**Интеграция:** Ring app, Alexa ("Alexa, talk to front door"), Ring Protect ($4–20/мес)  
**Питание:** Батарея (Quick Release) + Солнечный аксессуар + Hardwire опция  
**Цена:** ~$180–230  
**Лучше для:** Гибкость батареи, Alexa дома, односторонние штаты

### 3. Лучший локальный/без-облака: Reolink Video Doorbell PoE / Reolink Argus 4 Pro
**Аудио:** Полудуплекс, 1W динамик, одиночный MEMS микрофон (PoE) / двухмикрофонный (Argus 4 Pro)  
**Шумоподавление:** Базовый спектральный вычитание — приемлемо в тихих пригородах  
**Функции согласия:** **Без облака** — аудио записывается только на NVR/microSD; Вы контролируете хранение  
**Интеграция:** Reolink App, ONVIF, NVR, Home Assistant (через RTSP/ONVIF), нет HomeKit  
**Питание:** PoE (дверной звонок) / Батарея+Солнечная (Argus 4 Pro)  
**Цена:** ~$130 (PoE) / ~$220 (Argus 4 Pro + солнечная)  
**Лучше для:** Абсолютисты приватности, владельцы NVR, без-подписки

### 4. Лучший внутренний фулл-дуплекс: Google Nest Cam (Wired, 2nd Gen) / Nest Cam IQ (выпуск прекращен, но доступен)
**Аудио:** Фулл-дуплекс, 3-микрофонный массив, 2W динамик, Google TPU DNN шумоподавление  
**Сценарий:** Няня-монитор, проверка пожилых, разговор с питомцами — **телефонное качество**  
**Приватность:** Аппаратный LED индикатор при активном микрофоне; Аудиозапись выкл по умолчанию  
**Цена:** ~$100  
**Лучше для:** Внутренний двусторонний разговор, Google Home

### 5. Лучший бизнес-интерком: UniFi G4 Doorbell Pro / 2N IP Verso / Aiphone IX
**Аудио:** Фулл-дуплекс, 3W+ динамик, акустическая эхо-компенсация, SIP/VoIP интеграция  
**Функции:** Реле открывания двери, клавиатура/RFID, контроль лифта, аудио лифта, мульти-тенант  
**Соответствие:** 2N/Aiphone созданы под коммерческий кодекс (ADA, интеграция с пожаротушением)  
**Цена:** G4 Doorbell Pro ~$300; 2N IP Verso ~$800+; Aiphone IX ~$1,200+  
**Лучше для:** Офисы, многоквартирные дома, закрытые поселки, интеграция СКУД

### 6. Лучший бюджетный полудуплекс: Wyze Cam v4 / Blink Outdoor 4 / TP-Link Tapo C120
**Аудио:** PTT, 0.5–1W динамик, одиночный микрофон, базовый шумовой гейт  
**Качество:** "Рация в лучшем случае" — нормально для "посылка у двери", не для разговора  
**Облако:** Требуется для аудио (Wyze/Blink); Tapo поддерживает локальный microSD  
**Цена:** $30–60  
**Лучше для:** Вторичные камеры, односторонние штаты, низкорисковая коммуникация

## Чек-лист проверки аудио-качества (перед монтажом)

1. **Скачайте App, сопарите камеру**
2. **Встаньте 10 футов**, говорите нормально → партнер слушает на телефоне
3. **Встаньте 3 фута**, говорите → проверьте клиппинг/дисторшн
4. **Включите музыку у камеры** (60 dB) → говорите → проверьте шумоподавление
5. **Симулируйте ветер** (вентилятор на низком у микрофона) → говорите → проверьте ветроподавление
6. **Фулл-дуплекс тест:** Оба говорят одновременно → проверьте эхо/обратную связь
7. **Проверка задержки:** Считайте "1-2-3" → измерьте задержку до телефона (цель <200 ms полу, <100 ms фулл)
8. **Батарейная камера:** Говорите 60 сек → проверьте падение заряда (должно быть <2% для полудуплекса)
