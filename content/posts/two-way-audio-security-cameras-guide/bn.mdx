টু-ওয়ে অডিও একটি ক্যামেরাকে ইন্টারকমে পরিণত করে: ডেলিভারি ড্রাইভারের সাথে কথা বলুন, অনুপ্রবেশকারীদের ভয় দেখান, বাচ্চা/পোষা প্রাণী দেখুন বা কর্মচারীদের সাথে যোগাযোগ করুন। কিন্তু সব "টু-ওয়ে অডিও" সমান নয় — হাফ-ডুপ্লেক্স (ওয়াকি-টকি) বনাম ফুল-ডুপ্লেক্স (ফোন কল), নয়েজ ক্যানসেলেশন কোয়ালিটি এবং আইনি সম্মতির প্রয়োজনীয়তা ব্যাপকভাবে পরিবর্তিত হয়।

<Badge variant="outline">মূল পার্থক্য</Badge> **হাফ-ডুপ্লেক্স** = পুশ-টু-টক,
একবারে এক দিক (বেশিরভাগ বাজেট ক্যাম)। **ফুল-ডুপ্লেক্স** = ফোনের মতো একসাথে কথা
বলা/শোনা (প্রিমিয়াম ক্যাম, ডেডিকেটেড ইন্টারকম)।

## নয়েজ ক্যানসেলেশন: কী আসলে কাজ করে

<Callout type="note">

**ভালো ক্যামেরায় DSP চেইন:** বিমফর্মিং → স্টেশনারি নয়েজ সাপ্রেশন
(Wiener/Kalman) → নন-স্টেশনারি সাপ্রেশন (DNN) → AEC (ফুল-ডুপ্লেক্স) → AGC (অটো
গেইন) → লিমিটার

</Callout>

| নয়েজের ধরন                  | সাপ্রেশন পদ্ধতি                            | সাধারণ হ্রাস |
| ---------------------------- | ------------------------------------------ | ------------ |
| বাতাস                        | ফিজিক্যাল মেশ + স্পেকট্রাল গেটিং           | 15-25 dB     |
| HVAC / ফ্যান / ট্রাফিক       | স্পেকট্রাল সাবট্র্যাকশন / Wiener           | 10-20 dB     |
| কুকুরের ঘেউ / বাচ্চার চিৎকার | DNN-ভিত্তিক নন-স্টেশনারি (RNNoise, NSNet2) | 15-30 dB     |
| আপনার নিজের কণ্ঠ (ইকো)       | অ্যাডাপ্টিভ AEC (NLMS/Kalman)              | 25-40 dB     |
| স্পিকার ফিডব্যাক (হাউল)      | নচ ফিল্টারিং + গেইন মার্জিন                | প্রতিরোধ     |

**মার্কেটিং বনাম বাস্তবতা:** "AI নয়েজ ক্যানসেলেশন" = সাধারণত শুধু স্পেকট্রাল গেট। **প্রকৃত DNN নয়েজ সাপ্রেশনের** প্রয়োজন NPU (Ambarella CV25, Rockchip RV1126, Google TPU, Apple Neural Engine)। স্পেক চেক করুন: "RNNoise", "NSNet2", "DNN-based", "ডিপ লার্নিং নয়েজ রিডাকশন।"

## আইনি: এক-পক্ষ বনাম দুই-পক্ষ সম্মতি

<Callout type="warning">

**অডিও রেকর্ডিং আইন ভিডিওর চেয়ে কঠোর।** ভিডিও = গোপনীয়তার যুক্তিসঙ্গত
প্রত্যাশা (ভিন্ন)। অডিও = অনেক রাজ্যে ওয়্যারট্যাপ আইন। **সম্মতির
প্রয়োজনীয়তা _রেকর্ডিং_-এর জন্য প্রযোজ্য, লাইভ শোনার জন্য নয়।**

</Callout>

**সম্মতি বৈশিষ্ট্য:** "ভিজিটর অ্যানাউন্সমেন্ট" (রেকর্ডিংয়ের আগে চাইম + "দরজায় কেউ আছে"); Home অ্যাপে অডিও রেকর্ডিং টগল  
**ইন্টিগ্রেশন:** Google Home, Assistant, Nest Aware ($8/মাস 30 দিনের ভিডিও + অডিও)  
**পাওয়ার:** শুধু ওয়্যার্ড (16-24V AC) — কোনো ব্যাটারি নেই  
**মূল্য:** ~$180  
**সেরা জন্য:** দুই-পক্ষ রাজ্য (CA/FL/IL), ফুল-ডুপ্লেক্স কথোপকথন, Google বাড়ি

### 2. সেরা হাফ-ডুপ্লেক্স ডোরবেল: Ring ব্যাটারি ডোরবেল Pro (2024) / Ring Stick Up Cam Pro

**অডিও:** হাফ-ডুপ্লেক্স PTT, 1.5W স্পিকার, ডুয়াল-মাইক বিমফর্মিং  
**নয়েজ ক্যানসেল:** "Audio+" DSP (স্পেকট্রাল গেট + বেসিক DNN) — ভালো বাতাস রিজেকশন  
**সম্মতি বৈশিষ্ট্য:** প্রতি ক্যামে "অডিও রেকর্ডিং" টগল; "মোশন ভেরিফিকেশন" মিথ্যা অডিও ট্রিগার কমায়  
**ইন্টিগ্রেশন:** Ring অ্যাপ, Alexa ("Alexa, সামনের দরজার সাথে কথা বল"), Ring Protect ($4-20/মাস)  
**পাওয়ার:** ব্যাটারি (কুইক রিলিজ) + সোলার আনুষঙ্গিক + হার্ডওয়্যার অপশন  
**মূল্য:** ~$180-230  
**সেরা জন্য:** ব্যাটারি নমনীয়তা, Alexa বাড়ি, এক-পক্ষ রাজ্য

### 3. সেরা লোকাল/কোনো-ক্লাউড: Reolink ভিডিও ডোরবেল PoE / Reolink Argus 4 Pro

**সম্মতি বৈশিষ্ট্য:** **কোনো ক্লাউড নেই** — অডিও শুধু NVR/microSD-তে রেকর্ড করা হয়; আপনি ধারণ নিয়ন্ত্রণ করেন

## মাউন্টিংয়ের আগে অডিও কোয়ালিটি পরীক্ষা চেকলিস্ট

1. **অ্যাপ ডাউনলোড করুন, ক্যামেরা পেয়ার করুন**
2. **10 ফুট দূরে দাঁড়ান**, স্বাভাবিকভাবে কথা বলুন → সঙ্গীকে ফোনে শুনতে বলুন
3. **3 ফুট দূরে দাঁড়ান**, কথা বলুন → ক্লিপিং/ডিসটর্শন চেক করুন
4. **ক্যামেরার কাছে মিউজিক বাজান** (60 dB) → কথা বলুন → নয়েজ সাপ্রেশন চেক করুন
5. **বাতাস সিমুলেট করুন** (মাইকের কাছে কম গতির ফ্যান) → কথা বলুন → বাতাস রিজেকশন চেক করুন
6. **ফুল-ডুপ্লেক্স পরীক্ষা:** উভয়ই একসাথে কথা বলুন → ইকো/ফিডব্যাক চেক করুন
7. **লেটেন্সি পরীক্ষা:** "1-2-3" গুনুন → ফোনে বিলম্ব মাপুন (লক্ষ্য &lt;200 ms হাফ, &lt;100 ms ফুল)
8. **ব্যাটারি ক্যাম:** 60 সেকেন্ড কথা বলুন → ব্যাটারি ড্রপ চেক করুন (হাফ-ডুপ্লেক্সের জন্য &lt;2% হওয়া উচিত)
