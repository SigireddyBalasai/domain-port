## Güvenlik Kamerası Sorun Giderme

Bu rehberde güvenlik kameralarıyla karşılaştığınız yaygın sorunları ve bunları nasıl düzelteceğinizi bulacaksınız.

## 1. Kamera Açılmıyor

### Olası Nedenler:

- Güç kaynağı arızalı
- Kablo hasarı
- PoE anahtarı sorunu
- Sigorta atmış

### Çözümler:

1. Güç kablosunu kontrol edin
2. Farklı bir priz deneyin
3. PoE kablosunu yeniden takın
4. Güç kaynağını değiştirin

## 2. Gece Görüşü Çalışmıyor

### Olası Nedenler:

- IR LED'leri arızalı
- Gece görüş ayarı kapalı
- Sensör engellenmiş
- Lens kirlenmiş

### Çözümler:

1. Kamerayı yeniden başlatın
2. Lensleri temizleyin
3. Gece görüş ayarını kontrol edin
4. IR LED'lerini test edin

## 3. WiFi Bağlantı Sorunları

### Olası Nedenler:

- Sinyal zayıf
- Parazit
- Yanlış şifre
- Kanal çakışması

### Çözümler:

1. WiFi sinyal gücünü kontrol edin (-65dBm veya daha iyi)
2. Kamerayı yönlendiriciye yaklaştırın
3. WiFi kanalını değiştirin
4. Kamerayı yeniden başlatın

## 4. Hareket Algılama Çalışmıyor

### Olası Nedenler:

- Hassasiyet çok düşük
- Yanlış bölge yapılandırması
- Yazılım hatası
- Sensör arızalı

### Çözümler:

1. Hassasiyeti artırın
2. Algılama bölgelerini yeniden ayarlayın
3. Kamerayı yeniden başlatın
4. Ürün yazılımını güncelleyin

## 5. Görüntü Kalitesi Sorunları

### Yaygın Sorunlar:

- Bulanık görüntü
- Titreşim
- Renk bozulması
- Düşük çözünürlük

### Çözümler:

1. Lensi temizleyin
2. Odağı ayarlayın (motorlu lenslerde)
3. Kamerayı sağlam monte edin
4. Bit hızını artırın

## 6. Ses Çalışmıyor

### Olası Nedenler:

- Mikrofon kapalı
- Hoparlör arızalı
- Ses seviyesi çok düşük
- Yazılım ayarı

### Çözümler:

1. Ses ayarlarını kontrol edin
2. Ses seviyesini artırın
3. Kamerayı yeniden başlatın
4. Uygulamayı yeniden yükleyin

## 7. Kayıt Sorunları

### Yaygın Sorunlar:

- SD kart dolu
- HDD arızalı
- Kayıt programı yanlış
- Kayıt yok

### Çözümler:

1. SD kartı biçimlendirin
2. HDD sağlığını kontrol edin
3. Kayıt programını ayarlayın
4. Depolama ayarlarını kontrol edin

## 8. Uzaktan Erişim Sorunları

### Olası Nedenler:

- Port yönlendirme yanlış
- Dinamik IP değişmiş
- Güvenlik duvarı engelliyor
- Uygulama hatası

### Çözümler:

1. Port yönlendirmeyi kontrol edin
2. DDNS kullanın
3. Güvenlik duvarı ayarlarını kontrol edin
4. Uygulamayı güncelleyin

## 9. Yanlış Alarmlar

### Olası Nedenler:

- Hassasiyet çok yüksek
- Yanlış bölge
- Böcekler / örümcek ağları
- Hava koşulları

### Çözümler:

1. Hassasiyeti azaltın
2. Algılama bölgelerini daraltın
3. Lensleri temizleyin
4. AI algılama kullanın (insan/araç)

## 10. Kamera Donuyor / Kilitleniyor

### Olası Nedenler:

- Yazılım hatası
- Aşırı ısınma
- Bellek sızıntısı
- Güç dalgalanması

### Çözümler:

1. Kamerayı güç döngüsüne sokun (fişi çekin, 30sn bekleyin, takın)
2. Ürün yazılımını güncelleyin
3. Havalandırmayı kontrol edin
4. Farklı güç kaynağı deneyin

## Genel Önleyici Bakım İpuçları

- **Lensleri düzenli temizleyin**: Ayda bir mikrofiber bezle
- **Ürün yazılımını güncel tutun**: Güvenlik düzeltmeleri için
- **Kabloları kontrol edin**: Aşınma, kemirgen hasarı
- **Pilleri değiştirin**: Pil ömrü dolduğunda
- **HDD sağlığını izleyin**: S.M.A.R.T. durumunu kontrol edin
- **Yedekleme yapın**: Kritik görüntüleri düzenli yedekleyin
- **Hava koşullarına karşı koruyun**: Kabloları sudan koruyun

## Donanım Yazılımı Güncelleme

### Güvenli Güncelleme Prosedürü

1. **Değişiklik günlüğünü okuyun** — Kararlıysanız ve sadece "güvenlik" ekliyorsa atlayın
2. **Yapılandırmayı yedekleyin** — NVR: Config dışa aktar; Kamera: Ayarları ekran görüntüsü al
3. **Önce NVR'yi güncelleyin** — Sonra kameralar (NVR uyumlu donanım yazılımı itebilir)
4. **Her seferinde bir kamera** — Her birini sonrakinden önce doğrulayın
5. **Gücü kapatmayın** — Kamera başına 5-10 dk
6. **Doğrulayın** — Canlı görüntü, kayıt, hareket, PTZ, ses

### Arızalı Donanım Yazılımını Geri Alma

| Marka               | Yöntem                                                           |
| ------------------- | ---------------------------------------------------------------- |
| **Reolink**         | Donanım yazılımı sayfası → "Önceki Sürüm" düğmesi                |
| **Dahua/Hikvision** | TFTP kurtarma (açılış + sıfırla basılı tut + TFTP sunucu)        |
| **UniFi**           | Protect → Cihazlar → Kamera → "Donanım Yazılımını Geri Al"       |
| **Genel**           | Kökte `firmware.bin` içeren SD kart; sıfırla basılı tutarak açın |

## PTZ ve Mekanik

### PTZ Hareket Etmiyor / Sıkıştı / Gıcırdıyor

| Neden                   | Düzeltme                                                 |
| ----------------------- | -------------------------------------------------------- |
| Mekanik sınıra ulaşıldı | Ön ayar sınırlarını kontrol edin; ev konumuna sıfırlayın |
| Bilezik halka arızası   | Kablo demeti aşınmış → kamerayı değiştirin               |
| Buz / Kalıntı           | Isıtıcı arızalı; manuel temizleyin                       |
| Protokol uyuşmazlığı    | Pelco-D/P vs ONVIF PTZ → protokolü eşleştirin            |

## Çevresel ve Fiziksel

### Su Sızması / Yoğunlaşma

| Belirti            | Düzeltme                                         |
| ------------------ | ------------------------------------------------ |
| Kubbe içinde buğu  | Kurutucu paket; havalandırma tapasını doğrulayın |
| Kablo girişinde su | Silikon + lastik burç; damlama halkası           |
| RJ45'te korozyon   | Dielektrik gres; konektörü değiştirin            |

### Vandalizm / Hırsızlık

- **Torx / Güvenlik vidaları** (çoğu profesyonel kamerayla birlikte gelir)
- **Kamera kanalı** (kablo kolay kesilemez)
- **Yükseklik** >10 ft (merdiven gerekli)
- **Kamerayı izleyen yedek kamera** (örtüşme)
- **Kurcalama Uyarısı** (NVR: Kamera → Kurcalama Algılama → Alarm)
