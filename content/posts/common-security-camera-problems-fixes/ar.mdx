كل نظام كاميرات يصاب بمشاكل. يغطي هذا الدليل 25 مشكلة شائعة مع تشخيص خطوة بخطوة وإصلاحات — منظمة حسب الأعراض لتجد الحل بسرعة.

<Badge variant="outline">طريقة التشخيص</Badge> **ابدأ بالبسيط:** الطاقة ← الكابل
← الشبكة ← الإعدادات ← الأجهزة. 80% من المشاكل هي طاقة أو كابل.

---

## مخطط التشخيص السريع

```
الكاميرا لا تعمل؟
    │
    ├─ لا يوجد ضوء طاقة؟ → مصدر الطاقة / PoE / بطارية (القسم 1)
    ├─ الطاقة جيدة، لا صورة؟ → كابل / اتصال / تعارض IP (القسم 2)
    ├─ الصورة جيدة، لا تسجيل؟ → NVR / تخزين / جدول (القسم 3)
    ├─ الصورة جيدة، إنذارات خاطئة؟ → مناطق حركة / حساسية / ذكاء اصطناعي (القسم 4)
    ├─ الرؤية الليلية سيئة؟ → IR / انعكاس / مصراع (القسم 5)
    └─ الوصول عن بُعد يفشل؟ → إعادة توجيه المنفذ / VPN / سحابة (القسم 6)
```

## 1. مشاكل الطاقة والتشغيل

### 1.1 الكاميرا ميتة — لا مؤشر LED، لا استجابة

| خطوة | افحص                   | الإصلاح                                                                      |
| ---- | ---------------------- | ---------------------------------------------------------------------------- |
| 1    | **مؤشر LED لمبدل PoE** | مطفأ = المنفذ معطل / تجاوز الميزانية → انقل إلى منفذ جيد معروف               |
| 2    | **مؤشر LED محقن PoE**  | مطفأ = المحقن تالف / لا تيار متردد → استبدل المحقن ($25)                     |
| 3    | **الجهد عند الكاميرا** | أفوميتر: 44–57V (PoE)، 12V±10% (تيار مستمر) → منخفض = انخفاض جهد / كابل تالف |
| 4    | **استمرارية الكابل**   | اختبار: كل 8 أسلاك → زوج مفتوح = أعد إنهاء الطرفين                           |
| 5    | **إعادة ضبط الكاميرا** | اضغط مع الاستمرار 10–30ث ← أعد الاعتماد في التطبيق/NVR                       |

**الكاميرات التي تعمل بالبطارية:** اشحن 24 ساعة عبر USB-C قبل اعتبارها ميتة. نظام إدارة البطاريات Li-ion يقفل بعد التفريغ العميق.

### 1.2 الكاميرا تعيد التشغيل ليلاً (عند تشغيل IR)

**السبب:** انخفاض الجهد عند تشغيل مصابيح IR (يضيف 1–2 واط).

| التشخيص                          | الإصلاح                                   |
| -------------------------------- | ----------------------------------------- |
| مبدل PoE عند حد الميزانية        | انقل الكاميرا إلى محقن؛ طور المبدل        |
| طول الكابل >250 قدماً على Cat6   | قصر المدى؛ Cat6a؛ محقن PoE++ عند الكاميرا |
| كابل CCA (نحاس مطلي بالألومنيوم) | **استبدل بكابل نحاسي صلب Cat6**           |

## 2. الشبكة والاتصال

### 2.1 الكاميرا "غير متصلة" في التطبيق/NVR

| خطوة | الاختبار                                    | الإصلاح                                         |
| ---- | ------------------------------------------- | ----------------------------------------------- |
| 1    | اختبار Ping لعنوان IP الكاميرا من الكمبيوتر | انتهاء المهلة = مشكلة طبقة 1/2 (كابل، مبدل، IP) |
| 2    | اختبار Ping للبوابة من الكاميرا             | انتهاء المهلة = شبكة محلية / جدار حماية / توجيه |
| 3    | تطابق عنوان MAC مع الكاميرا؟                | تعارض IP → IP ثابت / حجز DHCP                   |
| 4    | انتهاء عقد DHCP                             | IP ثابت أو حجز DHCP (الراوتر)                   |

### 2.2 كاميرا Wi-Fi تنقطع يومياً

| التشخيص             | الإصلاح                                               |
| ------------------- | ----------------------------------------------------- |
| RSSI < -75 dBm      | قرّب نقطة الوصول؛ أضف نقطة وصول سلكية؛ تحقق من القناة |
| ازدحام 2.4 جيجاهرتز | القناة 1/6/11 فقط؛ عرض 20 ميجاهرتز؛ عطل توجيه النطاق  |
| وضع توفير الطاقة    | عطل توفير طاقة Wi-Fi في تطبيق الكاميرا                |

## 3. التسجيل والتخزين

### 3.1 NVR لا يسجل / فجوات في الجدول الزمني

| افحص                            | الإصلاح                                             |
| ------------------------------- | --------------------------------------------------- |
| صحة القرص الصلب (S.M.A.R.T.)    | قطاعات معاد تخصيصها >0 → **استبدل فوراً**           |
| القرص ممتلئ / الكتابة فوق معطلة | فعّل "الكتابة فوق الأقدم" في إعدادات التخزين        |
| الجدول مضبوط بشكل خاطئ          | تحقق: مستمر 24/7 على الدفق الرئيسي؛ حركة على الفرعي |

## 4. كشف الحركة والتنبيهات

### 4.1 الكثير من الإنذارات الخاطئة (البريد العشوائي)

| الإعداد              | الموصى به                  | السبب                                    |
| -------------------- | -------------------------- | ---------------------------------------- |
| **نوع الكشف**        | شخص + مركبة فقط            | يستثني الحيوانات، الأشجار، الظلال، المطر |
| **مناطق الحركة**     | ارسم المنطقة المستهدفة فقط | يستثني الشارع، الأشجار، الجيران، السماء  |
| **الحساسية**         | متوسط (بداية) ← اضبط       | عالي = كل شيء يحفز                       |
| **الذكاء الاصطناعي** | فعّل (شخص/مركبة/طرد)       | تقليل الإنذارات الخاطئة بنسبة +90%       |

## 5. جودة الصورة والرؤية الليلية

### 5.1 الرؤية الليلية بيضاء / مغسولة

**السبب:** مصابيح IR تنعكس عن سطح قريب (جدار، طنف، زجاج، نسيج عنكبوت).

| الإصلاح                  | كيف                                             |
| ------------------------ | ----------------------------------------------- |
| **أمل للأسفل 15–30°**    | قوس إمالة؛ الكاميرا تنظر للأسفل، ليس إلى الحائط |
| **حرّك الكاميرا للخارج** | 12–18" من الحائط/الطنف (ذراع تمديد)             |
| **عطل IR المدمج**        | أضف **منير IR خارجي** (منفصل، موجه للهدف)       |

---

## 6. الوصول عن بُعد وتطبيق الجوال

### 6.1 التطبيق يعمل على Wi-Fi، يفشل على LTE/5G

| البنية                                   | الإصلاح                                                                  |
| ---------------------------------------- | ------------------------------------------------------------------------ |
| **السحابة** (Reolink, UniFi, Eufy, Ring) | تحقق من الإنترنت على الهاتف؛ صفحة حالة البائع                            |
| **إعادة توجيه المنفذ**                   | **لا تستخدمه.** CGNAT، NAT مزدوج، مزود الخدمة يمنع. انتقل إلى VPN/سحابة. |
| **VPN (Tailscale, WireGuard)**           | تأكد من أن VPN نشط؛ النفق المنفصل قد يستثني التطبيق                      |

### 6.2 البث المباشر يعمل، التسجيل يفشل عن بُعد

- التشغيل عن بُعد يستخدم منفذاً مختلفاً (غالباً HTTPS/443 بدلاً من RTSP/554)
- **تحقق:** NVR → الشبكة → منفذ HTTPS (الافتراضي 443) يمكن الوصول إليه عبر VPN
- **النطاق الترددي:** التشغيل يحتاج 4–8 Mbps مستدامة؛ LTE قد يخفض السرعة

## 7. PTZ والميكانيكا

### 7.1 PTZ لا يتحرك / عالق / صرير

| السبب                          | الإصلاح                                               |
| ------------------------------ | ----------------------------------------------------- |
| الوصول إلى الحد الميكانيكي     | تحقق من حدود preset؛ إعادة تعيين إلى الوضع المنزلي    |
| فشل حلقة الانزلاق (دوران 360°) | حزام الكابل تالف → استبدل الكاميرا (غير قابل للإصلاح) |
| جليد / حطام                    | سخان معطل؛ نظف يدوياً؛ تحقق من دائرة السخان           |

---

## 8. الصوت (ثنائي الاتجاه)

### 8.1 لا يوجد صوت / باتجاه واحد فقط

| افحص              | الإصلاح                                               |
| ----------------- | ----------------------------------------------------- |
| إذن التطبيق       | إعدادات الهاتف → التطبيق → الميكروفون: تشغيل          |
| صوت الكاميرا مفعل | واجهة الويب للكاميرا → الصوت → الإدخال/الإخراج: تشغيل |
| عدم تطابق الترميز | G.711 (PCM) الأكثر توافقاً؛ جرب G.726 / AAC           |

## 9. البرامج الثابتة والتحديثات

### 9.1 إجراء تحديث البرنامج الثابت الآمن

1. **اقرأ سجل التغييرات** — تخطى إذا كان "أمن فقط" وكنت مستقراً
2. **انسخ الإعدادات احتياطياً** — NVR: تصدير الإعدادات؛ الكاميرا: لقطة شاشة للإعدادات
3. **حدث NVR أولاً** — ثم الكاميرات (NVR قد يدفع برنامجاً ثابتاً متوافقاً)
4. **كاميرا واحدة في كل مرة** — تحقق من كل واحدة قبل التالية
5. **لا تفصل الطاقة** — 5–10 دقائق لكل كاميرا
6. **تحقق** — البث المباشر، التسجيل، الحركة، PTZ، الصوت

## 10. البيئة والمادية

### 10.1 تسرب المياه / التكثف

| العرض               | الإصلاح                                                    |
| ------------------- | ---------------------------------------------------------- |
| ضباب داخل القبة     | كيس مجفف (استبدل سنوياً)؛ تحقق من سدادة التهوية (Gore-Tex) |
| ماء عند مدخل الكابل | سيليكون + طوق حماية؛ حلقة تقطير؛ ماسورة                    |

### 10.2 الحشرات / العناكب / الطيور

| الطارد                                   | الفعالية              |
| ---------------------------------------- | --------------------- |
| زيت النعناع (كرة قطن بالقرب من الكاميرا) | عالية (العناكب تكرهه) |
| مسامير الطيور على الحامل                 | عالية (الطيور)        |

### 10.3 التخريب / السرقة

- **مسامير Torx / أمان** (مرفقة مع معظم الكاميرات الاحترافية)
- **ماسورة** إلى الكاميرا (يصعب قطع الكابل)
- **الارتفاع** >10 أقدام (يتطلب سلماً)
- **تنبيه العبث** (NVR: الكاميرا → كشف العبث → إنذار)
