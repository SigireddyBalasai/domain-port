## Solução de Problemas de Câmeras de Segurança

Encontre aqui os problemas comuns de câmeras de segurança e como corrigi-los.

## 1. Câmera Não Liga

### Causas Possíveis:

- Fonte de alimentação com defeito
- Dano no cabo
- Problema no switch PoE
- Disjuntor desarmado

### Soluções:

1. Verifique o cabo de alimentação
2. Teste outra tomada
3. Reconnecte o cabo PoE
4. Substitua a fonte de alimentação

## 2. Visão Noturna Não Funciona

### Causas Possíveis:

- LEDs IR com defeito
- Configuração desativada
- Sensor obstruído
- Lente suja

### Soluções:

1. Reinicie a câmera
2. Limpe as lentes
3. Verifique a configuração de visão noturna
4. Teste os LEDs IR

## 3. Problemas de Conexão WiFi

### Causas Possíveis:

- Sinal fraco
- Interferência
- Senha incorreta
- Conflito de canal

### Soluções:

1. Verifique a intensidade do sinal (-65dBm ou melhor)
2. Aproxime a câmera do roteador
3. Mude o canal WiFi
4. Reinicie a câmera

## 4. Detecção de Movimento Não Funciona

### Causas Possíveis:

- Sensibilidade muito baixa
- Zona configurada incorretamente
- Bug de software
- Sensor com defeito

### Soluções:

1. Aumente a sensibilidade
2. Reajuste as zonas de detecção
3. Reinicie a câmera
4. Atualize o firmware

## 5. Problemas de Qualidade de Imagem

### Problemas Comuns:

- Imagem borrada
- Tremulação
- Distorção de cor
- Baixa resolução

### Soluções:

1. Limpe a lente
2. Ajuste o foco (lentes motorizadas)
3. Monte a câmera firmemente
4. Aumente o bitrate

## 6. Áudio Não Funciona

### Causas Possíveis:

- Microfone desligado
- Alto-falante com defeito
- Volume muito baixo
- Configuração de software

### Soluções:

1. Verifique as configurações de áudio
2. Aumente o volume
3. Reinicie a câmera
4. Reinstale o aplicativo

## 7. Problemas de Gravação

### Problemas Comuns:

- Cartão SD cheio
- HDD com defeito
- Agenda de gravação errada
- Sem gravação

### Soluções:

1. Formate o cartão SD
2. Verifique a saúde do HDD
3. Ajuste a agenda de gravação
4. Verifique as configurações de armazenamento

## 8. Problemas de Acesso Remoto

### Causas Possíveis:

- Encaminhamento de porta incorreto
- IP dinâmico alterado
- Firewall bloqueando
- Erro no aplicativo

### Soluções:

1. Verifique o encaminhamento de porta
2. Use DDNS
3. Verifique as configurações do firewall
4. Atualize o aplicativo

## 9. Alarmes Falsos

### Causas Possíveis:

- Sensibilidade muito alta
- Zona errada
- Insetos / teias de aranha
- Condições climáticas

### Soluções:

1. Reduza a sensibilidade
2. Ajuste as zonas de detecção
3. Limpe as lentes
4. Use detecção por IA (pessoa/veículo)

## 10. Câmera Congela / Trava

### Causas Possíveis:

- Bug de software
- Superaquecimento
- Vazamento de memória
- Flutuação de energia

### Soluções:

1. Ciclo de energia (desconecte, aguarde 30s, reconecte)
2. Atualize o firmware
3. Verifique a ventilação
4. Teste outra fonte de alimentação

## Dicas de Manutenção Preventiva

- **Limpe as lentes regularmente**: Mensalmente com pano de microfibra
- **Mantenha o firmware atualizado**: Para correções de segurança
- **Verifique os cabos**: Desgaste, danos por roedores
- **Substitua as baterias**: Quando a vida útil acabar
- **Monitore a saúde do HDD**: Verifique o status S.M.A.R.T.
- **Faça backup**: Grave eventos críticos regularmente
- **Proteja contra intempéries**: Proteja cabos da água

## Problemas de Qualidade de Imagem & Visão Noturna

### Visão Noturna Estourada / Lavada

**Causa:** LEDs IR refletindo em superfície próxima (parede, beiral, vidro, teia de aranha).

| Correção                      | Como                                                                |
| ----------------------------- | ------------------------------------------------------------------- |
| **Incline para baixo 15-30°** | Suporte angular; câmera olha para baixo, não para a parede          |
| **Afaste a câmera**           | 12-18" da parede/beiral (braço de extensão)                         |
| **Desative o IR embutido**    | Adicione **iluminador IR externo** (separado, apontado para o alvo) |
| **Limpe a lente/dome**        | Teias de aranha = dispersão IR → microfibra + limpador de lentes    |
| **IR Inteligente**            | Ajusta automaticamente a intensidade IR pela distância              |

### Imagem Diurna: Neblina / Baixo Contraste / Alteração de Cor

| Causa                     | Correção                                               |
| ------------------------- | ------------------------------------------------------ |
| Dome sujo / arranhado     | Limpe com microfibra; substitua o dome ($15)           |
| Filtro IR Preso           | Ciclo de energia; atualização de firmware; toque suave |
| Balanço de Branco Errado  | Defina "Externo" / "Automático" / Kelvin personalizado |
| Contraluz (sol no quadro) | Reposicione; ative WDR; adicione protetor solar        |

### Oscilação / Linhas Rolando (50/60 Hz)

| Causa                               | Correção                                                               |
| ----------------------------------- | ---------------------------------------------------------------------- |
| **Incompatibilidade de Frequência** | Câmera definida para 60 Hz (EUA) / 50 Hz (BR) — corresponda à sua rede |
| Loop de Terra                       | Todos equipamentos mesma terra; isolador PoE                           |
| Oscilação Fluorescente/LED          | Defina obturador para 1/100 (50 Hz) ou 1/120 (60 Hz)                   |

## 7. PTZ & Mecânico

### 7.1 PTZ Não Move / Travado / Rangendo

| Causa                          | Correção                                                |
| ------------------------------ | ------------------------------------------------------- |
| Limite mecânico atingido       | Verifique limites predefinidos; redefina para home      |
| Falha no anel coletor (360°)   | Chicote desgastado → substitua câmera (não reparável)   |
| Gelo / Detritos                | Aquecedor falhou; limpe manualmente; verifique circuito |
| Incompatibilidade de Protocolo | Pelco-D/P vs ONVIF PTZ → Corresponda protocolo NVR      |

### 7.2 Tour Predefinido Não Funciona

- Agendamento ativado? (NVR → PTZ → Tour Schedule)
- Tempo de permanência >0? (10–30s por predefinição)
- Home predefinido definido? (Tour retorna ao home entre loops)

## 8. Áudio (Dois Sentidos)

### 8.1 Sem Áudio / Apenas Unidirecional

| Verificação           | Correção                                             |
| --------------------- | ---------------------------------------------------- |
| Permissão do app      | Config. Telefone → App → Microfone: LIGADO           |
| Áudio da câmera ativo | Interface web → Áudio → Entrada/Saída: LIGADO        |
| Codec incompatível    | G.711 (PCM) mais compatível; tente G.726 / AAC       |
| Half-duplex vs Full   | Maioria câmeras = PTT (half); não fale enquanto ouve |

### 8.2 Eco / Feedback / Estático

| Causa                                    | Correção                                                          |
| ---------------------------------------- | ----------------------------------------------------------------- |
| Alto-falante muito alto / ganho mic alto | Abaixe volume do alto-falante no app; reduza ganho do microfone   |
| Sem AEC (Cancelamento Eco Acústico)      | Câmeras full-duplex precisam AEC; half-duplex = mute mic ao falar |
| Vento no microfone                       | Ative filtro de vento; espuma antivent; reposicione               |

## 9. Firmware & Atualizações

### 9.1 Procedimento Seguro de Atualização de Firmware

1. **Leia o changelog** — Pule se "apenas segurança" e estiver estável
2. **Backup da configuração** — NVR: Exportar config; Câmera: Print das configurações
3. **Atualize o NVR primeiro** — Depois câmeras (NVR pode enviar firmware compatível)
4. **Uma câmera por vez** — Verifique cada uma antes da próxima
5. **Não desligue** — 5–10 min por câmera
6. **Verifique** — Visualização ao vivo, gravação, movimento, PTZ, áudio

### 9.2 Reverter Firmware com Falha

| Marca               | Método                                                            |
| ------------------- | ----------------------------------------------------------------- |
| **Reolink**         | Página Firmware → Botão "Versão Anterior"                         |
| **Dahua/Hikvision** | Recuperação TFTP (boot + reset pressionado + servidor TFTP)       |
| **UniFi**           | Protect → Dispositivos → Câmera → "Reverter Firmware"             |
| **Genérico**        | Cartão SD com `firmware.bin` na raiz; ligue com reset pressionado |

## 10. Ambiental & Físico

### 10.1 Infiltração de Água / Condensação

| Sintoma                 | Correção                                                              |
| ----------------------- | --------------------------------------------------------------------- |
| Névoa dentro do dome    | Pacote dessecante (troque anualmente); verifique plugue de ventilação |
| Água na entrada do cabo | Silicone + grommet; drip loop; corpo de conduíte                      |
| Corrosão no RJ45        | Graxa dielétrica nos contatos; substitua conector; cabo blindado      |

### 10.2 Insetos / Aranhas / Pássaros

| Dissuasão                                 | Eficácia                              |
| ----------------------------------------- | ------------------------------------- |
| Óleo de hortelã-pimenta (perto da câmera) | Alta (aranhas odeiam)                 |
| Repelente ultrassônico                    | Média                                 |
| Espinhos anti-pássaros no suporte         | Alta (pássaros)                       |
| Cobertura de lã sobre anel IR             | Alta (aranhas constroem no IR quente) |
| Limpeza regular (mensal)                  | Essencial                             |

### 10.3 Vandalismo / Roubo

- **Parafusos Torx / Segurança** (incluídos na maioria das câmeras profissionais)
- **Conduíte** até a câmera (não corta cabo facilmente)
- **Altura** >3 m (requer escada)
- **Câmera de backup** vigiando a câmera (sobreposição)
- **Alerta de Violação** (NVR: Câmera → Detecção de Violação → Alarme)
