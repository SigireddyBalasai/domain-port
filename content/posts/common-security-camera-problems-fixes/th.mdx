ทุกระบบกล้องมีปัญหา คู่มือนี้ครอบคลุม 25 ปัญหาที่พบบ่อยพร้อมการวินิจฉัยและวิธีแก้ไขทีละขั้นตอน

<Badge variant="outline">กฎข้อ 1</Badge> **เริ่มจากสิ่งง่าย ๆ ก่อน:** รีบูตกล้อง
ตรวจสอบสายเคเบิล และตรวจสอบไฟ PoE ก่อนที่จะเปลี่ยนฮาร์ดแวร์

---

## ผังการวินิจฉัยด่วน

1. **ไม่มีไฟ / ไม่ทำงาน** → ตรวจสอบ PoE → ตรวจสอบสาย → ตรวจสอบอะแดปเตอร์
2. **ออฟไลน์ในแอป** → Ping กล้อง → ตรวจสอบ IP → ตรวจสอบ VLAN / firewall
3. **ภาพกลางคืนไม่ทำงาน** → ตรวจสอบ IR LED → ตรวจสอบสิ่งกีดขวาง → ตรวจสอบโหมดกลางคืน
4. **NVR ไม่บันทึก** → ตรวจสอบ HDD → ตรวจสอบพื้นที่ว่าง → ตรวจสอบกำหนดการ
5. **แจ้งเตือนปลอม** → ปรับ灵敏度 → ตั้งโซนเคลื่อนไหว → ใช้ AI

## 1. ปัญหาเปิดเครื่อง

### 1.1 กล้องไม่ทำงาน ไม่มีไฟ ไม่ตอบสนอง

| ขั้นตอน | ตรวจสอบ             | วิธีแก้ไข                                   |
| ------- | ------------------- | ------------------------------------------- |
| 1       | **ไฟพอร์ต PoE**     | ดับ = พอร์ตเสีย/เกินงบ → ย้ายไปพอร์ตที่ดี   |
| 2       | **ไฟ Injektor PoE** | ดับ = Injektor เสีย → เปลี่ยน Injektor      |
| 3       | **แรงดันที่กล้อง**  | มัลติมิเตอร์: 44–57V (PoE), 12V±10% (DC)    |
| 4       | **สายครบทุกเส้น**   | ทดสอบ: 8 เส้น → เปิดคู่ = ครอบปลายใหม่      |
| 5       | **รีเซ็ตกล้อง**     | กด reset ค้าง 10–30 วินาที → เพิ่มในแอป/NVR |

### 1.2 PoE เกินงบ

**อาการ:** กล้องบางตัวดับเมื่อเพิ่มกล้องใหม่ หรือกล้องกระพริบเป็นจังหวะ

| ขั้นตอน | ตรวจสอบ                         | วิธีแก้ไข                         |
| ------- | ------------------------------- | --------------------------------- |
| 1       | รวมการใช้พลังงานของกล้องทั้งหมด | คำนวณ Σ วัตต์ของกล้อง             |
| 2       | เปรียบเทียบกับงบ PoE switch     | ต้องว่าง >20%                     |
| 3       | แบ่งกล้องไปยัง switch อื่น      | เพิ่ม PoE switch หรือใช้ injector |

## 2. เครือข่ายและการเชื่อมต่อ

### 2.1 กล้อง "ออฟไลน์" ในแอป/NVR

| ขั้นตอน | ทดสอบ                 | วิธีแก้ไข                           |
| ------- | --------------------- | ----------------------------------- |
| 1       | Ping IP กล้องจาก PC   | Timeout = ปัญหา Layer 1/2           |
| 2       | Ping gateway จากกล้อง | Timeout = VLAN / firewall / routing |
| 3       | ARP → MAC ตรงกัน?     | IP ซ้ำ → IP คงที่ / สำรอง DHCP      |

### 2.2 Wi-Fi หลุดบ่อย

| ขั้นตอน | ตรวจสอบ            | วิธีแก้ไข                                           |
| ------- | ------------------ | --------------------------------------------------- |
| 1       | ความแรงสัญญาณ RSSI | ต่ำกว่า -60 dBm = ย้าย Access Point ให้ใกล้ขึ้น     |
| 2       | ช่องสัญญาณ         | เลือกช่องที่ไม่ซ้ำกับเพื่อนบ้าน (ใช้ WiFi Analyzer) |
| 3       | แบนด์วิดท์         | 2.4 GHz ทะลุกำแพงได้ดีกว่า 5 GHz                    |
| 4       | รีบูตเราเตอร์      | ถอดปลั๊ก 30 วินาที → เสียบใหม่                      |

## 3. การบันทึก

### 3.1 NVR ไม่บันทึก / ช่องว่างในไทม์ไลน์

| ตรวจสอบ                   | วิธีแก้ไข                                |
| ------------------------- | ---------------------------------------- |
| สุขภาพ HDD (S.M.A.R.T.)   | เซกเตอร์รีอัลโลเคท >0 → **เปลี่ยนทันที** |
| พื้นที่เต็ม / เขียนทับปิด | เปิด "เขียนทับไฟล์เก่าสุด" ในการตั้งค่า  |
| กำหนดการบันทึก            | ตรวจสอบว่าได้ตั้งค่า "บันทึกต่อเนื่อง"   |
| โหมดกล้อง                 | กล้องบางรุ่นต้องเปิด "บันทึก" แยกต่างหาก |

### 3.2 microSD ไม่บันทึก

| ตรวจสอบ        | วิธีแก้ไข                                       |
| -------------- | ----------------------------------------------- |
| รูปแบบการ์ด    | FAT32 (สูงสุด 32GB) หรือ exFAT (>32GB)          |
| ความเร็วการ์ด  | ต้องเป็น U3 / V30 สำหรับ 4K                     |
| อายุการ์ด      | การ์ด microSD สำหรับ Surveillance มีอายุ 1–3 ปี |
| ฟอร์แมตในการ์ด | ฟอร์แมตในการ์ดผ่านแอปหรือ NVR                   |

## 4. ตรวจจับการเคลื่อนไหว

### 4.1 แจ้งเตือนปลอมมากเกินไป

| การตั้งค่า         | คำแนะนำ                 | เหตุผล                      |
| ------------------ | ----------------------- | --------------------------- |
| **ประเภทตรวจจับ**  | คน+รถเท่านั้น           | ไม่รวมสัตว์ ต้นไม้ เงา ฝน   |
| **โซนเคลื่อนไหว**  | วาดเฉพาะพื้นที่เป้าหมาย | ไม่รวมถนน ต้นไม้ เพื่อนบ้าน |
| **ความไว**         | ปานกลาง (เริ่ม) → ปรับ  | สูงเกิน = ทุกอย่างแจ้งเตือน |
| **เวลาเริ่ม-หยุด** | กลางคืน = แมว/กวาง      | จำกัดเวลาที่สัตว์ออกหากิน   |

### 4.2 ไม่มีการแจ้งเตือนเคลื่อนไหว

| ตรวจสอบ             | วิธีแก้ไข                            |
| ------------------- | ------------------------------------ |
| เปิดใช้งานหรือยัง   | ตรวจสอบว่าเปิด "แจ้งเตือนเคลื่อนไหว" |
| โซนเคลื่อนไหว       | โซนว่างเปล่า = วาดโซนใหม่            |
| ความไว              | เพิ่มความไวชั่วคราวเพื่อทดสอบ        |
| การอัปเดตเฟิร์มแวร์ | อัปเดตเฟิร์มแวร์กล้องเป็นรุ่นล่าสุด  |

## 5. ปัญหาภาพ

### 5.1 ภาพกลางคืนมืดหรือไม่ชัด

| ขั้นตอน | ตรวจสอบ             | วิธีแก้ไข                                 |
| ------- | ------------------- | ----------------------------------------- |
| 1       | มี IR LEDs สว่างไหม | ไม่สว่าง = เปิดโหมดกลางคืน / เปลี่ยนกล้อง |
| 2       | แสงสะท้อน           | ย้ายกล้องออกจากผนัง/เพดาน หรือใช้遮光罩   |
| 3       | ใยแมงมุม            | เช็ดเลนส์—ใยแมงมุมสะท้อน IR               |
| 4       | ระยะ IR             | IR พลังงานสูงสุด 30 ม. เกินกว่านั้นมืด    |
| 5       | โฟกัส               | ปรับโฟกัสเลนส์ (กล้อง varifocal)          |
| 6       | หมอก/ฝน             | ใช้กล้องกันน้ำ (IP67) หรือเช็ดเลนส์       |

### 5.2 ภาพเบลอ / ไม่ชัด

| สาเหตุ            | วิธีแก้ไข                                      |
| ----------------- | ---------------------------------------------- |
| โฟกัสผิด          | ปรับโฟกัสเลนส์ (มอเตอร์หรือ manual)            |
| คราบเลนส์         | เช็ดด้วยผ้าไมโครไฟเบอร์                        |
| อัตราเฟรมต่ำ      | ลด FPS เหลือ 15 → ความคมชัดดีขึ้น              |
| Bitrate ต่ำเกินไป | เพิ่ม CBR เป็น 4 Mbps (4K) หรือ 2 Mbps (1080p) |
| กล้องสั่น         | ขันสกรู mount ให้แน่น                          |

## 6. ปัญหาเสียง

### 6.1 ไม่มีเสียง (กล้องที่มีไมโครโฟน)

| ตรวจสอบ          | วิธีแก้ไข                          |
| ---------------- | ---------------------------------- |
| เปิดเสียงในแอป   | เปิดใช้งาน "บันทึกเสียง"           |
| ระดับเสียง       | เพิ่ม音量เป็น 100%                 |
| กันน้ำ           | กล้องIP67—ไมค์อยู่ใต้ซีล ตรวจสอบรู |
| อัปเดตเฟิร์มแวร์ | อัปเดตเป็นรุ่นล่าสุด               |

## 7. ปัญหาการเข้าถึงระยะไกล

### 7.1 เข้าถึงจากนอกบ้านไม่ได้

| ตรวจสอบ          | วิธีแก้ไข                                    |
| ---------------- | -------------------------------------------- |
| อินเทอร์เน็ต NVR | NVR ต้องต่ออินเทอร์เน็ต                      |
| พอร์ต forwarding | เปิดพอร์ต HTTP/HTTPS/RTSP ในเราเตอร์         |
| UPnP             | เปิด UPnP (หรือตั้งค่า Forwarding ด้วยตนเอง) |
| VPN              | ตั้งค่า VPN (Tailscale, WireGuard)           |
| DDNS             | ตั้งค่า DDNS ถ้า IP เปลี่ยนบ่อย              |
