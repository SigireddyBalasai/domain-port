Kamera keamanan memiliki pola kegagalan yang dapat diprediksi. Panduan ini mencakup perbaikan untuk 10 masalah teratas — diperbarui untuk 2026. Sebagian besar dapat diperbaiki tanpa mengganti kamera.

dahulu.** Lebih dari 50% "kamera mati" adalah masalah daya.

## Masalah #1: Kamera Offline / Tidak Merespons (Paling Umum)

### Apakah Kamera Mendapat Daya?

<Accordion type="single" collapsible>
  <AccordionItem value="power-check">
    <AccordionTrigger>Cara Memeriksa Daya Kamera</AccordionTrigger>
    <AccordionContent>

<strong>PoE (Power over Ethernet):</strong>
<ol>
  <li>Cari lampu tautan di switch PoE → Mati = Tidak ada daya. Menyala = Daya aktif.</li>
  <li>Gunakan tester PoE (seperti LinkRunner AT 2000) di ujung kamera: Baca tegangan. Seharusnya >44V (PoE) / >50V (PoE+).</li>
  <li>Multimeter: Ukur antara pin 4–5 (+) dan 7–8 (-) pada kabel yang tidak tersambung. Seharusnya 44–57V DC.</li>
</ol>

<strong>12V DC:</strong>
<ol>
  <li>Multimeter di colokan daya kamera: Seharusnya 11.5–12.5V DC.</li>
  <li>
    Di bawah beban (kamera menyala + IR): Jangan turun di bawah 11V. Jika turun,
    ganti adaptor daya.
  </li>
</ol>

<strong>Baterai:</strong>
<ol>
  <li>Periksa aplikasi kamera → tingkat baterai.</li>
  <li>Jika <10%, isi daya. Jika tidak mau mengisi daya (musim dingin, <0°C), bawa masuk untuk menghangatkannya.</li>
</ol>

    </AccordionContent>

  </AccordionItem>
</Accordion>

### Perbaikan Cepat Matikan-Hidupkan

1. **Putuskan daya kamera** (cabut PoE / adaptor 12V / baterai) — tunggu **minimal 30 detik**. Kamera membutuhkan waktu 10–30 detik untuk melepaskan muatan kapasitor.
2. **Sambungkan kembali daya** — tunggu 2 menit. Kamera PoE boot dalam 30–90 detik.
3. **Periksa aplikasi** — jika masih offline, lanjutkan ke langkah berikutnya.

### Perbaikan Lebih Lanjut

| Masalah                                             | Perbaikan                                                                         |
| --------------------------------------------------- | --------------------------------------------------------------------------------- |
| **Kamera reboot dalam siklus**                      | Penurunan tegangan — uji di bawah beban. Ganti kabel / catu daya.                 |
| **Lampu tautan switch menyala, kamera tidak aktif** | Kabel rusak — crimp ulang / ganti kabel. Uji dengan tester.                       |
| **Kamera boot, lalu mati setelah 10 detik**         | Pasokan daya PoE tidak mencukupi (kamera + IR > anggaran PoE). Tingkatkan switch. |
| **Firmware macet**                                  | Pemulihan firmware (tombol reset 10 detik) → perbarui lagi.                       |
| **Kamera beku — gambar diam**                       | Cabut 30 detik. Jika berulang, mungkin sensor/CPU rusak → ganti.                  |

## Masalah #2: Gambar Buram / Berdebu / Berjaring Laba-laba

**Lensa kamera perlu dibersihkan.** Debu, serbuk sari, dan sarang laba-laba adalah penyebab #1 gambar buram.

<Accordion type="single" collapsible>
  <AccordionItem value="lens-cleaning">
    <AccordionTrigger>Cara Membersihkan Lensa Kamera</AccordionTrigger>
    <AccordionContent>

<strong>Alat:</strong> Lap microfiber, pembersih lensa (atau alkohol isopropil
70%), sikat lembut, udara bertekanan.

<ol>
  <li>
    <strong>Periksa secara visual</strong> — Sarang laba-laba di sekitar
    kubah/lensa. Jaring di depan lensa = gambar buram — bersihkan dengan sikat
    lembut terlebih dulu.
  </li>
  <li>
    <strong>Udara bertekanan</strong> — Semprot debu dari kubah/lensa.
  </li>
  <li>
    <strong>Lap microfiber</strong> — Basahi sedikit dengan pembersih lensa.
    Usap dari dalam ke luar. Jangan menekan (tekanan dapat merusak permukaan
    kubah).
  </li>
  <li>
    <strong>Periksa lagi</strong> — Nyalakan umpan langsung. Fokus? Jika masih
    buram setelah dibersihkan, goresan kubah → ganti kubah.
  </li>
  <li>
    <strong>Pencegahan:</strong> Semprot anti-laba-laba (Web-Off, Permanone) di
    sekitar braket kamera setiap 3 bulan. Potong tanaman yang menyentuh kamera.
  </li>
</ol>

    </AccordionContent>

  </AccordionItem>
</Accordion>

### Goresan Kubah / Lensa

**Kubah gores = ganti.** Poles tidak akan memperbaiki goresan yang dalam (mengaburkan gambar). Cari nomor model kubah, pesan pengganti ($10–$30). Kamera bulat: sering kali satu rakitan dengan lampu IR — kubah + IR ring.

**Goresan kecil:** Coba poles dengan pasta gigi (non-gel) atau PlastX. Gosok melingkar dengan kain microfiber selama 60 detik. Bilas. Periksa. Jika masih gores — ganti.

## Masalah #3: Malam Hari Berwarna Putih (IR Whiteout)

**Penyebab umum:** Lampu IR kamera memantulkan permukaan di dekatnya.

| Permukaan Pemantul                     | Jarak      | Perbaikan                                                                             |
| -------------------------------------- | ---------- | ------------------------------------------------------------------------------------- |
| **Dinding / Papan / Atap**             | 6–24 inci  | Miringkan kamera ke bawah 15–30°. Jauhkan 12–18 inci (lengan ekstensi).               |
| **Kaca / Jendela**                     | Berapa pun | **Jangan** bidik melalui kaca. Pasang kamera di luar atau matikan IR + gunakan sorot. |
| **Sarang laba-laba / Debu pada kubah** | Pada lensa | Bersihkan. Tutup cincin IR dengan kain bulu domba (buatan sendiri).                   |
| **Hujan / Salju / Kabut**              | Udara      | Illuminator IR eksternal (dipasang jauh dari lensa). Matikan IR kamera.               |
| **Trim putih / talang**                | 12–36 inci | Cat hitam doff (cat panggangan) atau selotip.                                         |
| **Pelat nomor (reflektif)**            | 20–50 kaki | Miringkan ke bawah; IR eksternal; kamera LPR dengan polarisator.                      |

Lihat [Panduan Lengkap Perbaikan Penglihatan Malam](/blog/night-vision-not-working-troubleshooting-guide) untuk diagnostik langkah demi langkah.

## Masalah #4: Penurunan Tegangan (Kamera Reboot di Malam Hari)

**Paling umum dengan PoE.** Kamera berfungsi di siang hari (daya rendah), reboot di malam hari (IR menyala, menarik daya 2–5W ekstra).

### Diagnosis

| Gejala                         | Kemungkinan Penyebab                                                                 |
| ------------------------------ | ------------------------------------------------------------------------------------ |
| Kamera reboot setiap malam     | Penurunan tegangan — kabel terlalu panjang / terlalu tipis / PoE anggaran terlampaui |
| Kamera reboot saat IR menyala  | Pasokan daya 12V tidak mencukupi amps                                                |
| Kamera reboot saat lampu sorot | Sorot LED menarik 10–20W — PoE+ diperlukan                                           |

### Perbaikan

| Skenario                               | Perbaikan                                                                     |
| -------------------------------------- | ----------------------------------------------------------------------------- |
| **PoE (802.3af) dengan IR**            | Upgrade ke PoE+ (802.3at, 30W). Ganti switch / injector.                      |
| **Kabel Cat5e panjang >100 kaki**      | Ganti dengan Cat6. Batasi panjang 300 kaki. Atau pasang switch PoE perantara. |
| **12V DC dengan kabel 18 AWG panjang** | Ganti dengan 14 AWG. Atau pasang catu daya 12V lokal di kamera.               |
| **Adaptor daya 12V 1A**                | Ganti dengan 12V 2A+ (arus lebih tinggi).                                     |
| **Switch PoE anggaran terlampaui**     | Hitung total daya kamera. Upgrade switch dengan anggaran PoE lebih besar.     |

<Callout type="warning">

**Penurunan tegangan adalah penyebab #1 "kamera mati" yang sebenarnya masih berfungsi.** Uji tegangan di kamera saat IR menyala. Tegangan PoE: seharusnya >44V (PoE), >50V (PoE+). 12V: seharusnya >11,5V. Jika tidak, Anda mengalami penurunan tegangan.

</Callout>

## Masalah #5: Deteksi Gerakan Salah / Tidak Ada Deteksi

| Gejala                                                 | Perbaikan                                                                                       |
| ------------------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| **Terlalu banyak peringatan (pohon, mobil, bayangan)** | Turunkan sensitivitas dari Tinggi ke Sedang. Gambar zona ketat — kecualikan pohon/jalan.        |
| **Tidak mendeteksi orang/kendaraan**                   | Naikkan sensitivitas. Aktifkan deteksi AI (orang + kendaraan) jika tersedia. Perbarui firmware. |
| **Deteksi berhenti di malam hari**                     | Mode tidur baterai — bangunkan dengan gerakan. Uji jarak — PIR hanya 25–30 kaki. Pindahkan.     |
| **Deteksi hanya di siang hari**                        | Filter IR menghalangi malam. Periksa filter potong IR macet — siklus daya.                      |
| **Penundaan sebelum peringatan**                       | Kopi pra-rekaman: Atur ke 5–10 detik (menangkap pendekatan).                                    |

## Masalah #6: Koneksi Terputus (Wi-Fi / Nirkabel)

| Gejala                                     | Perbaikan                                                                                          |
| ------------------------------------------ | -------------------------------------------------------------------------------------------------- |
| **Kamera offline setiap hari**             | RSSI < -70 dBm — pindahkan AP lebih dekat. Ubah saluran Wi-Fi. Uji dengan ponsel di lokasi kamera. |
| **Umpan langsung berputar / gagal**        | Bandwidth upload < 1 Mbps — uji Speedtest di lokasi kamera. Upgrade AP.                            |
| **Rekaman terputus-putus / hilang**        | Packet loss — periksa interferensi (microwave, tetangga). Ubah saluran. Gunakan 20 MHz saja.       |
| **"Tidak Ada Wi-Fi" setelah mati listrik** | UPS pada router + AP. Nyalakan ulang semuanya. Atur kamera agar otomatis terhubung kembali.        |

## Masalah #7: Penyimpanan Penuh (microSD / HDD)

| Gejala                                | Perbaikan                                                                                                                                |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **microSD penuh — tidak merekam**     | Hapus otomatis diaktifkan? Periksa pengaturan. Format kartu (keluarkan + masukkan kembali). Ganti dengan kartu yang lebih besar (256GB). |
| **HDD NVR penuh**                     | Hapus rekaman lama. Perluas HDD (ganti dengan 4TB+). Aktifkan timpa otomatis (hapus yang terlama).                                       |
| **HDD gagal — kesalahan penyimpanan** | Ganti HDD. GunakN HDD kelas CCTV (WD Purple, Seagate SkyHawk) — dirancang untuk perekaman 24/7.                                          |

<Callout type="note">

**HDD CCTV vs HDD desktop:** HDD desktop (Barracuda, Blue) memiliki waktu kerja 5.000 jam/tahun. HDD CCTV (Purple, SkyHawk) 8.760 jam/tahun + peredam getaran + cache yang dioptimalkan untuk streaming video. HDD desktop di NVR = gagal dalam 1–2 tahun.

</Callout>

## Masalah #8: Korosi / Oksidasi / Karat pada Konektor

**Gejala:** Kamera berfungsi sebentar-sebentar, kehilangan tautan saat hujan/lembab, LED berkedip.

### Perbaikan

| Metode                                           | Biaya | Efektivitas        |
| ------------------------------------------------ | ----- | ------------------ |
| **Bersihkan dengan alkohol isopropil**           | $0    | ✅✅✅ (ringan)    |
| **Semprotan kontak listrik** (DeoxIT D5, CRC QD) | $12   | ✅✅✅✅ (sedang)  |
| **Ganti konektor**                               | $5    | ✅✅✅✅✅ (parah) |
| **Gunakan pelumas dielektrik**                   | $8    | ✅✅✅✅✅ (cegah) |
| **Potong ujung kabel, crimp ulang**              | $2    | ✅✅✅✅✅ (parah) |

### Pencegahan

- Oleskan **pelumas dielektrik** (NoOx, CRC) ke pin RJ45 sebelum memasukkan ke kamera.
- Tutup konektor dengan **pita kedap sendiri** (Rescue Tape, Nashua Stretch & Seal).
- Pasang **conduit** ke kamera — segel ujung conduit dengan但il.
- Periksa konektor setiap tahun. Bersihkan + olesi ulang.

## Masalah #9: Sarang Laba-laba pada Lensa (Masalah Abadi)

Laba-laba suka kamera lampu IR — hangat, terlindung, dan penerangan untuk mangsa.

### Pencegahan

| Metode                                                           | Biaya | Efektivitas | Durasi   |
| ---------------------------------------------------------------- | ----- | ----------- | -------- |
| **Pembersihan bulanan**                                          | $0    | ✅✅        | 1 bulan  |
| **Semprot anti laba-laba** (Web-Off, Permanone)                  | $12   | ✅✅✅✅    | 3 bulan  |
| **Penyegar udara peppermint** (beberapa tetes di braket)         | $3    | ✅✅        | 2 minggu |
| **Lingkaran IR menjauh dari dinding** (ekstensi 6")              | $10   | ✅✅✅✅✅  | Permanen |
| **Bungkus braket dengan pita tembaga** (sengatan listrik ringan) | $15   | ✅✅✅✅    | 1 tahun  |

## Masalah #10: Had上限 Kelelahan Wi-Fi / Terlalu Banyak Perangkat

**Gejala:** Perangkat baru terhubung, yang lama terputus. Router rumah membatasi koneksi simultan.

### Perbaikan

| Solusi                          | Detail                                                                                                                               |
| ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ |
| **Kurangi perangkat**           | Jangan menghubungkan 20+ perangkat IoT ke satu router rumahan. Pisahkan: satu router untuk komputer, AP terpisah untuk kamera/IoT.   |
| **AP khusus untuk kamera**      | Pasang AP Wi-Fi kedua (UniFi U6 Lite, $109) hanya untuk kamera. Didedikasikan, tidak ada persaingan.                                 |
| **Pindahkan ke PoE**            | Kamera nirkabel semuanya → ganti dengan PoE. Tidak ada kelelahan Wi-Fi.                                                              |
| **Batasi bandwidth per kamera** | Atur bitrate ke 4 Mbps (4K) alih-alih 8 Mbps. Batasi umpan tampilan langsung (mengalirkan semua kamera = membebani AP).              |
| **Router yang lebih baik**      | Router kelas konsumen (TP-Link Archer, Asus) menangani 20–30 perangkat. Router kelas bisnis (UniFi, Omada) menangani 100+ perangkat. |
