## দ্রুত ডায়াগনস্টিক ফ্লোচার্ট

```
ক্যামেরা কাজ করছে না?
    │
    ├─ পাওয়ার LED নেই? → পাওয়ার সাপ্লাই / PoE / ব্যাটারি (সেকশন 1)
    ├─ পাওয়ার ঠিক আছে, ছবি নেই? → কেবল / লিংক / IP কনফ্লিক্ট (সেকশন 2)
    ├─ ছবি ঠিক, রেকর্ডিং নেই? → NVR / স্টোরেজ / শিডিউল (সেকশন 3)
    ├─ ছবি ঠিক, ভুল অ্যালার্ট? → মোশন জোন / সেন্সিটিভিটি / AI (সেকশন 4)
    ├─ নাইট ভিশন খারাপ? → IR / রিফ্লেকশন / শাটার (সেকশন 5)
    └─ রিমোট অ্যাক্সেস ব্যর্থ? → পোর্ট ফরওয়ার্ড / VPN / ভেন্ডর ক্লাউড (সেকশন 6)
```

## 1. পাওয়ার ও স্টার্টআপ সমস্যা

### 1.1 ক্যামেরা ডেড — কোনো LED নেই, কোনো সাড়া নেই

| ধাপ | চেক                    | সমাধান                                                                              |
| --- | ---------------------- | ----------------------------------------------------------------------------------- |
| 1   | **PoE সুইচ পোর্ট LED** | বন্ধ = সুইচ পোর্ট ডেড / বাজেট অতিক্রান্ত → পরিচিত ভালো পোর্টে সরান                  |
| 2   | **PoE ইনজেক্টর LED**   | বন্ধ = ইনজেক্টর নষ্ট / AC নেই → ইনজেক্টর প্রতিস্থাপন ($25)                          |
| 3   | **ক্যামেরায় ভোল্টেজ** | মাল্টিমিটার: 44–57V (PoE), 12V±10% (DC) → কম = ভোল্টেজ ড্রপ / খারাপ কেবল            |
| 4   | **কেবল কন্টিনিউইটি**   | টেস্টার: সব 8 তার → ওপেন পেয়ার = উভয় প্রান্ত পুনরায় টার্মিনেট                    |
| 5   | **ক্যামেরা রিসেট**     | রিসেট 10–30 সেকেন্ড চেপে ধরে রাখুন (ম্যানুয়াল চেক) → অ্যাপ/NVR এ পুনরায় অ্যাডাপ্ট |

**ব্যাটারি ক্যামেরা:** ডেড ঘোষণা করার আগে USB-C দিয়ে 24 ঘন্টা চার্জ করুন। Li-ion BMS গভীর ডিসচার্জের পর লক হয়ে যায়।

### 1.2 রাতে ক্যামেরা রিবুট (IR চালু)

**কারণ:** IR LED চালু হলে ভোল্টেজ ড্রপ (1–2W যোগ হয়)।

| রোগ নির্ণয়                            | সমাধান                                           |
| -------------------------------------- | ------------------------------------------------ |
| PoE সুইচ বাজেট লিমিটে                  | ক্যামেরা ইনজেক্টরে সরান; সুইচ আপগ্রেড            |
| Cat6 এ কেবল রান >250 ফুট               | রান ছোট করুন; Cat6a; ক্যামেরায় PoE++ ইনজেক্টর   |
| CCA (কপার-ক্ল্যাড অ্যালুমিনিয়াম) কেবল | **সলিড কপার Cat6 দিয়ে প্রতিস্থাপন করুন**        |
| ব্যাটারি ক্যাম: Li-ion ঠান্ডায় স্যাগ  | LiFePO₄ আপগ্রেড; ইনসুলেটেড বক্স; ভিতরে এনে চার্জ |

### 1.3 PoE সুইচ ওভারহিটিং / পোর্ট ফ্ল্যাপিং

- **PoE বাজেট চেক:** Σ ক্যামেরা সর্বোচ্চ ড্র ≤ সুইচ বাজেট × 0.8
- **থার্মাল:** সুইচ বন্ধ ক্যাবিনেটে → ফ্যান যোগ করুন বা ভেন্টিলেটেড র্যাকে সরান
- **ফার্মওয়্যার:** সুইচ ফার্মওয়্যার আপডেট (PoE নেগোশিয়েশন বাগ সাধারণ)

## 2. নেটওয়ার্ক ও কানেক্টিভিটি

### 2.1 অ্যাপ/NVR এ ক্যামেরা "অফলাইন"

| ধাপ | পরীক্ষা                             | সমাধান                                        |
| --- | ----------------------------------- | --------------------------------------------- |
| 1   | PC থেকে ক্যামেরা IP পিং             | টাইমআউট = লেয়ার 1/2 সমস্যা (কেবল, সুইচ, IP)  |
| 2   | ক্যামেরা থেকে গেটওয়ে পিং (NVR শেল) | টাইমআউট = VLAN / ফায়ারওয়াল / রাউটিং         |
| 3   | `arp -a` → MAC ক্যামেরার সাথে মেলে? | IP কনফ্লিক্ট → স্ট্যাটিক IP / DHCP রিজার্ভেশন |
| 4   | DHCP লিজ মেয়াদ শেষ                 | স্ট্যাটিক IP বা DHCP রিজার্ভেশন (রাউটার)      |
| 5   | IP কনফ্লিক্ট                        | দুটি ডিভাইস একই IP → ইউনিক IP রিজার্ভ         |

### 2.2 Wi-Fi ক্যামেরা প্রতিদিন ড্রপ

| রোগ নির্ণয়         | সমাধান                                                      |
| ------------------- | ----------------------------------------------------------- |
| RSSI < -75 dBm      | AP কাছাকাছি সরান; ওয়্যার্ড AP যোগ করুন; চ্যানেল চেক        |
| 2.4 GHz কনজেশন      | শুধু চ্যানেল 1/6/11; 20 MHz প্রস্থ; ব্যান্ড-স্টিয়ারিং বন্ধ |
| DFS চ্যানেল (5 GHz) | 2.4 GHz বাধ্য করুন; 5 GHz রাডার = সংযোগ বিচ্ছিন্ন           |
| পাওয়ার সেভ মোড     | ক্যামেরা অ্যাপে Wi-Fi পাওয়ার সেভ নিষ্ক্রিয়                |
| রাউটার রিবুট শিডিউল | অটো-রিবুট নিষ্ক্রিয়; রাউটারে UPS                           |

### 2.3 RTSP স্ট্রিম খুলছে না (VLC / Frigate / NVR)

```
টেস্ট URL: rtsp://user:pass@192.168.1.100:554/stream1
```

| ত্রুটি             | কারণ                | সমাধান                                                                                                                           |
| ------------------ | ------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| 401 Unauthorized   | ভুল ব্যবহারকারী/পাস | NVR/অ্যাপ চেক: প্রায়ই `admin` / `123456` বা খালি                                                                                |
| 404 Not Found      | ভুল পাথ             | ব্র্যান্ড পাথ: Hikvision `/Streaming/Channels/101`, Dahua `/cam/realmonitor?channel=1&subtype=0`, Reolink `/h264Preview_01_main` |
| Connection Refused | RTSSP নিষ্ক্রিয়    | ক্যামেরা Web UI → Network → RTSP সক্রিয় করুন                                                                                    |
| টাইমআউট            | ফায়ারওয়াল / VLAN  | NVR VLAN থেকে Camera VLAN এ TCP 554 অনুমতি দিন                                                                                   |

**ব্র্যান্ড RTSP পাথ (সাধারণ):**

- **Reolink:** `/h264Preview_01_main` (মেইন), `/h264Preview_01_sub` (সাব)
- **Dahua/Amcrest:** `/cam/realmonitor?channel=1&subtype=0`
- **Hikvision:** `/Streaming/Channels/101` (মেইন), `/Streaming/Channels/102` (সাব)
- **UniFi:** `rtsp://user:pass@ip:7447/...` (Protect সেটিংস → RTSP)
- **Amcrest:** Dahua এর মতো (OEM)

## 3. রেকর্ডিং ও স্টোরেজ

### 3.1 NVR রেকর্ড করছে না / টাইমলাইনে ফাঁক

| চেক                        | সমাধান                                                          |
| -------------------------- | --------------------------------------------------------------- |
| HDD স্বাস্থ্য (S.M.A.R.T.) | রিঅ্যালোকেটেড সেক্টর >0 → **অবিলম্বে প্রতিস্থাপন**              |
| HDD পূর্ণ / ওভাররাইট বন্ধ  | স্টোরেজ সেটিংসে "সবচেয়ে পুরনো ওভাররাইট" সক্রিয় করুন           |
| শিডিউল ভুল কনফিগার         | যাচাই: মেইন স্ট্রিমে 24/7 একটানা; সাবে মোশন                     |
| স্ট্রিম অমিল               | NVR মেইন আশা করছে; ক্যামেরা সাব পাঠাচ্ছে → এনকোড সেটিংস মেলান   |
| টাইমস্ট্যাম্প ড্রিফট       | NVR + ক্যামেরায় NTP সক্রিয় করুন (একই সার্ভার: `pool.ntp.org`) |

### 3.2 microSD রেকর্ড করছে না / নষ্ট

| লক্ষণ                    | কারণ                  | সমাধান                                                         |
| ------------------------ | --------------------- | -------------------------------------------------------------- |
| "ফরম্যাট প্রয়োজন" লুপ   | কার্ড নষ্ট / ভুল FS   | **U3/V30 A2** দিয়ে প্রতিস্থাপন (SanDisk Extreme, Samsung Pro) |
| ফাইল 0 বাইট / প্লেবল নয় | লেখার সময় পাওয়ার লস | অ্যাপে "সেফ ইজেক্ট" সক্রিয়; প্লাগ-ইন ক্যামে UPS               |
| 4 GB এ থেমে যায়         | FAT32 সীমা            | ক্যামেরা অটো-exFAT করবে; না হলে PC তে exFAT ফরম্যাট            |
| লুপ কাজ করছে না          | কার্ড পূর্ণ / লক      | লক সুইচ চেক; পুরনো ফাইল মুছুন; ক্যামেরায় ফরম্যাট              |

### 3.3 প্লেব্যাক ফ্রোজেন / পিক্সেলেটেড / সবুজ পর্দা

| কারণ                       | সমাধান                                                           |
| -------------------------- | ---------------------------------------------------------------- |
| H.265/HEVC সমর্থিত নয়     | প্লেয়ারে HEVC কোডেক দরকার (Windows: $0.99 স্টোর; VLC: বিল্ট-ইন) |
| সাব-স্ট্রিম নির্বাচিত      | প্লেব্যাকে মেইন স্ট্রিমে স্যুইচ                                  |
| কিফ্রেম ইন্টারভাল খুব বেশি | কিফ্রেম ইন্টারভাল = FPS (যেমন 15 fps → 15)                       |
| VBR বিটরেট স্পাইক          | CBR বা VBR Max Cap সেট করুন (4K এর জন্য 4–6 Mbps)                |

## 4. মোশন ডিটেকশন ও অ্যালার্ট

### 4.1 অনেক বেশি ভুল অ্যালার্ট (স্প্যাম)

| সেটিং                    | সুপারিশকৃত                        | কারণ                                 |
| ------------------------ | --------------------------------- | ------------------------------------ |
| **ডিটেকশন টাইপ**         | শুধু Person + Vehicle             | প্রাণী, গাছ, ছায়া, বৃষ্টি বাদ দেয়  |
| **মোশন জোন**             | শুধু টার্গেট এলাকা আঁকুন          | রাস্তা, গাছ, প্রতিবেশী, আকাশ বাদ দিন |
| **সেন্সিটিভিটি**         | মাঝারি (শুরু) → সামঞ্জস্য         | উচ্চ = সবকিছু ট্রিগার                |
| **রিট্রিগার ইন্টারভাল**  | 30–60 সেকেন্ড                     | একই গাড়ির 50টি ক্লিপ প্রতিরোধ       |
| **অবজেক্ট সাইজ ফিল্টার** | ফ্রেমের ন্যূনতম 5% / সর্বোচ্চ 50% | পোকা, পাখি, দূরের গাড়ি উপেক্ষা      |
| **AI/ডিপ লার্নিং**       | সক্রিয় (Person/Vehicle/Package)  | 90%+ ভুল অ্যালার্ট হ্রাস             |

### 4.2 মিসড ইভেন্ট (ঘটনা ঘটলেও কোনো অ্যালার্ট নেই)

| চেক                        | সমাধান                                                               |
| -------------------------- | -------------------------------------------------------------------- |
| জোন ইভেন্ট এলাকা কভার করে? | জোন চওড়া করুন; হেঁটে পরীক্ষা                                        |
| সেন্সিটিভিটি খুব কম?       | এক নচ বাড়ান; পরীক্ষা                                                |
| রাতে "Person" শনাক্ত হয়নি | IR রিফ্লেকশন AI কে বিভ্রান্ত করে → ক্যামেরা নিচে কোণ; বাহ্যিক IR যোগ |
| ক্যামেরা "স্লিপ" মোডে      | ব্যাটারি ক্যাম: স্লিপ নিষ্ক্রিয় / PIR সেন্সিটিভিটি বাড়ান           |
| NVR মোশন ডিটেকশন বন্ধ      | NVR সাইডে সক্রিয় করুন (ক্যামেরা সাইড বন্ধ থাকতে পারে)               |

### 4.3 অ্যালার্ট বিলম্ব >10 সেকেন্ড

| কারণ                            | সমাধান                                                |
| ------------------------------- | ----------------------------------------------------- |
| ক্লাউড রিলে (Ring, Arlo, Blink) | সহজাত — লোকাল NVR/Frigate ব্যবহার করুন &lt;1s এর জন্য |
| দুর্বল Wi-Fi (RSSI < -75)       | সিগন্যাল উন্নত করুন (সেকশন 2.2)                       |
| NVR CPU সর্বোচ্চ                | অ্যানালিটিক্স চ্যানেল কমান; NVR আপগ্রেড               |
| MQTT/Webhook লেটেন্সি           | প্রথমে সরাসরি অ্যাপ পুশ পরীক্ষা                       |

## 5. ছবির গুণমান ও নাইট ভিশন

### 5.1 নাইট ভিশন হোয়াইটআউট / ফ্যাকাশে

**কারণ:** IR LED কাছের পৃষ্ঠ (দেয়াল, ইভ, কাচ, মাকড়সার জাল) থেকে প্রতিফলিত হচ্ছে।

| সমাধান                       | পদ্ধতি                                                     |
| ---------------------------- | ---------------------------------------------------------- |
| **15–30° নিচের দিকে কোণ**    | ব্র্যাকেট কাত; ক্যামেরা নিচে তাকায়, দেয়ালে নয়           |
| **ক্যামেরা বাইরে সরান**      | দেয়াল/ইভ থেকে 12–18" (এক্সটেনশন আর্ম)                     |
| **বিল্ট-ইন IR নিষ্ক্রিয়**   | **বাহ্যিক IR ইলুমিনেটর** যোগ করুন (আলাদা, লক্ষ্যের দিকে)   |
| **ডোম/লেন্স পরিষ্কার**       | মাকড়সার জাল = IR বিচ্ছুরণ → মাইক্রোফাইবার + লেন্স ক্লিনার |
| **স্মার্ট IR (যদি সমর্থিত)** | দূরত্ব অনুযায়ী IR তীব্রতা অটো-সামঞ্জস্য                   |

### 5.2 রাতে লাইসেন্স প্লেট অপঠনযোগ্য

| প্রয়োজনীয়তা   | স্পেক                                                    |
| --------------- | -------------------------------------------------------- |
| **শাটার স্পিড** | ন্যূনতম 1/100s (1/250s আদর্শ) — Exposure → Shutter এ সেট |
| **রেজোলিউশন**   | 4K/8MP + অপটিক্যাল জুম OR 2MP LPR ক্যামেরা               |
| **IR রেঞ্জ**    | প্লেট দূরত্ব ≤ ক্যামেরা IR রেটিং এর 50%                  |
| **WDR**         | 120 dB+ (হেডলাইট হ্যান্ডল করে)                           |
| **কোণ**         | লম্ব থেকে ≤30° (প্লেট সোজা)                              |

**সমাধান:** শাটার বাড়ান → ছবি গাঢ় কিন্তু প্লেট তীক্ষ্ণ। খুব গাঢ় হলে বাহ্যিক IR যোগ করুন।

### 5.3 দিনের ছবি: কুয়াশাচ্ছন্ন / কম কনট্রাস্ট / রঙের ভুল

| কারণ                     | সমাধান                                                     |
| ------------------------ | ---------------------------------------------------------- |
| ডোম নোংরা / আঁচড়        | মাইক্রোফাইবার পরিষ্কার; ডোম প্রতিস্থাপন ($15)              |
| IR কাট ফিল্টার আটকে      | পাওয়ার সাইকেল; ফার্মওয়্যার আপডেট; হালকা টোকা (যান্ত্রিক) |
| হোয়াইট ব্যালেন্স ভুল    | "Outdoor" / "Auto" / কাস্টম Kelvin (5500K) সেট             |
| ব্যাকলাইট (ফ্রেমে সূর্য) | পুনঃস্থাপন; WDR সক্রিয়; সান শিল্ড যোগ                     |

### 5.4 ফ্লিকার / ঢেউ খেলানো লাইন (50/60 Hz)

| কারণ                                | সমাধান                                                      |
| ----------------------------------- | ----------------------------------------------------------- |
| **পাওয়ার লাইন ফ্রিকোয়েন্সি অমিল** | ক্যামেরা 60 Hz (US) / 50 Hz (EU) — আপনার গ্রিডের সাথে মেলান |
| গ্রাউন্ড লুপ                        | সব যন্ত্রপাতি একই গ্রাউন্ড; PoE আইসোলেটর (Transtector)      |
| ফ্লুরোসেন্ট/LED ফ্লিকার             | শাটার 1/100 (50 Hz) বা 1/120 (60 Hz) সেট                    |

## 6. রিমোট অ্যাক্সেস ও মোবাইল অ্যাপ

### 6.1 Wi-Fi তে অ্যাপ কাজ করে, LTE/5G এ ব্যর্থ

| আর্কিটেকচার                                    | সমাধান                                                               |
| ---------------------------------------------- | -------------------------------------------------------------------- |
| **ভেন্ডর ক্লাউড** (Reolink, UniFi, Eufy, Ring) | ফোনের ইন্টারনেট চেক; ভেন্ডর স্ট্যাটাস পৃষ্ঠা                         |
| **পোর্ট ফরওয়ার্ড**                            | **ব্যবহার করবেন না।** CGNAT, ডাবল NAT, ISP ব্লক। VPN/ক্লাউডে স্যুইচ। |
| **VPN (Tailscale, WireGuard)**                 | ফোন VPN সক্রিয় আছে কিনা নিশ্চিত; স্প্লিট-টানেল অ্যাপ বাদ দিতে পারে  |
| **Cloudflare Tunnel**                          | `cloudflared` সার্ভিস স্বাস্থ্যকর কিনা যাচাই                         |

### 6.2 রিমোটে "কানেকশন ব্যর্থ" / টাইমআউট

| পরীক্ষা                          | প্রত্যাশিত | সমাধান                      |
| -------------------------------- | ---------- | --------------------------- |
| ফোন (VPN অন) থেকে NVR LAN IP পিং | &lt;50 ms  | VPN রাউটিং সমস্যা           |
| ফোন থেকে `curl http://nvr:80`    | HTML       | পোর্ট ব্লক / ফায়ারওয়াল    |
| ভেন্ডর ক্লাউড স্ট্যাটাস পৃষ্ঠা   | সবুজ       | অপেক্ষা / লোকাল VPN ব্যবহার |

### 6.3 লাইভ ভিউ কাজ করে, প্লেব্যাক রিমোটে ব্যর্থ

- প্লেব্যাক ভিন্ন পোর্ট/স্ট্রিম ব্যবহার করে (প্রায়ই HTTPS/443 vs RTSP/554)
- **চেক:** NVR → Network → HTTPS পোর্ট (ডিফল্ট 443) VPN এর মাধ্যমে অ্যাক্সেসযোগ্য
- **ব্যান্ডউইথ:** প্লেব্যাকের জন্য 4–8 Mbps টেকসই প্রয়োজন; LTE থ্রটল করতে পারে

## 7. PTZ ও যান্ত্রিক

### 7.1 PTZ নড়ে না / আটকে / ঘষে

| কারণ                            | সমাধান                                                        |
| ------------------------------- | ------------------------------------------------------------- |
| যান্ত্রিক সীমায় পৌঁছেছে        | প্রিসেট লিমিট চেক; হোমে রিসেট                                 |
| স্লিপ রিং ব্যর্থ (360° অন্তহীন) | কেবল হার্নেস নষ্ট → ক্যামেরা প্রতিস্থাপন (মেরামত করা যায় না) |
| বরফ / ধ্বংসাবশেষ                | হিটার ব্যর্থ; ম্যানুয়ালি পরিষ্কার; হিটার সার্কিট যাচাই       |
| প্রোটোকল অমিল                   | Pelco-D/P vs ONVIF PTZ → NVR প্রোটোকল ক্যামেরার সাথে মেলান    |

### 7.2 প্রিসেট টুর চলছে না

- শিডিউল সক্রিয়? (NVR → PTZ → Tour Schedule)
- ডুয়েল টাইম >0? (প্রতি প্রিসেটে 10–30s)
- হোম প্রিসেট সংজ্ঞায়িত? (টুর লুপের মধ্যে হোমে ফিরে)

## 8. অডিও (টু-ওয়ে)

### 8.1 কোনো অডিও নেই / একমুখী

| চেক                   | সমাধান                                                |
| --------------------- | ----------------------------------------------------- |
| অ্যাপ অনুমতি          | ফোন সেটিংস → অ্যাপ → মাইক্রোফোন: চালু                 |
| ক্যামেরা অডিও সক্রিয় | ক্যামেরা Web UI → Audio → Input/Output: চালু          |
| কোডেক অমিল            | G.711 (PCM) সবচেয়ে সামঞ্জস্যপূর্ণ; G.726 / AAC ট্রাই |
| হাফ-ডুপ্লেক্স vs ফুল  | বেশিরভাগ ক্যাম = PTT (হাফ); কথা বলার সময় শুনবেন না   |

### 8.2 ইকো / ফিডব্যাক / স্ট্যাটিক

| কারণ                                   | সমাধান                                                                  |
| -------------------------------------- | ----------------------------------------------------------------------- |
| স্পিকার খুব জোরে / মাইক গেইন বেশি      | অ্যাপে স্পিকার ভলিউম কমান; মাইক গেইন কমান                               |
| AEC (অ্যাকোস্টিক ইকো ক্যান্সেলেশন) নেই | ফুল-ডুপ্লেক্স ক্যামেরায় AEC দরকার; হাফ-ডুপ্লেক্স = বলার সময় মাইক মিউট |
| মাইকে বাতাস                            | উইন্ড ফিল্টার সক্রিয়; ফোম উইন্ডস্ক্রিন; পুনঃস্থাপন                     |

## 9. ফার্মওয়্যার ও আপডেট

### 9.1 নিরাপদ ফার্মওয়্যার আপগ্রেড পদ্ধতি

1. **চেঞ্জলগ পড়ুন** — "শুধু নিরাপত্তা" এবং স্থিতিশীল থাকলে এড়ান
2. **কনফিগ ব্যাকআপ** — NVR: কনফিগ এক্সপোর্ট; ক্যামেরা: সেটিংস স্ক্রিনশট
3. **প্রথমে NVR আপডেট** — তারপর ক্যামেরা (NVR কম্প্যাটিবল ফার্মওয়্যার পুশ করতে পারে)
4. **একবারে একটি ক্যামেরা** — প্রতিটি যাচাই করে পরেরটিতে যান
5. **পাওয়ার বন্ধ করবেন না** — প্রতি ক্যামেরায় 5–10 মিনিট
6. **যাচাই** — লাইভ ভিউ, রেকর্ডিং, মোশন, PTZ, অডিও

### 9.2 ব্যর্থ ফার্মওয়্যার রোলব্যাক

| ব্র্যান্ড           | পদ্ধতি                                              |
| ------------------- | --------------------------------------------------- |
| **Reolink**         | ফার্মওয়্যার পৃষ্ঠা → "পূর্ববর্তী সংস্করণ" বোতাম    |
| **Dahua/Hikvision** | TFTP রিকভারি (বুট + রিসেট ধরে + TFTP সার্ভার)       |
| **UniFi**           | Protect → Devices → Camera → "Rollback Firmware"    |
| **জেনেরিক**         | SD কার্ডে রুটে `firmware.bin`; রিসেট ধরে পাওয়ার অন |

## 10. পরিবেশগত ও শারীরিক

### 10.1 পানি প্রবেশ / কনডেনসেশন

| লক্ষণ               | সমাধান                                                                     |
| ------------------- | -------------------------------------------------------------------------- |
| ডোমের ভিতর কুয়াশা  | ডেসিক্যান্ট প্যাক (বার্ষিক প্রতিস্থাপন); ভেন্ট প্লাগ খোলা যাচাই (Gore-Tex) |
| কেবল এন্ট্রিতে পানি | সিলিকন + গ্রোমেট; ড্রিপ লুপ; কন্ডুইট বডি                                   |
| RJ45 জারা           | কন্টাক্টে ডাইলেকট্রিক গ্রিজ; কানেক্টর প্রতিস্থাপন; শিল্ডেড কেবল            |

### 10.2 পোকামাকড় / মাকড়সা / পাখি

| প্রতিরোধক                                | কার্যকারিতা                      |
| ---------------------------------------- | -------------------------------- |
| পেপারমিন্ট তেল (ক্যামেরার কাছে তুলার বল) | উচ্চ (মাকড়সা ঘৃণা করে)          |
| আল্ট্রাসোনিক পেস্ট রিপেলার               | মাঝারি                           |
| ব্র্যাকেটে বার্ড স্পাইক                  | উচ্চ (পাখি)                      |
| IR রিং এর উপর ফ্লিস কভার                 | উচ্চ (মাকড়সা গরম IR এ জাল বোনে) |
| নিয়মিত পরিষ্কার (মাসিক)                 | অপরিহার্য                        |

### 10.3 ভাঙচুর / চুরি

- **Torx / সিকিউরিটি স্ক্রু** (বেশিরভাগ প্রো ক্যামেরার সাথে আসে)
- **কন্ডুইট** ক্যামেরা পর্যন্ত (সহজে কেবল কাটা যায় না)
- **উচ্চতা** >10 ফুট (মই প্রয়োজন)
- **ব্যাকআপ ক্যামেরা** ক্যামেরাটি পর্যবেক্ষণ করছে (ওভারল্যাপ)
- **টেম্পার অ্যালার্ট** (NVR: Camera → Tamper Detection → Alarm)
