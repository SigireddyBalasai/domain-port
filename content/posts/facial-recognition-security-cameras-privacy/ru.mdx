Распознавание лиц в камерах безопасности перешло из «роскошного облачного API» в «стандарт на устройстве». Но не все функции лиц равны: обнаружение ≠ распознавание ≠ идентификация. А законы о приватности (BIPA, CCPA, GDPR, штатные биометрические акты) имеют зубы. Это руководство разбирает, что возможно, что законно, и какие камеры делают это правильно.

<Badge variant="outline">Проверка терминологии</Badge>
- **Обнаружение лица:** «В этом прямоугольнике есть лицо» — никакой личности
- **Распознавание лица:** «Это лицо совпадает с известным лицом в *моей* базе» — 1:1 или 1:few
- **Идентификация лица:** «Это лицо = Джон Доу в *глобальной* базе» — 1:many (масштаб полиции)

Потребительские камеры делают **обнаружение + распознавание (только локальная база)**. Идентификация требует облака + ордеров.

## Топовые камеры с локальным распознаванием лиц (2026)

### 1. EufyCam 3 S330 + HomeBase 3 — Лучший потребительский универсал
**NPU:** HomeBase 3 (двухъядерный NPU, 2 TOPS)  
**База данных:** 100 лиц на HomeBase  
**Распознавание:** 1:1 (заенролленные) + 1:few (семья)  
**Приватность:** **Полностью локально** — никакого облака для распознавания; опциональный облачный бэкап *зашифрован*  
**Функции:** Лицо + человек + транспорт + питомец + посылка; Зоны активности; Apple HomeKit Secure Video  
**Компромиссы:** Проприетарная экосистема; HomeBase должно оставаться включенным; 4K@15fps макс  
**Цена:** ~$550 (2-кам + HomeBase)  
**Лучше для:** Домовладельцев, желающих нулевое-облако распознавание лиц + HomeKit

### 2. Reolink Argus 4 Pro / RLC-1212A (FW 3.0+) — Лучшая интеграция с локальным NVR
**NPU:** В камере (Ambarella CV25)  
**База данных:** 100 лиц на камеру (синхронизируется через NVR)  
**Распознавание:** 1:1 локально; NVR агрегирует между камерами  
**Приватность:** Только локально; NVR хранит эмбеддинги, не изображения (настраиваемо)  
**Функции:** Человек/транспорт/питомец; Пользовательские зоны обнаружения; ONVIF Profile S/G/T  
**Компромиссы:** UX приложения менее полированный; NVR обязателен для мульти-камерного синка лиц; нет HomeKit  
**Цена:** ~$130–230/кам + NVR  
**Лучше для:** DIY-щиков с Reolink NVR, смешанных PoE/беспроводных флотов

### 3. Ubiquiti UniFi Protect (G4/G5 + AI Port) — Лучший проsumer/малый бизнес
**NPU:** Камера (G4/G5) или AI Port (добавляет к любой ONVIF камере)  
**База данных:** 200 лиц на Protect консоль  
**Распознавание:** 1:1 локально; Классификация «Известное/Неизвестное»; Поиск по фото лица  
**Приватность:** Полностью локально; Protect консоль on-prem; никакого облачного аккаунта для распознавания лиц  
**Функции:** Человек/транспорт/животное; Распознавание номеров (LPR); Подсчет посещаемости; MQTT/вебхуки  
**Компромиссы:** Лок-ин в экосистему UniFi; AI Port = $199 доп.; Нужен консольное железо  
**Цена:** G4 Bullet $230, G5 Bullet $330, AI Port $199, Cloud Key Gen2+ $300  
**Лучше для:** UniFi сетевых магазинов, мульти-сайт, IT-комфортных владельцев

### 4. Rhombus R2/R360 + Console — Лучший энтерпрайз (Нет облачной зависимости)
**NPU:** Камера (Ambarella CV52) + Console (Intel)  
**База данных:** 10,000+ лиц на консоль  
**Распознавание:** 1:1 локально; Списки наблюдения; Таймстемпы «First seen»; API/SIEM экспорт  
**Приватность:** **Нулевое облако для распознавания**; SOC 2 Type II; NDAA compliant; Только on-prem  
**Функции:** Обнаружение tailgating; Детекция масок; Загрузка; LPR; Аудио аналитика  
**Компромиссы:** Подписочная модель железа ($10–20/кам/мес включает гарантию); Нет потребительского приложения  
**Цена:** Железо $200–500/кам + подписка  
**Лучше для:** Школы, медицина, ритейл-цепочки, гос — приоритет соответствию

### 5. Ring Stick Up Cam Pro / Floodlight Cam Pro — Лучшая интеграция с экосистемой
**NPU:** Камера (Amazon AZ2)  
**База данных:** 100 лиц на аккаунт (синхронизируется между Ring камерами)  
**Распознавание:** 1:1 локальный вывод; облачная синхронизация эмбеддингов (зашифрованная)  
**Приватность:** **Облачная синхронизация по умолчанию** — отключение в Control Center; Ring Protect обязателен ($4–20/мес) для записи  
**Функции:** 3D Обнаружение движения (радар); Bird's Eye View; Alexa объявления («Джон у входной двери»)  
**Компромиссы:** Подписка обязательна для записей; Облачная синхронизация лиц = шире поверхность атаки; Нет ONVIF  
**Цена:** $180–250/кам + Protect план  
**Лучше для:** Alexa дома, Ring экосистема, пользователи принимающие облачный компромисс

### 6. Axis Q1656 / Q3536-LVE + ACAP — Лучшая открытая платформа
**NPU:** ARTPEC-8 (Axis кастом)  
**База данных:** Через ACAP приложение (третьи: Cognitec, Irisity, или кастом)  
**Распознавание:** Зависит от приложения; 1:1, 1:N, списки наблюдения  
**Приватность:** Полностью локально; Axis Edge Vault (TPM); Подписанная прошивка; NDAA compliant  
**Функции:** Lightfinder 2.0; Forensic WDR; Zipstream; Открытый VAPIX API  
**Компромиссы:** Требует ACAP разработки/интегратора; Дорого; Нет «потребительского» приложения  
**Цена:** $800–1,500/кам + ACAP лицензии  
**Лучше для:** Критической инфраструктуры, интеграторов, кастомной аналитики

### Международные
- **GDPR (EU/UK):** Биометрия = особые категории данных → Art. 9 явное согласие или существенный публичный интерес; требуется DPIA; DPO при большом масштабе
- **PIPEDA (Канада):** Согласие + разумные цели; Биометрия = чувствительные
- **LGPD (Бразилия):** Аналогично GDPR

## Точность и предвзятость: Что означают цифры

**NIST FRVT (Face Recognition Vendor Test) — 2024/2025 результаты:**
- Топ коммерческие SDK (Cognitec, NEC, Rank One, Paravision, Idemia): **FNIR <0.5% при FPIR 0.1%** (1:1 верификация)
- **Демографические дифференциалы сохраняются:** Ложные не-совпадения 2–10× выше для темнокожих женщин vs светлокожих мужчин у некоторых алгоритмов
- **Производительность с маской:** 2024+ алгоритмы восстанавливают до <2% FNIR с хирургическими масками; N95 все еще 5–15%
- **Старение:** 5-летний разрыв → 3–8% рост FNIR; пере-енролл ежегодно для высокой безопасности

**Реальность потребительских камер:**
- Eufy/Reolink/UniFi используют встроенные MobileFaceNet/ArcFace варианты — **не NIST-топ**
- Ожидайте **5–15% FNIR** (пропуск известных лиц) при разумных порогах
- **Ложные срабатывания (незнакомец = известный):** 1–3% при 100-лицевой БД; растет с размером БД
- **Смягчение:** Установите порог консервативно (алерт на «неизвестный», не на «известный»); Используйте «известное лицо» как *дополнительный* сигнал, не единственный триггер

## Спафинг и живость: Не верьте лицу для доступа

| Атака | Сложность | Защита |
|--------|------------|---------|
| Распечатанное фото | Тривиальная | Анализ текстуры, IR глубина, челлендж моргания |
| Телефон/видео реплей | Легко | Моаре-детекция, глубина, челленж-ответ |
| 3D маска | Сложно | Структурированный свет / ToF / тепловизор + AI |
| Deepfake инъекция | Средне (цифровая) | Безопасный элемент, подписанный видео-пайплайн (Axis Edge Vault) |

**Правило:** Распознавание лиц = **уведомление/идентификация**. Лицо + PIN / Лицо + Карта / Лицо + QR = **контроль доступа**. Никогда однфакторное лицо для дверей/ворот.
