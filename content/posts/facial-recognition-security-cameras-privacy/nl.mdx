Gezichtsherkenning in beveiligingscamera's is van "cloud API luxe" naar "on-device standaard" gegaan. Maar niet alle gezichtsfuncties zijn gelijk: detectie ≠ herkenning ≠ identificatie. En privacywetten (BIPA, AVG, CCPA, biometrische wetten per staat) hebben tanden. Deze gids legt uit wat mogelijk is, wat legaal is en welke camera's het goed doen.

<Badge variant="outline">Terminologie Check</Badge>- **Gezichtsdetectie:** "Er
is een gezicht in dit beeld" — geen identiteit - **Gezichtsherkenning:** "Dit
gezicht komt overeen met een bekend gezicht in *mijn* database" — 1:1 of
1:weinig - **Gezichtsidentificatie:** "Dit gezicht = Jan Jansen in *globale*
database" — 1:veel (politie schaal)

Consumentencamera's doen **detectie + herkenning (alleen lokale database)**. Identificatie vereist cloud + huiszoekingsbevelen.

## Top Camera's met On-Device Gezichtsherkenning (2026)

### 1. EufyCam 3 S330 + HomeBase 3 — Beste Consumenten Allrounder

**NPU:** HomeBase 3 (dual-core NPU, 2 TOPS)
**Database:** 100 gezichten per HomeBase
**Herkenning:** 1:1 (ingeschreven) + 1:weinig (familie)
**Privacy:** **Volledig lokaal** — geen cloud voor herkenning; optionele cloud back-up _versleuteld_
**Functies:** Gezicht + persoon + voertuig + huisdier + pakket; Activiteitszones; Apple HomeKit Secure Video
**Nadelen:** Propriëtair ecosysteem; HomeBase moet aan blijven; 4K@15fps max
**Prijs:** ~€550 (2-cam + HomeBase)
**Beste Voor:** Huiseigenaren die nul-cloud gezichtsherkenning + HomeKit willen

### 2. Reolink Argus 4 Pro / RLC-1212A (FW 3.0+) — Beste Lokale NVR Integratie

**NPU:** Camera-zijde (Ambarella CV25)
**Database:** 100 gezichten per camera (gesynchroniseerd via NVR)
**Herkenning:** 1:1 lokaal; NVR aggregeert over cams
**Privacy:** Alleen lokaal; NVR slaat embeddings op, geen afbeeldingen (configureerbaar)
**Functies:** Persoon/voertuig/huisdier; Aangepaste detectiezones; ONVIF Profile S/G/T
**Nadelen:** App UX minder gepolijst; NVR nodig voor multi-cam gezichtssync; geen HomeKit
**Prijs:** ~€130–230/cam + NVR
**Beste Voor:** DIY-ers met Reolink NVR, gemengde PoE/draadloze vloten

### 3. Ubiquiti UniFi Protect (G4/G5 + AI Port) — Beste Prosumer/Klein Bedrijf

**NPU:** Camera (G4/G5) of AI Port (voegt toe aan elke ONVIF cam)
**Database:** 200 gezichten per Protect console
**Herkenning:** 1:1 lokaal; "Bekend/Onbekend" classificatie; Zoeken op gezichtsminiatuur
**Privacy:** Volledig lokaal; Protect console on-prem; geen cloud account nodig voor gezichtsherkenning
**Functies:** Persoon/voertuig/dier; Kenteken (LPR); Bezettingsgraad; MQTT/webhooks
**Nadelen:** UniFi ecosysteem lock-in; AI Port = €199 extra; Console hardware nodig
**Prijs:** G4 Bullet €230, G5 Bullet €330, AI Port €199, Cloud Key Gen2+ €300
**Beste Voor:** Ubiquiti netwerkgebruikers, multi-site, IT-vaardige eigenaren

### 4. Rhombus R2/R360 + Console — Beste Enterprise (Geen Cloud Afhankelijkheid)

**NPU:** Camera (Ambarella CV52) + Console (Intel)
**Database:** 10.000+ gezichten per console
**Herkenning:** 1:1 lokaal; Watchlists; "Eerst gezien" tijdstempels; API/SIEM export
**Privacy:** **Nul cloud voor herkenning**; SOC 2 Type II; NDAA conform; Alleen on-prem
**Functies:** Tailgating detectie; Maskerdetectie; Bezetting; LPR; Audio-analyse
**Nadelen:** Subscription hardware model ($10–20/cam/maand incl. garantie); Geen consumenten-app
**Prijs:** Hardware $200–500/cam + abonnement
**Beste Voor:** Scholen, zorg, winkelketens, overheid — compliance-first

### 5. Ring Stick Up Cam Pro / Floodlight Cam Pro — Beste Ecosysteem Integratie

**NPU:** Camera (Amazon AZ2)
**Database:** 100 gezichten per account (gesynchroniseerd over Ring cams)
**Herkenning:** 1:1 lokale inferentie; cloud sync van embeddings (versleuteld)
**Privacy:** **Cloud-sync standaard** — opt-out in Control Center; Ring Protect vereist ($4–20/maand) voor opname
**Functies:** 3D Bewegingsdetectie (radar); Bird's Eye View; Alexa aankondigingen ("Jan bij voordeur")
**Nadelen:** Abonnement verplicht voor opnames; Cloud gezichtssync = breder aanvalsoppervlak; Geen ONVIF
**Prijs:** $180–250/cam + Protect plan
**Beste Voor:** Alexa-huizen, Ring ecosysteem, gebruikers die cloud trade-off accepteren

### 6. Axis Q1656 / Q3536-LVE + ACAP — Beste Open Platform

**NPU:** ARTPEC-8 (Axis custom)
**Database:** Via ACAP app (third-party: Cognitec, Irisity, of custom)
**Herkenning:** App-afhankelijk; 1:1, 1:N, watchlists
**Privacy:** Volledig lokaal; Axis Edge Vault (TPM); Ondertekende firmware; NDAA conform
**Functies:** Lightfinder 2.0; Forensic WDR; Zipstream; Open VAPIX API
**Nadelen:** Vereist ACAP ontwikkelaar/integrator; Duur; Geen "consumenten" app
**Prijs:** $800–1.500/cam + ACAP licenties
**Beste Voor:** Kritieke infrastructuur, integratoren, aangepaste analyses

### Internationaal

- **AVG (EU/EER):** Biometrisch = bijzondere categorie persoonsgegevens → Art. 9 expliciete toestemming of substantieel publiek belang; DPIA vereist; FG bij grootschalig
- **PIPEDA (Canada):** Toestemming + redelijke doeleinden; Biometrisch = gevoelig
- **LGPD (Brazilië):** Vergelijkbaar met AVG

## Nauwkeurigheid & Vooroordelen: Wat de Cijfers Betekenen

**NIST FRVT (Face Recognition Vendor Test) — 2024/2025 Resultaten:**

- Top commerciële SDK's (Cognitec, NEC, Rank One, Paravision, Idemia): **FNIR &lt;0,5% bij FPIR 0,1%** (1:1 verificatie)
- **Demografische verschillen blijven bestaan:** Vals-niet-match percentages 2–10× hoger voor donkerdere vrouwen vs lichtere mannen bij sommige algoritmen
- **Maskerprestaties:** 2024+ algoritmen herstellen tot &lt;2% FNIR met chirurgische maskers; N95 nog 5–15%
- **Veroudering:** 5-jaar gat → 3–8% FNIR toename; jaarlijks opnieuw inschrijven voor hoge beveiliging

**Consumentencamera Realiteit:**

- Eufy/Reolink/UniFi gebruiken ingebedde MobileFaceNet/ArcFace varianten — **niet NIST-top-tier**
- Verwachte **5–15% FNIR** (gemiste bekende gezichten) bij redelijke drempels
- **Vals-positieven (vreemdeling = bekend):** 1–3% met 100-gezichten DB; stijgt met DB-grootte
- **Mitigatie:** Stel drempel conservatief in (meld op "onbekend" niet "bekend"); Gebruik "bekend gezicht" als _aanvullend_ signaal, niet als enige trigger

## Spoofing & Liveness: Vertrouw Niet Alleen op Gezicht voor Toegang

| Aanval                | Moeilijkheid         | Verdediging                                                      |
| --------------------- | -------------------- | ---------------------------------------------------------------- |
| Afgedrukte foto       | Triviaal             | Textuuranalyse, IR diepte, oogknipper uitdaging                  |
| Telefoon/video replay | Makkelijk            | Moiré detectie, diepte, challenge-response                       |
| 3D masker             | Moeilijk             | Gestructureerd licht / ToF / thermisch + AI                      |
| Deepfake injectie     | Gemiddeld (digitaal) | Beveiligd element, ondertekende video pipeline (Axis Edge Vault) |

**Regel:** Gezichtsherkenning = **melding/identificatie**. Gezicht + PIN / Gezicht + Kaart / Gezicht + QR = **toegangscontrole**. Nooit enkele factor gezicht voor deuren/poorten.
