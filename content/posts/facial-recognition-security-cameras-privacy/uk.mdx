Розпізнавання облич у камерах безпеки перейшло з "хмарної API розкоші" на "стандарт на пристрої". Але не всі функції облич однакові: виявлення ≠ розпізнавання ≠ ідентифікація. І закони про приватність (BIPA, CCPA, GDPR) мають зуби. Цей посібник розбирає, що можливо, що законно та які камери роблять це правильно.

<Badge variant="outline">Перевірка термінології</Badge>- **Виявлення обличчя:**
"У цьому кадрі є обличчя" — без ідентифікації - **Розпізнавання обличчя:** "Це
обличчя збігається з відомим у *моїй* базі" — 1:1 або 1:декілька -
**Ідентифікація обличчя:** "Це обличчя = Іван у *глобальній* базі" — 1:багато
(масштаб правоохоронних органів)

Побутові камери роблять **виявлення + розпізнавання (тільки локальна база)**. Ідентифікація потребує хмари + ордера.

## Топ камери з розпізнаванням облич на пристрої (2026)

### 1. EufyCam 3 S330 + HomeBase 3 — найкращий універсальний

**NPU:** HomeBase 3 (двоядерний NPU, 2 TOPS)
**База:** 100 облич на HomeBase
**Приватність:** **Повністю локальне** — без хмари для розпізнавання
**Ціна:** ~$550 (2 камери + HomeBase)
**Найкраще для:** Власники будинків, які хочуть нуль-хмарне розпізнавання + HomeKit

### 2. Reolink (FW 3.0+) — найкраща інтеграція з локальним NVR

**NPU:** На камері (Ambarella CV25)
**База:** 100 облич на камеру (синхронізовано через NVR)
**Приватність:** Тільки локально
**Ціна:** ~$130–230/кам + NVR
**Найкраще для:** DIY з Reolink NVR, змішані PoE/бездротові системи

### 3. Ubiquiti UniFi Protect (G4/G5 + AI Port) — найкращий просумер/малий бізнес

**NPU:** На камері або AI Port
**База:** 200 облич на консоль Protect
**Приватність:** Повністю локально
**Ціна:** G4 Bullet $230, G5 Bullet $330, AI Port $199
**Найкраще для:** Мережі Ubiquiti, багатооб'єктні, власники з IT-навичками

### 4. Rhombus R2/R360 — найкращий корпоративний (без хмари)

**NPU:** Камера (Ambarella CV52) + Console (Intel)
**База:** 10,000+ облич на консоль
**Приватність:** Нуль хмари для розпізнавання
**Ціна:** $200–500/кам + підписка
**Найкраще для:** Школи, охорона здоров'я, роздріб, уряд — відповідність перша

### 5. Ring Stick Up Cam Pro — найкраща інтеграція екосистеми

**NPU:** Камера (Amazon AZ2)
**База:** 100 облич на акаунт
**Приватність:** Синхронізація в хмарі за замовчуванням (можна відключити)
**Ціна:** $180–250/кам + Protect план
**Найкраще для:** Alexa будинки, екосистема Ring

### 6. Axis Q1656 / Q3536-LVE + ACAP — найкраща відкрита платформа

**NPU:** ARTPEC-8 (власний)
**Приватність:** Повністю локально
**Ціна:** $800–1,500/кам + ACAP ліцензії
**Найкраще для:** Критична інфраструктура, інтегратори

## Відповідність законам про приватність

<Callout type="warning">

<strong>Це не юридична консультація.</strong> Проконсультуйтеся з юристом.
Закони про біометрію мають штрафи ($1,000–5,000/порушення) та ризик колективних
позовів.

</Callout>

### Ключові закони США (2026)

| Закон         | Юрисдикція | Застосовується                           | Ключові вимоги                                          | Штрафи                              |
| ------------- | ---------- | ---------------------------------------- | ------------------------------------------------------- | ----------------------------------- |
| **BIPA**      | Іллінойс   | Приватні суб'єкти, що збирають біометрію | Письмова згода; політика зберігання; право на видалення | $1,000 недбалість / $5,000 навмисне |
| **CUBI**      | Техас      | Комерційні суб'єкти                      | Згода; графік зберігання; без розголошення              | $25,000/порушення                   |
| **CCPA/CPRA** | Каліфорнія | Бізнес >$25M або 100K споживачів         | Повідомлення; право відмови; право видалення            | $2,500–7,500/порушення              |

### Міжнародні

- **GDPR (ЄС/UK):** Біометрія = спеціальна категорія → Art. 9 явна згода
- **PIPEDA (Канада):** Згода + розумні цілі

### Контрольний список відповідності

**1. Згода та повідомлення:**

- Табличка "Використовується розпізнавання облич" на всіх входах
- Письмова згода для співробітників, орендарів
- Шлях відмови / видалення

**2. Управління даними:**

- Політика зберігання (автовидалення через 30–90 днів)
- Шифрування AES-256 у спокої + TLS 1.3 у транзиті
- Контроль доступу на основі ролей

**3. Технічний захист:**

- Локальний AI на пристрої (Eufy, Reolink, UniFi, Rhombus, Axis)
- Зберігання тільки векторів ознак (не зображень)
- Виявлення живості для контролю доступу

**Розпізнавання облич на дверях орендарів у квартирі — магніт для позовів BIPA/CCPA.** Множинні колективні позови 2023–2025. Використовуйте тільки виявлення людини + пакунка. Без бази облич для орендарів.

</Callout>

## Точність та упередженість

**Споживчі камери:**

- Eufy/Reolink/UniFi використовують вбудовані MobileFaceNet/ArcFace — **не рівень NIST**
- Очікуйте **5–15% FNIR** (пропущені відомі обличчя)
- **Хибні спрацювання (незнайома людина = відома):** 1–3% з 100-обличчям БД

## Захист від підробки

| Атака                | Складність | Захист                                  |
| -------------------- | ---------- | --------------------------------------- |
| Друкована фотографія | Тривіальна | Аналіз текстури, ІЧ глибина, моргання   |
| Відео/телефон        | Легка      | Детекція муару, глибина                 |
| 3D маска             | Складна    | Структуроване світло / ToF / тепловізор |

**Правило:** Розпізнавання облич = сповіщення/ідентифікація. Обличчя + PIN / Обличчя + Карта = контроль доступу. Ніколи не використовуйте однофакторне обличчя для дверей/воріт.
