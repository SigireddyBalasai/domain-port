Small businesses face security challenges that consumer cameras can't solve: liability protection, employee safety, inventory shrinkage, multi-location management, and compliance with workplace surveillance laws. A $200 wireless kit won't cut it when you need 30-day retention, audit trails, and centralized management across two warehouses and a storefront.

This guide covers systems designed for commercial use—PoE NVR kits, cloud-managed platforms, and hybrid solutions that scale from 4 to 64 cameras without rewiring your network.

<Badge variant="outline">Compliance Note</Badge> Workplace surveillance laws
vary by state. See our [Privacy Laws by State for
CCTV](/blog/privacy-laws-by-state-cctv-guide) before installing cameras in break
rooms, restrooms, or areas with reasonable expectation of privacy.

## Commercial vs Consumer: What Changes

<Callout type="note">

<strong>Key differentiators for business use:</strong> 24/7 continuous recording
(not motion-only), NVR/local storage redundancy, multi-user access with role
permissions, ONVIF compliance for mixed-camera fleets, and warranty/SLA backing.

</Callout>

| Requirement | Consumer Approach            | Commercial Approach                              |
| ----------- | ---------------------------- | ------------------------------------------------ |
| Recording   | Motion-triggered clips       | Continuous + motion events                       |
| Storage     | Cloud subscription / microSD | NVR (2–16 TB) + RAID + cloud backup              |
| Access      | Single app account           | Role-based: Owner, Manager, Security, View-only  |
| Camera Mix  | Single brand ecosystem       | ONVIF — mix Reolink, Hikvision, Dahua, Axis      |
| Power       | Battery / Wi-Fi / single PoE | PoE switch (8/16/24 port) + UPS                  |
| Warranty    | 1–2 years, mail-in           | 3–5 years, advanced replacement, on-site options |
| Compliance  | None built-in                | Audit logs, watermarked export, privacy masks    |

## System Types for Small Business

### 1. PoE NVR Kits (Best Value, 4–32 Cameras)

**How it works:** Cameras → PoE Switch → NVR → Monitor/Network. All power and data over single Cat6 run.

- **Pros:** Zero monthly fees, reliable 24/7 recording, local network only (secure), expandable
- **Cons:** Requires cable runs, on-premises hardware, remote access needs VPN/port forwarding
- **Best for:** Single location, 8–24 cameras, IT-comfortable owner

### 2. Cloud-Managed NVR (Best Remote Management, 4–64 Cameras)

**How it works:** Cameras → PoE Switch → Cloud NVR Appliance (Ubiquiti UniFi Protect, Rhombus, Verkada, Arcules) → Cloud Dashboard.

- **Pros:** Browser/app access anywhere, auto updates, AI search (people/vehicles), multi-site dashboard, no port forwarding
- **Cons:** Monthly/annual per-camera fees ($5–20/cam/mo), vendor lock-in, internet required for management
- **Best for:** Multi-location, non-technical owners, franchises, remote monitoring priority

### 3. Hybrid: Local NVR + Cloud Backup (Best of Both)

**How it works:** PoE NVR records locally 24/7; critical clips auto-sync to cloud (Wasabi, Backblaze B2, AWS S3) via NVR firmware or script.

- **Pros:** No cloud subscription for live viewing; offsite backup for liability events; control over retention
- **Cons:** Setup complexity; cloud egress costs on large exports; manual restore
- **Best for:** Businesses with IT support wanting cloud safety without per-camera fees

## Storage Planning: How Much Do You Need?

<Callout type="info">

<strong>Retention Formula:</strong> Cameras × Resolution Bitrate × Hours/Days ÷
Compression = Total TB

</Callout>

| Cameras | Resolution | Bitrate (H.265) | 14 Days | 30 Days | 90 Days |
| ------- | ---------- | --------------- | ------- | ------- | ------- |
| 8       | 4K (8MP)   | 4 Mbps          | 4.8 TB  | 10.3 TB | 31 TB   |
| 16      | 4K (8MP)   | 4 Mbps          | 9.6 TB  | 20.6 TB | 62 TB   |
| 8       | 5MP        | 3 Mbps          | 3.6 TB  | 7.7 TB  | 23 TB   |
| 16      | 5MP        | 3 Mbps          | 7.2 TB  | 15.4 TB | 46 TB   |
| 24      | 4MP        | 2.5 Mbps        | 9.0 TB  | 19.3 TB | 58 TB   |

**Practical tips:**

- Use **sub-streams** (D1/720p) for live view / mobile — saves 80% bandwidth
- Enable **motion-only recording** for low-traffic zones (stockroom at night) — cuts storage 50–70%
- **RAID 5/6** on NVR: lose 1–2 drives worth of capacity for redundancy
- **Cloud backup:** Sync only "event" clips (person/vehicle detected) — 5–10% of full stream

| Approach                    | Hardware                | Management             | Cost                             | Best For                        |
| --------------------------- | ----------------------- | ---------------------- | -------------------------------- | ------------------------------- |
| VPN + Local NVRs            | NVR per site            | Desktop client / VPN   | Hardware only                    | 2–3 sites, IT staff             |
| UniFi Site Manager          | Cloud Key/UNVR per site | UniFi cloud portal     | Hardware only                    | 3–20 sites, Ubiquiti ecosystem  |
| Rhombus / Verkada / Arcules | Cloud appliances        | Single cloud dashboard | $10–20/cam/mo                    | 5–100+ sites, no IT staff       |
| Nx Witness / DW Spectrum    | NVR per site            | Centralized server     | License per cam (~$150 one-time) | 10–500 cams, integrator-managed |

<Callout type="warning">

<strong>Don't port-forward NVRs.</strong> Use ZeroTier, Tailscale, or vendor
cloud (UniFi, Rhombus) for remote access. Exposed NVRs are a top ransomware
vector.

</Callout>
