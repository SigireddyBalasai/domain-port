Малий бізнес стикається з викликами безпеки, які побутові камери не можуть вирішити: захист від відповідальності, безпека співробітників, крадіжки інвентарю, керування кількома об'єктами та дотримання законів про нагляд на робочому місці. Бездротовий набір за $200 не підійде, коли потрібно 30 днів зберігання, аудиторські сліди та централізоване керування двома складами та магазином.

Цей посібник охоплює системи, розроблені для комерційного використання — PoE NVR набори, хмарні платформи та гібридні рішення, які масштабуються від 4 до 64 камер без перекладання мережі.

<Badge variant="outline">Примітка про відповідність</Badge> Закони про нагляд на
робочому місці відрізняються залежно від штату. Див. [Privacy Laws by State for
CCTV](/blog/privacy-laws-by-state-cctv-guide) перед встановленням камер у
кімнатах відпочинку, ванних кімнатах або зонах з обґрунтованим очікуванням
приватності.

## Комерційне vs Побутове: Що змінюється

<Callout type="note">

<strong>Ключові відмінності для бізнесу:</strong> цілодобовий безперервний запис
(не тільки по руху), NVR/локальне резервування, багатокористувацький доступ з
правами, сумісність ONVIF для змішаних систем та гарантія/SLA.

</Callout>

| Вимога         | Побутовий підхід           | Комерційний підхід                                    |
| -------------- | -------------------------- | ----------------------------------------------------- |
| Запис          | Кліпи по руху              | Безперервний + події руху                             |
| Сховище        | Хмарна підписка / microSD  | NVR (2–16 ТБ) + RAID + хмарне резервування            |
| Доступ         | Один акаунт додатку        | Рольовий: Власник, Менеджер, Охорона, Тільки перегляд |
| Змішання камер | Один бренд                 | ONVIF — мікс Reolink, Hikvision, Dahua, Axis          |
| Живлення       | Батарея / Wi-Fi / один PoE | PoE комутатор (8/16/24 портів) + UPS                  |
| Гарантія       | 1–2 роки, поштою           | 3–5 років, заміна, onsite опції                       |
| Відповідність  | Немає                      | Журнали аудиту, водяні знаки, маски приватності       |

## Типи систем для малого бізнесу

### 1. PoE NVR набори (найкраща ціна, 4–32 камери)

**Як працює:** Камери → PoE комутатор → NVR → Монітор/Мережа. Все живлення та дані через один Cat6.

- **Плюси:** Без щомісячної плати, надійний 24/7 запис, тільки локальна мережа (безпечно), розширюваність
- **Мінуси:** Потрібна прокладка кабелю, обладнання на місці, віддалений доступ потребує VPN/порт форвардинг
- **Найкраще для:** Один об'єкт, 8–24 камери, власник з IT-навичками

### 2. Хмарний NVR (найкраще віддалене керування, 4–64 камери)

**Як працює:** Камери → PoE комутатор → Хмарний NVR пристрій (Ubiquiti UniFi Protect, Rhombus, Verkada, Arcules) → Хмарна панель.

- **Плюси:** Доступ з браузера/додатку звідусіль, автооновлення, AI пошук (люди/транспорт), дашборд кількох об'єктів, без порт форвардингу
- **Мінуси:** Щомісячна/річна плата за камеру ($5–20/кам/міс), прив'язка до вендора, інтернет потрібен для керування
- **Найкраще для:** Кілька об'єктів, власники без технічних знань, франшизи

### 3. Гібрид: локальний NVR + хмарне резервування (найкраще з обох світів)

**Як працює:** PoE NVR записує локально 24/7; критичні кліпи автоматично синхронізуються в хмару (Wasabi, Backblaze B2, AWS S3) через прошивку NVR або скрипт.

- **Плюси:** Без хмарної підписки для живого перегляду; віддалене резервування для подій відповідальності; контроль над зберіганням
- **Мінуси:** Складність налаштування; витрати на вивантаження; ручне відновлення
- **Найкраще для:** Бізнес з IT-підтримкою, який хоче хмарну безпеку без щомісячної плати за камеру

## Топ-пікери за категорією

### Найкращий PoE NVR набір: Reolink RLK16-800D8 (16CH, 8 камер включено)

**Характеристики:** 16-канальний NVR, 8× 4K (8MP) PoE bullet, HDD 4 ТБ встановлено, H.265, 4K HDMI/VGA вихід
**Плюси:** Plug-and-play; виявлення людини/транспорту; безкоштовний додаток/десктоп; гарантія 2 роки; розширення до 16 камер
**Мінуси:** Тільки камери Reolink (не ONVIF для змішування); сповіщення додатку можуть затримуватися; немає RAID
**Ціна:** ~$1,100 (набір 8 камер) | **Додайте:** 8 RLC-810A bullet (~$100/шт) для заповнення 16 каналів
**Найкраще для:** Роздріб, офіси, склади, які хочуть готову 4K систему до $2K

<Callout type="tip">

<strong>Професійний крок:</strong> Купуйте NVR окремо (RLN16-410, ~$350) і
підбирайте камери à la carte. Змішайте 4K bullet для входів, 5MP dome для
інтер'єрів, PTZ для парковки.

</Callout>

### Найкращий ONVIF NVR: Amcrest NV4108E-HS (8CH) / NV4116E-HS (16CH)

**Характеристики:** 8/16 каналів, 4K декодування, 2 відсіки SATA (до 16 ТБ кожен), RAID 0/1/5/10, ONVIF Profile S/G/T, гарантія 3 роки
**Плюси:** Працює з Hikvision, Dahua, Axis, Hanwha, Reolink, Amcrest; RAID для захисту від відмови дисків; HDMI 4K вихід
**Мінуси:** Камери не включені; складніше навчання; базовий мобільний додаток
**Ціна:** $280 (8CH) / $380 (16CH) — додайте камери окремо
**Найкраще для:** Змішані бренди, IT-команди, які хочуть RAID надійність

### Найкращий хмарний: Ubiquiti UniFi Protect (G4/G5 + Cloud Key Gen2 Plus / UNVR)

**Характеристики:** 4K/5MP камери, UniFi OS консоль (Cloud Key Gen2 Plus ~$300 або UNVR ~$500), без ліцензійних платежів
**Плюси:** Нульових повторюваних витрат; відмінний інтерфейс; AI виявлення (людина/транспорт/пакунок); кілька об'єктів через UniFi Site Manager; локальне сховище
**Мінуси:** Тільки камери Ubiquiti; UNVR макс 15 ТБ; Cloud Key макс 2 ТБ; обмежена інтеграція сторонніх
**Ціна:** G4 Bullet 4K ~$230, G5 Bullet 4K ~$330, G5 Dome 4K ~$280, G5 Flex ~$180
**Найкраще для:** Мережі Ubiquiti, багатооб'єктний роздріб, технічні власники

### Найкращий корпоративний (бюджет малого бізнесу): Rhombus R2 / R360 / R400

**Характеристики:** Хмарний, 5MP–4K, edge AI, 30–365 днів хмарного зберігання, гарантія 10 років, опція LTE
**Плюси:** Найкращий AI пошук (колір одягу, тип транспорту); нульова конфігурація; автооновлення; SOC 2 / NDAA
**Мінуси:** $10–20/кам/міс підписка; контракт на 3 роки; тільки власне обладнання
**Ціна:** Обладнання $200–500/кам + підписка
**Найкраще для:** Галузі з високими вимогами до відповідності (охорона здоров'я, фінанси, держконтракти), франшизи

## Планування сховища: скільки потрібно?

<Callout type="info">

<strong>Формула зберігання:</strong> Камери × Бітрейт × Години/Дні ÷ Стиск =
Загальний ТБ

</Callout>

| Камери | Роздільна здатність | Бітрейт (H.265) | 14 днів | 30 днів | 90 днів |
| ------ | ------------------- | --------------- | ------- | ------- | ------- |
| 8      | 4K (8MP)            | 4 Мбіт/с        | 4.8 ТБ  | 10.3 ТБ | 31 ТБ   |
| 16     | 4K (8MP)            | 4 Мбіт/с        | 9.6 ТБ  | 20.6 ТБ | 62 ТБ   |
| 8      | 5MP                 | 3 Мбіт/с        | 3.6 ТБ  | 7.7 ТБ  | 23 ТБ   |
| 16     | 5MP                 | 3 Мбіт/с        | 7.2 ТБ  | 15.4 ТБ | 46 ТБ   |
| 24     | 4MP                 | 2.5 Мбіт/с      | 9.0 ТБ  | 19.3 ТБ | 58 ТБ   |

**Практичні поради:**

- Використовуйте **суб-потоки** (D1/720p) для живого перегляду — економить 80% пропускної здатності
- Увімкніть **запис тільки по руху** для зон з низьким трафіком — скорочує сховище на 50–70%
- **RAID 5/6** на NVR: втратьте 1–2 диски для надійності
- **Хмарне резервування:** Синхронізуйте тільки кліпи "подій" — 5–10% від повного потоку

## Бюджетні рівні

**Рівень 1 — DIY PoE NVR ($1,500–4,000)**

- Amcrest NV4116E-HS (16CH) + 8× Reolink RLC-810A (4K) + 2× 10 ТБ WD Purple = ~$2,200
- Додайте ще 8 камер за бюджетом
- Ви керуєте, ви обслуговуєте, нульова щомісячна плата

**Рівень 2 — Prosumer хмарний ($3,000–8,000 + $0/міс)**

- 2× Ubiquiti UNVR + 12× G5 Bullets/Domes + 2× Cloud Key Gen2 Plus = ~$6,500
- Єдина панель, AI пошук, мобільні сповіщення, без підписок

**Рівень 3 — Повністю керований хмарний ($5,000–15,000 + $200–800/міс)**

- Rhombus R2/R360 × 16 камер + 3-річна підписка
- Нульове обслуговування, готовність до відповідності, LTE, гарантія 10 років
