PoE (پاور اوور ایتھرنیٹ) کیمرے وائرڈ سیکیورٹی کیمرے ہیں جو بیک وقت ایک ہی ایتھرنیٹ کیبل پر ڈیٹا اور بجلی منتقل کرتے ہیں۔ PoE کیمروں کا سب سے بڑا چیلنج **کیبل چلانا** ہے۔ یہ گائیڈ کیبل کو صحیح طریقے سے منتخب کرنے، چلانے اور ختم کرنے کے بارے میں بتاتی ہے - چاہے وہ اٹاری، کرال اسپیس، ایو، یا گیراج سے گزرے۔

---

## فوری جائزہ

| مرحلہ                | تفصیل                                      |
| -------------------- | ------------------------------------------ |
| کیبل کا انتخاب       | Cat6 بیرونی / پلینم                        |
| راستے کی منصوبہ بندی | رن کو پیمائش کریں، رکاوٹوں کی نشاندہی کریں |
| کیبل چلائیں          | اٹاری / ایو / کرال اسپیس                   |
| کیبل ختم کریں        | RJ45 کنیکٹر (جعلی کنیکٹرز سے بچیں)         |
| کیبل ٹیسٹ کریں       | کیبل ٹیسٹر / PoE ٹیسٹر                     |
| کیمرہ منسلک کریں     | PoE سوئچ، کیمرہ آن                         |

## کیبل کا انتخاب

| کیبل قسم | بہترین کے لیے            | 🎯 لمبائی زیادہ | 🎯 لاگت   |
| -------- | ------------------------ | --------------- | --------- |
| Cat5e    | SSD کیفے / UW            | 100m            | کم        |
| Cat6     | PoE کیمرے                | 100m            | درمیانہ   |
| Cat6a    | پاور ڈیٹا بیک بون / 10Gb | 100m            | زیادہ     |
| Cat7     | بیک بون / پیچ کیبل       | 100m            | بہت زیادہ |

**گھر کے کیمروں کے لیے بہترین انتخاب: بیرونی Siamese Cat6** — ایک یا دو کور، پانی کے خلاف مزاحمت، اور UV مزاحمت والی جیکٹ۔

---

## راستے کی منصوبہ بندی

1. **کیمرہ کی جگہیں:** مطلوبہ جگہوں پر پینٹ / چاک لگائیں۔ ہر ایک کی ضرورت ہے:
   - 100m سے کم Cat6 رن
   - PoE سوئچ یا پاور انجیکٹر تک رسائی
   - لائن آف سائٹ → ویونگ اینگل

2. **کیبل رن:** دیوار / چھت پر بچھائیں، گرمی / نمی سے بچیں، زیادہ سے زیادہ 50m انتظام کریں۔

**عام رکاوٹیں:**

- ایئر کنڈیشن ڈکٹ (گرمی، نمی)
- پانی کے پائپ (رساؤ، نمی)
- برقی لائنیں (EMI)
- کیل / پیچ (میکانکی نقصان)

3. **کیبل انٹری:** دیوار / چھت کے ذریعے ڈرل۔
   - **بیرونی دیوار:** سلیکون سیلنٹ سے سیل کریں
   - **ایو:** ویدر پروف گرومٹ، ڈرپ لوپ
   - **اٹاری:** اٹاری وینٹ کے قریب انٹری

## کیبل کی لمبائی کا حساب

Cat6 کیبل کی مناسب لمبائی کا تعین کرنے کے لیے:

1. **رن کی پیمائش:** Naچ کہ اٹاری سے کیمرہ تک
2. **سرپلس:** 10–15% اضافی
3. **کنیکٹر:** RJ45 لمبائی شامل کریں

**مثال:** اگر رن 30m ہے تو 35m + 5m RJ45 = 40m کیبل درکار۔

---

## RJ45 کو ختم کرنا: ٹرمینل B

بیلا/ٹرمینل B وائر آرڈر:

| پن  | تار کا رنگ  | تار جوڑا  |
| --- | ----------- | --------- |
| 1   | سفید/نارنجی | TX+       |
| 2   | نارنجی      | TX-       |
| 3   | سفید/سبز    | RX+       |
| 4   | نیلا        | PoE+ (V+) |
| 5   | سفید/نیلا   | PoE+ (V+) |
| 6   | سبز         | RX-       |
| 7   | سفید/براؤن  | PoE- (V-) |
| 8   | براؤن       | PoE- (V-) |

## PoE کیبل کی مرمت

اگر کوئی کیبل منقطع ہو جائے یا خراب ہو جائے:

1. **خراب حصہ کاٹیں** — نقصان والی جگہ سے 6 انچ باہر
2. **نئے سرے سے ختم کریں** — دونوں طرف RJ45 لگائیں
3. **کپلر استعمال کریں** — دو حصوں کو جوڑتا ہے
4. **ٹیسٹ کریں** — کیبل ٹیسٹر سے تصدیق کریں
5. **موسم بندی** — کپلر کو سلیکون سے سیل کریں

## PoE کے عمومی سوالات

**س: کیا Cat5e PoE کے لیے کافی ہے؟**
ج: جی ہاں، 100m تک PoE کام کرتا ہے۔ Cat6 بہتر ہے خاص طور پر 4K کیمروں کے لیے۔

**س: کیا PoE سوئچ کے بغیر PoE کیمرہ استعمال کر سکتے ہیں؟**
ج: جی ہاں — PoE انجیکٹر (انفرادی) استعمال کریں۔ سوئچ بہتر ہے اگر ایک سے زیادہ کیمرے ہوں۔

**س: کیبل کی زیادہ سے زیادہ لمبائی کتنی ہے؟**
ج: Ethernet کے لیے 100m (328 فٹ)۔ PoE کی حد بھی 100m ہے۔

**س: کیا کیبل کو زیر زمین دفن کیا جا سکتا ہے؟**
ج: جی ہاں — ڈائریکٹ بریل Cat6 کیبل استعمال کریں (UV اور پانی سے محفوظ)۔

**س: کیا PoE کیمرے بجلی کی بچت کرتے ہیں؟**
ج: جی ہاں — PoE (802.3af) صرف 15.4W فی کیمرہ استعمال کرتا ہے۔
