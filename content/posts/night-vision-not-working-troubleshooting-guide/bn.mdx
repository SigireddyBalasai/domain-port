| **ফার্মওয়্যার বাগ**             | সাম্প্রতিক আপডেট অটো-IR ভেঙেছে                      | ফার্মওয়্যার রোলব্যাক; ভেন্ডর সাপোর্ট টিকিট                                      |

### 1.3 IR লোডের নিচে ভোল্টেজ ড্রপ (গুরুত্বপূর্ণ)

**IR LED অতিরিক্ত 2–5W টানে।** ক্যামেরা দিনে চলে, রাতে ক্র্যাশ করে।

<Callout type="warning">

**পরীক্ষা:** ক্যামেরা পাওয়ার ইনপুটে মাল্টিমিটার (PoE: PoE টেস্টার ব্যবহার করুন)। দিনের ভোল্টেজ বনাম রাতের (IR চালু) ভোল্টেজ। **>0.5V (12V) বা >2V (PoE 48V) ড্রপ = কেবল খুব লম্বা / গেজ খুব ছোট / সুইচ বাজেট অতিক্রান্ত।**

</Callout>

| আর্কিটেকচার                 | সর্বোচ্চ IR লোড         | সমাধান                                                       |
| --------------------------- | ----------------------- | ------------------------------------------------------------ |
| **PoE (802.3af 15.4W)**     | ~4W IR                  | PoE+ (30W) ইনজেক্টর/সুইচে আপগ্রেড                            |
| **12V DC (18 AWG, 50 ফুট)** | 2A IR                   | রান ছোট / 14 AWG / লোকাল 12V সাপ্লাই                         |
| **ব্যাটারি ক্যাম**          | IR ঘন্টায় ব্যাটারি শেষ | IR নিষ্ক্রিয়; পরিবেষ্টিত আলো / বাহ্যিক IR ইলুমিনেটর ব্যবহার |

### 2.1 হোয়াইটআউট সমাধানের ধাপ

1. **রিফ্লেক্টর শনাক্ত:** রাতে বাইরে যান, পৃষ্ঠে IR এর আভা দেখুন। ফোন ক্যামেরা দিয়ে IR স্পিল দেখুন।
2. **পুনরায় কোণ:** অনুভূমিক থেকে **15–30° নিচে কাত** করুন। লেভেল + প্রটেক্টর ব্যবহার করুন।
3. **দূরত্ব যোগ:** এক্সটেনশন ব্র্যাকেট (12–24") IR শঙ্কুকে কাছের বস্তু অতিক্রম করতে সাহায্য করে।
4. **বাহ্যিক IR ইলুমিনেটর:** 10–20 ফুট দূরে মাউন্ট, টার্গেট জোনের দিকে। ক্যামেরা IR **বন্ধ**। লাইসেন্স প্লেট, গেটের জন্য সেরা।
5. **স্মার্ট IR (যদি সমর্থিত):** ক্যামেরা অটোমেটিক কাছের বস্তুর জন্য IR তীব্রতা কমায়। সেটিংসে সক্রিয় করুন।
6. **IR কাট ফিল্টার চেক:** "আউট" এ আটকে (নাইট মোড) = দিনে রঙিন ছবি। "ইন" এ আটকে (ডে মোড) = IR নেই। পাওয়ার সাইকেল।

## 3. কালো / নয়জি / দানাদার রাতের ছবি

**লক্ষণ:** অন্ধকার, ছিটছিটে, "তুষারময়" ভিডিও। কোনো বিবরণ নেই।

| কারণ                                  | রোগ নির্ণয়                     | সমাধান                                                              |
| ------------------------------------- | ------------------------------- | ------------------------------------------------------------------- |
| **দূরত্বের জন্য IR খুব দুর্বল**       | টার্গেট > রেটেড IR রেঞ্জের 50%  | বাহ্যিক IR ইলুমিনেটর; ক্যামেরা কাছাকাছি সরান                        |
| **শাটার খুব ধীর (মোশন ব্লার)**        | ভুতুড়ে ট্রেইল, উজ্জ্বল দাগ     | **ন্যূনতম শাটার: 1/100s** (প্লেটের জন্য 1/250s) সেট                 |
| **গেইন/ISO সর্বোচ্চে**                | নয়েজ = "তুষার"                 | আলো যোগ (বাহ্যিক IR, স্ট্রিটলাইট); সর্বোচ্চ গেইন কমান               |
| **ওয়াইড ডায়নামিক রেঞ্জ (WDR) বন্ধ** | উজ্জ্বল আলো ক্যামেরাকে অন্ধ করে | **WDR: 120 dB+** সক্রিয় (True WDR, DWDR নয়)                       |
| **ডোম নোংরা / আঁচড়**                 | আলোর চারপাশে হ্যালো             | ডোম প্রতিস্থাপন ($15); মাইক্রোফাইবার + লেন্স ক্লিনার দিয়ে পরিষ্কার |
| **সেন্সর ডিগ্রেডিং**                  | বছরের পর বছর ধরে ক্রমশ খারাপ    | ক্যামেরা প্রতিস্থাপন (CMOS বয়স হয়, হট পিক্সেল বাড়ে)              |
| **কম্প্রেশন খুব বেশি**                | রাতে ব্লকি আর্টিফ্যাক্ট         | বিটরেট: CBR 4–6 Mbps (4K) / 2–3 Mbps (2K)                           |

## 4. কালার নাইট ভিশন ব্যর্থতা

**কালার নাইট ভিশন (CNV)** = বড় সেন্সর (1/1.8", 1/1.2") + লো-লাইট ISP + **পরিবেষ্টিত আলো প্রয়োজন**।

| ব্যর্থতা মোড                  | কারণ                        | সমাধান                                                                                                         |
| ----------------------------- | --------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **B&W তে থাকে (IR চালু)**     | পরিবেষ্টিত &lt;1 lux        | CNV এর ~3–5 lux প্রয়োজন (স্ট্রিটলাইট, বারান্দার আলো)। **উষ্ণ সাদা (3000K) LED** 10–20 ফুট দূরে যোগ করুন।      |
| **রঙিন নয়জি/দানাদার**        | গেইন খুব বেশি               | সর্বোচ্চ গেইন কমান; আলো যোগ; B&W IR গ্রহণ                                                                      |
| **রঙ পরিবর্তন (বেগুনি/সবুজ)** | হোয়াইট ব্যালেন্স বিভ্রান্ত | WB "Outdoor" / "Streetlight" / কাস্টম 3000K সেট                                                                |
| **IR + CNV দ্বন্দ্ব**         | উভয় চালু = অদ্ভুত রঙ       | CNV সক্রিয় থাকলে IR নিষ্ক্রিয় (ভালো ক্যামেরায় অটো)                                                          |
| **সমর্থিত নয়**               | ক্যামেরায় CNV সেন্সর নেই   | স্পেস চেক: 1/1.8"+ সেন্সর প্রয়োজন (Reolink ColorX, Hikvision ColorVu, Dahua Full-Color, Axis Lightfinder 2.0) |

## 5. লাইসেন্স প্লেট ক্যাপচার (LPC) — বিশেষায়িত রাতের সমাধান

**LPC রাতে 90% ব্যর্থ হয় কারণ:** প্লেট থেকে IR প্রতিফলন + ধীর শাটার + দুর্বল IR।

### 5.1 LPC রাতের চেকলিস্ট

- [ ] **ক্যামেরা:** 4K/8MP + অপটিক্যাল জুম (ভ্যারিফোকাল 2.8–12mm বা 5–50mm) বা ডেডিকেটেড LPR (2MP, 100+ fps, গ্লোবাল শাটার)
- [ ] **শাটার:** **ন্যূনতম 1/250s** (ম্যানুয়াল)
- [ ] **IR:** বাহ্যিক ইলুমিনেটর (850 nm, 30–50° বিম) **ক্যামেরার বাইরে**, প্লেট জোনে লক্ষ্য
- [ ] **কোণ:** লম্ব থেকে ≤30°; উচ্চতা 3–4 ফুট (প্লেট লেভেল)
- [ ] **WDR:** 120 dB+ True WDR (হেডলাইট হ্যান্ডেল)
- [ ] **IR কাট:** স্যুইচিং যাচাই (দিনে রঙ / রাতে B&W)
- [ ] **জোন:** মোশন জোন = শুধু প্লেট পাথ (ভুল ট্রিগার কমান)
- [ ] **রেকর্ড:** CBR 6–8 Mbps (4K) / 4 Mbps (2MP LPR)

```
NIGHT VISION TRIAGE FLOWCHART

  1. IR GLOW? (phone cam test)
   NO → Check voltage (night) → Power fix
   YES → Step 2

  2. IMAGE WHITE/GRAY?
   YES → Angle down 15-30° / Move out 12"+ / External IR
   NO → Step 3

  3. IMAGE BLACK/NOISY?
   YES → Shutter 1/100s min / WDR On / External IR
   NO → Step 4

  4. COLOR NIGHT VISION OFF?
   YES → Add 3000K light / Check sensor size
   NO → Step 5

  5. PLATE UNREADABLE?
   YES → Shutter 1/250s / External IR / Varifocal zoom / LPR cam
   NO → DONE ✓

VOLTAGE CHECK (night, IR on):
PoE: >48V (af) / >50V (at) / >52V (bt)
  12V: >11.5V at camera
```
