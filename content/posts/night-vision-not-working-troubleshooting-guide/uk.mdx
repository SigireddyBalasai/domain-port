Несправності нічного бачення — це найпоширеніша скарга "камера працює вдень, але марна вночі". Цей посібник охоплює всі режими відмов з діагностикою, виправленнями та коли варто замінити камеру.

<Badge variant="outline">Швидкий тріаж</Badge>
1. **IR-світлодіоди горять?** (червоне світіння вночі) → Ні = Живлення/плата /
Так = Оптика/Налаштування 2. **Зображення біле/сіре?** → Відбиття ІЧ
(кут/відстань) 3. **Зображення чорне/шумне?** → ІЧ занадто слабке / витримка
занадто довга / сенсор виходить з ладу 4. **Кольорове нічне бачення вимкнене?**
→ Недостатньо зовнішнього світла / налаштування вимкнено

## 1. ІЧ-світлодіоди не світяться (немає червоного світіння)

### 1.1 Перевірка статусу ІЧ

<Accordion type="single" collapsible>
  <AccordionItem value="check-ir">
    <AccordionTrigger>Як перевірити ІЧ-світлодіоди</AccordionTrigger>
    <AccordionContent>

      **Візуально:** Станьте перед камерою вночі → шукайте **слабке червоне світіння** (850 нм) або **невидиме світіння** (940 нм). **Тест камерою телефона:** Спрямуйте камеру телефона на камеру — ІЧ з'явиться як **яскраве фіолетове/біле** світло (сенсори телефона бачать ближній ІЧ). **Статус у додатку:** Веб-інтерфейс / NVR → "Статус ІЧ" / "Нічний режим" → має показувати "Увімкнено" або "Авто".

    </AccordionContent>

  </AccordionItem>
</Accordion>

### 1.2 Причини та виправлення

| Причина                                  | Діагностика                                                    | Виправлення                                                                                |
| ---------------------------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| **IR Cut Filter застряг (денний режим)** | Зображення кольорове вночі, немає ІЧ світіння                  | Перезавантажте живлення (12V/PoE вимкніть на 30 с). Обережно постукайте. Оновіть прошивку. |
| **Фоторезистор (LDR) вийшов з ладу**     | ІЧ ніколи не вмикається автоматично; ручне "Увімк" працює      | Накрийте LDR (маленьке віконце) → якщо ІЧ спрацьовує, замініть LDR ($2) або камеру         |
| **Плата драйвера ІЧ вийшла з ладу**      | 12V/PoE присутні, світлодіоди холодні, ручне "Увімк" не працює | Замініть камеру (плата не підлягає ремонту)                                                |
| **Недостатнє живлення для ІЧ**           | Вдень OK, вночі перезавантажується / ІЧ блимає                 | Перевірте напругу на камері під навантаженням ІЧ (див. §1.3)                               |
| **Помилка прошивки**                     | Останнє оновлення зламало авто-ІЧ                              | Відкотити прошивку; звернутися до підтримки                                                |
| **Невидимий ІЧ (940 нм)**                | Немає червоного світіння, телефон показує фіолетове            | Нормально — 940 нм невидимий для ока                                                       |

### 1.3 Падіння напруги під навантаженням ІЧ (критично)

**ІЧ-світлодіоди споживають додатково 2–5 Вт.** Камера працює вдень, вимикається вночі.

<Callout type="warning">

**Тест:** Мультиметр на вході живлення камери (PoE: використовуйте PoE тестер). Напруга вдень проти ночі (ІЧ увімкнено). **Падіння >0.5В (12V) або >2В (PoE 48V) = кабель занадто довгий / переріз занадто малий / комутатор недостатньо потужний.**

</Callout>

| Архітектура                   | Макс. навантаження ІЧ       | Виправлення                                                          |
| ----------------------------- | --------------------------- | -------------------------------------------------------------------- |
| **PoE (802.3af 15.4W)**       | ~4W ІЧ                      | Оновіть до PoE+ (30W) інжектора/комутатора                           |
| **12V DC (18 AWG, 50 футів)** | 2A ІЧ                       | Скоротіть кабель / 14 AWG / локальне джерело 12V                     |
| **Акумуляторна камера**       | ІЧ вбиває батарею за години | Вимкніть ІЧ; використовуйте зовнішнє світло / зовнішній ІЧ-прожектор |

## 2. ІЧ засвічення / розмиття / сіре зображення

**Симптом:** Нічне зображення = яскрава біла/сіра пляма, без деталей. ІЧ відбивається від близької поверхні.

<Accordion type="single" collapsible>
  <AccordionItem value="whiteout-causes">
    <AccordionTrigger>Поширені джерела відбиття</AccordionTrigger>
    <AccordionContent>

      | Поверхня | Відстань | Виправлення |
      |----------|----------|-------------|
      | **Стіна / Сайдинг / Фасад** | 6–24" | Нахиліть камеру **вниз 15–30°**; відсуньте на 12–18" (кронштейн-подовжувач) |
      | **Карниз / Підшивка** | 0–12" | Монтуйте **під** підшивкою, не врівень з нею |
      | **Вікно / Скло** | Будь-яка | **Ніколи** не спрямовуйте через скло (ІЧ відбивається 100%) |
      | **Павутина / Пил на куполі** | На об'єктиві | Чистіть щомісяця; накрийте ІЧ кільце чохлом |
      | **Дощ / Сніг / Туман** | У повітрі | Зовнішній ІЧ-прожектор (встановлений окремо від об'єктива) |
      | **Білий молдинг / Риштування** | 12–36" | Пофарбуйте матовою чорною фарбою або закрийте стрічкою |
      | **Номерний знак (світловідбивний)** | 20–50 футів | Нахиліть вниз; зовнішній ІЧ; LPR камера з поляризатором |

    </AccordionContent>

  </AccordionItem>
</Accordion>

### 2.1 Покрокове виправлення засвічення

1. **Визначте відбивач:** Вийдіть вночі, спостерігайте світіння ІЧ на поверхнях. Використовуйте камеру телефона, щоб побачити розсіювання ІЧ.
2. **Змініть кут:** Нахиліть **вниз 15–30°** від горизонталі. Використовуйте рівень + транспортир.
3. **Додайте відстань:** Кронштейн-подовжувач (12–24") переміщує конус ІЧ за близькі об'єкти.
4. **Зовнішній ІЧ-прожектор:** Розташуйте на 10–20 футів від камери, спрямований на цільову зону. ІЧ камери **ВИМК**. Найкраще для номерних знаків, воріт.
5. **Smart IR (якщо підтримується):** Камера автоматично зменшує інтенсивність ІЧ для близьких об'єктів. Увімкніть у налаштуваннях.
6. **Перевірка IR Cut Filter:** Застряг "ззовні" (нічний режим) = кольорове денне зображення. Застряг "всередині" (денний режим) = немає ІЧ. Перезавантажте живлення.

## 3. Чорне / шумне / зернисте нічне зображення

**Симптом:** Темне, плямисте, "засніжене" відео. Без деталей.

| Причина                                 | Діагностика                            | Виправлення                                                                 |
| --------------------------------------- | -------------------------------------- | --------------------------------------------------------------------------- |
| **ІЧ занадто слабке для відстані**      | Ціль >50% від номінальної дальності ІЧ | Зовнішній ІЧ-прожектор; перемістіть камеру ближче                           |
| **Витримка занадто довга (змазування)** | Примарні сліди, яскраві смуги          | Встановіть **Мін. витримку: 1/100с** (1/250с для номерів)                   |
| **Посилення/ISO на максимумі**          | Шум = "сніг"                           | Додайте світло (зовнішній ІЧ, вуличне освітлення); зменшіть макс. посилення |
| **WDR вимкнено**                        | Яскраве світло засліплює камеру        | Увімкніть **WDR: 120 дБ+** (True WDR, не DWDR)                              |
| **Купол брудний / подряпаний**          | Ореол навколо ліхтарів                 | Замініть купол ($15); протріть мікрофіброю + засобом для лінз               |
| **Сенсор деградує**                     | Прогресивне погіршення за роки         | Замініть камеру (CMOS старіє, з'являються гарячі пікселі)                   |
| **Занадто високий стиск**               | Блокові артефакти вночі                | Бітрейт: CBR 4–6 Мбіт/с (4K) / 2–3 Мбіт/с (2K)                              |

### 3.1 Шпаргалка з витримки

| Ціль               | Мін. витримка       | Примітки                           |
| ------------------ | ------------------- | ---------------------------------- |
| Загальна територія | 1/50с (1/60с США)   | Баланс світла та змазування        |
| Людина, що йде     | 1/100с              | Чіткі кінцівки                     |
| **Номерний знак**  | **1/250с – 1/500с** | **Критично** — встановлюйте вручну |
| Швидкий автомобіль | 1/500с – 1/1000с    | Потрібен зовнішній ІЧ              |

<Callout type="note">

**Як налаштувати:** Веб-інтерфейс камери → Зображення → Експозиція → Витримка → Вручну → Введіть значення. Вимкніть "Авто" витримку.

</Callout>

## 4. Збої кольорового нічного бачення

**Кольорове нічне бачення (КНБ)** = Великий сенсор (1/1.8", 1/1.2") + ISP для слабкого освітлення + **потрібне зовнішнє світло**.

| Режим відмови                          | Причина                          | Виправлення                                                                                                                 |
| -------------------------------------- | -------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **Залишається Ч/Б (ІЧ увімкнено)**     | Освітлення <1 люкс               | КНБ потребує ~3–5 люкс (вуличне світло, світло на ганку). Додайте **теплий білий (3000K) світлодіод** на 10–20 футів.       |
| **Кольорове шумне/зернисте**           | Посилення занадто високе         | Зменшіть макс. посилення; додайте світло; прийміть Ч/Б ІЧ                                                                   |
| **Зсув кольорів (фіолетовий/зелений)** | Баланс білого збитий             | Встановіть баланс білого "Вулиця" / "Вуличне світло" / Користувацький 3000K                                                 |
| **Конфлікт ІЧ + КНБ**                  | Обидва ввімкнені = дивні кольори | Вимкніть ІЧ, коли КНБ активне (авто в хороших камерах)                                                                      |
| **Не підтримується**                   | Камера не має сенсора КНБ        | Перевірте специфікації: потрібен сенсор 1/1.8"+ (Reolink ColorX, Hikvision ColorVu, Dahua Full-Color, Axis Lightfinder 2.0) |

## 5. Захоплення номерних знаків — нічні виправлення

**Захоплення номерів вночі не працює у 90% через:** відбиття ІЧ від знаку + повільна витримка + слабкий ІЧ.

### 5.1 Контрольний список для нічного захоплення номерів

- [ ] **Камера:** 4K/8MP + оптичний зум (варіфокальний 2.8–12 мм або 5–50 мм) АБО спеціалізована LPR (2MP, 100+ fps, глобальний затвор)
- [ ] **Витримка:** **1/250с мінімум** (вручну)
- [ ] **ІЧ:** Зовнішній прожектор (850 нм, 30–50° промінь) **окремо від камери**, спрямований на зону номера
- [ ] **Кут:** ≤30° від перпендикуляра; висота 3–4 фути (рівень номерного знака)
- [ ] **WDR:** 120 дБ+ True WDR (справляється з фарами)
- [ ] **IR Cut:** Перевірте перемикання (вдень кольорове / вночі Ч/Б)
- [ ] **Зона руху:** Зона руху = тільки шлях номерів (зменшити хибні спрацювання)
- [ ] **Запис:** CBR 6–8 Мбіт/с (4K) / 4 Мбіт/с (2MP LPR)

## 6. Нічні проблеми дверних дзвінків

| Проблема                             | Причина                          | Виправлення                                                           |
| ------------------------------------ | -------------------------------- | --------------------------------------------------------------------- |
| **Обличчя відвідувача засвічене**    | ІЧ відбивається від ганку/навісу | Кутовий клин (15–25° вниз); вимкніть ІЧ → використовуйте світло ганку |
| **Немає нічного бачення**            | Режим економії батареї           | Провідне живлення (16–24V AC) → увімкне постійне ІЧ                   |
| **Кольорове нічне бачення вимкнене** | Недостатньо зовнішнього світла   | Додайте **світлодіод 3000K** у світильник ганку (не холодний білий)   |
| **Пропуски руху вночі**              | Дальність PIR < дальності ІЧ     | PIR ~25 футів, ІЧ ~30 футів → зменшіть чутливість ІЧ / звужте зону    |

## 7. Особливості акумуляторних / сонячних камер вночі

| Проблема                           | Причина                           | Виправлення                                                           |
| ---------------------------------- | --------------------------------- | --------------------------------------------------------------------- |
| **ІЧ вбиває батарею за 1 тиждень** | ІЧ = 2–4W безперервно             | Вимкніть ІЧ; покладайтесь на зовнішнє світло + прожектор (Eufy, Ring) |
| **Немає нічного бачення взимку**   | Li-ion <32°F перестає заряджатись | Оновіть до LiFePO₄ (Reolink Go PT Plus); ізольований бокс + обігрівач |
| **Рух працює тільки вдень**        | PIR заблокований вікном/склом     | Розташуйте **поза** склом; PIR потребує різниці температур            |
| **Прожектор не спрацьовує**        | Неправильна зона руху             | Перемалюйте зону, щоб покрити шлях підходу; пройдіться для тесту      |

### 8.2 Вимірювання потужності ІЧ

| Інструмент           | Вимірювання                 | Критерій проходження                 |
| -------------------- | --------------------------- | ------------------------------------ |
| **PoE тестер**       | Напруга на камері (вночі)   | >48V (PoE+), >52V (PoE++)            |
| **Мультиметр (12V)** | Напруга на камері (вночі)   | >11.5V (12V номінал)                 |
| **Кліщі**            | Струм на лінії 12V (вночі)  | <2A (на камеру)                      |
| **Тепловізор**       | Температура ІЧ світлодіодів | 40–60°C (працює), <30°C (несправний) |

### 8.3 Аудит прошивки та конфігурації

```

☐ Витримка: Вручну 1/100с мін (1/250с для номерів)
☐ WDR: True WDR Увімк (120 дБ+)
☐ Макс. посилення: Обмежено (не "Авто Макс")
☐ Бітрейт: CBR 4–6 Мбіт/с (4K)
☐ IR Cut: Перемикання перевірено (день/ніч)
☐ Часовий пояс/NTP: Правильний (розклад ІЧ)
```

## 9. Коли замінити проти ремонту

| Симптом                             | Можливий ремонт?                      | Замінити, коли                                             |
| ----------------------------------- | ------------------------------------- | ---------------------------------------------------------- |
| Немає ІЧ світіння, живлення OK      | Плата ІЧ ($50) / LDR ($2)             | Камера >3 років, потрапляння води                          |
| IR Cut Filter застряг               | Постукати/перезавантажити (тимчасово) | Механізм зношений; >2 років                                |
| Засвічення не виправляється кутом   | Н/Д                                   | Неправильна камера для місця (використовуйте зовнішній ІЧ) |
| Шумний сенсор (гарячі пікселі)      | Мапування пікселів (деякі камери)     | >5% мертвих пікселів; >4 років                             |
| Кольорове нічне бачення не працює   | Проблема сенсора/ISP                  | Сенсор занадто малий (<1/2.8"); оновіть                    |
| Переривчасте перезавантаження вночі | Падіння напруги                       | Виправте кабель/живлення; якщо постійно → замініть         |

## 10. Картка швидкої довідки (роздрукуйте для роботи)

```
ТРІАЖ НІЧНОГО БАЧЕННЯ — БЛОК-СХЕМА

  1. ІЧ СВІТІННЯ? (тест камерою телефона)
```
