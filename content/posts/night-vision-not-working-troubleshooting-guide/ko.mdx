## 1. IR LED가 켜지지 않음 (붉은 빛 없음)

### 1.1 IR 상태 확인

<Accordion type="single" collapsible>
  <AccordionItem value="check-ir">
    <AccordionTrigger>IR LED 확인 방법</AccordionTrigger>
    <AccordionContent>

      **육안:** 밤에 카메라 앞에 서서 **희미한 붉은 빛**(850 nm) 또는 **보이지 않는 빛**(940 nm "은밀형") 확인. **휴대폰 카메라 테스트:** 휴대폰 카메라를 카메라에 비추면 IR이 화면에서 **밝은 보라색/흰색**으로 보임 (휴대폰 센서가 근적외선을 봄). **앱 상태:** 카메라 웹 UI / NVR → "IR 상태" / "야간 모드" → "켜짐" 또는 "자동"으로 표시되어야 함.

    </AccordionContent>

  </AccordionItem>

### 1.2 원인 및 해결책

| 원인                            | 진단                                               | 수정                                                                    |
| ------------------------------- | -------------------------------------------------- | ----------------------------------------------------------------------- |
| **IR 컷 필터 고정 (주간 모드)** | 밤에도 컬러 이미지, IR 빛 없음                     | 전원 재부팅 (12V/PoE 30초 오프). 카메라를 살짝 두드림. 펌웨어 업데이트. |
| **포토레지스터/LDR 고장**       | IR이 자동으로 켜지지 않음; 수동 "켜짐"은 작동      | LDR(작은 투명 창) 가림 → IR이 켜지면 LDR 교체 ($2) 또는 카메라 교체     |
| **IR LED 드라이버 보드 고장**   | 12V/PoE 전원 있음, LED 차가움, 수동 "켜짐"도 안 됨 | 카메라 교체 (보드 수리 불가)                                            |
| **IR에 전원 부족**              | 낮엔 정상, 밤에 재부팅 / IR 깜박임                 | IR 부하 시 카메라 전압 확인 (§1.3 참조)                                 |
| **펌웨어 버그**                 | 최근 업데이트로 자동 IR 고장                       | 펌웨어 롤백; 공급업체 지원 티켓                                         |
| **은밀형 IR (940 nm)**          | 붉은 빛 없음, 휴대폰 테스트에서 보라색             | 정상 — 940 nm는 육안에 보이지 않음                                      |

### 1.3 IR 부하 시 전압 강하 (중요)

**IR LED는 추가로 2–5W를 소모합니다.** 낮에는 실행되다 밤에 꺼집니다.

<Callout type="warning">

**테스트:** 카메라 전원 입력에서 멀티미터 (PoE: PoE 테스터 사용). 주간 전압 vs 야간(IR 켜짐) 전압. **강하 0.5V(12V) 또는 2V(PoE 48V) 초과 = 케이블이 너무 김 / 게이지가 너무 작음 / 스위치 예산 초과.**

</Callout>

| 구성                       | 최대 IR 부하                  | 수정                                      |
| -------------------------- | ----------------------------- | ----------------------------------------- |
| **PoE (802.3af 15.4W)**    | ~4W IR                        | PoE+ (30W) 인젝터/스위치로 업그레이드     |
| **12V DC (18 AWG, 50 ft)** | 2A IR                         | 케이블 단축 / 14 AWG / 로컬 12V 공급      |
| **배터리 캠**              | IR이 몇 시간 만에 배터리 소진 | IR 비활성화; 주변광 / 외부 IR 조명기 사용 |

## 2. IR 화이트아웃 / 세탁 / 회색 이미지

**증상:** 야간 이미지 = 밝은 흰색/회색 덩어리, 디테일 없음. IR이 가까운 표면에 반사됨.

  <AccordionItem value="whiteout-causes">
    <AccordionTrigger>일반적인 반사 원인</AccordionTrigger>
    <AccordionContent>

      | 표면 | 거리 | 수정 | |---------|----------|-----| | **벽 / 사이딩 / 파시아** | 6–24" | 카메라 각도를 **아래로 15–30°** 조정; 12–18" 밖으로 이동 (연장 암) | | **처마 / 처마및** | 0–12" | 처마밑 **아래**에 장착, 밀착하지 않음 | | **창문 / 유리** | 모든 거리 | **절대** 유리를 통해 조준하지 마세요 (IR 100% 반사) | | **거미줄 / 돔 먼지** | 렌즈 위 | 매월 청소; IR 링 위 플리스 커버 | | **비 / 눈 / 안개** | 공중 | 외부 IR 조명기 (렌즈에서 떨어져 장착) | | **흰색 트림 / 홈통** | 12–36" | 무광 검정으로 도색 (BBQ 페인트) 또는 테이프로 마스킹 | | **번호판 (재귀반사)** | 20–50 ft | 아래로 각도 조정; 외부 IR; 편광판이 있는 LPR 카메라 |

    </AccordionContent>

  </AccordionItem>
</Accordion>

### 2.1 단계별 화이트아웃 수정

1. **반사체 식별:** 밤에 나가서 표면의 IR 빛 관찰. 휴대폰 카메라로 IR 확산 확인.
2. **각도 재조정:** 수평에서 **아래로 15–30°** 기울임. 레벨 + 각도기 사용.
3. **거리 추가:** 연장 브래킷(12–24")이 IR 콘을 가까운 물체 너머로 이동.
4. **외부 IR 조명기:** 10–20 ft 떨어져 장착, 목표 영역 조준. 카메라 IR **OFF**. 번호판, 대문에 최적.
5. **스마트 IR (지원 시):** 카메라가 가까운 물체에 대해 IR 강도 자동 감소. 설정에서 활성화.
6. **IR 컷 필터 확인:** "밖으로" 고정 (야간 모드) = 컬러 주간 이미지. "안으로" 고정 (주간 모드) = IR 없음. 전원 재부팅.

## 3. 검은색 / 노이즈 / 거친 야간 이미지

**증상:** 어둡고, 얼룩덜룩하고, "눈 오는" 비디오. 디테일 없음.

| 원인                               | 진단                           | 수정                                               |
| ---------------------------------- | ------------------------------ | -------------------------------------------------- |
| **IR이 거리에 비해 너무 약함**     | 대상이 정격 IR 범위의 50% 초과 | 외부 IR 조명기; 카메라를 더 가까이 이동            |
| **셔터가 너무 느림 (모션 블러)**   | 잔상, 밝은 줄무늬              | **최소 셔터: 1/100s** 설정 (번호판은 1/250s)       |
| **게인/ISO 최대치**                | 노이즈 = "눈"                  | 빛 추가 (외부 IR, 가로등); 최대 게인 낮춤          |
| **광역 다이내믹 레인지(WDR) 꺼짐** | 밝은 빛이 카메라를 무력화      | **WDR: 120 dB+** 활성화 (True WDR, DWDR 아님)      |
| **돔 더러움 / 긁힘**               | 빛 주변 후광                   | 돔 교체 ($15); 마이크로파이버 + 렌즈 클리너로 청소 |
| **센서 열화**                      | 수년에 걸쳐 점진적             | 카메라 교체 (CMOS 노화, 핫픽셀 증가)               |
| **압축률 너무 높음**               | 야간에 블록 아티팩트           | 비트레이트: CBR 4–6 Mbps (4K) / 2–3 Mbps (2K)      |

### 3.1 셔터 속도 치트 시트

| 대상       | 최소 셔터           | 비고                 |
| ---------- | ------------------- | -------------------- |
| 일반 영역  | 1/50s (미국 1/60s)  | 빛과 블러 균형       |
| 걷는 사람  | 1/100s              | 선명한 팔다리        |
| **번호판** | **1/250s – 1/500s** | **중요** — 수동 설정 |
| 빠른 차량  | 1/500s – 1/1000s    | 외부 IR 필요         |

<Callout type="note">

**설정 방법:** 카메라 웹 UI → 이미지 → 노출 → 셔터 속도 → 수동 → 값 입력. "자동" 셔터 비활성화.

</Callout>

## 4. 컬러 나이트 비전 고장

**컬러 나이트 비전(CNV)** = 대형 센서(1/1.8", 1/1.2") + 저조도 ISP + **주변광 필요**.

| 고장 모드                   | 원인                     | 수정                                                                                                     |
| --------------------------- | ------------------------ | -------------------------------------------------------------------------------------------------------- |
| **흑백으로 유지 (IR 켜짐)** | 주변광 1 lux 미만        | CNV는 ~3–5 lux 필요 (가로등, 현관등). **따뜻한 백색(3000K) LED**를 10–20 ft 떨어져 추가.                 |
| **컬러 노이즈/거침**        | 게인 너무 높음           | 최대 게인 낮춤; 빛 추가; 흑백 IR 수용                                                                    |
| **색상 변이 (보라/초록)**   | 화이트 밸런스 혼동       | WB를 "야외" / "가로등" / 사용자 설정 3000K로 설정                                                        |
| **IR + CNV 충돌**           | 둘 다 켜짐 = 이상한 색상 | CNV 활성화 시 IR 비활성화 (좋은 카메라는 자동)                                                           |
| **지원되지 않음**           | 카메라에 CNV 센서 없음   | 사양 확인: 1/1.8"+ 센서 필요 (Reolink ColorX, Hikvision ColorVu, Dahua Full-Color, Axis Lightfinder 2.0) |

## 5. 번호판 캡처 (LPC) — 전문 야간 수정

**LPC는 밤에 90% 실패 원인:** 번호판의 IR 반사 + 느린 셔터 + 약한 IR.

### 5.1 LPC 야간 확인 목록

- [ ] **카메라:** 4K/8MP + 광학 줌 (가변 초점 2.8–12mm 또는 5–50mm) 또는 전용 LPR (2MP, 100+ fps, 글로벌 셔터)
- [ ] **셔터:** **최소 1/250s** (수동)
- [ ] **IR:** 외부 조명기 (850 nm, 30–50° 빔) **카메라 외부**, 번호판 영역 조준
- [ ] **각도:** 수직에서 ≤30°; 높이 3–4 ft (번호판 높이)
- [ ] **WDR:** 120 dB+ True WDR (헤드라이트 처리)
- [ ] **IR 컷:** 전환 확인 (주간 컬러 / 야간 흑백)
- [ ] **영역:** 모션 영역 = 번호판 경로만 (거짓 트리거 감소)
- [ ] **녹화:** CBR 6–8 Mbps (4K) / 4 Mbps (2MP LPR)

| **Dahua ITC237-PW6M-IRLZF**        | 2MP LPR        | 60 fps, 140 dB WDR, 164 ft IR               | 주차장, 게이트 출입   |
| **Reolink RLC-1212A**              | 12MP 가변 초점 | 5배 광학 줌, 190 ft IR, 1/1.8"              | 가성비 4K + 번호판 줌 |
| **Axis Q1700-LE**                  | 2MP LPR        | 60 fps, Forensic WDR, 131 ft IR             | 기업용, 증거 등급     |

## 6. 초인종 카메라 야간 문제

| 문제                       | 원인                    | 수정                                                           |
| -------------------------- | ----------------------- | -------------------------------------------------------------- |
| **방문자 얼굴 화이트아웃** | IR이 현관/처마에서 반사 | 웨지 키트 각도 조정 (아래로 15–25°); IR 비활성화 → 현관등 사용 |
| **야간 투시 없음**         | 배터리 절약 모드        | 유선 연결 (16–24V AC) → 상시 IR 활성화                         |
| **컬러 나이트 비전 꺼짐**  | 주변광 너무 낮음        | 현관 조명기구에 **3000K LED** 추가 (차가운 백색 아님)          |
| **밤에 모션 미감지**       | PIR 범위 < IR 범위      | PIR ~25 ft, IR ~30 ft → IR 감도 감소 / 영역 좁힘               |

## 7. 배터리 / 태양광 카메라 야간 특이사항

| 문제                               | 원인                           | 수정                                                      |
| ---------------------------------- | ------------------------------ | --------------------------------------------------------- |
| **IR이 1주일 만에 배터리 소진**    | IR = 연속 2–4W 소모            | IR 비활성화; 주변광 + 모션 스포트라이트 활용 (Eufy, Ring) |
| **겨울에 야간 투시 안 됨**         | Li-ion 32°F 미만에서 충전 중단 | LiFePO₄ 업그레이드 (Reolink Go PT Plus); 단열 박스 + 히터 |
| **모션이 낮에만 작동**             | PIR이 창문/유리에 차단됨       | 유리 **밖**에 장착; PIR은 온도 차이 필요                  |
| **스포트라이트가 트리거되지 않음** | 모션 영역 잘못됨               | 접근로를 덮도록 영역 다시 그리기; 걸어서 테스트           |

## 8. 진단 도구 및 절차

### 8.1 야간 투시 테스트 카드 (인쇄 및 코팅)

```
┌─────────────────────────────────────┐
│   NIGHT VISION TEST CARD            │
│  ████████████████████████████████   │  ← 100% 흰색 (IR 반사)
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   │  ← 50% 회색
│  ████████████████████████████████   │  ← 0% 검은색
│                                     │
│  Plate: ABC-1234  (20/40/60ft에서 테스트)│
│  Face: [Outline]                    │
└─────────────────────────────────────┘
```

**절차:** 목표 거리에 장착 → 녹화 → 번호판/얼굴 선명도 검토.

### 8.2 IR 전력 측정

| 도구               | 측정                 | 합격 기준                       |
| ------------------ | -------------------- | ------------------------------- |
| **PoE 테스터**     | 카메라 전압 (야간)   | >48V (PoE+), >52V (PoE++)       |
| **멀티미터 (12V)** | 카메라 전압 (야간)   | >11.5V (12V 공칭)               |
| **클램프 미터**    | 12V 라인 전류 (야간) | 카메라당 <2A                    |
| **열화상 카메라**  | IR LED 온도          | 40–60°C (작동 중), <30°C (고장) |

### 8.3 펌웨어 및 설정 감사

```
☐ 펌웨어: 최신 안정 버전 (베타 아님)
☐ IR 모드: 자동 (24/7 "켜짐" 아님)
☐ 스마트 IR: 활성화
☐ 셔터: 수동 최소 1/100s (번호판 1/250s)
☐ WDR: True WDR 켜짐 (120 dB+)
☐ 최대 게인: 제한됨 ("자동 최대" 아님)
☐ 비트레이트: CBR 4–6 Mbps (4K)
☐ IR 컷: 전환 확인 (주야간)
☐ 시간대/NTP: 정확 (예약 IR용)
```

## 9. 교체 vs 수리 시점

| 증상                               | 수리 가능?                | 교체 시점                            |
| ---------------------------------- | ------------------------- | ------------------------------------ |
| IR 빛 없음, 전원 정상              | IR 보드 ($50) / LDR ($2)  | 카메라 3년 초과, 물 침투             |
| IR 컷 필터 고정                    | 두드림/전원 재부팅 (임시) | 메커니즘 마모; 2년 초과              |
| 각도 조정으로 화이트아웃 해결 불가 | 해당 없음                 | 위치에 잘못된 카메라 (외부 IR 사용)  |
| 노이즈 센서 (핫픽셀)               | 픽셀 매핑 (일부 카메라)   | 5% 초과 데드 픽셀; 4년 초과          |
| 컬러 나이트 비전 고장              | 센서/ISP 문제             | 센서 너무 작음 (<1/2.8"); 업그레이드 |
| 간헐적 야간 재부팅                 | 전압 강하                 | 케이블/전원 수정; 지속 시 → 교체     |

## 10. 빠른 참조 카드 (차량용 인쇄)

```
야간 투시 분류 순서도

  1. IR 빛? (휴대폰 카메라 테스트)
   아니오 → 전압 확인 (야간) → 전원 수정
   예 → 2단계

  2. 이미지 흰색/회색?
   예 → 아래로 15-30° 각도 조정 / 12"+ 밖으로 이동 / 외부 IR
   아니오 → 3단계

  3. 이미지 검은색/노이즈?
   예 → 셔터 최소 1/100s / WDR 켜짐 / 외부 IR
   아니오 → 4단계

  4. 컬러 나이트 비전 꺼짐?
   예 → 3000K 빛 추가 / 센서 크기 확인
   아니오 → 5단계

  5. 번호판 판독 불가?
   예 → 셔터 1/250s / 외부 IR / 가변 초점 줌 / LPR 카메라
   아니오 → 완료 ✓

전압 확인 (야간, IR 켜짐):
PoE: >48V (af) / >50V (at) / >52V (bt)
  12V: 카메라에서 >11.5V
```
