ความล้มเหลวของการมองเห็นตอนกลางคืนคือข้อร้องเรียนอันดับ 1 ที่ว่า "กล้องใช้ได้ตอนกลางวัน ไร้ประโยชน์ตอนกลางคืน" คู่มือนี้อธิบายทุกโหมดความล้มเหลวพร้อมการวินิจฉัย การแก้ไข และเมื่อต้องเปลี่ยน

ปัญหาเลนส์/การตั้งค่า 2. **ภาพขาว/เทา?** → การสะท้อน IR (มุม/ระยะทาง) 3.

**ภาพสีตอนกลางคืนไม่ทำงาน?** → แสงโดยรอบน้อยเกินไป / ปิดการตั้งค่า

### 1.2 สาเหตุและการแก้ไข

| สาเหตุ                             | การวินิจฉัย                              | การแก้ไข                                                              |
| ---------------------------------- | ---------------------------------------- | --------------------------------------------------------------------- |
| **ฟิลเตอร์ IR ค้าง (โหมดกลางวัน)** | ภาพมีสีตอนกลางคืน ไม่มีแสง IR            | รีเซ็ตไฟ (12V/PoE ปิด 30 วินาที) เคาะกล้องเบาๆ อัปเดตเฟิร์มแวร์       |
| **โฟโตรีซีสเตอร์/LDR เสีย**        | IR ไม่เปิดอัตโนมัติ เปิดด้วยมือได้       | ปิด LDR (หน้าต่างใสเล็ก) → ถ้า IR ทำงาน ให้เปลี่ยน LDR ($2) หรือกล้อง |
| **แผงขับ IR เสีย**                 | มี 12V/PoE, LED เย็น, เปิดด้วยมือไม่ได้  | เปลี่ยนกล้อง (แผงซ่อมไม่ได้)                                          |
| **ไฟไม่พอสำหรับ IR**               | กลางวันใช้ได้ กลางคืนรีสตาร์ท/IR กระพริบ | ตรวจสอบแรงดันที่กล้องขณะ IR ทำงาน (ดู §1.3)                           |
| **บั๊กเฟิร์มแวร์**                 | อัปเดตล่าสุดทำให้ IR อัตโนมัติเสีย       | ย้อนกลับเฟิร์มแวร์; แจ้งฝ่ายสนับสนุนผู้ขาย                            |
| **IR แบบซ่อนเร้น (940 nm)**        | ไม่มีแสงสีแดง ทดสอบโทรศัพท์แสดงสีม่วง    | ปกติ — 940 nm มองไม่เห็นด้วยตา                                        |

### 1.3 แรงดันตกภายใต้โหลด IR (สำคัญ)

**LED IR กินไฟเพิ่ม 2–5W** กล้องใช้ได้ตอนกลางวัน ดับตอนกลางคืน

<Callout type="warning">

**ทดสอบ:** มัลติมิเตอร์ที่อินพุตไฟกล้อง (PoE: ใช้เครื่องทดสอบ PoE) แรงดันกลางวัน vs แรงดันกลางคืน (IR เปิด) **ตก >0.5V (12V) หรือ >2V (PoE 48V) = สายยาวเกินไป / เกจเล็กเกินไป / สวิตช์ไม่พอ**

</Callout>

| สถาปัตยกรรม                | โหลด IR สูงสุด                | การแก้ไข                                        |
| -------------------------- | ----------------------------- | ----------------------------------------------- |
| **PoE (802.3af 15.4W)**    | ~4W IR                        | อัปเกรดเป็น PoE+ (30W) อินเจกเตอร์/สวิตช์       |
| **12V DC (18 AWG, 15 ม.)** | 2A IR                         | ลดความยาวสาย / 14 AWG / แหล่งจ่าย 12V เฉพาะจุด  |
| **กล้องแบตเตอรี่**         | IR ทำให้แบตหมดในไม่กี่ชั่วโมง | ปิด IR; ใช้แสงโดยรอบ / อิลลูมิเนเตอร์ IR ภายนอก |

### 2.1 การแก้ไขภาพขาวโพลนทีละขั้นตอน

1. **ระบุตัวสะท้อน:** ออกไปตอนกลางคืน ดูแสง IR บนพื้นผิว ใช้กล้องโทรศัพท์เพื่อดูการกระจาย IR
2. **ปรับมุมใหม่:** เอียง **ลง 15–30°** จากแนวนอน ใช้ระดับ + ไม้โปรแทรกเตอร์
3. **เพิ่มระยะห่าง:** ขายึดต่อขยาย (30–60 ซม.) ย้ายกรวย IR ผ่านวัตถุใกล้
4. **อิลลูมิเนเตอร์ IR ภายนอก:** ติดตั้งห่าง 3–6 ม. หันไปยังโซนเป้าหมาย ปิด IR **กล้อง** เหมาะที่สุดสำหรับป้ายทะเบียน ประตูรั้ว
5. **IR อัจฉริยะ (หากรองรับ):** กล้องลดความเข้ม IR อัตโนมัติสำหรับวัตถุใกล้ เปิดใช้งานในการตั้งค่า
6. **ตรวจสอบฟิลเตอร์ IR:** ค้าง "ออก" (โหมดกลางคืน) = ภาพสีตอนกลางวัน ค้าง "เข้า" (โหมดกลางวัน) = ไม่มี IR รีเซ็ตไฟ

## 3. ภาพกลางคืนดำ / มีสัญญาณรบกวน / เป็นเม็ด

**อาการ:** วิดีโอมืด เป็นจุด "เหมือนหิมะ" ไม่มีรายละเอียด

| สาเหตุ                                          | การวินิจฉัย                       | การแก้ไข                                                              |
| ----------------------------------------------- | --------------------------------- | --------------------------------------------------------------------- |
| **IR อ่อนเกินไปสำหรับระยะทาง**                  | เป้าหมาย >50% ของระยะ IR ที่กำหนด | อิลลูมิเนเตอร์ IR ภายนอก; ขยับกล้องให้ใกล้ขึ้น                        |
| **ชัตเตอร์ช้าเกินไป (ภาพเบลอจากการเคลื่อนไหว)** | รอยผี แถบสว่าง                    | ตั้ง **ชัตเตอร์ขั้นต่ำ: 1/100s** (1/250s สำหรับป้าย)                  |
| **เกน/ISO สูงสุด**                              | สัญญาณรบกวน = "หิมะ"              | เพิ่มแสง (IR ภายนอก, ไฟถนน); ลดเกนสูงสุด                              |
| **WDR ปิด**                                     | ไฟสว่างทำให้กล้องบอด              | เปิด **WDR: 120 dB+** (True WDR, ไม่ใช่ DWDR)                         |
| **โดมสกปรก / มีรอยขีดข่วน**                     | รัศมีรอบไฟ                        | เปลี่ยนโดม ($15); ทำความสะอาดด้วยไมโครไฟเบอร์ + น้ำยาทำความสะอาดเลนส์ |
| **เซ็นเซอร์เสื่อม**                             | แย่ลงเรื่อยๆ หลายปี               | เปลี่ยนกล้อง (CMOS เสื่อม, ฮอตพิกเซลเพิ่มขึ้น)                        |
| **การบีบอัดสูงเกินไป**                          | ภาพแตกเป็นบล็อกตอนกลางคืน         | บิตเรต: CBR 4–6 Mbps (4K) / 2–3 Mbps (2K)                             |

### 3.1 ตารางความเร็วชัตเตอร์

| เป้าหมาย        | ชัตเตอร์ขั้นต่ำ      | หมายเหตุ                  |
| --------------- | -------------------- | ------------------------- |
| พื้นที่ทั่วไป   | 1/50s (1/60s สหรัฐฯ) | สมดุลแสง vs เบลอ          |
| คนเดิน          | 1/100s               | แขนขาคม                   |
| **ป้ายทะเบียน** | **1/250s – 1/500s**  | **สำคัญ** — ตั้งด้วยตนเอง |
| รถเร็ว          | 1/500s – 1/1000s     | ต้องใช้ IR ภายนอก         |

<Callout type="note">

**วิธีตั้ง:** หน้าเว็บกล้อง → รูปภาพ → การเปิดรับแสง → ความเร็วชัตเตอร์ →
แมนนวล → ป้อนค่า ปิดชัตเตอร์ "อัตโนมัติ"

</Callout>

## 4. ความล้มเหลวของภาพสีตอนกลางคืน

**ภาพสีตอนกลางคืน (CNV)** = เซ็นเซอร์ใหญ่ (1/1.8", 1/1.2") + ISP แสงน้อย + **ต้องใช้แสงโดยรอบ**

| โหมดความล้มเหลว              | สาเหตุ                  | การแก้ไข                                                                                                          |
| ---------------------------- | ----------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **ค้างเป็นขาวดำ (IR ติด)**   | แสงแวดล้อม &lt;1 ลักซ์  | CNV ต้องการ ~3–5 ลักซ์ (ไฟถนน, ไฟเฉลียง) เพิ่ม **LED สีขาวอุ่น (3000K)** ห่าง 3–6 ม.                              |
| **สีมีสัญญาณรบกวน/เป็นเม็ด** | เกนสูงเกินไป            | ลดเกนสูงสุด; เพิ่มแสง; ยอมรับ IR ขาวดำ                                                                            |
| **สีเพี้ยน (ม่วง/เขียว)**    | สมดุลขาวสับสน           | ตั้ง WB เป็น "กลางแจ้ง" / "ไฟถนน" / กำหนดเอง 3000K                                                                |
| **IR + CNV ขัดแย้ง**         | ทั้งคู่ติด = สีเพี้ยน   | ปิด IR เมื่อ CNV ทำงาน (อัตโนมัติในกล้องดี)                                                                       |
| **ไม่รองรับ**                | กล้องไม่มีเซ็นเซอร์ CNV | ตรวจสอบสเปก: ต้องใช้เซ็นเซอร์ 1/1.8"+ (Reolink ColorX, Hikvision ColorVu, Dahua Full-Color, Axis Lightfinder 2.0) |

## 5. การจับภาพป้ายทะเบียน (LPC) — การแก้ไขกลางคืนเฉพาะทาง

**LPC ล้มเหลวตอนกลางคืน 90% เนื่องจาก:** การสะท้อน IR จากป้าย + ชัตเตอร์ช้า + IR อ่อน

### 5.1 รายการตรวจสอบกลางคืน LPC

- [ ] **กล้อง:** 4K/8MP + ซูมออปติคอล (varifocal 2.8–12mm หรือ 5–50mm) หรือ LPR เฉพาะ (2MP, 100+ fps, global shutter)
- [ ] **ชัตเตอร์:** **1/250s ขั้นต่ำ** (แมนนวล)
- [ ] **IR:** อิลลูมิเนเตอร์ภายนอก (850 nm, ลำแสง 30–50°) **นอกกล้อง** หันไปยังโซนป้าย
- [ ] **มุม:** ≤30° จากแนวตั้งฉาก; ความสูง 0.9–1.2 ม. (ระดับป้าย)
- [ ] **WDR:** 120 dB+ True WDR (จัดการไฟหน้ารถ)
- [ ] **ฟิลเตอร์ IR:** ตรวจสอบการสลับ (สีกลางวัน / ขาวดำกลางคืน)
- [ ] **โซน:** โซนเคลื่อนไหว = เส้นทางป้ายเท่านั้น (ลดการแจ้งเตือนปลอม)
- [ ] **บันทึก:** CBR 6–8 Mbps (4K) / 4 Mbps (2MP LPR)

| ---------------------------------- | -------------- | --------------------------------------------- | -------------------------- |
| **Hikvision DS-2CD4A26FWD-IZHS/P** | 2MP LPR        | 100 fps, global shutter, 120 dB WDR, 30 ม. IR | จับภาพป้ายโดยเฉพาะ         |

| **Reolink RLC-1212A**              | 12MP Varifocal | ซูมออปติคอล 5×, 58 ม. IR, 1/1.8"              | 4K ประหยัด + ซูมสำหรับป้าย |
| **Axis Q1700-LE**                  | 2MP LPR        | 60 fps, Forensic WDR, 40 ม. IR                | องค์กร, ระดับหลักฐาน       |

## 6. ปัญหากล้องกริ่งประตูตอนกลางคืน

| ปัญหา                                      | สาเหตุ                   | การแก้ไข                                              |
| ------------------------------------------ | ------------------------ | ----------------------------------------------------- |
| **ใบหน้าผู้มาเยือนขาวโพลน**                | IR สะท้อนจากเฉลียง/ชายคา | ชุดลิ่มปรับมุม (15–25° เอียงลง); ปิด IR → ใช้ไฟเฉลียง |
| **ไม่มีภาพกลางคืน**                        | โหมดประหยัดแบตเตอรี่     | เดินสายไฟ (16–24V AC) → เปิดใช้งาน IR ตลอดเวลา        |
| **ภาพสีตอนกลางคืนไม่ทำงาน**                | แสงแวดล้อมน้อยเกินไป     | เพิ่ม **LED 3000K** ในโคมเฉลียง (ไม่ใช่สีขาวเย็น)     |
| **การตรวจจับความเคลื่อนไหวพลาดตอนกลางคืน** | ระยะ PIR < ระยะ IR       | PIR ~7.6 ม., IR ~9 ม. → ลดความไว IR / แคบโซน          |

## 7. ปัญหากล้องแบตเตอรี่/โซลาร์เซลล์ตอนกลางคืนโดยเฉพาะ

| ปัญหา                                    | สาเหตุ                          | การแก้ไข                                                                    |
| ---------------------------------------- | ------------------------------- | --------------------------------------------------------------------------- |
| **IR ทำให้แบตหมดใน 1 สัปดาห์**           | IR = 2–4W ต่อเนื่อง             | ปิด IR; ใช้แสงโดยรอบ + สปอตไลต์ตรวจจับความเคลื่อนไหว (Eufy, Ring)           |
| **ไม่มีภาพกลางคืนในฤดูหนาว**             | Li-ion &lt;0°C หยุดชาร์จ        | อัปเกรดเป็น LiFePO₄ (Reolink Go PT Plus); กล่องหุ้มฉนวน + เครื่องทำความร้อน |
| **ตรวจจับความเคลื่อนไหวได้เฉพาะกลางวัน** | PIR ถูกปิดกั้นโดยหน้าต่าง/กระจก | ติดตั้ง **นอก** กระจก; PIR ต้องการความแตกต่างของอุณหภูมิ                    |
| **สปอตไลต์ไม่ทำงาน**                     | โซนเคลื่อนไหวผิด                | วาดโซนใหม่ให้ครอบคลุมเส้นทางเข้า; ทดสอบเดิน                                 |

## 8. เครื่องมือและขั้นตอนการวินิจฉัย

### 8.1 บัตรทดสอบการมองเห็นตอนกลางคืน (พิมพ์และเคลือบ)

```
┌─────────────────────────────────────┐
│   บัตรทดสอบการมองเห็นตอนกลางคืน      │
│  ████████████████████████████████   │  ← ขาว 100% (สะท้อน IR)
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   │  ← เทา 50%
│  ████████████████████████████████   │  ← ดำ 0%
│                                     │
│  ป้าย: ABC-1234  (ทดสอบที่ 6/12/18 ม.)│
│  ใบหน้า: [โครงร่าง]                    │
└─────────────────────────────────────┘
```

**ขั้นตอน:** ติดตั้งที่ระยะเป้าหมาย → บันทึก → ตรวจสอบความคมชัดของป้าย/ใบหน้า

### 8.2 การวัดกำลัง IR

| เครื่องมือ               | การวัด                   | เกณฑ์ผ่าน                        |
| ------------------------ | ------------------------ | -------------------------------- |
| **เครื่องทดสอบ PoE**     | แรงดันที่กล้อง (กลางคืน) | >48V (PoE+), >52V (PoE++)        |
| **มัลติมิเตอร์ (12V)**   | แรงดันที่กล้อง (กลางคืน) | >11.5V (12V nominal)             |
| **แคลมป์มิเตอร์**        | กระแสบนสาย 12V (กลางคืน) | &lt;2A (ต่อกล้อง)                |
| **กล้องถ่ายภาพความร้อน** | อุณหภูมิ LED IR          | 40–60°C (ทำงาน), &lt;30°C (เสีย) |

### 8.3 การตรวจสอบเฟิร์มแวร์และการตั้งค่า

```
☐ เฟิร์มแวร์: เวอร์ชันเสถียรล่าสุด (ไม่ใช่เบต้า)
☐ โหมด IR: อัตโนมัติ (ไม่ใช่ "เปิด" ตลอด 24 ชม.)
☐ IR อัจฉริยะ: เปิดใช้งาน
☐ ชัตเตอร์: แมนนวลขั้นต่ำ 1/100s (1/250s สำหรับป้าย)
☐ WDR: True WDR เปิด (120 dB+)
☐ เกนสูงสุด: จำกัด (ไม่ใช่ "อัตโนมัติสูงสุด")
☐ บิตเรต: CBR 4–6 Mbps (4K)
☐ ฟิลเตอร์ IR: ตรวจสอบการสลับ (กลางวัน/กลางคืน)
☐ เขตเวลา/NTP: ถูกต้อง (IR ตามกำหนดเวลา)
```

## 9. เมื่อต้องเปลี่ยน vs ซ่อม

| อาการ                              | ซ่อมได้?                 | เปลี่ยนเมื่อ                                |
| ---------------------------------- | ------------------------ | ------------------------------------------- |
| ไม่มีแสง IR, ไฟปกติ                | แผง IR ($50) / LDR ($2)  | กล้อง >3 ปี, น้ำเข้า                        |
| ฟิลเตอร์ IR ค้าง                   | เคาะ/รีเซ็ตไฟ (ชั่วคราว) | กลไกสึก; >2 ปี                              |
| ภาพขาวโพลนแก้ไม่ได้ด้วยมุม         | N/A                      | กล้องผิดประเภทสำหรับตำแหน่ง (ใช้ IR ภายนอก) |
| เซ็นเซอร์มีสัญญาณรบกวน (ฮอตพิกเซล) | การแมปพิกเซล (บางรุ่น)   | >5% พิกเซลเสีย; >4 ปี                       |
| ภาพสีตอนกลางคืนล้มเหลว             | ปัญหาเซ็นเซอร์/ISP       | เซ็นเซอร์เล็กเกินไป (&lt;1/2.8"); อัปเกรด   |
| รีสตาร์ทกลางคืนเป็นระยะ            | แรงดันตก                 | แก้ไขสาย/ไฟ; ถ้ายังเป็นอยู่ → เปลี่ยน       |

## 10. บัตรอ้างอิงด่วน (พิมพ์พกติดรถ)

```
ผังการคัดแยกปัญหาการมองเห็นตอนกลางคืน

  1. มีแสง IR? (ทดสอบด้วยกล้องโทรศัพท์)
   ไม่ → ตรวจสอบแรงดัน (กลางคืน) → แก้ไขไฟ
   ใช่ → ขั้นตอน 2

  2. ภาพขาว/เทา?
   ใช่ → เอียงลง 15-30° / ขยับออก 30 ซม.+ / IR ภายนอก
   ไม่ → ขั้นตอน 3

  3. ภาพดำ/มีสัญญาณรบกวน?
   ใช่ → ชัตเตอร์ขั้นต่ำ 1/100s / เปิด WDR / IR ภายนอก
   ไม่ → ขั้นตอน 4

  4. ภาพสีตอนกลางคืนไม่ทำงาน?
   ใช่ → เพิ่มแสง 3000K / ตรวจสอบขนาดเซ็นเซอร์
   ไม่ → ขั้นตอน 5

  5. ป้ายทะเบียนอ่านไม่ออก?
   ใช่ → ชัตเตอร์ 1/250s / IR ภายนอก / ซูม varifocal / กล้อง LPR
   ไม่ → เสร็จ ✓

ตรวจสอบแรงดัน (กลางคืน, IR เปิด):
PoE: >48V (af) / >50V (at) / >52V (bt)
  12V: >11.5V ที่กล้อง
```
