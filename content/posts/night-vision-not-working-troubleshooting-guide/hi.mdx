| **IR कट फ़िल्टर अटका (डे मोड)** | रात में रंगीन छवि, कोई IR चमक नहीं                      | पावर साइकिल (12V/PoE 30 सेकंड बंद)। कैमरे को हल्के से थपथपाएं। फ़र्मवेयर अपडेट करें। |

### 1.3 IR लोड के तहत वोल्टेज ड्रॉप (महत्वपूर्ण)

**IR LED 2–5W अतिरिक्त खींचते हैं।** कैमरा दिन में चलता है, रात में क्रैश होता है।

<Callout type="warning">

**परीक्षण:** कैमरा पावर इनपुट पर मल्टीमीटर (PoE: PoE टेस्टर का उपयोग करें)। दिन का वोल्टेज बनाम रात (IR चालू) का वोल्टेज। **>0.5V (12V) या >2V (PoE 48V) ड्रॉप = केबल बहुत लंबी / गेज बहुत छोटा / स्विच बजट पार हो गया।**

</Callout>

| आर्किटेक्चर                 | अधिकतम IR लोड                      | समाधान                                                            |
| --------------------------- | ---------------------------------- | ----------------------------------------------------------------- |
| **PoE (802.3af 15.4W)**     | ~4W IR                             | PoE+ (30W) इंजेक्टर/स्विच में अपग्रेड करें                        |
| **12V DC (18 AWG, 50 फीट)** | 2A IR                              | रन छोटा करें / 14 AWG / स्थानीय 12V आपूर्ति                       |
| **बैटरी कैम**               | IR बैटरी को घंटों में खत्म करता है | IR अक्षम करें; परिवेश प्रकाश / बाहरी IR इल्युमिनेटर का उपयोग करें |

### 2.1 चरण-दर-चरण व्हाइटआउट फ़िक्स

1. **रिफ़्लेक्टर पहचानें:** रात में बाहर जाएं, सतहों पर IR चमक देखें। IR फैलाव देखने के लिए फ़ोन कैमरे का उपयोग करें।
2. **पुनः-कोण करें:** क्षैतिज से **15–30° नीचे** झुकाएं। लेवल + प्रोट्रैक्टर का उपयोग करें।
3. **दूरी जोड़ें:** एक्सटेंशन ब्रैकेट (12–24") IR कोन को पास की वस्तुओं से आगे ले जाता है।
4. **बाहरी IR इल्युमिनेटर:** 10–20 फीट दूर माउंट करें, लक्ष्य क्षेत्र पर निशाना लगाएं। कैमरा IR **बंद**। लाइसेंस प्लेट, गेट के लिए सर्वश्रेष्ठ।
5. **स्मार्ट IR (यदि समर्थित):** कैमरा पास की वस्तुओं के लिए IR तीव्रता अपने आप कम करता है। सेटिंग्स में सक्षम करें।
6. **IR कट फ़िल्टर जांच:** "बाहर" अटका (नाइट मोड) = रंगीन दिन की छवि। "अंदर" अटका (डे मोड) = कोई IR नहीं। पावर साइकिल।

## 3. काली / शोर भरी / दानेदार रात की छवि

**लक्षण:** अंधेरा, धब्बेदार, "बर्फीला" वीडियो। कोई विवरण नहीं।

| कारण                           | निदान                            | समाधान                                                    |
| ------------------------------ | -------------------------------- | --------------------------------------------------------- |
| **IR दूरी के लिए बहुत कमज़ोर** | लक्ष्य रेटेड IR रेंज का >50%     | बाहरी IR इल्युमिनेटर; कैमरे को करीब ले जाएं               |
| **शटर बहुत धीमा (मोशन ब्लर)**  | घोस्ट ट्रेल्स, चमकीली धारियां    | **न्यूनतम शटर: 1/100s** सेट करें (प्लेट के लिए 1/250s)    |
| **गेन/ISO अधिकतम**             | शोर = "बर्फ"                     | प्रकाश जोड़ें (बाहरी IR, स्ट्रीटलाइट); अधिकतम गेन कम करें |
| **WDR बंद**                    | तेज़ रोशनी कैमरे को अंधी करती है | **WDR: 120 dB+** चालू करें (True WDR, DWDR नहीं)          |
| **डोम गंदा / खरोंचदार**        | रोशनी के चारों ओर हेलो           | डोम बदलें ($15); माइक्रोफ़ाइबर + लेंस क्लीनर से साफ़ करें |
| **सेंसर डिग्रेड हो रहा**       | वर्षों में धीरे-धीरे बिगड़ता है  | कैमरा बदलें (CMOS पुराना होता है, हॉट पिक्सेल बढ़ते हैं)  |
| **कंप्रेशन बहुत अधिक**         | रात में ब्लॉकी कलाकृतियां        | बिटरेट: CBR 4–6 Mbps (4K) / 2–3 Mbps (2K)                 |

### 3.1 शटर स्पीड चीट शीट

| लक्ष्य            | न्यूनतम शटर         | नोट्स                              |
| ----------------- | ------------------- | ---------------------------------- |
| सामान्य क्षेत्र   | 1/50s (1/60s US)    | प्रकाश बनाम धुंधलापन संतुलित करें  |
| चलता हुआ व्यक्ति  | 1/100s              | तेज़ अंग                           |
| **लाइसेंस प्लेट** | **1/250s – 1/500s** | **महत्वपूर्ण** — मैन्युअल सेट करें |
| तेज़ वाहन         | 1/500s – 1/1000s    | बाहरी IR की आवश्यकता               |

<Callout type="note">

**कैसे सेट करें:** कैमरा वेब UI → इमेज → एक्सपोज़र → शटर स्पीड → मैन्युअल →
मान दर्ज करें। "ऑटो" शटर अक्षम करें।

</Callout>

## 4. कलर नाइट विज़न विफलताएं

**कलर नाइट विज़न (CNV)** = बड़ा सेंसर (1/1.8", 1/1.2") + कम-रोशनी ISP + **परिवेश प्रकाश आवश्यक**।

| विफलता मोड                    | कारण                     | समाधान                                                                                                          |
| ----------------------------- | ------------------------ | --------------------------------------------------------------------------------------------------------------- |
| **B&W (IR चालू) रहता है**     | परिवेश &lt;1 लक्स        | CNV को ~3–5 लक्स चाहिए (स्ट्रीटलाइट, पोर्च लाइट)। 10–20 फीट दूर **गर्म सफ़ेद (3000K) LED** जोड़ें।              |
| **रंग शोर भरा/दानेदार**       | गेन बहुत अधिक            | अधिकतम गेन कम करें; प्रकाश जोड़ें; B&W IR स्वीकार करें                                                          |
| **रंग बदलता है (बैंगनी/हरा)** | व्हाइट बैलेंस गड़बड़     | WB को "आउटडोर" / "स्ट्रीटलाइट" / कस्टम 3000K पर सेट करें                                                        |
| **IR + CNV संघर्ष**           | दोनों चालू = अजीब रंग    | CNV सक्रिय होने पर IR अक्षम करें (अच्छे कैमरों में ऑटो)                                                         |
| **समर्थित नहीं**              | कैमरे में CNV सेंसर नहीं | स्पेक्स जांचें: 1/1.8"+ सेंसर चाहिए (Reolink ColorX, Hikvision ColorVu, Dahua Full-Color, Axis Lightfinder 2.0) |

## 5. लाइसेंस प्लेट कैप्चर (LPC) — विशेष रात फ़िक्सेस

**LPC रात में 90% विफल होता है:** प्लेट से IR रिफ़्लेक्शन + धीमा शटर + कमज़ोर IR।

### 5.1 LPC रात चेकलिस्ट

- [ ] **कैमरा:** 4K/8MP + ऑप्टिकल ज़ूम (वेरीफ़ोकल 2.8–12mm या 5–50mm) या समर्पित LPR (2MP, 100+ fps, ग्लोबल शटर)
- [ ] **शटर:** **न्यूनतम 1/250s** (मैन्युअल)
- [ ] **IR:** बाहरी इल्युमिनेटर (850 nm, 30–50° बीम) **कैमरे से अलग**, प्लेट ज़ोन पर निशाना लगाकर
- [ ] **कोण:** लंबवत से ≤30°; ऊंचाई 3–4 फीट (प्लेट स्तर)
- [ ] **WDR:** 120 dB+ True WDR (हेडलाइट्स संभालता है)
- [ ] **IR कट:** स्विचिंग सत्यापित करें (दिन रंग / रात B&W)
- [ ] **ज़ोन:** मोशन ज़ोन = केवल प्लेट पथ (गलत ट्रिगर कम करें)
- [ ] **रिकॉर्ड:** CBR 6–8 Mbps (4K) / 4 Mbps (2MP LPR)

## 6. डोरबेल कैमरा नाइट इश्यू

| समस्या                        | कारण                                  | समाधान                                                               |
| ----------------------------- | ------------------------------------- | -------------------------------------------------------------------- |
| **आगंतुक का चेहरा व्हाइटआउट** | IR पोर्च/ओवरहैंग से परावर्तित होता है | वेज किट (15–25° नीचे) करें; IR अक्षम करें → पोर्च लाइट का उपयोग करें |
| **कोई नाइट विज़न नहीं**       | बैटरी सेविंग मोड                      | हार्डवायर (16–24V AC) करें → पूर्णकालिक IR सक्षम करता है             |
| **कलर नाइट विज़न बंद**        | परिवेश बहुत कम                        | पोर्च फ़िक्सचर में **3000K LED** जोड़ें (कूल व्हाइट नहीं)            |
| **रात में मोशन मिस**          | PIR रेंज < IR रेंज                    | PIR ~25 फीट, IR ~30 फीट → IR संवेदनशीलता कम करें / ज़ोन संकीर्ण करें |

## 7. बैटरी / सोलर कैमरा नाइट स्पेसिफ़िक्स

| मुद्दा                                  | कारण                                   | समाधान                                                               |
| --------------------------------------- | -------------------------------------- | -------------------------------------------------------------------- |
| **IR एक सप्ताह में बैटरी खत्म करता है** | IR = 2–4W निरंतर                       | IR अक्षम करें; परिवेश + मोशन स्पॉटलाइट पर निर्भर करें (Eufy, Ring)   |
| **सर्दी में नाइट विज़न नहीं**           | Li-ion &lt;32°F चार्ज करना बंद करता है | LiFePO₄ अपग्रेड (Reolink Go PT Plus); इंसुलेटेड बॉक्स + हीटर         |
| **मोशन केवल दिन में काम करता है**       | PIR खिड़की/कांच से अवरुद्ध             | **कांच के बाहर** माउंट करें; PIR को तापमान अंतर की आवश्यकता है       |
| **स्पॉटलाइट ट्रिगर नहीं होता**          | मोशन ज़ोन गलत                          | ज़ोन को दृष्टिकोण पथ को कवर करने के लिए फिर से बनाएं; वॉक टेस्ट करें |

## 8. डायग्नोस्टिक टूल्स और प्रक्रियाएं

### 8.1 नाइट विज़न टेस्ट कार्ड (प्रिंट और लैमिनेट करें)

```
┌─────────────────────────────────────┐
│   नाइट विज़न टेस्ट कार्ड            │
│  ████████████████████████████████   │  ← 100% सफ़ेद (IR रिफ़्लेक्शन)
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   │  ← 50% ग्रे
│  ████████████████████████████████   │  ← 0% काला
│                                     │
│  प्लेट: ABC-1234  (20/40/60 फीट पर टेस्ट करें)│
│  चेहरा: [रूपरेखा]                    │
└─────────────────────────────────────┘
```

**प्रक्रिया:** लक्ष्य दूरी पर माउंट करें → रिकॉर्ड करें → प्लेट/चेहरे की स्पष्टता समीक्षा करें।

### 8.2 IR पावर माप

| **मल्टीमीटर (12V)** | कैमरे पर वोल्टेज (रात) | >11.5V (12V नाममात्र)                |

| **थर्मल कैमरा**     | IR LED तापमान          | 40–60°C (काम कर रहा), &lt;30°C (मृत) |

☐ IR कट: स्विचिंग सत्यापित (दिन/रात)

## 10. त्वरित संदर्भ कार्ड (ट्रक के लिए प्रिंट करें)

```
नाइट विज़न ट्राइएज फ़्लोचार्ट

  1. IR चमक? (फ़ोन कैम टेस्ट)
   नहीं → वोल्टेज जांचें (रात) → पावर फ़िक्स
   हां → चरण 2

  2. इमेज सफ़ेद/ग्रे?
   हां → 15-30° नीचे करें / 12"+ बाहर ले जाएं / बाहरी IR
   नहीं → चरण 3

  3. इमेज काली/शोर भरी?
   हां → शटर 1/100s न्यूनतम / WDR चालू / बाहरी IR
   नहीं → चरण 4

  4. कलर नाइट विज़न बंद?
   हां → 3000K लाइट जोड़ें / सेंसर साइज़ जांचें
   नहीं → चरण 5

  5. प्लेट अपठनीय?
   हां → शटर 1/250s / बाहरी IR / वेरीफ़ोकल ज़ूम / LPR कैम
   नहीं → DONE ✓

वोल्टेज जांच (रात, IR चालू):
PoE: >48V (af) / >50V (at) / >52V (bt)
  12V: >11.5V कैमरे पर
```
