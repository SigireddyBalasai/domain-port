ความปลอดภัยของ RV และบ้านเคลื่อนที่มีข้อจำกัดเฉพาะ: ไฟ DC 12V/24V, การสั่นสะเทือนตลอดเวลา, อุณหภูมิ -29°C ถึง 60°C, พื้นที่หลังคาจำกัด และมักไม่มีอินเทอร์เน็ตถาวร กล้องบ้านทั่วไปพังเร็ว คู่มือนี้ครอบคลุมกล้องที่ออกแบบมาเพื่อชีวิตบนล้อ

<Badge variant="outline">ข้อแตกต่างหลักของ RV</Badge> **ห้ามเดินสายถาวร**
ในที่เช่า/สวน RV ส่วนใหญ่ แบตเตอรี่/โซลาร์ + 12V DC คือตัวเลือกเดียวของคุณ Wi-Fi
คือ Wi-Fi สวน (ไม่น่าเชื่อถือ) หรือมือถือ

## Checklist ข้อกำหนดเฉพาะสำหรับ RV

<Callout type="note">

<strong>สิ่งที่ต้องมีสำหรับ RV/เคลื่อนที่:</strong> ไฟ 12V DC โดยตรง
(ไม่มีอะแดปเตอร์ AC), อุปกรณ์ยึดกันสั่นสะเทือน, กันน้ำ IP66+, ช่วงอุณหภูมิ -20°C
ถึง 60°C, ตัวเลือกมือถือ/LTE หรือเสาอากาศ Wi-Fi แรง, บันทึก microSD ในเครื่อง
(คลาวด์ ล้มเหลวเมื่อนอกระบบ)

</Callout>

| คุณสมบัติ              | ทำไมถึงสำคัญใน RV               | ค่าเป้าหมาย                             |
| ---------------------- | ------------------------------- | --------------------------------------- |
| แหล่งจ่ายไฟ            | แบตเตอรี่บ้านคือ 12V DC         | 12V DC โดยตรง (ไม่สูญเสียอินเวอร์เตอร์) |
| ทนทานต่อการสั่นสะเทือน | ถนนทำลายอุปกรณ์ยึดทั่วไป        | MIL-STD-810G หรือเทียบเท่า              |
| ช่วงอุณหภูมิ           | วันที่ทะเลทราย ค่ำคืนบนดอย      | -29°C ถึง 60°C                          |
| การเชื่อมต่อ           | Wi-Fi สวนแออัด/ไม่ปลอดภัย       | 4G LTE / 5G + Wi-Fi สองโหมด             |
| หน่วยเก็บข้อมูล        | ไม่มีอินเทอร์เน็ต = ไม่มีคลาวด์ | microSD (128–512 GB) + บันทึกเวียน      |
| การติดตั้ง             | ไม่เจาะหลังคาบ้านเช่า           | ดูด, แม่เหล็ก, หนีบ, 3M VHB             |

## ตัวเลือกอันดับต้นตามหมวดหมู่

### 1. โดยรวมดีที่สุด: Reolink Go PT Plus (4G LTE + แบตเตอรี่/โซลาร์)

**รูปแบบ:** PTZ กะทัดรัด (14×9.4×9.4 ซม.), สีขาว
**พลังงาน:** แบตเตอรี่แบบชาร์จได้ (7800 mAh) + แผงโซลาร์ 6W ไม่บังคับ; **รับ 12V DC ผ่านสาย USB-C PD trigger**
**การเชื่อมต่อ:** 4G LTE (nano SIM) + Wi-Fi 2.4 GHz สองโหมด
**สเปก:** 4K (8MP), 355° pan / 140° tilt, IR 10 ม., ตรวจจับคน/รถ/สัตว์, microSD สูงสุด 128 GB
**ข้อดีใน RV:** ชาร์จ 12V จริงขณะขับรถ (USB-C PD), สำรองมือถือเมื่อ Wi-Fi สวนล้ม, PTZ ครอบคลุม 360° ด้วยการติดตั้งเดียว
**ข้อเสีย:** ต้องใช้ซิมดาต้า ($5–15/เดือน); แผงโซลาร์เพิ่มพื้นที่หลังคา; มอเตอร์ pan/tilt อาจแข็งตัวต่ำกว่า -23°C
**ราคา:** ~$230 กล้อง + ~$50 โซลาร์ | **เหมาะสำหรับ:** ผู้อยู่เต็มเวลา, ชอบป่ามากกว่า, RV ที่มีสัญญาณมือถือ

### 2. 12V แบบมีสายดีที่สุด: Reolink RLC-1212A (PoE) + Injektor 12V-เป็น-PoE

**พลังงาน:** PoE (802.3af) — ใช้ **injektor 12V-เป็น-PoE** (Tycon POE-12-48V)
**สเปก:** 12MP, มุมมอง 122°, IR 58 ม., ตรวจจับคน/รถ, microSD + NVR
**ข้อดีใน RV:** รายละเอียดสูงสุดสำหรับป้ายทะเบียน; ไม่มีการรบกวนไร้สาย; บันทึก 24/7
**ราคา:** ~$130 กล้อง + ~$25 injector + NVR | **เหมาะสำหรับ:** รถบ้าน Class A/B มี Ethernet

### 3. ไร้สายประหยัดที่สุด: EufyCam 3 S330 + HomeBase 3

**พลังงาน:** แบตเตอรี่ (13,000 mAh) + โซลาร์ 2.4W ในตัว; HomeBase 3 เสียบ 12V ผ่าน USB-C PD
**สเปก:** 4K, มุมมอง 140°, สปอตไลต์+IR, 16 GB eMMC + SSD ขยายได้, รู้จำใบหน้า
**ข้อดีใน RV:** แผงโซลาร์ในตัว (ดูเรียบร้อย), HomeBase 3 ใช้ไฟ 12V, AI ในเครื่อง
**ราคา:** ~$550 (2 กล้อง+HomeBase) | **เหมาะสำหรับ:** นักผจญภัยวันหยุดสุดสัปดาห์

### 4. ดีที่สุดสำหรับ Starlink: Ubiquiti G4 Instant + UniFi Protect

**รูปแบบ:** ลูกบาศก์จิ๋ว (4×4×4 ซม.), สีขาว
**สเปก:** 4K (8MP), 122° FOV, IR 4.5 ม.
**ราคา:** ~$130 | **เหมาะสำหรับ:** คนที่มี UniFi Dream Router + Starlink

### 5. ดีที่สุดสำหรับกริ่งประตู: Ring Door Cam 2 / Ring Stick Up Cam Battery

**ข้อดี:** แบรนด์ดัง = ขออนุมัติจากสวน RV ง่าย; ตัวเลือกสำรอง LTE; ใช้กับ Alexa
**ข้อเสีย:** ต้องสมัครสมาชิกเพื่อบันทึก; 1080p เท่านั้น
**ราคา:** ~$100–180 | **เหมาะสำหรับ:** เสาในสวน RV, RV เช่า

### 6. สไตล์ Dash-Cam: Viofo A229 Pro (สองช่อง)

**ข้อดี:** บันทึกขณะขับรถ _และ_ จอด; GPS; supercapacitor (ไม่บวมในความร้อน)
**ราคา:** ~$300 | **เหมาะสำหรับ:** รถตู้ Class B/C, รถลากพ่วง

## การเชื่อมต่อ: Wi-Fi สวน vs มือถือ vs Starlink

| สถานการณ์                   | หลัก                           | สำรอง                     | อุปกรณ์                     |
| --------------------------- | ------------------------------ | ------------------------- | --------------------------- |
| **เต็มเวลา มีสัญญาณมือถือ** | 4G/5G LTE (Reolink Go PT Plus) | Wi-Fi สวน (Travel Router) | กล้อง + SIM ($10/เดือน)     |
| **วันหยุดสุดสัปดาห์ สวนดี** | Wi-Fi สวน → Travel Router      | Hotspot มือถือ            | Router + Wi-Fi กล้อง        |
| **ห่างไกล**                 | Starlink Roam / Mini           | กล้อง 4G                  | Starlink + Router 12V DC    |
| **สวน RV ถาวร**             | Wi-Fi สวน/ Cable / Fiber       | Failover 4G               | Ubiquiti Dream Router + LTE |

**เคล็ดลับ Travel Router:** GL.iNet GL-MT3000 — USB, ใช้ OpenWrt, ต่อ Wi-Fi สวน, สร้างเครือข่ายย่อยส่วนตัว

## การติดตั้งโดยไม่ต้องเจาะ

| ชนิดการติดตั้ง              | เหมาะสำหรับ                  | แรงยึด     | การถอด                    |
| --------------------------- | ---------------------------- | ---------- | ------------------------- |
| **เทป 3M VHB** (5952, 4910) | หลังคาไฟเบอร์กลาส/อลูมิเนียม | 60+ lb/in² | ปืนเป่าลมร้อน + ไหมขัดฟัน |
| **ถ้วยดูด** (SeaSucker)     | กระจก พื้นผิวเรียบ           | 100+ lb    | ดึง — ทันที               |
| **ฐานแม่เหล็ก**             | หลังคาเหล็ก (Class A)        | 20–50 lb   | ดึง — ทันที               |
| **หนีบ/ราง**                | บันได กันสาด ราวหลังคา       | 50+ lb     | คลายเกลียว — ไม่ทิ้งรอย   |
| **ขาตั้ง/เสา**              | พื้นดิน โต๊ะปิกนิก ชั่วคราว  | N/A        | เก็บ                      |

<Callout type="tip">

<strong>ทดสอบก่อนเชื่อถือ</strong> ติดตั้งกล้อง ขับ 80 กม. ทางหลวง + 16 กม.
ทางขรุขระ ตรวจสอบภาพสั่นไหว

</Callout>

## ระดับงบประมาณ

**ระดับ 1 — ขั้นต่ำ ($200–400):** 1× Reolink Go PT Plus (LTE) + โซลาร์พกพา 10W + SIM 20 GB
**ระดับ 2 — สมดุล ($500–900):** 2× EufyCam 3 + HomeBase 3 + โซลาร์หลังคา 50W
**ระดับ 3 — ครบวงจร ($1,200–2,500):** 4× Reolink RLC-1212A + Injektor PoE 12V + NVR 8 ช่อง + Starlink Mini
