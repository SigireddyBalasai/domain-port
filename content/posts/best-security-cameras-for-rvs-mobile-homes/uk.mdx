## Детальний огляд

### 1. eufyCam S330 (SoliCam) ($220)

**Найкраща сонячна камера без підписки.**

Вбудована сонячна панель + акумулятор 13000 мАг = без підзарядки. Локальне сховище на хабі (без щомісячної плати). Ідеально для RV.

- **Встановлення:** Магнітний кронштейн + 3M VHB
- **Сховище:** Локальне (HomeBase 2) — 16 ГБ
- **Живлення:** Сонячна панель (5 Вт) + акумулятор 13000 мАг
- **Роздільна здатність:** 2K
- **AI:** Людина, домашня тварина, автомобіль
- **Погодостійкість:** IP67

**Мінуси:** Потрібен HomeBase для локального сховища, Wi-Fi тільки.

### 2. Reolink Go PT Ultra ($250)

**Найкраща LTE камера для віддалених місць.**

Якщо ви паркуєтеся далеко від цивілізації, Reolink Go PT Ultra використовує 4G LTE стільниковий зв'язок для передачі відео. Жодного Wi-Fi не потрібно.

- **Встановлення:** Кронштейн + гвинти (або міцне клейке кріплення)
- **Сховище:** microSD до 128 ГБ
- **Живлення:** Акумулятор (5200 мАг) + сонячна панель (опціонально)
- **Роздільна здатність:** 4K (8MP)
- **Зв'язок:** 4G LTE (потрібна SIM-карта)
- **PTZ:** Pan 355°, Tilt 140°

**Мінуси:** Потрібен тарифний план даних, вища ціна.

### 3. Vantrue N4 Pro Dashcam ($200)

**Найкращий запис під час руху.**

Vantrue N4 Pro — це 3-канальна dashcam: перед, салон, зад. Вона записує все під час руху та має режим паркування для виявлення ударів.

- **Встановлення:** Присоска або клейке кріплення на лобове скло
- **Сховище:** microSD до 512 ГБ
- **Живлення:** 12 В (автомобільний прикурювач)
- **Роздільна здатність:** 4K (перед) + 1080p (салон) + 1080p (зад)
- **GPS:** Вбудований GPS для швидкості та маршруту
- **Нічне бачення:** Sony STARVIS 2

**Мінуси:** Тільки для використання в автомобілі, не для периметра.

### 4. Ring Car Cam ($200)

**Найкраща гібридна dashcam + безпека.**

Ring Car Cam — це dashcam з підтримкою LTE та штучного інтелекту. Вона виявляє зломи, надсилає сповіщення на телефон та може зв'язатися з власником через двостороннє аудіо.

- **Встановлення:** Клейке кріплення на лобове скло
- **Сховище:** Хмара (Ring Protect Go $6/міс)
- **Живлення:** 12 В (автомобільний прикурювач)
- **Роздільна здатність:** 1080p HDR
- **Зв'язок:** Wi-Fi + LTE (вбудований)

**Мінуси:** Потрібна підписка Ring Protect Go, тільки передній огляд.

### 5. Arlo Go 2 ($250)

**Найкраща гнучкість (LTE + Wi-Fi).**

Arlo Go 2 може перемикатися між Wi-Fi та LTE залежно від наявності. Коли ви в межах Wi-Fi — використовує його. Немає Wi-Fi? Автоматично перемикається на LTE.

- **Встановлення:** Магнітний кронштейн або гвинти
- **Сховище:** Хмара + локальне (Arlo SmartHub)
- **Живлення:** Акумулятор (3–6 місяців) + сонячна панель (опціонально)
- **Роздільна здатність:** 2K
- **Зв'язок:** 4G LTE + Wi-Fi 2.4/5 ГГц

**Мінуси:** Потрібна підписка Arlo для LTE та хмарного сховища.
