Camper- en stacaravanbeveiliging heeft unieke beperkingen: 12V/24V DC-stroom, constante trillingen, temperaturen van -30°C tot 60°C, beperkte dakruimte en vaak geen permanent internet. Standaard huis-camera's gaan snel kapot. Deze gids behandelt camera's ontworpen voor onderweg.

<Badge variant="outline">Belangrijk Camper Verschil</Badge> **Geen permanente
bedrading toegestaan** in de meeste huur/parken. Accu/zon + 12V DC vast bedraad
zijn uw enige opties.

## Camper-Specifieke Vereisten Checklist

<Callout type="note">

<strong>Must-haves voor camper/mobiel gebruik:</strong> 12V DC directe stroom,
trillingsbestendige montage, IP66+ waterdicht, -20°C tot 60°C werkbereik,
mobiel/LTE optie of sterke Wi-Fi-antenne, microSD lokale opname.

</Callout>

| Functie            | Waarom Belangrijk in Camper               | Doel Specificatie                     |
| ------------------ | ----------------------------------------- | ------------------------------------- |
| Stroomingang       | Huisaccu's zijn 12V DC                    | 12V DC direct (geen omvormerverlies)  |
| Trillingsbestendig | Snelweg schudden breekt standaard beugels | MIL-STD-810G of equivalent            |
| Temperatuurbereik  | Woestijndagen, bergnachten                | -29°C tot 60°C                        |
| Connectiviteit     | Park Wi-Fi is overbelast/onveilig         | 4G LTE / 5G + Wi-Fi dual mode         |
| Opslag             | Geen internet = geen cloud                | microSD (128–512 GB) + loop recording |
| Montage            | Kan dak niet boren op huur                | Zuignap, magnetisch, klem, 3M VHB     |

## Topkeuzes per Categorie

### 1. Beste Algeheel: Reolink Go PT Plus (4G LTE + Accu/Zon)

**Vorm:** Compacte PTZ (5.5" × 3.7" × 3.7"), wit
**Stroom:** Oplaadbare accu (7800 mAh) + optioneel 6W zonnepaneel; **accepteert ook 12V DC via USB-C PD triggerkabel**
**Connectiviteit:** 4G LTE (nano SIM) + 2.4 GHz Wi-Fi dual mode
**Specificaties:** 4K (8MP), 355° pan / 140° tilt, 33 ft IR, persoons-/voertuig-/huisdierdetectie, microSD tot 128 GB
**Camper Winsten:** Echt 12V opladen tijdens rijden, mobiele backup wanneer park Wi-Fi uitvalt, PTZ dekt 360° met één montage, geen abonnement voor lokale opname
**Afwegingen:** SIM-data-abonnement nodig ($5–15/mnd); zonnepaneel voegt dakvoetafdruk toe; pan/tilt motoren kunnen bevriezen onder -23°C
**Prijs:** ~$230 cam + ~$50 zon | **Beste Voor:** Full-timers, boondockers

### 2. Beste 12V Vast Bedraad: Reolink RLC-1212A (PoE) + 12V-naar-PoE Injector

**Vorm:** Bullet (7.5" × 3.1" × 3.1"), wit/zwart
**Stroom:** PoE (802.3af) — gebruik 12V-naar-PoE injector om op huisaccu's te draaien
**Connectiviteit:** Alleen Ethernet
**Specificaties:** 12MP, 122° FOV, 190 ft IR, persoons-/voertuigdetectie, microSD + NVR
**Camper Winsten:** Hoogste detail voor kentekens bij campingingang; nul draadloze interferentie; 24/7 opname naar NVR
**Prijs:** ~$130 cam + ~$25 injector + NVR | **Beste Voor:** Class A/B moterhomes, permanente parkmodellen

### 3. Beste Budget Draadloos: EufyCam 3 S330 + HomeBase 3

**Stroom:** Accu (13.000 mAh) + ingebouwde 2.4W zon; HomeBase 3 werkt op 12V via USB-C PD
**Connectiviteit:** Wi-Fi 6 (2.4/5 GHz) naar HomeBase
**Specificaties:** 4K, 140° FOV, spotlight + IR, 16 GB eMMC + SSD, gezichtsherkenning
**Prijs:** ~$550 (2-cam + HomeBase) | **Beste Voor:** Weekend-strijders, zonnige locaties

### 4. Beste voor Mobiele Hotspot/Starlink: Ubiquiti G4 Instant

**Vorm:** Klein blokje (1.6" × 1.6" × 1.6"), wit
**Stroom:** 5V USB-C (gebruik 12V→5V buck converter)
**Specificaties:** 4K (8MP), 122° FOV, 15 ft IR | **Prijs:** ~$130

### 5. Beste Deurbel/Ingang: Ring Door Cam 2 / Ring Stick Up Cam Battery

**Stroom:** Accu (6–12 mnd); 12V bedraad via Ring Plug-In Adapter
**Specificaties:** 1080p HDR, 155° FOV, tweerichtingsaudio
**Prijs:** ~$100–180 | **Beste Voor:** Campingpalen, huurcampers

### 6. Beste Dashcam-Stijl: Viofo A229 Pro (Dual Channel)

**Specificaties:** Voor 4K 60fps + Achter 2K; Sony STARVIS 2
**Prijs:** ~$300 | **Beste Voor:** Bestelwagens, trekkende voertuigen

## Stroomstrategieën voor Campers

Vuistregel: Paneel watts ≥ Camera verbruik (W) × 3 (voor winter/bewolkt).
Reolink Go PT Plus verbruikt ~3W gem → 10W paneel minimum, 20W aanbevolen.

Montage:

<ul>
  <li>
    <strong>Draagbaar opvouwbaar paneel</strong> (Renogy 20W): Opzetten bij

  </li>
  <li>
    <strong>Dak vast</strong> (Renogy 50W/100W): Laadt huisaccu's → voedt
    camera's via 12V-bus.
  </li>
  <li>
    <strong>Flexibel semi-stijf</strong> (SunPower 50W/100W): Buigt mee met dak,
    lijm (3M VHB).
  </li>
</ul>

<strong>Nooit AC-omvormer 24/7 gebruiken voor camera's.</strong> 120V AC → 12V

## Connectiviteit: Park Wi-Fi vs Mobiel vs Starlink

| Scenario                       | Primair                    | Backup                  | Hardware                    |
| ------------------------------ | -------------------------- | ----------------------- | --------------------------- |
| Full-time, mobiele dekking     | 4G/5G LTE camera           | Park Wi-Fi (reisrouter) | Camera + SIM ($10/mnd)      |
| Weekend-strijder, goede parken | Park Wi-Fi → Reisrouter    | Telefoon hotspot        | Router + cam Wi-Fi          |
| Afgelegen/boondocking          | Starlink Roam / Mini       | 4G camera               | Starlink + 12V DC router    |
| Permanent parkmodel            | Park Wi-Fi / Kabel / Fiber | 4G failover             | Ubiquiti Dream Router + LTE |

## Montage Zonder Boren

| Montagetype           | Beste Voor                     | Houdkracht | Verwijdering                 |
| --------------------- | ------------------------------ | ---------- | ---------------------------- |
| **3M VHB Tape**       | Fiberglast/rvs dak, glad       | 60+ lb/in² | Hetelucht + flos—geen residu |
| **Zuignap**           | Glas, gelcoat, tijdelijke mont | 100+ lb    | Loslaat lipje—direct         |
| **Magnetische Basis** | Stalen dak/zijkanten           | 20–50 lb   | Trek—direct                  |
| **Klem / Rail**       | Ladder, luifelarm, dakrek      | 50+ lb     | Losschroeven—geen sporen     |

<Callout type="tip">

Test de montage door 50 mijl snelweg + 10 mijl wasbord te rijden voordat u
erop vertrouwt.

</Callout>
