Armazenamento é onde sistemas de câmeras vingam ou morrem. Perdeu filmagem = perdeu evidência. Nuvem oferece conveniência mas custa para sempre e arrisca privacidade. Local (NVR, microSD, NAS) custa adiantado mas é seu. Híbrido dá ambos — se configurado certo. Este guia detalha cada opção com preços reais de 2026, cálculos de largura de banda e frameworks de decisão.

<Badge variant="outline">Resposta Rápida</Badge> **Maioria das casas:** PoE NVR
(local 24/7) + backup em nuvem apenas para alertas. **Apartamentos/Inquilinos:**
Câmera com microSD + assinatura de nuvem opcional. **Negócios:** NVR + NAS +
offsite (Wasabi/Backblaze B2) + clipes de alerta na nuvem.

| **microSD (Borda)**            | Todas câmeras wireless, algumas PoE                               | Cartão uma vez    | Dias-Semanas | Você tem chaves   | Não                 |
| **NAS / Servidor**             | Synology, QNAP, TrueNAS, Unraid + Frigate/Scrypted                | Hardware + discos | Anos         | Você tem chaves   | Opcional            |
| **Armazenamento Objeto Nuvem** | Wasabi, Backblaze B2, AWS S3, Cloudflare R2                       | $/TB/mês          | Infinito     | Você criptografa  | Sim (só backup)     |
| **Híbrido**                    | UniFi Protect + Cloud Backup, Reolink NVR + FTP, Frigate + rclone | Misto             | Em camadas   | Você controla     | Parcial             |

<Callout type="warning">

**Realidade da Largura de Banda:** 8× câmeras 4K @ 8 Mbps H.265 = 64 Mbps **upload contínuo**. Maioria das residências tem 10-35 Mbps de upload. Nuvem 24/7 completa é impossível em internet típica. Nuvem = apenas clipes de movimento ou sub-stream.

</Callout>

## Arquiteturas Híbridas

### 1. UniFi Protect + Backup na Nuvem (Mais Fácil)

- **Local:** UNVR (7× HDD, RAID 6) → 24/7 4K
- **Nuvem:** Protect Cloud Backup ($0.50/cam/mês) → só clipes de evento
- **Custo:** $1,000 hardware + $30/ano (5 cams) = **Ano 1 $1,150, depois $30/ano**

### 2. Reolink NVR + Upload FTP/S3 (Híbrido Econômico)

- **Local:** RLN16-410 (16 canais, 2× SATA) → 24/7
- **Nuvem:** Firmware NVR → FTP/S3 (Wasabi/B2) no movimento
- **Custo:** $300 NVR + $84/ano Wasabi (1TB) = **Ano 1 $384, $84/ano**

### 3. Frigate + rclone (DIY Controle Máximo)

- **Local:** Frigate (Intel N100 + 4× 10TB RAID 5) → 24/7 + eventos IA
- **Nuvem:** `rclone sync` cron diário → Wasabi (criptografado)
- **Custo:** $600 hardware + $84/ano Wasabi = **Ano 1 $684, $84/ano**

## Cenários de Ataque & Implicações na Retenção

| Cenário                           | Nuvem                      | NVR Local                |
| --------------------------------- | -------------------------- | ------------------------ |
| **Ladrão leva o NVR**             | ✔️ Vídeos seguros na nuvem | ❌ Perdeu tudo           |
| **Incêndio/inundação em casa**    | ✔️ Seguro fora do local    | ❌ Perda total           |
| **Internet cai por 3 dias**       | ❌ Sem carregamento        | ✔️ Gravação continua     |
| **Ladrão corta cabo de internet** | ❌ Sem carregamento        | ✔️ Sempre gravando (PoE) |
| **NVR com falha de HDD**          | ✔️ Vídeos seguros na nuvem | ❌ Perdeu gravação       |
| **Ladrão desliga energia**        | ❌ Se sem bateria backup   | ❌ Se sem bateria backup |
| **Servidor violado**              | ❌ Variável                | ✔️ Apenas local          |
