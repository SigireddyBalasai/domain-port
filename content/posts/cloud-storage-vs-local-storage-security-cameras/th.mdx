พื้นที่จัดเก็บคือจุดที่ระบบกล้องอยู่หรือตาย ภาพหาย = หลักฐานหาย คลาวด์สะดวกแต่เสียเงินตลอดไปและเสี่ยงความเป็นส่วนตัว Local (NVR, microSD, NAS) จ่ายครั้งเดียวแต่คุณเป็นเจ้าของ ไฮบริดให้ทั้งสองอย่าง—ถ้าตั้งค่าถูกต้อง

<Badge variant="outline">เส้นแบ่งสำคัญ</Badge> คลาวด์ = จ่ายรายเดือนไปเรื่อย ๆ
แต่ไม่ต้องจัดการฮาร์ดแวร์ Local = จ่ายครั้งเดียว แต่คุณต้องดูแลเอง ระยะเวลา 3–5
ปี คลาวด์อาจแพงกว่า NVR ถึง 2–3 เท่า

## ภาพรวมพื้นที่จัดเก็บ (2026)

| หมวดหมู่           | ตัวอย่าง                 | รูปแบบราคา          | เก็บกี่วัน  | ความเป็นส่วนตัว      | ต้องใช้อินเทอร์เน็ต   |
| ------------------ | ------------------------ | ------------------- | ----------- | -------------------- | --------------------- |
| สมัครคลาวด์        | Ring, Arlo, Nest, Eufy   | $/ตัว/เดือน         | 7–60 วัน    | ผู้ให้บริการถือกุญแจ | ใช่ (อัปโหลดตลอด)     |
| NVR/DVR (Local)    | Reolink, Amcrest, Dahua  | ฮาร์ดแวร์ครั้งเดียว | เดือน–ปี    | คุณถือกุญแจ          | ไม่                   |
| microSD (Edge)     | กล้องไร้สาย, PoE บางรุ่น | การ์ดครั้งเดียว     | วัน–สัปดาห์ | คุณถือกุญแจ          | ไม่                   |
| NAS / Server       | Synology, QNAP + Frigate | ฮาร์ดแวร์+ไดรฟ์     | ปี          | คุณถือกุญแจ          | ไม่บังคับ             |
| Cloud Object Store | Wasabi, B2, S3, R2       | $/TB/เดือน          | ไม่จำกัด    | คุณเข้ารหัส          | ใช่ (backup เท่านั้น) |

## ข้อดีข้อเสีย

### คลาวด์

| ข้อดี                                   | ข้อเสีย                            |
| --------------------------------------- | ---------------------------------- |
| ไม่ต้องจัดการฮาร์ดแวร์                  | ค่าใช้จ่ายรายเดือนตลอดไป           |
| เข้าถึงจากที่ไหนก็ได้                   | ต้องใช้อินเทอร์เน็ตความเร็วสูง     |
| อัปเดตและสำรองข้อมูลอัตโนมัติ           | ความเป็นส่วนตัว—บริษัทถือกุญแจ     |
| การแจ้งเตือนและ AI ขั้นสูง              | ค่าใช้จ่ายเพิ่มเมื่อมีกล้องมากขึ้น |
| ไม่มีความเสี่ยงสูญเสียจากไฟไหม้/น้ำท่วม | การพึ่งพาผู้ให้บริการ—อาจปิดบริการ |

### Local (NVR/NAS)

| ข้อดี                                  | ข้อเสีย                             |
| -------------------------------------- | ----------------------------------- |
| จ่ายครั้งเดียว ไม่มีค่าใช้จ่ายรายเดือน | ต้องตั้งค่าและบำรุงรักษาเอง         |
| ความเป็นส่วนตัวสมบูรณ์—คุณถือกุญแจ     | เสี่ยงสูญเสียจากไฟไหม้/น้ำท่วม/ขโมย |
| ไม่ต้องใช้อินเทอร์เน็ต                 | เข้าถึงจากระยะไกลต้องตั้งค่า VPN    |
| ไม่มีค่าใช้จ่ายแอบแฝง                  | ต้องเปลี่ยน HDD ทุก 3–5 ปี          |
| ความจุไม่จำกัด (เพิ่ม HDD ได้)         | AI และการแจ้งเตือนอาจจำกัด          |

## โซลูชันไฮบริดที่ดีที่สุด

### 1. Frigate + NAS + Wasabi

| ส่วนประกอบ                | ราคา        | วัตถุประสงค์                        |
| ------------------------- | ----------- | ----------------------------------- |
| เครื่อง Frigate (Mini PC) | $200–400    | AI ตรวจจับภายในเครื่อง, จัดการกล้อง |
| NAS (Synology/QNAP)       | $300–600    | จัดเก็บภาพภายในเครื่อง 30 วัน       |
| HDD สำหรับ NAS            | $200–400    | 8–16 TB การจัดเก็บ                  |
| Wasabi (สำรอง)            | $7/TB/เดือน | เก็บ 90 วันสำหรับเหตุการณ์สำคัญ     |

### 2. NVR + สำรองคลาวด์

1. NVR จัดเก็บบันทึกต่อเนื่อง 7–14 วัน
2. อัปโหลดเฉพาะเหตุการณ์สำคัญ (ตรวจจับคน/รถ) ไปยัง Wasabi หรือ B2
3. ต้นทุนคลาวด์: $5–15/เดือน สำหรับ ~500 GB
