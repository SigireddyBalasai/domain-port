Правильне встановлення камери безпеки вимагає більше ніж просто просвердлити отвори. Цей посібник охоплює весь процес: від планування та вибору обладнання до монтажу, прокладки кабелю, термінації та введення в експлуатацію.

<Badge variant="outline">Час виконання</Badge> **4–8 годин** для 4–8 камер DIY.
**1–2 дні** якщо потрібне свердління цегли, горище, канали.

---

## Фаза 1: Планування

### Контрольний список перед встановленням

- [ ] **Визначте зони покриття** — Намалюйте карту властивості. Позначте всі точки входу, сліпі зони, цінні активи.
- [ ] **Виберіть тип камери** — PoE (найнадійніший), Wi-Fi (найлегший), Акумулятор (гнучкий).
- [ ] **Перевірте Wi-Fi сигнал** (для бездротових) — RSSI > -60 dBm біля кожної камери.
- [ ] **Виміряйте маршрути кабелю** — Додайте 3 м запасу на кожен маршрут.
- [ ] **Перевірте бюджет PoE** — Σ споживання камер × 1.2 ≤ бюджет комутатора.
- [ ] **Перевірте місцеві норми** — Дозволи, правила HOA, закони про аудіозапис.
- [ ] **Складіть список інструментів** — Перфоратор, свердла, кримпер, тестер кабелю, детектор стійок.

| Зона               | Тип камери               | Кут огляду | Висота монтажу | Примітки                         |

| Вхідні двері       | Дверний дзвінок / Bullet | 120–180°   | 1.2–1.5 м      | Захоплення обличчя на рівні очей |
| Під'їзд / Дорога   | Bullet (варіфокальний)   | 60–90°     | 3–4.5 м        | Захоплення номерів               |
| Заднє подвір'я     | Dome / Turret            | 100–130°   | 3–4 м          | Широке покриття                  |
| Гараж              | Dome                     | 100–130°   | 3 м            | Виявлення дверей та вікон        |
| Сліпі зони (кути)  | PTZ або fisheye          | 180–360°   | 3–4.5 м        | Активне спостереження            |
| Внутрішній простір | Dome (непомітний)        | 90–110°    | 2.5–3 м        | Стеля, спрямований на вхід       |

1. **Позначте позицію** — Тримайте кронштейн на місці, позначте через монтажні отвори
2. **Пілотне свердління** — Використовуйте правильний біт для поверхні
3. **Кріплення кронштейна** — Шурупи в стійку або анкери в бетон
4. **Ущільнення** — Бутилова стрічка під кронштейном, силікон навколо кріплень
5. **Підключення кабелю** — Протягніть кабель через кронштейн, залиште 30 см запасу
6. **Встановлення камери** — Прикрутіть камеру до кронштейна, вирівняйте
7. **Регулювання кута** — Нахиліть вниз 15–30°, спрямуйте на цільову зону

| Ціль                   | Висота | Кут нахилу | Чому                                          |
| ---------------------- | ------ | ---------- | --------------------------------------------- |

## Фаза 4: Прокладка кабелю

### Маршрутизація

| Маршрут                    | Складність | Час на камеру | Інструменти                   |
| -------------------------- | ---------- | ------------- | ----------------------------- |
| Горище → Підшивка          | Легко      | 20–30 хв      | Стрижні, свердел ⅞"           |
| Підвал → Зовнішня стіна    | Середньо   | 30–45 хв      | Стрижні, свердел, вогнезахист |
| Внутрішня стіна → Зовнішня | Середньо   | 30–60 хв      | Стрижні, коробка, герметик    |
| Пряме закопування          | Важко      | 60–90 хв      | Лопата, PVC канал, стрічка    |

### Правила прокладки

- **Категорія:** Тільки Cat6 суцільна мідь (не CCA)
- **Максимальна довжина:** 100 м (328 футів) для PoE+
- **З'єднання:** Ніяких з'єднань — один безперервний кабель від комутатора до камери
- **Паралельно до силових:** Мінімум 30 см від силових кабелів
- **Перетин силових:** Тільки під кутом 90°
- **Скоби:** Кожні 1.2 м, не притискайте кабель

## Фаза 5: Термінація

### Обтискання RJ45 (T568B)

1. Зніміть 38 мм оболонки
2. Розкрутіть пари, розташуйте в порядку T568B
3. Вставте в конектор pass-through
4. Перевірте порядок через прозорий конектор
5. Обтисніть храповим інструментом
6. Тест на натяг (2.5 кг)

### Перевірка тестером

- **Усі 8 зелених** = прохід
- **Будь-який червоний** = перетермінуйте
- **PoE тест:** Напруга на кінці камери > 44V (PoE) / > 50V (PoE+)

## Фаза 6: Налаштування NVR

### Початкове налаштування

1. **Підключіть NVR** до монітора (HDMI), миші, мережі
2. **Увімкніть** — Зачекайте 2–3 хв для завантаження
3. **Майстер налаштування** — Мова, дата/час, пароль адміністратора
4. **Ініціалізуйте HDD** — Форматуйте встановлений диск
5. **Виявлення камер** — NVR автоматично знайде камери в тій же мережі
6. **Призначте IP** — Статичний IP або DHCP резервація

### Конфігурація запису

| Параметр            | Рекомендація                  |
| ------------------- | ----------------------------- |
| Розклад запису      | Безперервний 24/7             |
| Роздільна здатність | 4K (основний потік)           |
| Кодек               | H.265                         |
| Бітрейт             | CBR 4–6 Мбіт/с (4K)           |
| FPS                 | 15 (баланс якості та сховища) |
| Виявлення руху      | Людина + Транспорт            |
| Час передзапису     | 5–10 секунд                   |
| Час післязапису     | 10–15 секунд                  |

## Фаза 7: Введення в експлуатацію

### Контрольний список перевірки

- [ ] **Живий перегляд** — Всі камери відображаються та оновлюються
- [ ] **Кут огляду** — Кожна камера покриває заплановану зону
- [ ] **Нічне бачення** — ІЧ вмикається в темряві, зображення чітке
- [ ] **Запис** — NVR записує безперервно та по руху
- [ ] **Відтворення** — Перевірте відтворення записаного відео
- [ ] **Сповіщення** — Push сповіщення працюють на телефоні
- [ ] **Віддалений доступ** — Додаток або VPN показує камери
- [ ] **Аудіо** (якщо є) — Двостороннє аудіо працює
- [ ] **PTZ** (якщо є) — Pan, tilt, zoom, preseti
- [ ] **Документація** — Сфотографуйте термінації, запишіть паролі

## Усунення поширених проблем

| Проблема                    | Вирішення                                   |
| --------------------------- | ------------------------------------------- |
| Камера не виявляється       | Перевірте кабель, PoE живлення, IP конфлікт |
| Немає зображення вночі      | Перевірте ІЧ, кут, відбиття                 |
| Прогалини в записі          | Перевірте HDD, розклад запису               |
| Сповіщення не приходять     | Перевірте зони руху, дозволи додатку        |
| Повільний віддалений доступ | Перевірте вихідну швидкість, QoS            |
