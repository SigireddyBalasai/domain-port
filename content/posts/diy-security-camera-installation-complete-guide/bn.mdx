### 3. কেবল রুট পরিকল্পনা (PoE)

- **অভ্যন্তরীণ দেয়াল:** টপ প্লেট ড্রিল করুন, বেসমেন্ট/ক্রল স্পেসে ফিশ করুন, অ্যাটিকে উঠান
- **বহির্বহি দেয়াল:** রিম জোইস্ট বা সিল প্লেট দিয়ে ড্রিল করুন, সিলিকন দিয়ে সিল করুন
- **অ্যাটিক রুট:** রাফটার বরাবর চালান, প্রতি 4 ফুটে স্ট্যাপল করুন, ইনসুলেশন স্পর্শ এড়ান
- **কন্ডুইট:** এক্সপোজড রানের জন্য ¾" EMT বা PVC (অনেক এলাকায় কোড প্রয়োজন)
- **সর্বোচ্চ দৈর্ঘ্য:** Cat5e/6 প্রতি রানে 328 ফুট (100 মি)। এর বেশি দূরে PoE এক্সটেন্ডার বা ফাইবার ব্যবহার করুন।

### 4. টুল ও ম্যাটেরিয়াল চেকলিস্ট

**অত্যাবশ্যক টুলস:**

- [ ] কর্ডলেস ড্রিল/ড্রাইভার (18V+)
- [ ] ½" স্পেড বিট / ⅞" অগার বিট (কেবল ছিদ্র)
- [ ] ফিশ টেপ / গ্লো রড (25–50 ফুট)
- [ ] স্টাড ফাইন্ডার (AC সেন্সিং)
- [ ] লেভেল (টর্পেডো বা লেজার)
- [ ] মই (দোতলার জন্য এক্সটেনশন)
- [ ] ক্রিম্প টুল (RJ45 পাস-থ্রু) + কেবল টেস্টার
- [ ] কেবল স্ট্রিপার (Cat5e/6)
- [ ] সিলিকন সিল্যান্ট (বহির্বহি গ্রেড)
- [ ] কেবল স্ট্যাপল / J-হুক / জিপ টাই

**ম্যাটেরিয়াল (PoE):**

- [ ] Cat6 সলিড কপার (দেয়ালের ভেতরে CMR রাইজার, বাইরে CMX আউটডোর)
- [ ] RJ45 পাস-থ্রু কানেক্টর (50-প্যাক)
- [ ] Keystone জ্যাক + ওয়াল প্লেট (ঐচ্ছিক, পরিষ্কার টার্মিনেশনের জন্য)
- [ ] PoE সুইচ (8/16/24 পোর্ট, 802.3at/bt) বা PoE পোর্ট সহ NVR
- [ ] NVR + সুইচের জন্য UPS (1500VA ন্যূনতম)

**ম্যাটেরিয়াল (ওয়্যারলেস/ব্যাটারি):**

- [ ] মাউন্টিং ব্র্যাকেট (সহ আসে বা ইউনিভার্সাল)
- [ ] সোলার প্যানেল + মাউন্টিং আর্ম
- [ ] 120V আউটডোর আউটলেট / ওয়েদারপ্রুফ বক্স (প্লাগ-ইন হলে)

### সিলিং / সফিট / ইভ মাউন্ট

- **জাংশন বক্স** (ওয়েদারপ্রুফ, 4" গোল) → ক্যামেরা ব্র্যাকেট বক্সে স্ক্রু
- **কন্ডুইট** (½" EMT বা LFMC) বক্স থেকে ক্যামেরায় (কেবল উন্মুক্ত থাকলে)
- **সিল:** বক্সের ছিদ্র + কেবল এন্ট্রিতে সিলিকন

### পোল মাউন্ট (ড্রাইভওয়ে, গেট, ইয়ার্ড)

- **Schedule 40 স্টিল পাইপ** (1.5" বা 2" ID) → কংক্রিটে বসান (গভীর 18" × ব্যাস 8")
- **U-বোল্ট ব্র্যাকেট** ক্যামেরার জন্য
- **কন্ডুইট** পাইপের ভেতর বা সারফেস-মাউন্টেড LFMC
- **গ্রাউন্ডিং:** পোল বেসে #6 CU গ্রাউন্ড রডে (NEC 250.52)

| **বার্ষিক**   | বাহিরের সিলিকন পুনরায় সিল; মাউন্ট স্ক্রু শক্ত করুন; ক্যামেরা কোণ পুনরায় মূল্যায়ন (গাছ বড় হয়); UPS ব্যাটারি পরিবর্তন (3–5 বছর আয়ু) |
