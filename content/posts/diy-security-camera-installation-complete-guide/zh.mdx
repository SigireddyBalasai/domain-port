专业安防摄像头安装费用为 **$150–$400/摄像头** (人工 + 材料)。典型 8 摄像头 PoE 系统: **$1,200–$3,200 人工**。本指南 DIY: **$0 人工 + $100–$300 工具**。从规划到首次录像, 覆盖每一步。

<Badge variant="outline">诚实评估</Badge> **DIY 适合你如果:** 你习惯爬梯子、打孔、基础网络、看懂图纸。**找专业如果:** 多层屋顶布线、混凝土/砖墙打孔、需许可证、想要人工保修。

### 2. 选择架构

| 架构 | 适用场景 | 布线 | 供电 | 难度 |
|--------------|----------|------------|-------|------------|
| **PoE NVR 套件** | 4–16 机, 永久, 24/7 | 每机 Cat5e/6 | PoE (交换机/NVR) | 中等 |
| **Wi-Fi + NVR** | 2–8 机, 难布线点 | 仅供电 (插座/太阳能) | 120V / 电池 / 太阳能 | 低–中等 |
| **电池/太阳能** | 1–4 机, 无线, 租房 | 无 | 电池 + 太阳能板 | 低 |
| **混合** | 大多数家庭 | 混合 | 混合 | 中等 |

<Callout type="tip">
**能用 PoE 就用 PoE。** 最可靠、画质最高、长期成本最低。仅在布线真正不切实际时用 Wi-Fi/电池。
</Callout>

### 3. 线缆路由规划 (PoE)

- **内墙:** 钻顶板, 从地下室/地爬空间鱼线下, 再上阁楼
- **外墙:** 钻过檐口板或门槛板, 硅胶密封
- **阁楼走线:** 沿椽子, 每 4 英尺订书钉, 避开绝缘棉接触
- **管道:** ¾" EMT 或 PVC 用于明装 (多地规范要求)
- **最大长度:** Cat5e/6 单段 328 英尺 (100 米). 超过用 PoE 延长器或光纤.

**必备工具:**
- [ ] 无绳钻/驱动器 (18V+)
- [ ] ½" 铲形钻 / ⅞" 螺旋钻 (穿线孔)
- [ ] 钢丝牵引带 / 荧光杆 (25–50 英尺)
- [ ] 寻木器 (交流感应)
- [ ] 水平仪 (鱼雷或激光)
- [ ] 梯子 (双层用伸缩)
- [ ] RJ45 压线钳 (直通式) + 线缆测试仪
- [ ] 网线剥线器 (Cat5e/6)
- [ ] 硅胶密封剂 (外用级)
- [ ] 线缆订书钉 / J 型钩 / 扎带

**材料 (PoE):**
- [ ] Cat6 纯铜 (CMR 阻燃进墙, CMX 户外) 
- [ ] RJ45 直通连接器 (50 包)
- [ ] Keystone 插孔 + 面板 (可选, 终端整洁)
- [ ] PoE 交换机 (8/16/24 口, 802.3at/bt) 或带 PoE 口 NVR
- [ ] NVR + 交换机 UPS (1500VA 起)

**材料 (无线/电池):**
- [ ] 安装支架 (随机或通用)
- [ ] 太阳能板 + 安装臂

### 天花板/屋檐/屋檐口安装

- **接线盒** (防水, 4" 圆) → 摄像头支架拧盒
- **管道** (½" EMT 或 LFMC) 从盒到摄像头 (外露走线时)
- **密封:** 盒子穿墙处 + 进线口全硅胶

### 立杆安装 (车道、大门、院子)

- **Schedule 40 钢管** (1.5" 或 2" 内径) → 浇筑混凝土 (深 18" × 直径 8")
- **U 型螺栓支架** 固定摄像头
- **管道** 杆内穿或表面 LFMC
- **接地:** #6 铜线到立杆底部接地棒 (NEC 250.52)

## 第 3 阶段: 走线 (PoE 系统)

### 阁楼到屋檐 (最常见)

<Accordion type="single" collapsible>
  <AccordionItem value="cable-run">
    <AccordionTrigger>分步: 阁楼 → 屋檐摄像头</AccordionTrigger>
    <AccordionContent>
**工具:** ⅞" 铲钻/螺旋钻, 50' 荧光杆, 头灯, 护膝, 防尘口罩

1. **从阁楼内定位摄像头位置** — 从已知地标 (通风口、烟囱) 量距
2. **钻 ⅞" 孔** 穿顶板 + 屋檐板 (微微向下倾斜利于排水)
3. **推荧光杆** 从阁楼穿孔到屋檐外
4. **胶带粘 Cat6** 到杆头: 折回 6", 胶带抹平 (无挂住)
5. **轻柔拉** — 双人配合: 一人喂线, 一人拉
6. **摄像头端留 3 英尺松线** (维修环), 阁楼端 2 英尺
7. **密封:** 屋檐穿线处内外打硅胶
8. **每 4 英尺订线** 到椽子 (T25 绝缘订书钉)
    </AccordionContent>
  </AccordionItem>
</Accordion>

### 终端: RJ45 直通 (最快最稳)

<Callout type="note">
**用直通连接器.** 标准 RJ45 需精准剪线. 直通式压线前能核对色序. **工具:** 棘轮直通压线钳 (Klein VDV226-110, $45).
</Callout>

### T568B 线序 (PoE 标准)

| Pin | Color | Pin | Color |
|-----|-------|-----|-------|
| 1 | Orange/White | 5 | Blue/White |
| 2 | Orange | 6 | Green |
| 3 | Green/White | 7 | Brown/White |
| 4 | Blue | 8 | Brown |

**PoE 用全 4 对** (802.3bt). **8 根导线必须完美.**

### 压线步骤 (直通式)

1. 剥 1.5" 外皮 → 解绞仅需长度 → 排 T568B
2. 插入直通头 → 透明头前端看清色序
3. 齐头剪齐连接器面
4. 压线 **用力** (棘轮单次咔哒)
5. **每根测** — 8 灯全绿 = 过

### PoE 交换机 / NVR 连接

- **每机回程** 到交换机/NVR (别串联)
- **配线架** (可选但专业): 12/24 口 keystone → 短跳线到交换机
- **UPS:** NVR + PoE 交换机上电池备用. 断电摄像头仍供电.

## 第 4 阶段: 无线/电池摄像头设置

### Wi-Fi 优化

<Accordion type="single" collapsible>
  <AccordionItem value="wifi-opt">
    <AccordionTrigger>获取摄像头可靠 Wi-Fi</AccordionTrigger>
    <AccordionContent>
**位置:**
- 摄像头距路由/AP **50 英尺内** (穿 ≤2 墙)
- 避免: 金属侧板、烟囱、金属风管、镜子、鱼缸

**硬件升级:**
- **Mesh 系统** (Eero Pro 6E, UniFi Dream Router, Asus ZenWiFi) — 有线回程优先
- **户外 AP** (UniFi U6 Mesh, UAP-AC-M) — PoE 供电, 檐下安装
- **定向天线** (2.4 GHz Yagi) 超远单摄像头

**信道规划:**
- 2.4 GHz: 仅信道 1, 6, 11 (不重叠)
- 强制摄像头用 2.4 GHz (穿墙强) — App 里关 5 GHz
- 路由器 2.4 GHz 强制 **20 MHz 宽度** (非 40) 稳定

**测试:** 手机站摄像头位 → Speedtest → 单摄 **>5 Mbps 上传** (1080p) / **>15 Mbps** (4K)
    </AccordionContent>
  </AccordionItem>
</Accordion>

### 电池/太阳能调试

1. **安装前满充** (USB-C 12–24 小时)
2. **安装高度配对 App** (测 Wi-Fi 信号)
3. **装支架** → 装摄像头 → 调角度
4. **太阳能板:** **朝真南** (北半球), 倾角 = 纬度 ±15° (冬 +15°, 夏 -15°)
5. **验证充电:** 1 小时内阳光下 App 显示 ☀️ / ⚡
6. **设定计划:** 夜间仅运动触发; 关实时预览自动启动

## 第 5 阶段: NVR / VMS 配置

### 首次启动 (Reolink / Amcrest / Dahua / Hikvision / UniFi)

<Accordion type="single" collapsible>
  <AccordionItem value="nvr-setup">
    <AccordionTrigger>NVR 首启清单</AccordionTrigger>
    <AccordionContent>
1. **装硬盘** 先通电 (SATA, WD Purple / Seagate SkyHawk / Toshiba S300)
2. **开机** → 向导: 日期/时间 (NTP), 语言, 密码 (强!)
3. **网络:** NVR 静态 IP (如 192.168.1.50), 网关, DNS (1.1.1.1)
4. **PoE 口:** 全开 → "Auto Add" 摄像头 (逐个插入)
5. **编码设置 (每摄像头):**
   - **主流:** 4K/8MP @ 15–20 fps, H.265, VBR, 4–6 Mbps
   - **子流:** 720p/D1 @ 15 fps, H.265, 0.5–1 Mbps (手机实时看)
6. **录制计划:** 主流 24/7 连续; 子流动检 (省空间)
7. **运动区域:** 仅画相关区 (排除树、街道、云)
8. **智能事件:** 开 Person/Vehicle 检测 (误报降 90%)
9. **存储:** 格式化硬盘 → 覆盖模式 (最老优先) → 开 S.M.A.R.T. 报警
10. **远程访问:** **切勿端口映射.** 用:
    - **厂商云** (Reolink, Dahua, Hikvision, UniFi) — 最易
    - **VPN** (路由器 WireGuard, Tailscale, ZeroTier) — 最安全
    - **Tailscale Funnel / Cloudflare Tunnel** — 进阶, 免客户端
    </AccordionContent>
  </AccordionItem>
</Accordion>

### 硬盘选择与 RAID

| 硬盘 | 容量 | 年负载 | 保修 | 最适合 |
|-------|----------|---------------|----------|----------|
| WD Purple | 1–18 TB | 180 TB/yr | 3 yr | 标准 NVR |
| Seagate SkyHawk | 1–20 TB | 180 TB/yr | 3 yr | 标准 NVR |
| WD Purple Pro | 8–22 TB | 300 TB/yr | 5 yr | RAID 5/6, 8+ 盘位, AI 分析 |
| Seagate Exos X / IronWolf Pro | 4–24 TB | 300–550 TB/yr | 5 yr | 高容 RAID |
| Toshiba S300 / N300 | 4–18 TB | 180 TB/yr | 3 yr | 预算 RAID |

<Callout type="warning">
**切勿用:** 桌面盘 (WD Blue, Seagate Barracuda), NAS 盘 (WD Red, Seagate IronWolf) — 不适合 24/7 顺序写。**监控盘固件针对持续写流优化.**
</Callout>

**RAID 配置:** 4 盘位 → RAID 5 (坏 1 盘). 6+ 盘位 → RAID 6 (坏 2 盘). **永不 RAID 0.**

## 第 6 阶段: 网络安全 (关键)

<Callout type="danger">
**切勿端口映射摄像头/NVR 端口 (80, 443, 8000, 554, 8080).** Shodan/Censys 7×24 扫描. 默认密码 = 几分钟成僵尸网络节点.
</Callout>

**安全远程访问 (好 → 佳):**

| 方式 | 安全性 | 易用度 | 成本 | 平台 |
|--------|----------|------|------|----------|
| **厂商云 (UniFi, Reolink, Eufy, Ring)** | 好 (TLS, 认证) | 最易 | 免费 | 全部 |
| **Tailscale / ZeroTier** (VPN mesh) | 极佳 (WireGuard) | 中等 | 免费 (个人) | 全部 |
| **路由器 WireGuard** (OpenWrt, OPNsense, pfSense) | 极佳 | 较难 | 免费 | 全部 |
| **Cloudflare Tunnel** (Zero Trust) | 极佳 | 中等 | 免费 | 仅 Web UI |
| **路由器 OpenVPN Server** | 好 | 中等 | 免费 | 全部 |

**摄像头级加固:**
- 改 **admin 密码** (16+ 字符, 密码管理器)
- 禁用 **ONVIF** 若不用
- 禁用路由器 **UPnP**
- 强制 **HTTPS only** (自签名证书本地 OK)
- **固件季度更新** (厂商官网, 不要自动)

## 故障速查表

| 症状 | 可能原因 | 修复 |
|---------|--------------|-----|
| **无画面 / "No Signal"** | PoE 功率预算超 | 查交换机 PoE 预算; 换注入器 |
| **图像闪烁 / 滚动条** | 接地环路 / 接地差 | 所有设备共地; 隔离器 |
| **夜视白化** | IR 反射 | 向下倾 15–30° / 离墙檐 12–18" / 关内置 IR + 外置补光 |
| **运动警报刷屏** | 树、影子、雨、虫 | 收窄区域; 仅开 Person/Vehicle; 降灵敏度 |
| **Wi-Fi 掉线** | 信道拥堵 / 距离 | 仅 2.4 GHz; 20 MHz; 摄像头附近加 Mesh 节点; 定向天线 |
| **NVR 蜂鸣** | 硬盘故障 / 满 / 风扇 | 查 S.M.A.R.T.; 换盘; 清风扇 |
| **远程访问失败** | ISP CGNAT / 双层 NAT | 用 Tailscale / Cloudflare Tunnel / 厂商云 |
| **车牌夜间不可读** | 快门太慢 / IR 弱 | 加快快门 (1/100+); 加外置 IR 补光; 4K + 光学变焦 / LPR 摄像头 |
