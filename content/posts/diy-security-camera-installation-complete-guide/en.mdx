Professional security camera installation costs **$150–$400 per camera** (labor + materials). A typical 8-camera PoE system: **$1,200–$3,200 labor**. DIY with this guide: **$0 labor + $100–$300 tools**. This guide covers every step from planning to first recording.

<Badge variant="outline">Honest Assessment</Badge> **DIY is right for you if:**
You're comfortable with ladders, drilling, basic networking, and following
diagrams. **Hire a pro if:** Multi-story roof runs, concrete/brick drilling,
permits required, or you want warranty on labor.

### 2. Choose Your Architecture

| Architecture           | Best For                     | Cable Runs                | Power                  | Difficulty |
| ---------------------- | ---------------------------- | ------------------------- | ---------------------- | ---------- |
| **PoE NVR Kit**        | 4–16 cams, permanent, 24/7   | Cat5e/6 to each cam       | PoE (switch/NVR)       | Medium     |
| **Wi-Fi + NVR**        | 2–8 cams, hard-to-wire spots | Power only (outlet/solar) | 120V / Battery / Solar | Low–Medium |
| **Battery/Solar Only** | 1–4 cams, no wires, rentals  | None                      | Battery + Solar panel  | Low        |
| **Hybrid**             | Most homes                   | Mix                       | Mix                    | Medium     |

<Callout type="tip">

**Start with PoE where you can.** It's the most reliable, highest quality, and
lowest long-term cost. Use Wi-Fi/battery only where cable is truly
impractical.

</Callout>

### 3. Cable Route Planning (PoE)

- **Interior walls:** Drill top plate, fish down to basement/crawl, up to attic
- **Exterior walls:** Drill through rim joist or sill plate, seal with silicone
- **Attic routes:** Run along rafters, staple every 4 ft, avoid insulation contact
- **Conduit:** ¾" EMT or PVC for exposed runs (code in many areas)
- **Max length:** 328 ft (100 m) per Cat5e/6 run. Use PoE extender or fiber beyond.

### 4. Tool & Material Checklist

**Essential Tools:**

- [ ] Cordless drill/driver (18V+)
- [ ] ½" spade bit / ⅞" auger bit (cable holes)
- [ ] Fish tape / glow rods (25–50 ft)
- [ ] Stud finder (AC sensing)
- [ ] Level (torpedo or laser)
- [ ] Ladder (extension for 2-story)
- [ ] Crimp tool (RJ45 pass-through) + cable tester
- [ ] Cable stripper (Cat5e/6)
- [ ] Silicone sealant (exterior grade)
- [ ] Cable staples / J-hooks / zip ties

**Materials (PoE):**

- [ ] Cat6 solid copper (CMR riser for in-wall, CMX outdoor for exterior)
- [ ] RJ45 pass-through connectors (50-pack)
- [ ] Keystone jacks + wall plates (optional, for clean terminations)
- [ ] PoE switch (8/16/24 port, 802.3at/bt) or NVR with PoE ports
- [ ] UPS for NVR + switch (1500VA min)

**Materials (Wireless/Battery):**

- [ ] Mounting brackets (included or universal)
- [ ] Solar panels + mounting arms
- [ ] 120V outdoor outlets / weatherproof boxes (if plug-in)

### Ceiling / Soffit / Eave Mount

- **Junction box** (weatherproof, 4" round) → camera bracket screws to box
- **Conduit** (½" EMT or LFMC) from box to camera if cable exposed
- **Seal:** Silicone around box penetration + cable entry

### Pole Mount (Driveway, Gate, Yard)

- **Schedule 40 steel pipe** (1.5" or 2" ID) → set in concrete (18" deep × 8" dia)
- **U-bolt bracket** for camera
- **Conduit** up pole interior or surface-mounted LFMC
- **Grounding:** #6 CU to ground rod at pole base (NEC 250.52)

### Termination: RJ45 Pass-Through (Fastest, Most Reliable)

<Callout type="note">

**Use pass-through connectors.** Standard RJ45 requires precise wire trim.
Pass-through lets you verify color order _before_ crimping. **T568B standard**
(most common):

</Callout>

| Pin | Color        | Pin | Color       |
| --- | ------------ | --- | ----------- |
| 1   | Orange/White | 5   | Blue/White  |
| 2   | Orange       | 6   | Green       |
| 3   | Green/White  | 7   | Brown/White |
| 4   | Blue         | 8   | Brown       |

**Crimp Process:**

1. Strip 1.5" jacket → untwist pairs → arrange T568B
2. Insert into pass-through → verify colors visible at front
3. Trim flush with connector face
4. Crimp **firmly** (ratchet tool clicks once)
5. **Test every cable** — all 8 LEDs green = good

### PoE Switch / NVR Connection

- **Home run** each camera to switch/NVR (not daisy-chain)
- **Patch panel** (optional but pro): 12/24-port keystone → short patch cables to switch
- **UPS:** NVR + PoE switch on battery backup. Cameras stay powered during outage.

<ul>

  </li>
  <li>
    <strong>Outdoor AP</strong> (UniFi U6 Mesh, Ubiquiti UAP-AC-M) — PoE

  </li>
  <li>
    <strong>Directional antenna</strong> (2.4 GHz Yagi) for long-range single

  </li>

<strong>Channel Planning:</strong>

<ul>
  <li>2.4 GHz: Channels 1, 6, 11 only (non-overlapping)</li>
  <li>
    Force camera to 2.4 GHz (better penetration) — disable 5 GHz in camera app
  </li>
  <li>
    Set router 2.4 GHz to <strong>20 MHz width</strong> (not 40) for stability
  </li>
</ul>

<strong>Test:</strong> Phone at camera location → Speedtest → Need <strong>>5 Mbps upload</strong> per camera (1080p) / <strong>>15 Mbps</strong> (4K)

### Battery / Solar Commissioning

1. **Charge fully** before mount (12–24 hrs on USB-C)
2. **Pair to app** at install height (test Wi-Fi signal)
3. **Mount bracket** → attach camera → adjust angle
4. **Solar panel:** Face **true south** (NH), tilt = latitude ±15° (winter +15°, summer -15°)
5. **Verify charging:** App shows ☀️ / ⚡ icon within 1 hr of sun
6. **Set schedules:** Motion-only at night; disable live view auto-start

## Phase 6: Network Security (Critical)

<Callout type="danger">

**Never port-forward camera/NVR ports (80, 443, 8000, 554, 8080).**
Shodan/Censys scan 24/7. Default creds = botnet in minutes.

</Callout>

**Secure Remote Access Options (Best → Good):**

| Method                                               | Security              | Ease    | Cost            | Platform    |
| ---------------------------------------------------- | --------------------- | ------- | --------------- | ----------- |
| **Vendor Cloud (UniFi, Reolink, Eufy, Ring)**        | Good (TLS, auth)      | Easiest | Free            | All         |
| **Tailscale / ZeroTier** (VPN mesh)                  | Excellent (WireGuard) | Medium  | Free (personal) | All         |
| **WireGuard on Router** (OpenWrt, OPNsense, pfSense) | Excellent             | Harder  | Free            | All         |
| **Cloudflare Tunnel** (Zero Trust)                   | Excellent             | Medium  | Free            | Web UI only |
| **OpenVPN Server on Router**                         | Good                  | Medium  | Free            | All         |

**Camera-Level Hardening:**

- Change **admin password** (16+ chars, password manager)
- Disable **ONVIF** if not used
- Disable **UPnP** on router
- Enable **HTTPS only** (self-signed cert OK for local)
- **Firmware update** quarterly (vendor site, not auto)

## Troubleshooting Quick Reference

| Symptom                           | Likely Cause                  | Fix                                                                            |
| --------------------------------- | ----------------------------- | ------------------------------------------------------------------------------ |
| **No image / "No Signal"**        | PoE power budget exceeded     | Check switch PoE budget; move to injector                                      |
| **Image flicker / rolling lines** | Ground loop / bad ground      | Common ground for all equipment; isolator                                      |
| **Night vision whiteout**         | IR reflection                 | Angle camera down; move away from wall/eave; disable IR + external illuminator |
| **Motion alerts constant**        | Trees, shadows, rain, bugs    | Narrow zones; enable Person/Vehicle only; reduce sensitivity                   |
| **Wi-Fi drops**                   | Channel congestion / distance | 2.4 GHz only; 20 MHz; mesh node near camera; directional antenna               |
| **NVR beeping**                   | HDD failure / full / fan      | Check S.M.A.R.T.; replace drive; clean fan                                     |
| **Remote access fails**           | ISP CGNAT / double NAT        | Use Tailscale / Cloudflare Tunnel / Vendor Cloud                               |
| **Plate unreadable at night**     | Shutter too slow / IR weak    | Increase shutter (1/100+); add external IR illuminator; 4K + optical zoom      |
