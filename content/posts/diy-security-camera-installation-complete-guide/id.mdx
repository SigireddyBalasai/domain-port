Biaya pemasangan kamera keamanan profesional **$150–$400 per kamera** (tenaga kerja + bahan). Sistem PoE 8 kamera typical: **$1.200–$3.200 tenaga kerja**. DIY dengan panduan ini: **$0 tenaga kerja + $100–$300 alat**. Panduan ini mencakup setiap langkah dari perencanaan hingga rekaman pertama.

<Badge variant="outline">Penilaian Jujur</Badge> **DIY tepat untuk Anda jika:**
Anda nyaman dengan tangga, pengeboran, jaringan dasar, dan mengikuti diagram.
**Sewa profesional jika:** Jalur atap bertingkat, pengeboran beton/bata, izin
diperlukan, atau Anda menginginkan garansi pada tenaga kerja.

### 3. Perencanaan Rute Kabel (PoE)

- **Dinding interior:** Bor pelat atas, fish ke bawah ke basement/crawl, naik ke loteng
- **Dinding eksterior:** Bor melalui rim joist atau sill plate, segel dengan silikon
- **Rute loteng:** Jalankan di sepanjang kasau, staples setiap 1,2 m, hindari kontak isolasi
- **Conduit:** ¾" EMT atau PVC untuk jalur terbuka (kode di banyak area)
- **Panjang maks:** 100 m per jalur Cat5e/6. Gunakan ekstender PoE atau fiber untuk yang lebih jauh.

### 4. Daftar Periksa Alat & Bahan

**Alat Penting:**

- [ ] Bor/driver tanpa kabel (18V+)
- [ ] Mata bor spade ½" / auger ⅞" (lubang kabel)
- [ ] Fish tape / glow rods (7,5–15 m)
- [ ] Pencari stud (sensing AC)
- [ ] Level (torpedo atau laser)
- [ ] Tangga (ekstensi untuk 2 lantai)
- [ ] Alat crimp (RJ45 pass-through) + tester kabel
- [ ] Pengupas kabel (Cat5e/6)
- [ ] Sealant silikon (kelas eksterior)
- [ ] Staples kabel / J-hooks / ikatan zip

**Bahan (PoE):**

- [ ] Cat6 tembaga solid (CMR riser untuk dalam dinding, CMX outdoor untuk eksterior)
- [ ] Konektor RJ45 pass-through (50-pack)
- [ ] Keystone jack + pelat dinding (opsional, untuk terminasi bersih)
- [ ] Switch PoE (8/16/24 port, 802.3at/bt) atau NVR dengan port PoE
- [ ] UPS untuk NVR + switch (1500VA min)

**Bahan (Nirkabel/Baterai):**

- [ ] Braket pemasangan (termasuk atau universal)
- [ ] Panel surya + lengan pemasangan
- [ ] Stopkontak luar ruangan 120V / kotak tahan cuaca (jika colokan)

### Pemasangan Langit-langit / Soffit / Atap

- **Junction box** (tahan cuaca, bulat 4 inci) → braket kamera disekrup ke kotak
- **Conduit** (½" EMT atau LFMC) dari kotak ke kamera jika kabel terbuka
- **Segel:** Silikon di sekitar penetrasi kotak + masuknya kabel

### Pemasangan Tiang (Jalan Masuk, Gerbang, Halaman)

- **Pipa baja Schedule 40** (1,5" atau 2" ID) → set di beton (45 cm dalam × 20 cm dia)
- **Braket U-bolt** untuk kamera
- **Conduit** di dalam tiang atau LFMC terpasang di permukaan
- **Grounding:** #6 CU ke ground rod di dasar tiang (NEC 250.52)

### Terminasi: RJ45 Pass-Through (Tercepat, Paling Andal)

<Callout type="note">

**Gunakan konektor pass-through.** RJ45 standar memerlukan pemotongan kabel presisi.
Pass-through memungkinkan Anda memverifikasi urutan warna _sebelum_ crimping. **Standar T568B**
(paling umum):

</Callout>

| Pin | Warna        | Pin | Warna        |
| --- | ------------ | --- | ------------ |
| 1   | Putih/Oranye | 5   | Putih/Biru   |
| 2   | Oranye       | 6   | Hijau        |
| 3   | Putih/Hijau  | 7   | Putih/Coklat |
| 4   | Biru         | 8   | Coklat       |

**Proses Crimp:**

1. Kupas jaket 4 cm → buka pilinan pasangan → atur T568B
2. Masukkan ke pass-through → verifikasi warna terlihat di depan
3. Potong rata dengan permukaan konektor
4. Crimp **kuat** (alat ratchet klik sekali)
5. **Uji setiap kabel** — semua 8 LED hijau = baik

### Koneksi Switch PoE / NVR

- **Home run** setiap kamera ke switch/NVR (bukan daisy-chain)
- **Panel patch** (opsional tetapi pro): keystone 12/24-port → kabel patch pendek ke switch
- **UPS:** NVR + switch PoE pada cadangan baterai. Kamera tetap menyala saat pemadaman.

### Komisioning Baterai / Surya

1. **Isi penuh** sebelum dipasang (12–24 jam via USB-C)
2. **Pasangkan ke aplikasi** di ketinggian pemasangan (uji sinyal Wi-Fi)
3. **Pasang braket** → pasang kamera → atur sudut
4. **Panel surya:** Hadapkan ke **selatan sejati** (belahan bumi utara), kemiringan = lintang ±15° (musim dingin +15°, musim panas -15°)
5. **Verifikasi pengisian:** Aplikasi menunjukkan ikon ☀️ / ⚡ dalam 1 jam setelah matahari
6. **Atur jadwal:** Hanya gerakan di malam hari; nonaktifkan mulai otomatis tampilan langsung

## Fase 6: Keamanan Jaringan (Kritis)

<Callout type="danger">

**Jangan pernah port-forward port kamera/NVR (80, 443, 8000, 554, 8080).**
Shodan/Censys memindai 24/7. Kredensial default = botnet dalam hitungan menit.

</Callout>

**Opsi Akses Jarak Jauh Aman (Terbaik → Baik):**

| Metode                                               | Keamanan                | Kemudahan   | Biaya            | Platform    |
| ---------------------------------------------------- | ----------------------- | ----------- | ---------------- | ----------- |
| **Cloud Vendor (UniFi, Reolink, Eufy, Ring)**        | Baik (TLS, auth)        | Termudah    | Gratis           | Semua       |
| **Tailscale / ZeroTier** (VPN mesh)                  | Sangat Baik (WireGuard) | Sedang      | Gratis (pribadi) | Semua       |
| **WireGuard di Router** (OpenWrt, OPNsense, pfSense) | Sangat Baik             | Lebih Sulit | Gratis           | Semua       |
| **Cloudflare Tunnel** (Zero Trust)                   | Sangat Baik             | Sedang      | Gratis           | UI Web saja |
| **OpenVPN Server di Router**                         | Baik                    | Sedang      | Gratis           | Semua       |

**Penguatan Tingkat Kamera:**

- Ganti **sandi admin** (16+ karakter, manajer sandi)
- Nonaktifkan **ONVIF** jika tidak digunakan
- Nonaktifkan **UPnP** di router
- Aktifkan **HTTPS saja** (sertifikat self-signed OK untuk lokal)
- **Pembaruan firmware** triwulanan (situs vendor, bukan otomatis)

## Referensi Cepat Pemecahan Masalah

| Gejala                               | Kemungkinan Penyebab                     | Perbaikan                                                                                    |
| ------------------------------------ | ---------------------------------------- | -------------------------------------------------------------------------------------------- |
| **Tidak ada gambar / "No Signal"**   | Anggaran daya PoE terlampaui             | Periksa anggaran PoE switch; pindah ke injector                                              |
| **Gambar berkedip / garis bergulir** | Ground loop / ground buruk               | Ground umum untuk semua peralatan; isolator                                                  |
| **Whiteout night vision**            | Refleksi IR                              | Miringkan kamera ke bawah; jauhkan dari dinding/atap; nonaktifkan IR + illuminator eksternal |
| **Peringatan gerakan konstan**       | Pohon, bayangan, hujan, serangga         | Persempit zona; aktifkan Orang/Kendaraan saja; kurangi sensitivitas                          |

| **NVR berbunyi bip**                 | HDD gagal / penuh / kipas                | Periksa S.M.A.R.T.; ganti drive; bersihkan kipas                                             |
| **Akses jarak jauh gagal**           | ISP CGNAT / NAT ganda                    | Gunakan Tailscale / Cloudflare Tunnel / Cloud Vendor                                         |
| **Plat tidak terbaca di malam hari** | Kecepatan rana terlalu lambat / IR lemah | Tingkatkan rana (1/100+); tambahkan illuminator IR eksternal; 4K + zoom optik                |
