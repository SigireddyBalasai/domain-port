## ขั้นตอนที่ 1: วางแผนและออกแบบ

### แผนผังตำแหน่งกล้อง

**ระดับ 1 — ต้องมี (ทางเข้า/ออก):**

- ประตูหน้า (กริ่งประตูหรือเหนือประตู)
- ประตูหลัง/ด้านข้าง
- ทางเข้ารถยนต์ (ป้ายทะเบียน)

**ระดับ 2 — มูลค่าสูง:**

- หน้าต่างชั้นล่าง
- สนามหลังบ้าน / ลาน
- ประตูรั้วด้านข้าง

**กฎการวางตำแหน่ง:**

- **ความสูง:** 2.4–3 ม.
- **มุม:** 15–30° ลงจากแนวนอน
- **ซ้อนทับมุมมอง:** 20–30% ระหว่างกล้องที่ติดกัน

### รายการตรวจสอบก่อนติดตั้ง

- [ ] **กำหนดโซนครอบคลุม** — วาดแผนผังทรัพย์สิน ทำเครื่องหมายจุดเข้าทั้งหมด
- [ ] **เลือกประเภทกล้อง** — PoE (เชื่อถือได้มากที่สุด), Wi-Fi (ง่ายที่สุด), แบตเตอรี่ (ยืดหยุ่น)
- [ ] **ตรวจสอบสัญญาณ Wi-Fi** (ไร้สาย) — RSSI > -60 dBm ที่ตำแหน่งกล้องแต่ละตัว
- [ ] **วัดเส้นทางสาย** — เผื่อ 3 ม. ต่อเส้นทาง
- [ ] **ตรวจสอบงบ PoE** — Σ กำลังกล้อง × 1.2 ≤ งบ switch
- [ ] **ตรวจสอบกฎหมายท้องถิ่น** — ใบอนุญาต, กฎ HOA, กฎหมายบันทึกเสียง
- [ ] **เตรียมเครื่องมือ** — สว่าน, ดอกสว่าน, ที่ครอบ RJ45, ที่ทดสอบสาย, เครื่องหาไม้

## ขั้นตอนที่ 3: เดินสาย (ระบบ PoE)

### การ crimp RJ45 Pass-Through

มาตรฐาน T568B:

- เข็ม 1: ส้ม/ขาว | เข็ม 2: ส้ม
- เข็ม 3: เขียว/ขาว | เข็ม 4: น้ำเงิน
- เข็ม 5: น้ำเงิน/ขาว | เข็ม 6: เขียว
- เข็ม 7: น้ำตาล/ขาว | เข็ม 8: น้ำตาล

### กฎการเดินสาย

- **ชนิดสาย:** Cat6 ทองแดงแท้ (ไม่ใช่ CCA)
- **ความยาวสูงสุด:** 100 ม. (328 ฟุต) สำหรับ PoE+
- **รอยต่อ:** ห้ามต่อ—สายเดียวจาก switch ถึงกล้อง
- **ขนานกับไฟบ้าน:** ห่างอย่างน้อย 30 ซม.
- **ตัดไฟบ้าน:** ทำมุม 90° เท่านั้น
- **ยึดสาย:** ทุก 1.2 ม. อย่าขันแน่นเกินไป

### การเดินสายตามตำแหน่ง

| เส้นทาง                 | ความยาก | เวลาต่อกล้อง | เครื่องมือ               |
| ----------------------- | ------- | ------------ | ------------------------ |
| ใต้หลังคา → ชายคา       | ง่าย    | 20–30 นาที   | แท่ง pulling, ดอก ⅞"     |
| ชั้นใต้ดิน → ผนังภายนอก | ปานกลาง | 30–45 นาที   | แท่ง pulling, ดอก, กันไฟ |
| ผนังภายใน → ภายนอก      | ปานกลาง | 30–60 นาที   | แท่ง pulling, box, อุด   |
| ฝังดิน                  | ยาก     | 60–90 นาที   | จอบ, ท่อ PVC, เทป        |

## ขั้นตอนที่ 4: ติดตั้งกล้อง

### ขั้นตอนการติดตั้ง

1. **ทำเครื่องหมายตำแหน่ง** — ถือ mount จ่อ ผ่าเครื่องหมายผ่านรู
2. **เจาะนำ** — ใช้ดอกให้เหมาะกับพื้นผิว
3. **ยึด mount** — สกรูเข้าไม้หรือพุกเข้า concrete
4. **กันน้ำ** — เทป butyl ใต้ mount, ซิลิโคนรอบสกรู
5. **ต่อสาย** — ร้อยสายผ่าน mount, เหลือ 30 ซม.
6. **ติดตั้งกล้อง** — ขันกล้องเข้ากับ mount, ปรับระดับ
7. **ปรับมุม** — เอียงลง 15–30°, เล็งไปที่เป้าหมาย
8. **ล็อก** — ขันน็อตล็อกทั้งหมด

### ความสูงติดตั้งตามพื้นที่

| เป้าหมาย         | ความสูง | มุมเอียง | เหตุผล                             |
| ---------------- | ------- | -------- | ---------------------------------- |
| ใบหน้าผู้มาเยือน | 1.2 ม.  | 0°       | ระดับสายตา—ดีที่สุดสำหรับระบุตัวตน |
| ป้ายทะเบียน      | 3–4 ม.  | 15–20°   | สูงพอเห็นเหนือรถ                   |
| เฝ้าระวังทั่วไป  | 3–4 ม.  | 20–30°   | มุมกว้าง, จุดบอดน้อย               |
| รอบรั้ว          | 4–5 ม.  | 15°      | ป้องกันการก่อวินาศกรรม             |

## ขั้นตอนที่ 5: การรักษาความปลอดภัยเครือข่าย

| วิธีการ                            | ความปลอดภัย | ความง่าย   | ค่าใช้จ่าย                 |
| ---------------------------------- | ----------- | ---------- | -------------------------- |
| **คลาวด์ผู้ผลิต** (UniFi, Reolink) | ดี (TLS)    | ง่ายที่สุด | ฟรี                        |
| **Tailscale / ZeroTier**           | ยอดเยี่ยม   | ปานกลาง    | ฟรี                        |
| **WireGuard VPN**                  | ยอดเยี่ยม   | ยาก        | ฟรี                        |
| **พอร์ต Forwarding**               | แย่         | ปานกลาง    | ฟรี                        |
| **VLAN แยกกล้อง**                  | ดีที่สุด    | ยาก        | ฟรี (ถ้ามี managed switch) |

## ขั้นตอนที่ 6: การตั้งค่า NVR

### การตั้งค่าเริ่มต้น

1. **เชื่อมต่อ NVR** กับจอภาพ (HDMI), เมาส์, เครือข่าย
2. **เปิดเครื่อง** — รอ 2–3 นาทีให้บูต
3. **ตั้งค่าเริ่มต้น** — ภาษา, วันที่/เวลา, รหัสผ่าน admin
4. **ฟอร์แมต HDD** — จัดรูปแบบไดรฟ์ที่ติดตั้ง
5. **ตรวจจับกล้อง** — NVR จะเจอกล้องในเครือข่ายเดียวกันโดยอัตโนมัติ
6. **กำหนด IP** — IP คงที่หรือ DHCP reservation

### การตั้งค่าการบันทึก

| การตั้งค่า        | คำแนะนำ                           |
| ----------------- | --------------------------------- |
| ตารางบันทึก       | บันทึกต่อเนื่อง 24/7              |
| ความละเอียด       | 4K (สตรีมหลัก)                    |
| Codec             | H.265                             |
| Bitrate           | CBR 4–6 Mbps (4K)                 |
| FPS               | 15 (สมดุลคุณภาพและพื้นที่จัดเก็บ) |
| ตรวจจับเคลื่อนไหว | คน + ยานพาหนะ                     |
| ก่อนเหตุการณ์     | 5–10 วินาที                       |
| หลังเหตุการณ์     | 10–15 วินาที                      |

## ขั้นตอนที่ 7: การตรวจสอบระบบ

- [ ] **สด** — กล้องทุกตัวแสดงภาพและอัปเดต
- [ ] **มุมมอง** — แต่ละกล้องครอบคลุมพื้นที่ที่วางแผนไว้
- [ ] **กลางคืน** — IR เปิดในที่มืด ภาพชัด
- [ ] **บันทึก** — NVR บันทึกต่อเนื่องและตามการเคลื่อนไหว
- [ ] **เล่นย้อนหลัง** — ตรวจสอบการเล่นวิดีโอที่บันทึกไว้
- [ ] **แจ้งเตือน** — Push notification ทำงานบนโทรศัพท์
- [ ] **เข้าถึงระยะไกล** — แอปหรือ VPN แสดงกล้อง
- [ ] **เสียง** (ถ้ามี) — เสียงสองทางทำงาน
- [ ] **PTZ** (ถ้ามี) — Pan, tilt, zoom, preset
- [ ] **เอกสาร** — ถ่ายรูปจุด crimp, จดรหัสผ่าน
