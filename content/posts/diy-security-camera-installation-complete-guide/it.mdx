| Architettura             | Migliore Per                               | Cavi Necessari                    | Alimentazione              | Difficoltà  |
| ------------------------ | ------------------------------------------ | --------------------------------- | -------------------------- | ----------- |

<Callout type="tip">

**Inizia con PoE dove puoi.** È il più affidabile, di massima qualità e con il costo a lungo termine più basso. Usa Wi-Fi/batteria solo dove il cavo è veramente impraticabile.

</Callout>

### 3. Pianificazione Percorso Cavi (PoE)

- **Pareti interne:** Fora il doppio plinto, pesca fino a seminterrato/vespaio, su fino a sottotetto
- **Pareti esterne:** Fora attraverso traversa o soglia, sigilla con silicone
- **Percorsi sottotetto:** Corri lungo i travetti, ferma ogni 1.2 m, evita contatto con isolamento
- **Conduit:** ¾" EMT o PVC per tratti esposti (richiesto da codice in molte aree)
- **Lunghezza max:** 100 m per tratto Cat5e/6. Usa estensore PoE o fibra oltre.

### 4. Checklist Strumenti e Materiali

**Strumenti Essenziali:**

- [ ] Trapano avvitatore a batteria (18V+)
- [ ] Punta da ½" / punta elicoidale ⅞" (fori cavi)
- [ ] Nastro pesca / aste luminescenti (7.5–15 m)
- [ ] Cercamontanti (rilevamento AC)
- [ ] Livella (torpedine o laser)
- [ ] Scala (estensione per 2 piani)
- [ ] Pinza crimpatrice (RJ45 pass-through) + tester cavo
- [ ] Spellacavi (Cat5e/6)
- [ ] Sigillante siliconico (per esterni)
- [ ] Fermacavi / J-hook / fascette

**Materiali (PoE):**

- [ ] Cat6 rame massiccio (CMR per pareti, CMX per esterno)
- [ ] Connettori RJ45 pass-through (50 pezzi)
- [ ] Keystone jack + piastre a muro (opzionali, per terminazioni pulite)
- [ ] Switch PoE (8/16/24 porte, 802.3at/bt) o NVR con porte PoE
- [ ] UPS per NVR + switch (1500VA min)

**Materiali (Wireless/Batteria):**

- [ ] Staffe di montaggio (incluse o universali)
- [ ] Pannelli solari + bracci di montaggio
- [ ] Prese esterne 120V / scatole stagno (se con cavo)

### Montaggio a Soffitto / Gronda / Aggetto

- **Scatola di derivazione** (stagna, 4" tonda) → staffa telecamera si avvita alla scatola
- **Conduit** (½" EMT o LFMC) dalla scatola alla telecamera se cavo esposto
- **Sigilla:** Silicone intorno alla penetrazione scatola + ingresso cavo

### Montaggio su Palo (Vialetto, Cancello, Giardino)

- **Tubo acciaio Schedule 40** (1.5" o 2" diametro interno) → cementato (45 cm profondità × 20 cm diametro)
- **Staffa a U** per telecamera
- **Conduit** sul palo interno o LFMC montato in superficie
- **Messa a terra:** #6 CU a asta di terra alla base del palo (NEC 250.52)

### Terminazione: RJ45 Pass-Through (Più Veloce, Più Affidabile)

<Callout type="note">

**Usa connettori pass-through.** RJ45 standard richiede taglio preciso dei fili. Pass-through ti permette di verificare l'ordine dei colori _prima_ di crimpare. **Standard T568B** (più comune):

</Callout>

| Pin | Colore           | Pin | Colore         |
| --- | ---------------- | --- | -------------- |
| 1   | Bianco/Arancione | 5   | Bianco/Blu     |
| 2   | Arancione        | 6   | Verde          |
| 3   | Bianco/Verde     | 7   | Bianco/Marrone |
| 4   | Blu              | 8   | Marrone        |

**Processo di Crimpatura:**

1. Spella 4 cm di guaina → srotola coppie → dispone T568B
2. Inserisci in pass-through → verifica colori visibili davanti
3. Taglia a filo con la faccia del connettore
4. Crimpa **con decisione** (strumento a cricchetto scatta una volta)
5. **Testa ogni cavo** — tutti 8 LED verdi = buono

### Connessione Switch PoE / NVR

- **Collegamento diretto** ogni telecamera a switch/NVR (non daisy-chain)
- **Pannello di permutazione** (opzionale ma professionale): keystone 12/24 porte → cavi patch corti a switch
- **UPS:** NVR + switch PoE su backup batteria. Le telecamere rimangono alimentate durante l'interruzione.

### Messa in Servizio Batteria / Solare

1. **Carica completamente** prima del montaggio (12–24 ore su USB-C)
2. **Associa all'app** all'altezza di installazione (testa segnale Wi-Fi)
3. **Monta staffa** → attacca telecamera → regola angolo
4. **Pannello solare:** Orientamento **vero sud** (NH), inclinazione = latitudine ±15° (inverno +15°, estate -15°)
5. **Verifica carica:** App mostra icona ☀️ / ⚡ entro 1 ora dal sole
6. **Imposta programmi:** Solo movimento di notte; disabilita avvio automatico visione diretta

## Fase 6: Sicurezza Rete (Critico)

<Callout type="danger">

**Non fare mai port-forward delle porte telecamera/NVR (80, 443, 8000, 554, 8080).** Shodan/Censys scansionano 24/7. Credenziali default = botnet in minuti.

</Callout>

**Opzioni Accesso Remoto Sicuro (Migliore → Buono):**

| Metodo                                               | Sicurezza              | Facilità      | Costo                | Piattaforma |
| ---------------------------------------------------- | ---------------------- | ------------- | -------------------- | ----------- |
| **Cloud Fornitore (UniFi, Reolink, Eufy, Ring)**     | Buona (TLS, auth)      | Più facile    | Gratuito             | Tutti       |
| **Tailscale / ZeroTier** (VPN mesh)                  | Eccellente (WireGuard) | Media         | Gratuito (personale) | Tutti       |
| **WireGuard su Router** (OpenWrt, OPNsense, pfSense) | Eccellente             | Più difficile | Gratuito             | Tutti       |
| **Cloudflare Tunnel** (Zero Trust)                   | Eccellente             | Media         | Gratuito             | Solo Web UI |
| **OpenVPN Server su Router**                         | Buona                  | Media         | Gratuito             | Tutti       |

**Rafforzamento Livello Telecamera:**

- Cambia password **admin** (16+ caratteri, password manager)
- Disabilita **ONVIF** se non usato
- Disabilita **UPnP** sul router
- Abilita **solo HTTPS** (certificato auto-firmato OK per locale)
- **Aggiornamento firmware** trimestrale (sito fornitore, non automatico)

## Riferimento Rapido Risoluzione Problemi

| Sintomo                                 | Causa Probabile                     | Soluzione                                                                                |
| --------------------------------------- | ----------------------------------- | ---------------------------------------------------------------------------------------- |
| **Nessuna immagine / "Nessun Segnale"** | Budget PoE switch superato          | Controlla budget switch PoE; sposta su iniettore                                         |
| **Sfarfallio immagine / righe**         | Ground loop / terra difettosa       | Terra comune per tutte le apparecchiature; isolatore                                     |
| **Visione notturna bianca**             | Riflesso IR                         | Inclina telecamera giù; allontana da parete/gronda; disabilita IR + illuminatore esterno |
| **Allarmi movimento continui**          | Alberi, ombre, pioggia, insetti     | Restringi zone; abilita solo Persona/Veicolo; riduci sensibilità                         |
| **Wi-Fi cade**                          | Congestione canale / distanza       | Solo 2.4 GHz; 20 MHz; nodo mesh vicino telecamera; antenna direzionale                   |
| **NVR bip**                             | HDD guasto/pieno/ventola            | Controlla S.M.A.R.T.; sostituisci disco; pulisci ventola                                 |
| **Accesso remoto fallisce**             | CGNAT ISP / Doppio NAT              | Usa Tailscale / Cloudflare Tunnel / Cloud Fornitore                                      |
| **Targa illeggibile di notte**          | Otturatore troppo lento / IR debole | Aumenta otturatore (1/100+); aggiungi illuminatore IR esterno; 4K + zoom ottico          |
