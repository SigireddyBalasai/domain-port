पेशेवर सिक्योरिटी कैमरा इंस्टॉलेशन की लागत **$150–$400 प्रति कैमरा** (श्रम + सामग्री) होती है। एक सामान्य 8-कैमरा PoE सिस्टम: **$1,200–$3,200 श्रम**। इस गाइड के साथ DIY: **$0 श्रम + $100–$300 उपकरण**। यह गाइड प्लानिंग से लेकर पहली रिकॉर्डिंग तक के हर कदम को कवर करती है।

<Badge variant="outline">ईमानदार मूल्यांकन</Badge> **DIY आपके लिए सही है अगर:**
आप सीढ़ी, ड्रिलिंग, बेसिक नेटवर्किंग और डायग्राम का पालन करने में सहज हैं।
**प्रो किराए पर लें अगर:** बहु-मंज़िला छत रन, कंक्रीट/ईंट ड्रिलिंग, परमिट
आवश्यक, या आप श्रम पर वारंटी चाहते हैं।

### 3. केबल रूट प्लानिंग (PoE)

- **आंतरिक दीवारें:** टॉप प्लेट ड्रिल करें, बेसमेंट/क्रॉल तक फिश करें, अटारी तक ऊपर
- **बाहरी दीवारें:** रिम जॉइस्ट या सिल प्लेट के माध्यम से ड्रिल करें, सिलिकॉन से सील करें
- **अटारी रूट:** राफ्टर्स के साथ चलाएं, हर 4 फीट पर स्टेपल करें, इंसुलेशन संपर्क से बचें
- **कंड्यूट:** एक्सपोज़्ड रन के लिए ¾" EMT या PVC (कई क्षेत्रों में कोड)
- **अधिकतम लंबाई:** 328 फीट (100 मी) प्रति Cat5e/6 रन। उससे आगे PoE एक्सटेंडर या फ़ाइबर का उपयोग करें।

### 4. उपकरण और सामग्री चेकलिस्ट

**आवश्यक उपकरण:**

- [ ] कॉर्डलेस ड्रिल/ड्राइवर (18V+)
- [ ] ½" स्पेड बिट / ⅞" ऑगर बिट (केबल होल)
- [ ] फ़िश टेप / ग्लो रॉड्स (25–50 फीट)
- [ ] स्टड फ़ाइंडर (AC सेंसिंग)
- [ ] लेवल (टारपीडो या लेज़र)
- [ ] सीढ़ी (2-मंज़िला के लिए एक्सटेंशन)
- [ ] क्रिम्प टूल (RJ45 पास-थ्रू) + केबल टेस्टर
- [ ] केबल स्ट्रिपर (Cat5e/6)
- [ ] सिलिकॉन सीलेंट (बाहरी ग्रेड)
- [ ] केबल स्टेपल / J-हुक / ज़िप टाई

**सामग्री (PoE):**

- [ ] Cat6 सॉलिड कॉपर (दीवार के अंदर के लिए CMR राइज़र, बाहरी के लिए CMX आउटडोर)
- [ ] RJ45 पास-थ्रू कनेक्टर (50-पैक)
- [ ] कीस्टोन जैक + वॉल प्लेट (वैकल्पिक, साफ़ टर्मिनेशन के लिए)
- [ ] PoE स्विच (8/16/24 पोर्ट, 802.3at/bt) या PoE पोर्ट वाला NVR
- [ ] NVR + स्विच के लिए UPS (1500VA न्यूनतम)

**सामग्री (वायरलेस/बैटरी):**

- [ ] माउंटिंग ब्रैकेट (शामिल या यूनिवर्सल)
- [ ] सोलर पैनल + माउंटिंग आर्म्स
- [ ] 120V आउटडोर आउटलेट / वेदरप्रूफ बॉक्स (प्लग-इन होने पर)

### छत / सॉफ़िट / ईव माउंट

- **जंक्शन बॉक्स** (वेदरप्रूफ, 4" गोल) → कैमरा ब्रैकेट बॉक्स में स्क्रू करता है
- अगर केबल एक्सपोज़्ड है तो बॉक्स से कैमरे तक **कंड्यूट** (½" EMT या LFMC)
- **सील:** बॉक्स पेनिट्रेशन + केबल एंट्री के चारों ओर सिलिकॉन

### पोल माउंट (ड्राइववे, गेट, यार्ड)

- **शेड्यूल 40 स्टील पाइप** (1.5" या 2" ID) → कंक्रीट में सेट करें (18" गहरा × 8" व्यास)
- कैमरे के लिए **U-बोल्ट ब्रैकेट**
- पोल इंटीरियर के ऊपर या सरफ़ेस-माउंटेड LFMC में **कंड्यूट**
- **ग्राउंडिंग:** पोल बेस पर #6 CU से ग्राउंड रॉड (NEC 250.52)

### टर्मिनेशन: RJ45 पास-थ्रू (सबसे तेज़, सबसे विश्वसनीय)

<Callout type="note">

**पास-थ्रू कनेक्टर का उपयोग करें।** मानक RJ45 को सटीक तार ट्रिम की आवश्यकता
होती है। पास-थ्रू आपको क्रिम्पिंग से पहले रंग क्रम सत्यापित करने देता है।
**T568B मानक** (सबसे आम):

</Callout>

| पिन | रंग          | पिन | रंग           |
| --- | ------------ | --- | ------------- |
| 1   | ऑरेंज/व्हाइट | 5   | ब्लू/व्हाइट   |
| 2   | ऑरेंज        | 6   | ग्रीन         |
| 3   | ग्रीन/व्हाइट | 7   | ब्राउन/व्हाइट |
| 4   | ब्लू         | 8   | ब्राउन        |

**क्रिम्प प्रक्रिया:**

1. 1.5" जैकेट स्ट्रिप करें → पेयर खोलें → T568B व्यवस्थित करें
2. पास-थ्रू में डालें → सामने रंग दिखाई देने चाहिए
3. कनेक्टर फेस के साथ फ्लश ट्रिम करें
4. **मजबूती से** क्रिम्प करें (रैचेट टूल एक बार क्लिक करता है)
5. **हर केबल की जांच करें** — सभी 8 LED हरे = अच्छा

### PoE स्विच / NVR कनेक्शन

- प्रत्येक कैमरा स्विच/NVR तक **होम रन** करें (डेज़ी-चेन नहीं)
- **पैच पैनल** (वैकल्पिक लेकिन प्रो): 12/24-पोर्ट कीस्टोन → स्विच तक छोटी पैच केबल
- **UPS:** बैटरी बैकअप पर NVR + PoE स्विच। बिजली जाने पर कैमरे चालू रहते हैं।

### बैटरी / सोलर कमीशनिंग

1. माउंट से पहले **पूरी तरह चार्ज करें** (USB-C पर 12–24 घंटे)
2. इंस्टॉल ऊंचाई पर **ऐप से पेयर करें** (Wi-Fi सिग्नल जांचें)
3. **ब्रैकेट माउंट करें** → कैमरा अटैच करें → एंगल एडजस्ट करें
4. **सोलर पैनल:** **ट्रू साउथ** (NH) की ओर रखें, झुकाव = अक्षांश ±15° (सर्दी +15°, गर्मी -15°)
5. **चार्जिंग सत्यापित करें:** ऐप 1 घंटे के भीतर ☀️ / ⚡ आइकन दिखाता है
6. **शेड्यूल सेट करें:** रात में मोशन-ओनली; लाइव व्यू ऑटो-स्टार्ट अक्षम करें

<Callout type="danger">

**कभी भी कैमरा/NVR पोर्ट (80, 443, 8000, 554, 8080) पोर्ट-फ़ॉरवर्ड न करें।**
Shodan/Censys 24/7 स्कैन करते हैं। डिफ़ॉल्ट क्रेडेंशियल = मिनटों में बॉटनेट।

</Callout>

**सुरक्षित रिमोट एक्सेस विकल्प (सर्वश्रेष्ठ → अच्छा):**

| विधि                                                | सुरक्षा              | सरलता     | लागत               | प्लेटफ़ॉर्म |
| --------------------------------------------------- | -------------------- | --------- | ------------------ | ----------- |
| **वेंडर क्लाउड (UniFi, Reolink, Eufy, Ring)**       | अच्छा (TLS, auth)    | सबसे आसान | मुफ़्त             | सभी         |
| **Tailscale / ZeroTier** (VPN मेश)                  | उत्कृष्ट (WireGuard) | मध्यम     | मुफ़्त (व्यक्तिगत) | सभी         |
| **राउटर पर WireGuard** (OpenWrt, OPNsense, pfSense) | उत्कृष्ट             | कठिन      | मुफ़्त             | सभी         |
| **Cloudflare Tunnel** (ज़ीरो ट्रस्ट)                | उत्कृष्ट             | मध्यम     | मुफ़्त             | केवल वेब UI |
| **राउटर पर OpenVPN सर्वर**                          | अच्छा                | मध्यम     | मुफ़्त             | सभी         |

**कैमरा-स्तरीय सख्तीकरण:**

- **एडमिन पासवर्ड** बदलें (16+ अक्षर, पासवर्ड मैनेजर)
- उपयोग में न होने पर **ONVIF अक्षम करें**
- राउटर पर **UPnP अक्षम करें**
- **केवल HTTPS** चालू करें (स्थानीय के लिए सेल्फ-साइन्ड सर्टिफिकेट ठीक है)
- तिमाही में **फ़र्मवेयर अपडेट** करें (वेंडर साइट, ऑटो नहीं)
