التركيب الاحترافي لكاميرات المراقبة يكلف **$150–$400 لكل كاميرا** (عمالة + مواد). نظام PoE نموذجي بـ 8 كاميرات: **$1,200–$3,200 عمالة**. التركيب بنفسك مع هذا الدليل: **$0 عمالة + $100–$300 أدوات**. يغطي هذا الدليل كل خطوة من التخطيط إلى أول تسجيل.

## المرحلة 1: التخطيط والتصميم

### 1. خريطة وضع الكاميرات

**المستوى 1 — إلزامي (مداخل/مخارج):** الباب الأمامي، الأبواب الخلفية/الجانبية، المرآب، الممر.
**المستوى 2 — قيمة عالية:** نوافذ الطابق الأرضي، الفناء الخلفي، البوابات الجانبية، منطقة تسليم الطرود.
**المستوى 3 — ظرفي:** المسبح، السقيفة، صندوق البريد.

**قواعد التنسيب:**

- **الارتفاع:** 8–10 أقدام (بعيداً عن المتناول، زاوية جيدة)
- **الزاوية:** 15–30° للأسفل من الأفقي
- **تداخل مجال الرؤية:** 20–30% بين الكاميرات المتجاورة

### 2. اختر بنيتك

| البنية          | الأفضل لـ                | تمديد الكابلات       |
| --------------- | ------------------------ | -------------------- |
| **طقم NVR PoE** | 4–16 كاميرا، دائمة، 24/7 | Cat5e/6 لكل كاميرا   |
| **Wi-Fi + NVR** | 2–8 كاميرا، أماكن صعبة   | طاقة فقط (مقبس/شمسي) |
| **بطارية/شمسي** | 1–4 كاميرا، بدون أسلاك   | لا شيء               |
| **هجين**        | معظم المنازل             | مختلط                |

### 3. تخطيط مسار الكابل (PoE)

- **جدران داخلية:** احفر اللوحة العلوية، اسحب للأسفل
- **جدران خارجية:** احفر عبر عتبة العتبة، أغلق بالسيليكون
- **العلية:** على طول العوارض الخشبية، دباسة كل 4 أقدام
- **الحد الأقصى:** 328 قدماً (100 م) لكل Cat5e/6

### 4. قائمة الأدوات والمواد

**الأدوات الأساسية:** مثقاب لاسلكي 18V+، شريط صيد، ميزان مستوى، سلم، أداة ضغط RJ45، اختبار كابلات، سيليكون.

**المواد (PoE):** Cat6 نحاسي صلب (CMR للجدران، CMX للخارج)، موصلات RJ45 pass-through، مقابس Keystone، مبدل PoE، UPS لـ NVR + المبدل.

## المرحلة 2: التركيب

### تركيب الجدار

- **انحياز الفينيل/الألومنيوم:** استخدم كتلة تثبيت انحياز الفينيل
- **الجص/الطوب/الخرسانة:** مثقاب مطرقة + لقمة بناء ← مسامير تثبيت بلاستيكية + براغي ستانلس ستيل
- **الخشب/Hardie:** ثقب تجريبي ← برغي سطح #8–#10 ستانلس 2–2.5 بوصة في العمود
- **الطنف/السقف:** برغي في ذيل العارضة الخشبية

---

## المرحلة 3: تمديد الكابلات (PoE)

### تقنية الصيد الداخلي

1. احفر ثقب ⅞ بوصة
2. ادفع شريط الصيد إلى فتحة الكاميرا
3. اربط Cat6، اسحب بلطف
4. اترك 3–4 أقدام من الركود
5. أغلق الفتحة الخارجية بالسيليكون
6. سم الطرفين
7. اختبر الاستمرارية باختبار الكابلات

### إنهاء RJ45 (T568B)

| الدبوس | اللون        | الدبوس | اللون     |
| ------ | ------------ | ------ | --------- |
| 1      | برتقالي/أبيض | 5      | أزرق/أبيض |
| 2      | برتقالي      | 6      | أخضر      |
| 3      | أخضر/أبيض    | 7      | بني/أبيض  |
| 4      | أزرق         | 8      | بني       |

## المرحلة 4: إعداد الكاميرات اللاسلكية/البطارية

### تحسين Wi-Fi

- الكاميرا ضمن 50 قدماً من الراوتر/نقطة الوصول (حد أقصى جداران)
- **نظام شبكي** (Eero, UniFi, Asus) — يُفضل التوصيل السلكي الخلفي
- **نقطة وصول خارجية** (UniFi U6 Mesh) — PoE، تحت الطنف
- **القناة:** 2.4 جيجاهرتز، القنوات 1/6/11، عرض 20 ميجاهرتز
- **الاختبار:** هاتف في موقع الكاميرا → Speedtest → >5 Mbps رفع لكل كاميرا (1080p) / >15 Mbps (4K)

### تشغيل البطارية/الشمسية

1. اشحن بالكامل قبل التركيب (12–24 ساعة عبر USB-C)
2. الاقتران بالتطبيق على ارتفاع التركيب
3. اللوح الشمسي: اتجه نحو **الجنوب الحقيقي** (نصف الكرة الشمالي)، الميل = خط العرض ±15°

---

## المرحلة 5: إعداد NVR

### أول تشغيل لـ NVR

1. ثبت الأقراص الصلبة قبل التشغيل (WD Purple / Seagate SkyHawk)
2. التشغيل → المعالج: الوقت (NTP)، كلمة المرور (قوية!)
3. الشبكة: IP ثابت لـ NVR
4. منافذ PoE: فعّل → "إضافة تلقائية" للكاميرات
5. الترميز: رئيسي 4K @ 15–20 fps, H.265; فرعي 720p
6. جدول التسجيل: مستمر 24/7 على الرئيسي؛ حركة على الفرعي
7. الوصول عن بُعد: **لا تعيد توجيه المنافذ.** استخدم سحابة البائع أو VPN

### اختيار القرص الصلب

| القرص           | السعة         | الضمان  | الأفضل لـ            |
| --------------- | ------------- | ------- | -------------------- |
| WD Purple       | 1–18 تيرابايت | 3 سنوات | NVR قياسي            |
| Seagate SkyHawk | 1–20 تيرابايت | 3 سنوات | NVR قياسي            |
| WD Purple Pro   | 8–22 تيرابايت | 5 سنوات | RAID / تيارات متعددة |

## المرحلة 6: أمن الشبكة

**لا تعيد توجيه منافذ الكاميرات/NVR أبداً (80, 443, 8000, 554).** Shodan/Censys تفحص 24/7.

| الطريقة                   | الأمان    | السهولة |
| ------------------------- | --------- | ------- |
| **سحابة البائع**          | جيد (TLS) | الأسهل  |
| **Tailscale / ZeroTier**  | ممتاز     | متوسط   |
| **WireGuard على الراوتر** | ممتاز     | صعب     |
| **Cloudflare Tunnel**     | ممتاز     | متوسط   |

**تحصين الكاميرا:** غيّر كلمة مرور المشرف (16+ حرفاً)، عطل ONVIF إذا لم يستخدم، عطل UPnP، فعّل HTTPS فقط، حدّث البرنامج الثابت فصلياً.

---

## المرحلة 7: الاختبار والتشغيل

**النهار:** صورة حية واضحة، مجال الرؤية يغطي المنطقة، حاد، كشف الحركة <5 ثوانٍ، التسجيل يعمل.
**الليل:** IR يعمل، لا انعكاس IR، لا "تبييض"، لوحة السيارة مقروءة.
**الشبكة:** IP الكاميرا يمكن الوصول إليه، RTSP يعمل في VLC، تسجيل NVR 24/7 بدون فجوات، الوصول عن بُعد يعمل.
**مادي:** البراغي مشدودة، حلقات الكابلات مؤمنة، السيليكون سليم.

## استكشاف الأخطاء وإصلاحها بسرعة

| العرض                              | السبب المحتمل              | الإصلاح                                                         |
| ---------------------------------- | -------------------------- | --------------------------------------------------------------- |
| **لا توجد صورة / "لا توجد إشارة"** | تجاوز ميزانية طاقة PoE     | تحقق من ميزانية PoE للمبدل؛ انقل إلى محقن                       |
| **وميض الصورة / خطوط متدحرجة**     | حلقة أرضية / تأريض سيئ     | تأريض مشترك لجميع المعدات؛ عازل                                 |
| **الرؤية الليلية بيضاء**           | انعكاس IR                  | أمل الكاميرا للأسفل؛ ابتعد عن الحائط؛ عطل IR + منير خارجي       |
| **ثبات تنبيهات الحركة**            | أشجار، ظلال، مطر، حشرات    | ضيق المناطق؛ فعّل شخص/مركبة فقط؛ قلل الحساسية                   |
| **انقطاع Wi-Fi**                   | ازدحام القناة / المسافة    | 2.4 GHz فقط؛ 20 MHz؛ عقدة شبكية قريبة من الكاميرا؛ هوائي اتجاهي |
| **إصدار صوت من NVR**               | فشل القرص / امتلاء / مروحة | تحقق من S.M.A.R.T.؛ استبدل القرص؛ نظف المروحة                   |
| **فشل الوصول عن بُعد**             | ISP CGNAT / NAT مزدوج      | استخدم Tailscale / Cloudflare Tunnel / سحابة البائع             |
| **لوحة السيارة غير مقروءة ليلاً**  | مصراع بطيء جداً / IR ضعيف  | زد سرعة المصراع (1/100+)؛ أضف منير IR خارجي؛ 4K + زوم بصري      |
