Профессиональная установка камер безопасности стоит **$150–$400 за камеру** (работа + материалы). Типичная 8-камерная PoE система: **$1,200–$3,200 за труд**. DIY по этому руководству: **$0 труд + $100–$300 инструменты**. Руководство охватывает каждый шаг от планирования до первой записи.

<Badge variant="outline">Честная оценка</Badge> **DIY для вас, если:** вы комфортно работаете с лестницами, дрелью, базовой сетью и следуете схемам. **Наймите профи если:** многоэтажные крышные пробежки, бетон/кирпич, нужны разрешения, или хотите гарантию на труд.

### 2. Выберите свою архитектуру

| Архитектура | Лучше для | Кабель-пробежки | Питание | Сложность |
|--------------|----------|------------|-------|------------|
| **PoE NVR Kit** | 4–16 камер, постоянно, 24/7 | Cat5e/6 к каждой камере | PoE (свич/NVR) | Средняя |
| **Wi-Fi + NVR** | 2–8 камер, трудные места | Только питание (розетка/солнечная) | 120V / Батарея / Солнечная | Низкая–Средняя |
| **Battery/Solar Only** | 1–4 камеры, никаких проводов, аренда | Никаких | Батарея + Солнечная панель | Низкая |
| **Гибрид** | Большинство домов | Смешанные | Смешанные | Средняя |

<Callout type="tip">
**Начинайте с PoE там, где можете.** Это самое надежное, лучшее качество и самая низкая долгосрочная стоимость. Используйте Wi-Fi/батарею только там, где кабель действительно непрактичен.
</Callout>

### 3. Планирование маршрута кабелей (PoE)

- **Внутренние стены:** Сверлите верхнюю плиту, протаскивайте вниз в подвал/ползу, наверх на чердак
- **Внешние стены:** Сверлите через крайовую балку или подоконник, заделывайте силиконом
- **Чердачные маршруты:** Поведите вдоль стропил, скобьте каждые 4 фута, избегайте контакта с утеплителем
- **Труба:** ¾" EMT или PVC для открытых пролётов (код во многих районах)
- **Макс длина:** 328 футов (100 м) на Cat5e/6. Используйте PoE удлинитель или волокно дальше.

### 4. Чек-лист инструментов и материалов

**Обязательные инструменты:**
- [ ] Аккумуляторная дрель/шуруповерт (18V+)
- [ ] ½" палец / ⅞" сверло-аугер (отверстия для кабеля)
- [ ] Проволочный вожак / светящиеся стержни (25–50 футов)
- [ ] Детектор балок (AC sensing)
- [ ] Уровень (торпеда или лазер)
- [ ] Лестница (выдвижная для 2-этажных)
- [ ] Обжимной инструмент (RJ45 pass-through) + тестер кабеля
- [ ] Стриппер (Cat5e/6)
- [ ] Силикон герметик (уличный)
- [ ] Кабельные скобы / J-хуки / стяжки

**Материалы (PoE):**
- [ ] Cat6 цельноколония медина (CMR райзер для в стенах, CMX уличный для наружного)
- [ ] RJ45 pass-through коннекторы (50-упаковка)
- [ ] Keystone джеки + настенные пластины (опционально, чистая терминация)
- [ ] PoE коммутатор (8/16/24 порт, 802.3at/bt) или NVR с PoE портами
- [ ] ИБП для NVR + свича (1500VA мин)

**Материалы (Wireless/Battery):**
- [ ] Крепежные кронштейны (в комплекте или универсальные)
- [ ] Солнечные панели + монтажные кронштейны
- [ ] 120V уличные розетки / влагозащищенные коробки (если plug-in)

### Потолок / Соффит / Карниз монтаж

- **Распределительная коробка** (влагозащищенная, 4" круг) → кронштейн камеры к коробке
- **Труба** (½" EMT или LFMC) от коробки к камере если кабель открыт
- **Герметизация:** Силикон вокруг прохода коробки + вход кабеля

### Столб монтаж (Подъезд, Ворота, Двор)

- **Schedule 40 стальная труба** (1.5" или 2" ID) → заливайте в бетон (18" глубина × 8" диаметр)
- **U-болт кронштейн** для камеры
- **Труба** внутри столба или поверхностно LFMC
- **Заземление:** #6 CU к заземляющему стержню у основания столба (NEC 250.52)

## Фаза 3: Прокладка кабелей (PoE системы)

### Внутренняя прокладка (Чердак → Стена → Камера)

<Accordion type="single" collapsible>
  <AccordionItem value="cable-run">
    <AccordionTrigger>Пошагово: Чердак → Наружная камера</AccordionTrigger>
    <AccordionContent>
1. **Локализуйте позицию камеры** изнутри чердака — измерьте от известной ориентировки (вент, труба)
2. **Сверлите ⅞" отверстие** через верхнюю плиту (внутренняя стена) или соффит (карниз)
3. **Протаскивайте вожака** из чердака вниз по полости стены к отверстию камеры
3. **Приклеивайте Cat6** к вожаку: согните 6" назад, изолентой гладко, тяните **аккуратно**
5. **Оставьте 3 фута запаса** в чердаке (сервисная петля), 2 фута у камеры
6. **Заглушите внешнее отверстие:** Силикон вокруг кабеля + проходник
7. **Маркируйте оба конца:** "CAM-01 Front Door" (Маркер + кабельная этикетка)
8. **Проверьте прерывность** тестером кабеля до терминации
    </AccordionContent>
  </AccordionItem>
</Accordion>

### Терминация: RJ45 Pass-Through (Самый быстрый, надежный)

<Callout type="note">
**Используйте pass-through коннекторы.** Стандартный RJ45 требует точной подрезки. Pass-through позволяет проверить цветовой порядок *перед* обжатием. **Инструмент:** Обжимная зажимная для pass-through (Klein VDV226-110, $45).
</Callout>

### T568B распиновка (PoE стандарт)

| Pin | Color | Pin | Color |
|-----|-------|-----|-------|
| 1 | Orange/White | 5 | Blue/White |
| 2 | Orange | 6 | Green |
| 3 | Green/White | 7 | Brown/White |
| 4 | Blue | 8 | Brown |

**PoE использует ВСЕ 4 пары** (802.3bt). **Все 8 проводников должны быть идеальны.**

### Процедура обжатия (Pass-through)

1. Снять 1.5" внешнюю оболочку → разкрутить пары только по необходимости → выложить T568B
2. Вставить в pass-through → проверить цвета через прозрачный фронт
3. Подстричь flush с лицом коннектора
4. Обжать **сильно** (трещеткий инструмент один клик)
4. **Тест каждого кабеля** — 8 зеленых LED = ок

### PoE Switch / NVR подключение

- **Домашний прогон** каждую камеру к свичу/NVR (не в цепочку)
- **Patch panel** (опционально но про): 12/24-портовый keystone → короткие патч-корды к свичу
- **UPS:** NVR + PoE свич на батарейном бэкапе. Камеры остаются под питанием при отключении.

### Wi-Fi оптимизация

**Размещение:**
- Камера в пределах **50 футов** от роутера/AP (через ≤2 стены)
- Избегайте: металлическая обшивка, труба, вентиляционный канал между камерой и AP

**Апгрейд железа:**
- **Mesh система** (Eero Pro 6E, UniFi Dream Router, Asus ZenWiFi) — проводной бэкхол предпочтителен

- Установите роутер 2.4 GHz на **20 MHz ширину** (не 40) для стабильности

**Тест:** Телефон в месте камеры → Speedtest → Нужно **>5 Mbps upload** на камеру (1080p) / **>15 Mbps** (4K)

### Батарея / Солнечная ввод в эксплуатацию

1. **Полностью зарядите** перед монтажом (12–24 ч на USB-C)
2. **Сопрягите с приложением** на высоте монтажа (проверьте Wi-Fi сигнал)
3. **Закрепите кронштейн** → прикрепите камеру → настройте угол
4. **Солнечная панель:** Лицо **строго на юг** (СХ), наклон = широта ±15° (зима +15°, лето -15°)
5. **Проверьте зарядку:** App показывает ☀️ / ⚡ иконку в течение 1 часа солнца
6. **Настройте расписания:** Только движение ночью; отключите авто-запуск живого просмотра

## Фаза 5: NVR / VMS Конфигурация

### Первоначальная настройка (Reolink / Amcrest / Dahua / Hikvision / UniFi)

<Accordion type="single" collapsible>
  <AccordionItem value="nvr-setup">
    <AccordionTrigger>NVR Первый запуск чек-лист</AccordionTrigger>
    <AccordionContent>
1. **Установите HDD** до подачи питания (SATA, WD Purple / Seagate SkyHawk / Toshiba S300)
2. **Включите** → Wizard: Дата/Время (NTP), Язык, Пароль (сильный!)
3. **Сеть:** Статический IP для NVR (напр. 192.168.1.50), Шлюз, DNS (1.1.1.1)
4. **PoE Порты:** Включите все → "Auto Add" камеры (втыкайте по одной)
5. **Настройки кодирования (на камеру):**
   - **Main Stream:** 4K/8MP @ 15–20 fps, H.265, VBR, 4–6 Mbps
   - **Sub Stream:** 720p/D1 @ 15 fps, H.265, 0.5–1 Mbps (мобильный live view)
6. **Расписание записи:** Непрерывно 24/7 на Main; Motion на Sub (экономит место)
7. **Зоны движения:** Рисуйте ТОЛЬКО релевантные области (исключайте деревья, улицу, облака)
8. **Умные события:** Включите Person/Vehicle детекцию (снижает ложные алерты 90%)
9. **Хранилище:** Форматируйте HDD → Overwrite (oldest first) → Enable S.M.A.R.T. alerts
10. **Удаленный доступ:** **НЕ ПРОБРАСЫВАЙТЕ ПОРТЫ.** Используйте:
    - **Vendor Cloud** (Reolink, Dahua, Hikvision, UniFi) — проще всего
    - **VPN** (WireGuard на роутере, Tailscale, ZeroTier) — безопаснее
    - **Tailscale Funnel / Cloudflare Tunnel** — продвинуто, без VPN клиента
    </AccordionContent>
  </AccordionItem>
</Accordion>

### Выбор жестких дисков и RAID

| Диск | Емкость | Нагрузка (TB/год) | Гарантия | Лучше для |
|-------|----------|---------------|----------|----------|
| WD Purple | 1–18 TB | 180 | 3 yr | Стандартный NVR |
| Seagate SkyHawk | 1–20 TB | 180 | 3 yr | Стандартный NVR |
| WD Purple Pro | 8–22 TB | 300 | 5 yr | RAID 5/6, 8+ бей, AI аналитика |
| Seagate Exos X / IronWolf Pro | 4–24 TB | 300–550 | 5 yr | Энтерпрайз, 24-бэй+, тяжелая аналитика |

<Callout type="warning">
**Никогда не используйте:** Десктопные диски (WD Blue, Seagate Barracuda), NAS диски (WD Red, Seagate IronWolf) — не для 24/7 последовательной записи. **Наблюдательные диски имеют прошивку под постоянный записываемый поток.**
</Callout>

**RAID конфиг:** 4-бэй → RAID 5 (1 диск выходит). 6+ бэй → RAID 6 (2 диска). **Никогда RAID 0.**

## Фаза 6: Сетевая безопасность (Критично)

<Callout type="danger">
**Никогда не пробрасывайте порты камеры/NVR (80, 443, 8000, 554, 8080).** Shodan/Censys сканируют 24/7. Дефолтные креды = ботнет за минуты.
</Callout>

**Безопасный удаленный доступ (Лучшее → Хорошее):**

| Метод | Безопасность | Легкость | Стоимость | Платформа |
|--------|----------|------|------|----------|
| **Vendor Cloud (UniFi, Reolink, Eufy, Ring)** | Хорошая (TLS, auth) | Самая простая | Бесплатно | Все |
| **Tailscale / ZeroTier** (VPN mesh) | Отличная (WireGuard) | Средняя | Бесплатно (личное) | Все |
| **WireGuard на роутере** (OpenWrt, OPNsense, pfSense) | Отличная | Сложнее | Бесплатно | Все |
| **Cloudflare Tunnel** (Zero Trust) | Отличная | Средняя | Бесплатно | Только Web UI |
| **OpenVPN Server на роутере** | Хорошая | Средняя | Бесплатно | Все |

**Укрепление уровня камеры:**
- Смените **admin пароль** (16+ символов, password manager)
- Отключите **ONVIF** если не используете
- Отключите **UPnP** на роутере
- Включите **HTTPS only** (self-signed cert ок для локального)
- **Прошивка quarterly** (сайт вендора, не auto)

| Симптом | Вероятная причина | Исправление |
|---------|--------------|-----|
| **Нет изображения / "No Signal"** | PoE бюджет превышен | Проверьте коммутатор PoE бюджет; пересадите на инжектор |
| **Мерцание / полосы** | Заземляющая петля / плохое заземление | Общее заземление для всех устройств; изолятор |
| **Ночная визия белая/серое** | IR отражение | Наклоните камеру вниз 15–30° / отодвиньте 12–18" / Откл встроенный IR + внешний iluminator |
| **Постоянные алерты движения** | Деревья, тени, дождь, жуки | Сузьте зоны; включите только Person/Vehicle; снизьте чувствительность |
| **Wi-Fi отваливается** | Перегруз канала / дальность | Только 2.4 GHz; 20 MHz; Mesh ноду ближе к камере; направленная антенна |
| **NVR пищит** | HDD сбой / полный / вентилятор | Проверьте S.M.A.R.T.; замените диск; почистите вентилятор |
| **Удаленный доступ не работает** | ISP CGNAT / двойной NAT | Используйте Tailscale / Cloudflare Tunnel / Vendor Cloud |
| **Номер ночью не читается** | Выдержка медленная / IR слабый | Увеличьте выдержку (1/100+); добавьте внешний IR illuminator; 4K + оптический зум / LPR камера |
