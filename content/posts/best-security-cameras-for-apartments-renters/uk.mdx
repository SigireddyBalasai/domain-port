## Детальний огляд

### 1. Wyze Cam v4 ($35+)

**Найкраща бюджетна камера для орендарів.**

Wyze Cam v4 встановлюється за 2 хвилини — приклейте кріплення, підключіть USB, налаштуйте через додаток. Жодних дротів, жодного свердління.

- **Встановлення:** Клейке кріплення
- **Сховище:** microSD до 256 ГБ (без підписки)
- **Живлення:** USB (5 Вт) — використовуйте будь-який адаптер
- **Роздільна здатність:** 2K (2560×1440)
- **Нічне бачення:** Кольорове (з підсвіткою)
- **Додаток:** Wyze (простіший у використанні)

**Мінуси:** Лише дротове живлення, аудіо тільки в один бік (базова модель).

### 2. Eufy SoloCam S340 ($130)

**Найкраще без підписки.**

Eufy пропонує 2K роздільну здатність + сонячну панель, що усуває потребу в підзарядці. Все сховище локальне, без щомісячної плати.

- **Встановлення:** Магнітний кронштейн або клей
- **Сховище:** Внутрішня пам'ять 8 ГБ (без підписки)
- **Живлення:** Сонячна панель в комплекті
- **Роздільна здатність:** 2K + 2K (подвійна лінза)
- **AI:** Виявлення людини пакетом

**Мінуси:** Вища початкова ціна, додаток не такий швидкий.

### 3. Ring Stick Up Cam Pro ($130)

**Найкраща інтеграція з Alexa.**

Ring ідеально працює з Alexa — "Alexa, show the front door" відображає камеру на Echo Show. Акумуляторна версія встановлюється без дротів.

- **Встановлення:** Магніт + гвинти (або клей)
- **Сховище:** Хмара (Ring Protect $3.99/міс)
- **Живлення:** Акумулятор (3–6 місяців)
- **Роздільна здатність:** 1080p HDR
- **Аудіо:** Двостороннє з шумозаглушенням

**Мінуси:** Потрібна підписка для запису, акумулятор потребує підзарядки.

### 4. Arlo Essential 2K ($100)

**Найкращий вибір на вулицю.**

Arlo Essential має рейтинг IP65 та працює при температурах від -20°C до 45°C. Ідеально для балконів та патіо.

- **Встановлення:** Клейке або гвинтове кріплення
- **Сховище:** Хмара (безкоштовно 7 днів) або Arlo SmartHub
- **Живлення:** Акумулятор (3–6 місяців)
- **Роздільна здатність:** 2K
- **Нічне бачення:** Кольорове

**Мінуси:** Безкоштовний план хмарного сховища обмежений, потрібен базовий хаб для локального сховища.

### 5. Blink Mini 2 ($40)

**Найбюджетніший вибір для всередині.**

Blink Mini 2 — найдешевший спосіб спостерігати за кімнатою. Працює від USB, встановлюється на полицю або кріпиться клеєм.

- **Встановлення:** Клейке кріплення + USB
- **Сховище:** Хмара (Blink підписка $3/міс)
- **Живлення:** USB (5 Вт)
- **Роздільна здатність:** 1080p HD
- **Аудіо:** Двостороннє

**Мінуси:** Тільки для всередині, потрібна підписка для запису.

3. **Настінний кронштейн** — Приклеюється 3M VHB, потім камера накручується на нього
4. **Поличне розміщення** — Просто поставте на полицю, комод, холодильник
5. **Двосторонній скотч** — Для тимчасових установок (1–2 тижні)

### Важливі поради

| Порада                            | Чому                                               |
| --------------------------------- | -------------------------------------------------- |
| Перевірте Wi-Fi перед покупкою    | 2.4 ГГц краще проходить крізь стіни                |
| Використовуйте гостьову мережу    | Не діліться основним паролем Wi-Fi                 |
| Вимкніть світлодіод               | Конфіденційність в квартирі                        |
| Локальне сховище = без підписки   | Економія $3–10/міс на камеру                       |
| Орендний пристрій = легко забрати | Вибирайте камери, які демонтуються за хвилину      |
| Дверний дзвінок посиленої безпеки | Клейке кріплення замість свердління                |
| Співмешканець не згоден?          | Камера тільки на вашій території (спальня/кабінет) |
