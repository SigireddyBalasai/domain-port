Беспроводные камеры обещают "никаких проводов, легкая установка". Реальность: мертвые зоны, тревога за батарею, перегрузка 2.4 GHz и солнечные панели, которые не заряжаются зимой. Это руководство превращает беспроводное из фрустрирующего в надежное.

<Badge variant="outline">Wireless ≠ Wire-Free</Badge> **Подключаемые к сети
беспроводные** (Reolink E1, Wyze v4, TP-Link Tapo) нуждаются в 120V розетке.
**Батарейные/Солнечные** (EufyCam 3, Reolink Argus 4 Pro, Ring Stick Up) —
действительно безпроводные. Оба используют Wi-Fi. Обоим нужна надежная
RF-планировка.

## Физика Wi-Fi: Почему ваша камера отваливается

### Частотные диапазоны

| Диапазон               | Каналы                       | Макс. скорость | Проникновение | Дальность  | Лучше для                      |
| ---------------------- | ---------------------------- | -------------- | ------------- | ---------- | ------------------------------ |
| **2.4 GHz**            | 1, 6, 11 (неперекрывающиеся) | 600 Mbps (ax)  | ★★★★★         | 150+ футов | **Все уличные камеры**         |
| **5 GHz**              | 36–165                       | 2.4 Gbps (ax)  | ★★☆☆☆         | 50 футов   | Внутренние, короткие дистанции |
| **6 GHz (Wi-Fi 6E/7)** | 1–233                        | 5+ Gbps        | ★☆☆☆☆         | 30 футов   | Future-proof, только indoor    |

**Правило:** Заставляйте камеры работать **только на 2.4 GHz**. 5/6 GHz не проходят стены/обшивку. Большинство камер и так не поддерживают 6 GHz.

### Планирование каналов (2.4 GHz)

```
Channels:  1       6       11
Freq:   2412    2437    2462  MHz
Width:  20 MHz each (non-overlapping)
```

**Сначала сканируйте:** Используйте **WiFi Analyzer (Android)** или **AirPort Utility (iOS)** → Scan → Запишите каналы соседей.

**Назначьте:**

- Ваш роутер/AP: Канал 1, 6 или 11 (наименее загруженный)
- Camera AP (если отдельный): Другой неперекрывающийся канал
- **Ширина: 20 MHz только** (40 MHz перекрывает соседей, вызывает ретраи)

## Сетевая архитектура: Mesh vs Экстендер vs Выделенный AP

| Архитектура                                              | Плюсы                                                       | Минусы                                                        | Лучше для                              |
| -------------------------------------------------------- | ----------------------------------------------------------- | ------------------------------------------------------------- | -------------------------------------- |
| **Mesh (Eero Pro 6E, UniFi Dream Router, Asus ZenWiFi)** | Бесшовный роуминг, единый SSID, управление через приложение | Цена ($300–600), весь трафик через mesh                       | Весь дом, 5+ камер                     |
| **Проводной AP (UniFi U6 Lite, U6+, U7 Pro)**            | Лучшая производительность, PoE питание, VLAN поддержка      | Требует Ethernet пролёт                                       | **Лучше для камер** — проводной бэкхол |
| **Powerline + Wi-Fi (TP-Link TL-WPA8630)**               | Никаких новых проводов, использует электропроводку          | 50–200 Mbps реально, шум на AFCI                              | Ретрофит там, где кабель невозможен    |
| **Range Extender (TP-Link RE605X)**                      | Дешево ($50)                                                | Урезает полосу пополам, добавляет латентность, отдельный SSID | **Избегайте для камер**                |
| **Уличный AP (UniFi U6 Mesh, UAP-AC-M)**                 | Влагозащищенный, PoE, mesh uplink                           | Стоимость ($180–300)                                          | Открепленный гараж, ворота, амбар      |

<Callout type="tip">

**Про-архитектура:** PoE Switch → UniFi U6+ AP (потолок, центр) → 2.4 GHz
выделенный SSID "CAM-2G" (VLAN 20) → Все камеры. Изолирует IoT, никаких
проблем с роумингом, проводной бэкхол.

</Callout>

## Wi-Fi требования камер

| Спека                | Минимум            | Рекомендуется         | Почему                               |
| -------------------- | ------------------ | --------------------- | ------------------------------------ |
| **Сигнал (RSSI)**    | -70 dBm            | **-55 dBm или лучше** | -70 = частые отключения; -55 = скала |
| **SNR**              | 15 dB              | **25+ dB**            | Сигнал vs шум запас                  |
| **Upload Bandwidth** | 1 Mbps/кам (1080p) | 5 Mbps/кам (4K)       | Устойчивый, не burst                 |
| **Latency**          | &lt;100 ms         | &lt;30 ms             | Отзывчивость живого просмотра        |
| **Packet Loss**      | &lt;2%             | &lt;0.5%              | Пропуски записи, зависшие кадры      |

**Тест в месте камеры:** Телефон на том же SSID → Speedtest → Запишите upload. Подойдите к месту камеры → перепроверьте. Если upload &lt;5 Mbps, сначала исправьте Wi-Fi.

## Mesh / AP Размещение для камер

<Accordion type="single" collapsible>
  <AccordionItem value="placement">
    <AccordionTrigger>Правила размещения AP</AccordionTrigger>
    <AccordionContent>

- **Центр, высоко** — Потолковый монтаж в коридоре > угловая полка
- **Прямая видимость** — Избегайте: труба, кирпичная стена, металлический сайдинг, HVAC, зеркало, аквариум
- **Макс одна стена** — Камера ↔ AP через ≤1 внутреннюю стену
- **Избегайте углов** — RF отражается, создает мультипуть
- **Высота** — 7–9 ft AGL (над уровнем земли) для лучшего паттерна
- **Ориентация антенны** — Вертикальная для горизонтального покрытия; наклон 45° для многоэтажных

**UniFi U6+ Пример:** Потолковый монтаж в центре коридора → 2.4 GHz покрывает 3,000 кв. футов дома + 50 футов двора. Камеры на -50 до -60 dBm.

## Оптимизация батарейных камер

### Размещение для Солнечного + Wi-Fi

| Фактор               | Идеально                                                     | Избегайте                                        |
| -------------------- | ------------------------------------------------------------ | ------------------------------------------------ |
| **Солнечная панель** | Истинный юг (СХ), наклон 30–45°, **нет тени 9am–3pm Dec 21** | Северная грань, под навесом, тень дерева         |
| **Камера**           | 10–15 футов от панели (длина кабеля), под навесом от дождя   | Прямое солнце на объективе (перегрев), ороситель |
| **Wi-Fi**            | ≤50 футов от AP, 1 стена макс                                | Металлический сайдинг, кирпичная труба между     |

### Расширение срока службы батареи

| Настройка              | По умолчанию | Оптимизировано               | Экономия           |
| ---------------------- | ------------ | ---------------------------- | ------------------ |
| **Motion Sensitivity** | High         | Medium / Custom Zone         | 30–50%             |
| **Clip Length**        | 30s          | 10–15s                       | 40–60%             |
| **Retrigger Interval** | 10s          | 30–60s                       | 20–30%             |
| **Night Vision**       | Auto IR      | Schedule: IR off если фонарь | 15–25%             |
| **Live View**          | On demand    | Disable auto-preview         | 10–20%             |
| **Firmware**           | Auto         | Manual (проверка ежемесячно) | Предотвращает баги |

**Реальность:** EufyCam 3 (13,000 mAh) @ Medium sensitivity, 15s клипы, 30s retrigger, 2.4 GHz -55 dBm → **4–6 месяцев** (заявлен 1 год).

### Зимняя забота о батарее (Зона 5 и холоднее)

- **Li-ion перестает заряжаться &lt;32°F (0°C).** Камера работает на батарее до смерти.
- **Фиксы:**
  - Приносите внутрь ежемесячно для теплой зарядки (больно, но работает)
  - **LiFePO₄ апгрейд** (Reolink Go PT Plus, кастомные) — заряжает до -4°F
  - **Изолированная коробка для батареи** + 5W нагревательный коврик (термостат 40°F) — DIY
  - **Большая панель** (20W вместо 6W) — больше зимнего сбора

## Настройка подключаемых беспроводных камер

### Соображения питания

| Камера                  | Потребление   | Тип розетки             | Управление кабелем                        |
| ----------------------- | ------------- | ----------------------- | ----------------------------------------- |
| Reolink E1 Outdoor      | 5W (12V/0.5A) | 120V → 12V адаптер      | Влагозащищенная коробка + прокладка       |
| Wyze Cam v4             | 3W (5V/0.6A)  | 120V → 5V USB           | USB удлинитель + силиконовая герметизация |
| TP-Link Tapo C120       | 4W (5V/0.8A)  | 120V → 5V USB           | Короткий USB, капающая петля              |
| Google Nest Cam (wired) | 7W (5V/1.4A)  | 120V → 5V проприетарный | Только проприетарный кабель               |

**Капающая петля:** Кабель образует **U-образную петлю ниже розетки** → вода капает с петли, а не в розетку.

### GFCI и Кодекс

- **Уличная розетка должна быть GFCI** (NEC 210.8(A)(3))
- **Влагозащитный чехол** (in-use / bubble cover) — обязателен для влажных мест
- **Удлинитель** — **Никогда** постоянно. Используйте нормальный уличный кабель + коробку.

## Troubleshooting беспроводных камер

| Симптом                                 | Диагноз                             | Исправление                                                                  |
| --------------------------------------- | ----------------------------------- | ---------------------------------------------------------------------------- |
| **"Camera Offline" ежедневно**          | RSSI < -75 dBm                      | Сдвиньте AP ближе; добавьте проводной AP; проверьте канал                    |
| **Live view крутится / падает**         | Upload < 1 Mbps                     | Тест Speedtest у камеры; апгрейд AP                                          |
| **Клипы пропадают ночью**               | IR отражение / whiteout             | Наклоните вниз 15°; отодвиньте от стены; выкл IR + внешний illuminator       |
| **Батарея умирает за 2 недели**         | Зона высокого трафика / плохой RSSI | Снизьте чувствительность; сузьте зоны; добавьте солнечную; проверьте -60 dBm |
| **Live view лаг > 5 сек**               | Высокая латентность / bufferbloat   | QoS на роутере (приоритет камера VLAN); проводной AP бэкхол                  |
| **"No Wi-Fi" после отключения питания** | Камера загружается раньше роутера   | UPS на роутере + AP; камера авто-переподключается (проверьте прошивку)       |
| **Не подключается к 2.4 GHz**           | Роутер band-steering                | Отключите band-steering; разделите SSIDs ("HOME-2G", "HOME-5G")              |

## Безопасность: Изолируйте ваши камеры

<Callout type="danger">

**Камеры — это IoT — самый высокий риск взлома.** Изолируйте в VLAN с **нет
доступа к интернету** (кроме NTP/прошивки). Заблокируйте camera → LAN.
Разрешите LAN → camera (только просмотр).

</Callout>

**UniFi / VLAN Пример:**

```
VLAN 10 (LAN)     192.168.10.x  → Полный доступ
VLAN 20 (CAMERAS) 192.168.20.x  → Block internet, Block LAN access
                    Allow: NTP (udp/123), DNS (udp/53) to router
                    Allow: NVR (192.168.10.50) → Cameras (tcp/554, 8000)
```

**Правила фаервола роутера (Ubiquiti / OPNsense / pfSense):**

- **Allow** CAMERAS → Router (NTP, DNS)
- **Allow** NVR → CAMERAS (RTSP, HTTP, ONVIF)
- **Block** CAMERAS → LAN
- **Block** CAMERAS → WAN (internet)
- **Allow** LAN → CAMERAS (management only, restricted IPs)
