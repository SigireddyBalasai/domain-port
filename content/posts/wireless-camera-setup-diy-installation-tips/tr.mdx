## Kablosuz Kamera Kurulum Rehberi

Bu kılavuz, kablosuz güvenlik kameralarınızı doğru şekilde kurmanıza yardımcı olur. Sinyal optimizasyonu, pil yönetimi, parazit giderme ve güvenlik ipuçları.

## 1. Kurulum Öncesi Planlama

### WiFi Sinyal Gücü

Kurulumdan önce, kamera konumunda WiFi sinyal gücünü test edin:

| Sinyal Gücü | dBm     | Kalite         | Öneri                  |
| ----------- | ------- | -------------- | ---------------------- |
| Mükemmel    | -30 dBm | En iyi         | Sorunsuz 4K akış       |
| İyi         | -50 dBm | İyi            | Sorunsuz 1080p akış    |
| Orta        | -60 dBm | Kullanılabilir | Ara sıra donma         |
| Zayıf       | -70 dBm | Sınırlı        | Sık bağlantı kesilmesi |
| Çok zayıf   | -80 dBm | Çalışmaz       | Kamera bağlanamaz      |

**Hedef**: -65 dBm veya daha iyi

### WiFi Kanal Seçimi

- **2.4GHz**: Daha uzun menzil, daha yavaş, daha fazla parazit
- **5GHz**: Daha kısa menzil, daha hızlı, daha az parazit
- **6GHz (WiFi 6E)**: En yeni, en az parazit, sınırlı uyumluluk

**Öneri**: Çoğu kamera için 2.4GHz kullanın (daha iyi menzil). 5GHz'yi yoğun ağlarda tercih edin.

## 2. WiFi Optimizasyonu

### Yönlendirici Konumu

- Yönlendiriciyi merkezi bir konuma yerleştirin
- Yerden yüksekte tutun (raf, masa)
- Metal nesnelerden, aynalardan uzak tutun
- Akvaryum, mikrodalga gibi parazit kaynaklarından kaçının

### WiFi Kanalı Değiştirme

1. WiFi analizör uygulaması (Android) veya WiFi Explorer (Mac) kullanın
2. En az kullanılan kanalı belirleyin
3. Yönlendirici ayarlarından kanalı değiştirin
4. Kameraları yeniden bağlayın

### Mesh WiFi Sistemleri

- Geniş evler veya çoklu kamera için mesh sistem önerilir
- Her düğüm kameralara yakın yerleştirilmeli
- Kablolu backhaul (Ethernet) tercih edin

## 3. Kurulum

### Kamera Konumu

- **Yükseklik**: 2.4-3.6 metre
- **Açı**: 10-15° aşağı eğim
- **Yönlendiriciye görüş hattı**: Mümkünse
- **Parazit kaynaklarından uzak**: Metal, beton, su

### Uygulama Kurulumu

1. Kamera uygulamasını indirin
2. Hesap oluşturun
3. Kamerayı ekleyin (QR kod tarama)
4. WiFi şifresini girin
5. Kamerayı konumlandırın
6. Ürün yazılımını güncelleyin

### Pil Yönetimi

- **Pil ömrü**: 1-12 ay (kullanıma bağlı)
- **Sıcaklık etkisi**: Soğuk pili daha hızlı tüketir
- **Kayıt sıklığı**: Sık kayıt pili tüketir
- **Güneş paneli**: Pil ömrünü süresiz uzatabilir

**Pil tasarrufu ipuçları:**

- Hareket hassasiyetini optimize edin
- Kayıt süresini kısaltın (15-30 saniye)
- Gece görüşünü kısın veya kapatın
- WiFi sinyalini güçlendirin

## 4. Parazit Giderme

### Yaygın Parazit Kaynakları

| Kaynak              | Etki             | Çözüm                             |
| ------------------- | ---------------- | --------------------------------- |
| Mikrodalga          | 2.4GHz parazit   | Kamerayı mikrodalgadan uzak tutun |
| Bluetooth           | 2.4GHz parazit   | Kablosuz hoparlörleri kapatın     |
| Bebek telsizi       | 2.4GHz parazit   | Kanalı değiştirin                 |
| Komşu WiFi          | Kanal çakışması  | Daha az kullanılan kanal seçin    |
| Metal / beton       | Sinyal engelleme | Yönlendiriciye yaklaştırın        |
| Su (boru, akvaryum) | Sinyal emilimi   | Kamerayı sudan uzak tutun         |

### Parazit Azaltma

1. 5GHz bandına geçin (kamera destekliyorsa)
2. WiFi kanalını değiştirin
3. Yönlendiriciyi yükseltin (WiFi 6)
4. Mesh sistem kurun
5. Kablolu bağlantı kullanın (PoE)

## 5. Menzil Genişletme

### WiFi Genişletici

- Uygun fiyatlı
- Bant genişliğini yarıya düşürür
- Basit kurulum
- Orta menzil artışı

### Mesh WiFi

- Daha pahalı
- Bant genişliği kaybı yok
- Daha iyi performans
- Geniş evler için ideal

### Powerline Adaptör

- Elektrik tesisatı üzerinden ağ
- Değişken performans
- Ev içi kullanım için uygun

### Harici WiFi Anten

- Kamera antenini yükseltin
- Yönlü anten ile menzili artırın
- Teknik bilgi gerekli

## 6. Güvenlik İpuçları

### Varsayılan Ayarları Değiştirin

1. **Şifre**: Varsayılan şifreyi hemen değiştirin
2. **Kullanıcı adı**: Varsayılan "admin" kullanmayın
3. **Port**: Varsayılan portu (80, 8080, 554) değiştirin
4. **UPnP**: Kapatın (güvenlik riski)

### Ağ Güvenliği

- Kameralar için ayrı VLAN kullanın
- Misafir ağında çalıştırın
- Güçlü WiFi şifresi (WPA3)
- Ürün yazılımını güncel tutun

### Uzaktan Erişim Güvenliği

- VPN kullanın (port yönlendirme yerine)
- İki faktörlü kimlik doğrulamayı etkinleştirin
- Bulut erişimini sınırlayın
- Güvenilir olmayan ağlardan erişmeyin

## 7. Sorun Giderme

### Kamera Bağlanmıyor

1. WiFi şifresini kontrol edin
2. Kamerayı yönlendiriciye yaklaştırın
3. Yönlendiriciyi yeniden başlatın
4. Kamerayı fabrika ayarlarına sıfırlayın

### Sık Bağlantı Kesilmesi

1. Sinyal gücünü kontrol edin
2. Kanal parazitini kontrol edin
3. Ürün yazılımını güncelleyin
4. Kamerayı yeniden başlatın

### Yavaş / Donan Görüntü

1. Bant genişliğini kontrol edin
2. Çözünürlüğü düşürün
3. WiFi kanalını değiştirin
4. Kamerayı yönlendiriciye yaklaştırın
