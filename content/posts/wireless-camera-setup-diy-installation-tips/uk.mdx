Безпровідні камери обіцяють "без дротів, легке встановлення". Реальність: мертві зони, тривога за батарею, завантаженість 2.4 ГГц та сонячні панелі, які не заряджають взимку. Цей посібник перетворює безпровідне встановлення з проблемного на надійне.

<Badge variant="outline">Безпровідне ≠ Без дротів</Badge> **Мережеві
безпровідні** (Reolink E1, Wyze v4, TP-Link Tapo) потребують розетки 120V.
**Акумуляторні/Сонячні** (EufyCam 3, Reolink Argus 4 Pro, Ring Stick Up) справді
без дротів. Обидва використовують Wi-Fi. Обидва потребують надійного RF
планування.

## Фізика Wi-Fi: чому ваша камера від'єднується

### Частотні діапазони

| Діапазон               | Канали                  | Макс. швидкість | Проникнення | Дальність | Найкраще для                |
| ---------------------- | ----------------------- | --------------- | ----------- | --------- | --------------------------- |
| **2.4 ГГц**            | 1, 6, 11 (неперекривні) | 600 Мбіт/с (ax) | ★★★★★       | 45+ м     | **Усі вуличні камери**      |
| **5 ГГц**              | 36–165                  | 2.4 Гбіт/с (ax) | ★★☆☆☆       | 15 м      | Внутрішні, коротка відстань |
| **6 ГГц (Wi-Fi 6E/7)** | 1–233                   | 5+ Гбіт/с       | ★☆☆☆☆       | 9 м       | Майбутнє, тільки всередині  |

**Правило:** Примушуйте камери до **2.4 ГГц**. 5/6 ГГц не проникають через стіни/сайдинг. Більшість камер не підтримують 6 ГГц.

### Планування каналів (2.4 ГГц)

```
Канали:  1       6       11
Частота: 2412    2437    2462  МГц
Ширина:  20 МГц кожен (неперекривні)
```

**Спочатку скануйте:** Використовуйте **WiFi Analyzer (Android)** або **AirPort Utility (iOS)** → Скануйте → Занотуйте канали сусідів.

**Призначте:**

- Ваш роутер/AP: Канал 1, 6 або 11 (найменш завантажений)
- AP камер (якщо окремий): Інший неперекривний канал
- **Ширина: тільки 20 МГц** (40 МГц перекриває сусідів, викликає повторні передачі)

## Мережева архітектура: Mesh vs Подовжувач vs Виділений AP

| Архітектура                                              | Плюси                                              | Мінуси                                                           | Найкраще для                                |
| -------------------------------------------------------- | -------------------------------------------------- | ---------------------------------------------------------------- | ------------------------------------------- |
| **Mesh (Eero Pro 6E, UniFi Dream Router, Asus ZenWiFi)** | Безшовний роумінг, єдиний SSID, керування додатком | Ціна ($300–600), весь трафік через mesh                          | Весь дім, 5+ камер                          |
| **Дротовий AP (UniFi U6 Lite, U6+, U7 Pro)**             | Найкраща продуктивність, PoE живлення, VLAN        | Потребує Ethernet кабель                                         | **Найкраще для камер** — дротова магістраль |
| **Powerline + Wi-Fi (TP-Link TL-WPA8630)**               | Без нових дротів, використовує електропроводку     | 50–200 Мбіт/с реально, шум на AFCI                               | Ремонт, де кабель неможливий                |
| **Подовжувач діапазону (TP-Link RE605X)**                | Дешево ($50)                                       | Зменшує пропускну здатність удвічі, додає затримку, окремий SSID | **Уникайте для камер**                      |
| **Вуличний AP (UniFi U6 Mesh, UAP-AC-M)**                | Захищений від погоди, PoE, mesh uplink             | Ціна ($180–300)                                                  | Окремий гараж, ворота, сарай                |

<Callout type="tip">

**Професійна архітектура:** PoE комутатор → UniFi U6+ AP (стеля, центр) → виділений SSID 2.4 ГГц "CAM-2G" (VLAN 20) → Всі камери. Ізолює IoT, без проблем роумінгу, дротова магістраль.

</Callout>

## Вимоги камер до Wi-Fi

| Параметр                        | Мінімум              | Рекомендовано         | Чому                                     |
| ------------------------------- | -------------------- | --------------------- | ---------------------------------------- |
| **Сигнал (RSSI)**               | -70 dBm              | **-55 dBm або краще** | -70 = часті відключення; -55 = стабільно |
| **SNR**                         | 15 дБ                | **25+ дБ**            | Запас сигналу до шуму                    |
| **Вихідна пропускна здатність** | 1 Мбіт/с/кам (1080p) | 5 Мбіт/с/кам (4K)     | Постійна, не burst                       |
| **Затримка**                    | <100 мс              | <30 мс                | Чутливість живого перегляду              |
| **Втрата пакетів**              | <2%                  | <0.5%                 | Прогалини запису, заморожені кадри       |

**Тест на місці камери:** Телефон на тому ж SSID → Speedtest → занотуйте вихідну швидкість. Пройдіть до місця камери → перетестуйте. Якщо вихідна швидкість <5 Мбіт/с, спочатку виправте Wi-Fi.

## Розташування Mesh / AP для камер

<Accordion type="single" collapsible>
  <AccordionItem value="placement">
    <AccordionTrigger>Правила розташування AP</AccordionTrigger>
    <AccordionContent>

1. **Центрально, піднято** — стельовий монтаж у коридорі > кутова полиця
2. **Пряма видимість** — уникайте: димар, цегляна стіна, металевий сайдинг, HVAC, дзеркало, акваріум
3. **Максимум одна стіна** — камера ↔ AP через ≤1 внутрішню стіну
4. **Уникайте кутів** — RF відбивається, створює багатопроменевість
5. **Висота** — 2–3 м над рівнем землі для найкращої діаграми
6. **Орієнтація антени** — Вертикально для горизонтального покриття; нахил 45° для багатоповерхових будівель

**Приклад UniFi U6+:** Стельовий монтаж у центрі коридору → 2.4 ГГц покриває 3000 кв. футів дому + 15 м подвір'я. Камери на -50 до -60 dBm.

## Оптимізація акумуляторних камер

### Розташування для сонця та Wi-Fi

| Фактор             | Ідеально                                                                         | Уникайте                                           |
| ------------------ | -------------------------------------------------------------------------------- | -------------------------------------------------- |
| **Сонячна панель** | Справжній південь (Пн. півкуля), нахил 30–45°, **без тіні 9:00–15:00 21 грудня** | Північна сторона, під карнизом, тінь від дерева    |
| **Камера**         | 3–5 м від панелі, під карнизом для захисту від дощу                              | Пряме сонце на лінзі (перегрів), бризки спринклера |
| **Wi-Fi**          | ≤15 м від AP, макс 1 стіна                                                       | Металевий сайдинг, цегляна стіна між               |

### Продовження терміну служби батареї

| Налаштування                        | Стандарт  | Оптимізовано                            | Економія           |
| ----------------------------------- | --------- | --------------------------------------- | ------------------ |
| **Чутливість руху**                 | Висока    | Середня / Користувацька зона            | 30–50%             |
| **Тривалість кліпу**                | 30 с      | 10–15 с                                 | 40–60%             |
| **Інтервал повторного спрацювання** | 10 с      | 30–60 с                                 | 20–30%             |
| **Нічне бачення**                   | Авто ІЧ   | Розклад: ІЧ вимк, якщо є вуличне світло | 15–25%             |
| **Живий перегляд**                  | На вимогу | Вимкніть автопопередній перегляд        | 10–20%             |
| **Прошивка**                        | Авто      | Вручну (перевіряйте щомісяця)           | Запобігає помилкам |

**Реальний термін:** EufyCam 3 (13,000 мАг) @ Середня чутливість, кліпи 15 с, повтор 30 с, 2.4 ГГц -55 dBm → **4–6 місяців** (проти заявленого 1 року).

### Зимовий догляд за батареєю (зона 5 та холодніше)

- **Li-ion припиняє зарядку <32°F (0°C).** Камера працює від батареї до розрядження.
- **Виправлення:**
  - Щомісяця заносьте додому для теплої зарядки
  - **Оновіть до LiFePO₄** (Reolink Go PT Plus, custom) — заряджається до -20°C
  - **Ізольований бокс для батареї** + нагрівальна пластина 5W (термостат 40°F)
  - **Більша панель** (20W vs 6W) — більше зимового збору

## Встановлення мережевих безпровідних камер

### Міркування щодо живлення

| Камера                     | Споживання    | Тип розетки        | Керування кабелем                      |
| -------------------------- | ------------- | ------------------ | -------------------------------------- |
| Reolink E1 Outdoor         | 5W (12V/0.5A) | 120V → 12V адаптер | Непромокальна коробка + ущільнювач     |
| Wyze Cam v4                | 3W (5V/0.6A)  | 120V → 5V USB      | USB подовжувач + силіконове ущільнення |
| TP-Link Tapo C120          | 4W (5V/0.8A)  | 120V → 5V USB      | Короткий USB, drip петля               |
| Google Nest Cam (провідна) | 7W (5V/1.4A)  | 120V → 5V фірмовий | Тільки фірмовий кабель                 |

**Drip петля:** Кабель утворює **U-форму нижче розетки** → вода стікає з петлі, не потрапляє в розетку.

### GFCI та норми

- **Вулична розетка має бути GFCI** (NEC 210.8(A)(3))
- **Непромокальна кришка** — обов'язкова для вологих місць
- **Подовжувач** — **Ніколи** постійно. Використовуйте правильний вуличний кабель + коробку.

## Усунення проблем безпровідних камер

| Симптом                                      | Діагностика                          | Виправлення                                                               |
| -------------------------------------------- | ------------------------------------ | ------------------------------------------------------------------------- |
| **"Камера офлайн" щодня**                    | RSSI < -75 dBm                       | Перемістіть AP ближче; додайте дротовий AP; перевірте канал               |
| **Живий перегляд крутиться / не вмикається** | Вихідна швидкість < 1 Мбіт/с         | Тест Speedtest на місці камери; оновіть AP                                |
| **Кліпи відсутні вночі**                     | Відбиття ІЧ / засвічення             | Нахиліть вниз 15°; відсуньте від стіни; вимкніть ІЧ + зовнішній прожектор |
| **Батарея вмирає за 2 тижні**                | Зона високого трафіку / поганий RSSI | Зменшіть чутливість; звужте зони; додайте сонце; перевірте -60 dBm        |
| **Затримка живого перегляду > 5 с**          | Висока затримка / bufferbloat        | QoS на роутері; дротова магістраль AP                                     |
| **"Немає Wi-Fi" після відключення світла**   | Камера вмикається до роутера         | UPS на роутер + AP; автоперепідключення камери                            |
| **Не може підключитися до 2.4 ГГц**          | Band-steering роутера                | Вимкніть band-steering; окремі SSID ("HOME-2G", "HOME-5G")                |

## Безпека: ізолюйте ваші камери

<Callout type="danger">

**Камери — це IoT, найвищий ризик злому.** Ізолюйте на VLAN **без доступу до інтернету** (крім NTP/прошивка). Блокуйте камери → LAN. Дозвольте LAN → камери (тільки перегляд).

</Callout>

**UniFi / VLAN приклад:**

```
VLAN 10 (LAN)     192.168.10.x  → Повний доступ
VLAN 20 (КАМЕРИ) 192.168.20.x  → Блокувати інтернет, блокувати LAN
                    Дозволити: NTP (udp/123), DNS (udp/53) до роутера
                    Дозволити: NVR (192.168.10.50) → Камери (tcp/554, 8000)
```

**Правила брандмауера роутера (Ubiquiti / OPNsense / pfSense):**

1. **Дозволити** КАМЕРИ → Роутер (NTP, DNS)
2. **Дозволити** NVR → КАМЕРИ (RTSP, HTTP, ONVIF)
3. **Блокувати** КАМЕРИ → LAN
4. **Блокувати** КАМЕРИ → WAN (інтернет)
5. **Дозволити** LAN → КАМЕРИ (тільки управління, обмежені IP)
