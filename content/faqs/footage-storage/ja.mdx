保存期間はニーズによって異なります：

- **一般家庭**：7〜14日で通常十分です
- **小規模事業者**：30日を推奨
- **大企業**：30〜90日

保存要件に影響する要因：

| 要因 | 影響 |
|------|------|
| カメラの台数 | カメラが多いほど = 保存容量が必要 |
| 解像度 | 解像度が高いほど = より多くの容量 |
| 動体検知 | 保存容量を削減 |
| 地域の規制 | 最低保存期間が定められている場合あり |
